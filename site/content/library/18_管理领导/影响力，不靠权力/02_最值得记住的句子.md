{{< sentence >}}
影响力的本质是交换。你想要什么，对方想要什么，中间能不能达成交易。
{{< /sentence >}}

科恩把组织中的所有影响都建模成交换关系。你想让同事帮你赶一个项目——你能给他什么？可见度？学习机会？人情？如果你不知道对方在意什么，你就没有筹码。

{{< sentence >}}
每个人都有自己的"货币"。找到它，才能和他做交易。
{{< /sentence >}}

"货币"是科恩的核心概念。有人在意被认可，有人在意自主权，有人在意职业发展，有人在意工作和生活的平衡。同样的"报酬"对不同的人价值完全不同。你给了一个在意自主权的人表扬，等于给了他零元。

{{< sentence >}}
把你想影响的人当成潜在的盟友，而不是需要克服的障碍。
{{< /sentence >}}

一旦你把对方定义为障碍，你的策略就变成了"怎么绕过他"或"怎么压过他"。把对方当盟友，策略变成"怎么让合作对双方都有利"——后者的成功率高得多。

{{< sentence >}}
永远不要低估声誉的货币价值。
{{< /sentence >}}

在组织中，你的声誉是你的最大资产。信守承诺、按时交付、公平对待合作者——这些行为积累的信任是未来所有影响力交换的基础。一次失信可以毁掉多年的积累。

{{< sentence >}}
理解对方的世界，比改变对方的想法更有效。
{{< /sentence >}}

你觉得对方不合作是因为"不配合"或"不懂"。但从对方的角度看，他的不合作完全合理——他有自己的优先级、压力和考核指标。先理解他的世界，才能找到交换的切入点。
