## 环境变化时的失效

{{< sentence >}}
经验在旧环境中越有效，在新环境中越容易失效。
{{< /sentence >}}

你用一套方法成功了十年，环境突然变了——技术变了、市场变了、竞争格局变了。这时候老方法往往不仅无效，反而会成为包袱。

{{< sentence >}}
识别"环境已经变了"这件事，往往比改变本身还难。
{{< /sentence >}}

有经验的人容易陷入"这次也能用老办法"的思维。直到撞了南墙，才意识到环境变了。

## 信息过滤时的失效

{{< sentence >}}
经验让大脑自动做选择——哪些信息值得注意，哪些可以忽视。但这个自动的选择往往是错的。
{{< /sentence >}}

新手会认真看待每一条信息。老手则会快速判断"这不重要"。有时候，被忽视的那一条信息，正是最关键的。

{{< sentence >}}
听取建议的前提是，承认自己的信息可能不完整。
{{< /sentence >}}

很多有经验的人听不进建议，因为他们觉得"我都见过，没什么新的"。其实，真的新东西，往往就是被这样过滤掉的。

## 方法论陷阱时的失效

{{< sentence >}}
当你把一套成功的方法推导出理论后，就容易把理论当成了真理。
{{< /sentence >}}

一个成功的企业家总结了自己的方法论。这个方法论在他的公司里很有效。但当他用这个方法论去指导别人或开新的企业时，往往失效了。

为什么？因为成功往往有很多偶然因素。你以为成功的原因是 A，其实真的原因可能是 A+B+时间+运气。

## 超越经验的三条路

{{< sentence >}}
超越经验的最有效的方式，不是忽视经验，而是理解经验为什么有效。
{{< /sentence >}}

一旦你理解了原理，就能在新的环境中应用原理，而不是生硬地复制过往的做法。

{{< sentence >}}
引入不同背景的人和想法，是对抗经验局限最有效的办法。
{{< /sentence >}}

新人往往能看到有经验的人看不到的东西，因为他们没有被既有的经验"污染"。

{{< sentence >}}
定期强制自己离开熟悉的领域，在陌生领域重新做学生，能有效地刷新思维。
{{< /sentence >}}

在陌生领域，你的经验全无用处，必须空杯心态重新学。这个过程虽然痛苦，但对扩展思维很有效。
