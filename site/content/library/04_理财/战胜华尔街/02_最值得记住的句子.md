# 一记下买点子，就把这几句放旁边

## 观察力只有进了组合才开始值钱

{{< sentence no="1" quote="发现线索不稀奇，把线索做成组合才值钱。" >}}
灵感只是起点，真正能留下结果的是后面的组织能力。
{{< /sentence >}}

{{< sentence no="2" quote="先记下为什么买，再看后来为什么涨跌。" >}}
没有原始记录，事后复盘很容易全变成合理化。
{{< /sentence >}}

{{< sentence no="3" quote="公司类型不同，研究标准也该不同。" >}}
分类不清，组合里的判断就会越来越乱。
{{< /sentence >}}

{{< sentence no="4" quote="长期记录比事后解释更诚实。" >}}
它能帮你分清到底是判断对了，还是只是运气顺。
{{< /sentence >}}

{{< sentence no="5" quote="组合是在替观察力买时间。" >}}
单点看见机会不够，组合和时间才能把优势放大。
{{< /sentence >}}

{{< sentence no="6" quote="不是每个好点子都要重仓表达。" >}}
仓位首先是边界管理，不是情绪放大器。
{{< /sentence >}}

{{< sentence no="7" quote="日常世界给你的线索，常常比研报更早。" >}}
别低估个人投资者在现实生活里的信息前置优势。
{{< /sentence >}}

{{< sentence no="8" quote="只复盘盈亏，不复盘判断，最后什么也学不到。" >}}
真正值钱的复盘，回看的不是输赢，而是当时的逻辑。
{{< /sentence >}}

## 该翻这页的时候，通常是你开始只看结果的时候

- 发现一个新机会时，你会不会先记下原始买入逻辑
- 组合里每个仓位，你还说得清它属于哪一类吗
- 回头看一笔投资时，你会不会先复盘判断而不是盈亏
- 组合变热闹时，你还能不能收回到分散和记录
- 新线索很多时，你还会不会筛掉不值得进组合的点子

这页真正帮你的，是把“我早知道”改成“我当时到底为什么买”。
