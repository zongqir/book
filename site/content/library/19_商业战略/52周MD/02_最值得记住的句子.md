{{< sentence >}}
零售业最大的浪费不是卖不掉的库存——而是"该有的商品不在该在的时间出现在该在的位置"。机会成本比库存成本更贵。
{{< /sentence >}}

缺货和滞销是同一个问题的两面——不是货太多或太少，是时间和位置不对。

{{< sentence >}}
以周为单位做计划的核心不是"计划更细"——而是"反馈更快"。一个月才能发现问题太慢；一周发现问题，下周就能调整。
{{< /sentence >}}

计划周期的长短决定了你能多快纠错。缩短计划周期不是增加管理负担而是加速学习。

{{< sentence >}}
商品陈列不是装饰——是沟通。顾客进门前三秒看到什么决定了他会不会往里走。陈列是零售业最便宜也最被低估的营销工具。
{{< /sentence >}}

空间是零售商的媒体。每一个展示面都是一条信息——用好了不花钱但产生销售。

{{< sentence >}}
天气是零售业最重要的外部变量之一。降温那一周羽绒服销量比广告能拉动的高三倍。和天气同步比和营销日历同步更重要。
{{< /sentence >}}

零售的最大驱动力经常不在你的控制范围内——但可以在你的预案范围内。

{{< sentence >}}
库存周转率不是一个财务指标——是一个运营能力指标。周转越快说明你对需求的预判越准、调整越快、浪费越少。
{{< /sentence >}}

周转率低意味着你在用资金替代判断力——因为猜不准所以多囤。
