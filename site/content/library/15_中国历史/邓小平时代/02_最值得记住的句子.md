# 七句话看懂邓小平的决策逻辑和改革节奏

邓小平不是金句型领导人。他的话往往平实、直白，甚至粗糙。但正因为不追求漂亮，这些话反而精准地暴露了他的决策逻辑。

以下七句按判断校准力排列。

{{< sentence >}}
不管黑猫白猫，捉到老鼠就是好猫。
{{< /sentence >}}

这句话1962年就说了，但直到1978年以后才兑现。它的力量不在修辞，在于它直接否定了"先确认路线正确，再允许行动"的决策习惯。在意识形态高度敏感的环境里，这句话等于说：结果比立场重要。

**调用场景**：当团队在"该不该做"上争论不休，却没有人愿意先试一下看效果的时候。

{{< sentence >}}
摸着石头过河。
{{< /sentence >}}

经常被引用，很少被正确理解。它不是说没有方向，而是说不在看不清水底的时候假装看清了。邓小平的改革几乎没有一项是一步到位的——农村改革、特区、价格双轨制，全部是先在局部试，试出效果再扩大。

**调用场景**：当你面对一个复杂变革，有人要求你先交一份完整方案才能开始行动的时候。

{{< sentence >}}
改革开放胆子要大一些，看准了的，就大胆地试，大胆地闯。
{{< /sentence >}}

1992年南巡时说的。背景是改革进程在1989年后明显放缓，保守力量重新抬头。邓小平已经八十八岁。这句话不是空喊口号——他是在用最后的政治影响力，重新把改革推回轨道。

**调用场景**：当变革已经有了初步成果，但因为某次挫折或外部压力，整个组织倾向于收缩的时候。

{{< sentence >}}
发展才是硬道理。
{{< /sentence >}}

同样是南巡讲话。把所有争论压缩成一个判断标准：经济发展是检验一切的尺子。这句话的威力在于它终结了"姓社姓资"的路线争论——不是回答那个问题，而是宣布那个问题不重要。

**调用场景**：当组织被定义之争、路线之争消耗了大量精力，忘了原来要解决什么问题的时候。

{{< sentence >}}
稳定压倒一切。
{{< /sentence >}}

1989年之后反复强调的判断。它既是邓小平维护秩序的核心逻辑，也是他对改革节奏的根本约束条件。改革可以推进，但不能以社会失序为代价。这句话同时划出了改革的动力和改革的天花板。

**调用场景**：当你在推动变革但开始感觉到系统失控的信号时——这句话提醒你，变革速度不能超过系统的承受力。

{{< sentence >}}
中国要警惕右，但主要是防止"左"。
{{< /sentence >}}

南巡讲话中最精确的一句政治判断。邓小平的经验是：右的问题是走得太快、步子太大；左的问题是根本不让走。在中国的政治环境里，左的惯性远比右的冲动更危险，因为左可以用意识形态正确性为自己辩护。

**调用场景**：当组织中"不能出错"的声音盖过了"应该尝试"的声音时。

{{< sentence >}}
让一部分人先富起来。
{{< /sentence >}}

不是一句鼓励话，是一个政策设计原则。在资源有限的条件下，均匀分配意味着所有人一起慢；允许不均匀，才可能产生示范效应和追赶动力。这个原则后来的副作用——贫富分化——邓小平生前已经意识到，但他判断初期阶段必须接受这个代价。

**调用场景**：当你在资源有限的情况下必须做分配决策，面对"为什么不能人人平等"的压力时。
