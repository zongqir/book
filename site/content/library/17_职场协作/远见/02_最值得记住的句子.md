## 时间尺度决定判断质量

{{< sentence >}}
职业生涯是一场至少四十五年的马拉松，不是一场短跑冲刺。
{{< /sentence >}}

每次觉得"来不及了"的时候拿出来量一量。大多数焦虑在这个尺度下会自动降级。

## 燃料比职位更值得计算

{{< sentence >}}
你不是在选工作，你是在为职业储备燃料——可迁移技能、有意义的经验、持久的关系。
{{< /sentence >}}

下次评估 offer 时，把"这份工作给我加了什么燃料"放进判断清单。薪资会贬值，燃料不会。

## 可迁移技能是唯一的硬通货

{{< sentence >}}
行业会消失，岗位会重组，只有可迁移技能跟着人走。
{{< /sentence >}}

解决问题、说服他人、管理复杂项目——这些能力换了行业照样成立。锚定它们，不要锚定岗位名称。

## 甜蜜区不是规划出来的

{{< sentence >}}
职业的甜蜜区在三个圈的交集：你擅长的、你热爱的、世界愿意付费的。
{{< /sentence >}}

三十岁之前强求交集，容易把圈画小。前十五年的任务是把每个圈尽量撑大。

## 四十岁之前的决定没那么重

{{< sentence >}}
四十岁以前的大多数职业决策，几乎都可以修正。
{{< /sentence >}}

不是鼓励随意。是提醒：不要因为害怕选错，把精力耗在犹豫上而不是行动上。

## 关系网的质量不靠数量

{{< sentence >}}
人脉质量不取决于认识多少人，而是有多少人愿意在关键时刻帮你。
{{< /sentence >}}

维护五个深度关系，胜过收集五百张名片。关系需要持续投入，不能只在需要时才启动。
