{{< sentence >}}
乾隆时代的中国GDP占世界三分之一，但普通中国人的生活水平低于同期的英国穷人。
{{< /sentence >}}

总量和人均是两个完全不同的故事。GDP世界第一的帝国里，大部分人吃不饱——这个反差比任何理论都有力。

{{< sentence >}}
文字狱不是乾隆的性格缺陷，而是一套精心设计的思想控制技术。
{{< /sentence >}}

乾隆时期的文字狱案件数量远超前代。张宏杰指出这不是某个皇帝的偏执，而是一种系统性的治理工具——通过恐惧消除异议。

{{< sentence >}}
盛世最大的代价不是经济的，而是精神的——一个创造力被彻底压制的社会，在下一轮竞争中注定落后。
{{< /sentence >}}

乾隆盛世消灭了独立思考的空间。等到鸦片战争来临时，整个社会已经失去了应对外部冲击的智力资源。

{{< sentence >}}
乾隆的"十全武功"大部分是用远超必要的代价打赢的——胜利的成本全部转嫁给了底层纳税人。
{{< /sentence >}}

战争的代价不在朝廷的账簿上。远征缅甸、大小金川之役的巨额军费最终都变成了农民的负担。

{{< sentence >}}
一个帝国如果把主要资源用于维护统治者的控制力而不是提升民众的生产力，它的繁荣一定是空心的。
{{< /sentence >}}

这是张宏杰对乾隆盛世的总判断——资源分配的优先级决定了繁荣是实心的还是空心的。
