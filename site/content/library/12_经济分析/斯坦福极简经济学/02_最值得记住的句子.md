这六句话从36个关键词中筛出来。筛选标准只有一条：能不能在遇到具体经济现象时直接改变你的判断起点。能改变的留，只是"说得对"但用不上的不留。

{{< sentence >}}
价格不是商家定的，是供给和需求共同决定的。商家能做的只是在供需允许的范围内找一个位置。
{{< /sentence >}}

涨价时骂商家贪心，降价时夸商家良心——这两种反应都跳过了供需。矿泉水在暴雨天涨价，不是老板突然变坏了，是需求暴增而供给跟不上。

调用场景：任何价格变动引发愤怒或困惑的时刻。先找供需变化，再判断是不是有人在捣鬼。

{{< sentence >}}
价格管制不会消灭稀缺，只会把稀缺从价格转移到排队、关系和黑市。
{{< /sentence >}}

政府限价看起来保护了消费者。但稀缺没有消失，只是换了分配方式。限价之后，谁能买到取决于谁起得早、谁认识人、谁愿意付黑市溢价。

调用场景：看到"政府应该管一管XX价格"的呼声时。管价格容易，管住稀缺的重新分配方式才难。

{{< sentence >}}
外部性意味着有人在承担他没同意的代价，或者享受他没付费的好处。
{{< /sentence >}}

工厂排污是负外部性——周围居民承担了健康代价，但没人问过他们愿不愿意。打疫苗是正外部性——你的免疫力保护了身边没打疫苗的人，但没人给你付这份保护费。

调用场景：分析任何涉及"公平不公平"的经济争论时。先问一句——有没有人在承担他不知道的代价。

{{< sentence >}}
GDP量的是一年造了多少，不是攒了多少。流量高不代表存量厚。
{{< /sentence >}}

GDP增长8%听起来很好。但如果增长全靠基建投资，而投资建成的高速路没车跑，那8%只是数字好看。区分流量和存量，才能判断增长质量。

调用场景：读到GDP数据、经济增速对比时。先问：这些产出是变成了真实财富，还是变成了沉没投资。

{{< sentence >}}
通胀是一种隐性税——不需要立法，不需要投票，直接从存款里扣。
{{< /sentence >}}

通胀3%意味着你存在银行里的钱每年缩水3%。没人通知你，没人征求你同意。持有现金和固定收入的人受伤最重；持有资产的人反而受益。

调用场景：做储蓄和投资决策时。通胀不是宏观数据，是你的购买力在缩水。

{{< sentence >}}
垄断的核心问题不是价格高，是消费者没有别的选项可以走。
{{< /sentence >}}

价格高本身不是罪。如果消费者有替代品，高价企业自然会被市场淘汰。垄断危险在于它堵死了"用脚投票"的出口。没有退出选项，消费者就只能接受。

调用场景：判断某个行业是否需要反垄断监管时。不看价格绝对值，看消费者有没有替代选择。
