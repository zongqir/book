# 十句提醒，把管理者从忙碌拉回有效

## 时间现实

{{< sentence >}}
看不见时间真实去向的人，没有资格讨论效率。
{{< /sentence >}}

德鲁克把时间放在第一位，不是技巧偏好，而是管理岗位现实使然。

{{< sentence >}}
管理者最常犯的错，不是时间太少，而是对自己真实拥有的时间有幻觉。
{{< /sentence >}}

这句补出了第一句背后的错法：高估完整时间，低估打断和协作成本。

## 贡献焦点

{{< sentence >}}
管理工作的起点，不是手上有多少事，而是你为谁贡献了什么。
{{< /sentence >}}

这句能直接把人从事务堆里拽回来。

{{< sentence >}}
忙得很满，却说不出关键贡献，通常说明你是在被工作推着跑。
{{< /sentence >}}

这句把“无效忙碌”从一种感觉变成了可检验状态。

## 长处配置

{{< sentence >}}
组织真正能放大的，永远是长处，不是短板修补的平均值。
{{< /sentence >}}

这句是德鲁克关于人的管理里最该留下的一句。

{{< sentence >}}
平均分工看上去公平，很多时候只是回避了谁最适合结果责任这个更难的问题。
{{< /sentence >}}

这句适合用来纠正“按谁有空来分活”的习惯。

## 优先级

{{< sentence >}}
没有明确放弃清单的优先级，最后一定会输给临时事务。
{{< /sentence >}}

这句把优先级的另一半补全了。

{{< sentence >}}
要事优先真正考验的，不是判断力，而是拒绝力。
{{< /sentence >}}

很多管理者知道什么重要，只是没有把拒绝动作做出来。

## 决策纪律

{{< sentence >}}
把规律当例外处理，组织会被一次次重复问题拖住。
{{< /sentence >}}

这句是决策纪律的总开关。

{{< sentence >}}
有效管理者最可怕的地方，不是更强，而是他把关键动作练成了习惯。
{{< /sentence >}}

最后这句最好拿来收束全书。德鲁克真正想证明的，不是天赋，而是纪律可训练。
