这七句话有先后顺序。前面的是后面的地基。读的时候按顺序过一遍，用的时候可以单独调用任何一句。

{{< sentence >}}
需求定律不需要"其他条件不变"这个限制。只要价格定义精确，价格上升需求量必定下降。
{{< /sentence >}}

张五常剥掉了需求定律身上多余的假设。不需要理性人，不需要效用函数，不需要完全信息。只需要两个可观察的量：价格和需求量。简化不是偷懒，是把理论的可检验性推到极致。

调用场景：分析任何市场行为时，先确认"价格"是什么——有时候价格不是标签上的数字，而是总代价（包含时间、精力、风险）。

{{< sentence >}}
成本是你放弃的最高代价，不是你付出的金额。
{{< /sentence >}}

会计成本和经济成本完全不同。经济成本=机会成本。你放弃的那个最佳替代选择，才是实际代价。一家看起来利润丰厚的公司，把老板自己的时间和资金的机会成本算进去，可能在亏钱。

调用场景：做任何需要取舍的决策时。不要只看花了多少钱，要看放弃了什么。

{{< sentence >}}
经济利润为零不是没赚钱——是所有要素都得到了最佳替代回报。
{{< /sentence >}}

长期竞争均衡中，经济利润趋向零。不是企业不赚钱，而是利润刚好等于所有投入在其他用途中能赚到的。超额利润的存在意味着某种稀缺性——要么是独特资源，要么是进入壁垒。

调用场景：判断一个行业是否"暴利"时。先问超额利润的来源是什么，能不能持续。

{{< sentence >}}
交易费用决定合约形式。合约形式不是选择的结果，是约束条件的产物。
{{< /sentence >}}

为什么有的工作按件付、有的按月付？不是老板的管理风格，是监督成本的结构。按件付的监督成本低（产出好量化），按月付的谈判成本低（不用每天定价）。合约形式跟着交易费用走。

调用场景：分析雇佣关系、商业合作模式时。先看监督和执行的成本结构，而不是问"哪种更好"。

{{< sentence >}}
企业不是一个黑箱，是一组合约的集合。
{{< /sentence >}}

科斯问"企业为什么存在"。张五常的回答：企业是一种用层级合约替代市场合约的安排——当市场交易费用太高时，把交易放进企业内部反而更便宜。企业的边界不是固定的，取决于两种合约方式的费用对比。

调用场景：思考组织设计时。问"这件事是放在企业内部做便宜，还是外包给市场做便宜"。

{{< sentence >}}
产权不清楚，前面所有分析都悬空。
{{< /sentence >}}

需求定律要有价格，价格要有产权——只有知道"谁拥有什么"，交易才有起点。产权模糊的地方，价格信号失灵，交易费用飙升，合约安排扭曲。张五常所有分析的落脚点都在产权。

调用场景：分析任何资源分配问题时。先问：产权归谁？界定清楚吗？

{{< sentence >}}
理论的好坏不看逻辑是否自洽，看对可观察现象的解释力。
{{< /sentence >}}

这是张五常的科学哲学立场——来自弗里德曼和波普尔。理论不需要假设"真实"，只需要预测正确。一个假设"人是理性的"理论，如果预测结果跟观察一致，就是好理论——哪怕人其实不总是理性的。

调用场景：评估任何经济学理论时。不要问"假设合不合理"，要问"预测准不准"。
