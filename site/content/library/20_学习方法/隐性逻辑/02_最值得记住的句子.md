## 最高频的捷径

{{< sentence >}}
你不是在评估一件事有多好，你在评估它比你脑子里闪过的参照物好多少。
{{< /sentence >}}

锚定效应。一个不相关的数字会影响你对完全不同的问题的判断。你先看到一件标价一万的外套，再看一件标价三千的——三千突然觉得便宜。但如果你先看到的是一件标价八百的，三千就觉得很贵。你的判断被第一个看到的数字锚定了。

{{< sentence >}}
你更容易相信和你已有观点一致的信息，不是因为那些信息更可靠，而是因为大脑在省力。
{{< /sentence >}}

确认偏误。你已经觉得某个方案是对的，然后你开始"收集信息做验证"。但你不自觉地只关注支持你观点的信息，忽略反对的信息。你以为在做客观分析，实际上在给自己的直觉找证据。

## 群体和情境的影响

{{< sentence >}}
一个信息被重复得越多，你越觉得它是对的——不是因为证据变多了，而是因为大脑把熟悉感当成了真实性。
{{< /sentence >}}

可得性偏误的变体。广告之所以有效不是因为它说的内容让你信服。是因为你看了太多次。大脑把"我经常看到这个说法"翻译成了"这个说法大概是对的"。Naughton 把这个叫做"真相错觉效应"——重复本身就能制造可信度。

{{< sentence >}}
当周围的人都在做同一件事时，你做同一件事的概率会大幅增加——即使你知道那件事不合理。
{{< /sentence >}}

从众效应。开会时前三个人都支持方案 A，轮到你的时候你本来想支持方案 B——但你说了"我也赞成方案 A"。不是因为你被说服了，而是因为和多数人不同的心理成本太高。你用"避免社交风险"替代了"做出最好判断"。

## 行动层面

{{< sentence >}}
做完一个判断之后，主动搜索一条反对证据。如果你找不到，不是因为你的判断完美，而是因为你的搜索方式被偏误控制了。
{{< /sentence >}}

这是 Naughton 给出的最实用的操作建议。对抗确认偏误的方法不是"更客观地思考"——这太抽象了。而是一个具体动作：做完判断之后，花五分钟专门搜索"为什么我可能是错的"。

{{< sentence >}}
大脑不会告诉你它在走捷径。你能做的不是关掉捷径，是在走完捷径之后做一次检查。
{{< /sentence >}}

这句话定义了隐性逻辑的管理策略。你没法阻止大脑使用捷径——那是它的默认运行方式。但你可以在捷径产出了一个结论之后，用慢思考做一次校验。校验不需要推翻捷径的结论，只需要确认捷径这次没有带偏。
