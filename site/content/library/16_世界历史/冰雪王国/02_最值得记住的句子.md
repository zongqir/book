{{< sentence >}}
德隆相信北极有一片温暖的开放水域——这个假设在出发前就是错的。但当时没有人能证明它是错的，因为没有人去过那里。无法证伪的假设是最危险的假设。
{{< /sentence >}}

无法被证伪的信念有一种特殊的危险——你会一直相信它直到它把你杀死。珍妮特号的悲剧始于一个不可能在出发前被推翻的错误假设。

{{< sentence >}}
船被冰层困住的那一刻，所有人都知道情况不对——但没有人知道该怎么办。计划是为已知风险设计的；当风险超出所有人的经验时，计划就是废纸。
{{< /sentence >}}

准备工作只能覆盖你预见到的风险。真正致命的是你根本没想到的那些。

{{< sentence >}}
在冰原上行走时，德隆每天记日志——详细到每一顿饭、每一个人的身体状况、每一次争吵。在一切都在崩溃的时候，记录本身变成了一种秩序的象征。
{{< /sentence >}}

极端环境中维持日常仪式不是浪费精力——它是抵抗混乱的最后手段。

{{< sentence >}}
三条救生艇分开后只有一条到达了有人居住的地方。另外两条的命运取决于风向、洋流和纯粹的运气。能力只决定你能走多远，运气决定你能不能活着到达。
{{< /sentence >}}

在极端环境中承认运气的作用不是消极——而是对活下来的人保持谦卑的前提。

{{< sentence >}}
珍妮特号的残骸漂流到了格陵兰海岸——证明了北极存在洋流。这次灾难性的探险意外地为后来的北极研究提供了关键数据。失败的探险也可以产生有价值的知识。
{{< /sentence >}}

失败不等于没有价值——但这个价值通常由后来者收获而不是当事人自己。
