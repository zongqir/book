{{< sentence >}}
不加分别地把所有书都精读，是对有限注意力的浪费。
{{< /sentence >}}

读完一年后，还记得的知识大概率只有全部内容的 5-10%。把有限的精力分配到能高效记忆的部分，比盲目全精读更理性。

{{< sentence >}}
读之前明确一个具体的记忆目标，而不是"想读这本书"。
{{< /sentence >}}

"我想读《经济学原理》"和"读完《经济学原理》后我需要解释供求关系如何影响房价"，后者有具体的记忆靶子。前者期待读完自动记住，结果往往一场空。

{{< sentence >}}
记忆不是读书完成的标志，记忆固化才是。
{{< /sentence >}}

读完只是第一步。三天后、一周后、一个月后还能调用，记忆才算真的进来了。短期记忆和长期记忆需要不同的操作。

{{< sentence >}}
用间隔重复打破遗忘曲线。
{{< /sentence >}}

读完两天内复习一次，一周后再复习一次，一个月后再复习一次。三次复习后，短期记忆会转化成长期记忆。第一次复习的时机最关键。

{{< sentence >}}
合上书时最需要的是主动回想，而不是重新翻页。
{{< /sentence >}}

即时复习的最高效形式是不看原文，逼自己用自己的话复述核心内容。这种努力本身就是强化记忆的过程。

{{< sentence >}}
输出倒推输入，而不是输入期待自动输出。
{{< /sentence >}}

先决定"读完后我需要用这个知识去做什么"，再从目标倒推"我需要重点记哪些"，最后在阅读时围绕这些重点去过滤。

{{< sentence >}}
分级阅读是为了分配努力，不是为了品味优越感。
{{< /sentence >}}

一级浏览的书不是"低级"，只是"这个阶段不需要深度理解"。如果有个入门概念就够了，二十分钟浏览比花五小时精读更聪明。

{{< sentence >}}
建立自己的"知识备忘单"，而不是依赖笔记软件的搜索功能。
{{< /sentence >}}

那些你持续调用的知识才值得存。一本书读完，把能用上的三到五个具体判断或技巧写进一个持久的列表。每次用时就能快速定位，而不是翻翻笔记记得有这么回事。
