# 《卢瑟经济学》最值得记住的句子

## 最值得记住的句子

{{< sentence no="1" quote="分配结果先由位置决定，再由努力修正。" >}}
这句是总开关。以后遇到“只要更努力就能拿到同等回报”的说法，先用它做第一道过滤。
{{< /sentence >}}

{{< sentence no="2" quote="交易形式可以很自由，交易地位却可能极不对称。" >}}
它纠正“双方都签字就等于公平”的误判，适合放在工资谈判、平台分成、租住合同这些场景里。
{{< /sentence >}}

{{< sentence no="3" quote="谁控制关键要素，谁就更接近定价权。" >}}
这句能把注意力从价格波动拉回到控制权本身，适合看土地、渠道、牌照、平台入口和资本结构。
{{< /sentence >}}

{{< sentence no="4" quote="危机常常不是偶发故障，而是扩张逻辑的阶段性兑现。" >}}
它提醒你少把危机看成“倒霉的一次下跌”，多去追问前期收益和风险是怎样被分配与延后的。
{{< /sentence >}}

{{< sentence no="5" quote="金融可以延后矛盾，但很少真正消除矛盾。" >}}
看见宽松、加杠杆、资产再定价时，用这句防止自己只盯短期缓和，不看中长期压力转移。
{{< /sentence >}}

{{< sentence no="6" quote="把所有问题都归因于个人能力，是结构分析最常见的失焦点。" >}}
这句适合拿来对冲“全责归己”叙事，尤其在职业焦虑和理财失败复盘时有用。
{{< /sentence >}}

{{< sentence no="7" quote="一套理论是否中立，要看它默认替谁解释世界。" >}}
这是阅读钥匙。它会迫使你在看任何经济评论时先问: 这段话默认站在谁的位置上。
{{< /sentence >}}

## 这些句子怎么用

- `记录项`: 每周选 1 条句子，配 1 个现实案例，写 3 句“这条句子如何改写了我的判断”。
- `判断项`: 读到一条理财建议时，先用第 1、2、3 句做三连问: 位置是谁、控制权在哪、代价由谁承担。
- `顺序`: 先用第 7 句判断叙事立场，再用第 4、5 句判断周期风险，最后用第 6 句回到自己的可执行动作。
- `补充项`: 当一句话只能制造情绪、不能产生判断时，暂时不用它，回到能改变行动选择的句子。
