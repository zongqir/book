{{< sentence >}}
好产品会说话，不需要被卖出去。
{{< /sentence >}}

传统营销的逻辑是"制造好产品→通过广告和销售渠道卖出去"。新营销的逻辑是"制造好产品→用户主动分享和传播"。这个转变看起来很小，但影响是根本性的。

{{< sentence >}}
品牌的价值不在于企业说自己有多好，而在于用户相信你有多好。
{{< /sentence >}}

一个品牌说自己质量好，不如十个用户说好。用户的口碑比企业的广告更有信任力。所以新营销的核心是如何赢得用户的信任和推荐。

{{< sentence >}}
透明是最好的营销。
{{< /sentence >}}

互联网时代，隐瞒成本很高。用户可以轻松地找到真相。反而是那些愿意透明、愿意承认不足、愿意和用户坦诚沟通的企业，获得了更多的信任。

{{< sentence >}}
用户参与是新营销的核心。
{{< /sentence >}}

不是企业单向地给用户推送信息，而是邀请用户参与产品设计、价格定位、营销创意。这个参与过程本身就成了营销的一部分。

{{< sentence >}}
营销的本质是关系，不是交易。
{{< /sentence >}}

传统营销追求的是单次交易的转化率。新营销追求的是和用户建立长期的关系。一个用户如果信任你的品牌，他会一次次地来。
