# 一想频繁调整，就先把这几句翻出来

## 把赢面放在配置、成本和纪律上

{{< sentence >}}
大多数人最该先承认的，不是自己不聪明，而是自己很难长期稳赢市场。
{{< /sentence >}}

这句不是让你放弃，而是让你先把游戏规则看清。

{{< sentence >}}
能控制的变量，通常只有成本、税和配置。
{{< /sentence >}}

一想做复杂动作时，先回到这句。

{{< sentence >}}
资产配置比单只股票更决定长期体验。
{{< /sentence >}}

真正影响你能不能拿住的，往往不是选股，而是整体结构。

{{< sentence >}}
再平衡是在和情绪对着干，不是在追求漂亮动作。
{{< /sentence >}}

它的价值不在刺激，而在把你拉回计划。

{{< sentence >}}
热门资产最擅长让你忘记分散。
{{< /sentence >}}

每次风口起来，这句都该放在最前面。

{{< sentence >}}
指数化不是放弃思考，是把思考放回自己能赢的地方。
{{< /sentence >}}

普通投资者真正的优势，不一定在选股，而在管理自己。

{{< sentence >}}
频繁调整常常只是把控制感买得很贵。
{{< /sentence >}}

成本和税会悄悄把这种控制感收费。

{{< sentence >}}
真正难的不是知道长期持有，而是熬得过中间那段无聊。
{{< /sentence >}}

长期方案最常被破坏的时刻，往往是它看起来太普通的时候。

## 账户开始躁动时，回来翻这页

- 市场大涨时，你还会不会按原计划配置，而不是把仓位追向热门资产
- 调仓前，你会不会先看成本和税
- 账户无聊时，你会不会为了有动作而加新策略
- 再平衡时，你是不是在按规则做，而不是按情绪做
- 想选股跑赢时，你会不会先问自己有没有稳定优势

这页不是让你更勤快，而是让你把控制欲收回到成本、配置和再平衡上。
