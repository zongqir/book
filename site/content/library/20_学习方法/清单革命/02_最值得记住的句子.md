{{< sentence >}}
人类的失败分两种：无知之错和无能之错。清单解决的是后者。
{{< /sentence >}}

无知之错是我们不知道该怎么做。无能之错是我们知道该怎么做但没做到。在现代世界的很多领域——医疗、航空、建筑——无能之错远比无知之错更致命。清单不教你新东西，只是确保你不遗漏已经知道的东西。

{{< sentence >}}
清单不是操作手册，是安全网。
{{< /sentence >}}

好的清单不会告诉你怎么做手术——而是确保你在做手术之前检查了关键步骤。它不替代专业判断——而是在专业判断之前设一道关卡。

{{< sentence >}}
越是专家越觉得自己不需要清单。
{{< /sentence >}}

经验丰富的飞行员和外科医生最抵制清单——"我做了二十年了还需要看清单？"但数据表明，资深专家犯低级错误的概率和新手一样高。经验增加的是判断力，不是记忆力。

{{< sentence >}}
好清单不超过九项，六十秒内可以完成。
{{< /sentence >}}

清单太长就没人用。葛文德的设计原则：只列最关键的"杀手步骤"——漏掉会导致严重后果的步骤。其他步骤交给专业训练和常规流程。

{{< sentence >}}
清单最大的副作用是让专业人士觉得被冒犯。
{{< /sentence >}}

推行清单制度的最大障碍不是技术——是人的自尊心。"你是在暗示我不够专业吗？"克服这个障碍需要数据和文化——用证据证明清单有效，用领导示范表明使用清单不是可耻的。
