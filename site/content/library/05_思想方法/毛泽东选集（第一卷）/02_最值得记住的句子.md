# 八句纠偏提醒：从拆位置到改判断

这些句子不是拿来背的，是拿来纠偏的。

一想跳过调查、跳过分层、跳过主次判断，就该把它们调出来。

{{< sentence no="1" quote="位置关系拆不清，后面的站位和策略都会漂。" source="阶级分析方法线" >}}
总开关。任何复杂局面里，参与者没有先拆清，站位、组织和策略全部悬空。

适合在你准备直接评论、直接站队、直接出手之前先压一下自己。
{{< /sentence >}}

{{< sentence no="2" quote="没有调查就没有发言权。" source="《反对本本主义》" >}}
专门对付那种看了几篇材料就急着下重判断的冲动。没到现场，没碰具体差异，很多确定感都是借来的。

当你说「已经差不多明白了」的时候，这句最该翻出来。
{{< /sentence >}}

{{< sentence no="3" quote="主要矛盾抓不准，力量摊在次要战场，什么都打不穿。" source="《矛盾论》方法线" >}}
适合放在项目失焦、团队什么都想做、议题铺得太开的时候。先压缩战场，再谈推进。

纠正的是「看见什么都重要」的失焦。
{{< /sentence >}}

{{< sentence no="4" quote="表面弱不等于没有生路；条件在变，要追问裂缝和支点。" source="《星星之火，可以燎原》" >}}
想因为外部强大就直接判死局时，用这句先挡一下。它会把你推回条件判断，不让你停在强弱印象。

也提醒你：支点得是真的，不是喊出来的。
{{< /sentence >}}

{{< sentence no="5" quote="书本给方向，现实给校正；两者脱节时，先回现实。" source="《实践论》" >}}
纠正两种偏差：书本崇拜和经验自满。更稳的做法是不断来回校正。

最适合在概念争得很热、可谁都说不清现实细部的时候调用。
{{< /sentence >}}

{{< sentence no="6" quote="没进过真实处境，替别人定义的问题往往只是你自己的投射。" source="调查研究方法线" >}}
适合对冲那种替别人安排优先级的管理冲动。没有进入真实处境，很多「代表」其实只是代言。

做组织、做产品、做写作，都能帮你挡住「我懂他们需要什么」的傲慢。
{{< /sentence >}}

{{< sentence no="7" quote="方向错了还把执行力往上加，只会偏得更快。" source="《关于纠正党内的错误思想》" >}}
适合放在复盘里。很多失败的根源在判断歪了以后越拼越糟。

团队开始靠更密的会议、更硬的纪律掩盖方向问题，这句就该顶到最前面。
{{< /sentence >}}

{{< sentence no="8" quote="认识不是一次想通的，要在实践里一轮轮修出来。" source="《实践论》" >}}
因为一次试错就想否定整条线，或者一次顺利就想把经验固化时，用这句给自己降温。

提醒你把一次反馈放进更长的校正链里看。
{{< /sentence >}}

## 按场景调用

- **读现实议题**：先用第 1、2、3 句。先拆参与者，再补调查，再压缩主次。
- **看弱势局面**：先用第 4、8 句。别被表面强弱吓住，也别把一轮结果当成终局。
- **带团队或做复盘**：先用第 6、7 句。先问真实处境，再问方向是不是已经歪了。
- **遇到理论争论**：先用第 5 句。概念争得越热，越要回去看现实支不支持。
- **准备大干一场**：把第 2、3、7 句连着看。材料薄、主次不清、方向可疑时，动作越猛越危险。
