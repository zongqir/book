## 关于确定性的陷阱

{{< sentence >}}
以为你知道的就是全部。在无知中，未知的往往比已知的更重要。
{{< /sentence >}}

{{< sentence >}}
听了某个专家的意见就以为真的了解了。专家错过的，往往和他知道的一样多。
{{< /sentence >}}

## 关于概率的误解

{{< sentence >}}
把"很可能"当成了一定会发生。一个 90% 确定性的预测，仍然有 10% 的概率完全错了。
{{< /sentence >}}

{{< sentence >}}
忘记了平均值背后隐藏的波动。"平均回报 10%"可能意味着一半的时间是 50%，一半的时间是亏 30%。
{{< /sentence >}}

## 关于容错的忽视

{{< sentence >}}
为了追求最高的收益，选了一个最脆弱的方案。一旦出错，就彻底崩盘。
{{< /sentence >}}

{{< sentence >}}
没有想过"如果我错了"会怎样。最坏的决策，就是没有准备最坏情况的决策。
{{< /sentence >}}

## 关于信息的陷阱

{{< sentence >}}
你掌握的信息越多，越容易陷入"我现在知道"的错觉。其实新信息只是改变了你不知道的东西。
{{< /sentence >}}

{{< sentence >}}
追求完美信息而延迟决策。在很多情况下，现在就决定（基于有限信息）比等待完美信息更好。
{{< /sentence >}}
