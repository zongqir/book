# 颠覆来临前最该记住的判断句

## 这组句子怎么用

这里收的不是名言摘抄，而是能在会议里及时踩刹车的提醒句。

选句标准只有三个：能早点识别颠覆信号，能戳破报表上的虚假安全感，能把问题指回系统，不急着怪个人。

排序也按决策顺序来：先看市场和客户，再看技术轨迹，最后看组织和资源。

## 看市场和客户时要记的句子

{{< sentence no="1" quote="主流客户要得越多，过度服务的风险越大。" >}}
{{< /sentence >}}

当指标已经远超客户真实需求时，多半在给颠覆者让路。

{{< sentence no="2" quote="客户说暂时用不上，不代表未来也用不上。" >}}
{{< /sentence >}}

客户抱怨的是今天的问题，颠覆解决的是明天的问题。

{{< sentence no="3" quote="只从最大客户收集需求，视野就被价值网络锁死。" >}}
{{< /sentence >}}

利润最好的客户，也是最会拖慢你转向的人。

{{< sentence no="4" quote="愿意忍受缺点的那批边缘用户，往往站在下一轮标准上。" >}}
{{< /sentence >}}

如果一个市场只能在大公司报表里当尾注，它很可能在孕育新规则。

{{< sentence no="5" quote="当你凭经验觉得某个新产品“不够专业”时，要先问一句：对谁来说。" >}}
{{< /sentence >}}

主流客户说"太简单"，边缘客户说"刚好"，这是危险信号。

## 看技术轨迹时要记的句子

{{< sentence no="6" quote="技术的危险不在今天的性能，而在改善速度。" >}}
{{< /sentence >}}

当新技术的进步曲线比客户需求上升更陡时，跨线只是时间问题。

{{< sentence no="7" quote="便宜又差的东西，只要在持续变好，就比贵而稳定的东西可怕。" >}}
{{< /sentence >}}

一项技术一开始只满足低端用途，很可能在长出新的价值网络。

{{< sentence no="8" quote="当工程师觉得产品性能已经过剩时，要立刻问：哪种更简单的方案正在逼近。" >}}
{{< /sentence >}}

如果新技术只能在最挑剔客户那里证明价值，它不太可能是颠覆。

{{< sentence no="9" quote="颠覆型技术最初的竞争对手不是你，而是非消费。" >}}
{{< /sentence >}}

当很多用户第一次有条件用上一种产品时，行业的重心已经在移动。

{{< sentence no="10" quote="技术路线图上没有故意保留的低端选项，未来就没有退路。" >}}
{{< /sentence >}}

## 看组织和资源时要记的句子

{{< sentence no="11" quote="预算表上最亮眼的业务，也是最难被质疑的业务。" >}}
{{< /sentence >}}

当所有评审标准都围绕利润率时，颠覆性机会已经被预先判死。

{{< sentence no="12" quote="资源分配会议里没有人替小项目说话，它们就等于不存在。" >}}
{{< /sentence >}}

只要新业务还被当作"项目"，它就继承主业的全部束缚。

{{< sentence no="13" quote="组织最擅长服务的客户，也是最能绑住组织的客户。" >}}
{{< /sentence >}}

销售渠道如果只能卖高毛利产品，很难为颠覆性产品留出口。

{{< sentence no="14" quote="当不同业务共享一套业绩考核，新业务通常输给旧业务。" >}}
{{< /sentence >}}

如果一个团队要一边守住主业指标，一边探索新市场，它多半两头都顾不好。

{{< sentence no="15" quote="流程为了效率而定型的那一刻，适应性就开始下降。" >}}
{{< /sentence >}}

## 做每个重大决策前的复读句

{{< sentence no="16" quote="今天这个决定，是不是进一步把资源锁在现有价值网络里。" >}}
{{< /sentence >}}

{{< sentence no="17" quote="我们有没有有意识地为小市场保留决策权，而不是让它在大流程里被稀释。" >}}
{{< /sentence >}}

{{< sentence no="18" quote="如果一个小公司以远低价格满足了部分客户的“够用”需求，我们准备好承认那是真竞争了吗。" >}}
{{< /sentence >}}

{{< sentence no="19" quote="当团队用“现在不急”解释某个方向时，有没有讨论过它的改善速度。" >}}
{{< /sentence >}}

{{< sentence no="20" quote="这次立项会，如果完全听从最重要客户的声音，结果会偏向哪里。" >}}
{{< /sentence >}}

{{< sentence no="21" quote="五年后回头看，现在放过的这个低端市场，会不会成为无法追赶的缺口。" >}}
{{< /sentence >}}
