经济增长的原因不在统计公报里。在土地出让年限、税收分成比例、承包责任书的利润分配条款里。

《中国的经济制度》提供了一组从产权和合约微观结构出发的分析准则。掌握这组准则，可以拆解任何经济体的制度安排；中国只是展开示范的案例。下面七条，前四条是分析基座，后三条是运作机制。

{{< principle no="1" type="core" statement="产权看功能，不看名分——使用权、经营权、收益权到位，效率就到位" >}}

西方产权理论的标准推理：明晰的私有产权→低交易成本→高效率。产权模糊的经济体，按这条逻辑不该有高增长。

中国的土地国有，企业产权复杂，股权结构经常说不清。但使用权可以转让，经营权可以承包，收益权可以按合同分配。三项权利各自有明确的合约载体，激励就传导到位了。

判断产权安排是否有效，直接问三件事：谁在用？谁在经营？收益怎么分？三个答案清楚，效率就有保障。

名义上的私有产权也可能失效。管制、审批、不可预期的执法层层侵蚀，行使不了的产权等于不存在。分析制度时，所有制标签可以先放一边。

{{< /principle >}}

{{< principle no="2" type="core" statement="交易费用决定合约形式——制度是约束条件的产物，不是理想蓝图的实现" >}}

为什么农村选了家庭联产承包制？为什么工业用了各种承包合同？

答案是交易费用。度量产权的费用太高，就用合约替代；监督个人的费用太高，就改分成制。在当时的约束条件下，被选中的安排最省成本。

制度分析的操作要点：不评判某种制度"先进"或"落后"。先估算可选安排各自的交易费用，再判断现行安排是否合理。

日本终身雇佣制、硅谷期权合约、德国学徒制——都对应特定的交易费用条件。条件变了，安排也该跟着变。

{{< /principle >}}

{{< principle no="3" type="core" statement="层级分成合约对齐纵向激励——收益按比例切到每一层，每层都有动力推产出" >}}

中央和地方的关系不只是命令与执行。分税制之后，税收收入按比例在中央、省、市、县之间分配。每一层拿到的份额与本层经济产出挂钩。

佃农理论提供了现成的类比。佃农交固定地租，产量多少跟自己无关，干活动力有限。改成比例分成——产量越高，佃农分到越多——激励就对了。同样的逻辑放到行政层级上：县级GDP越高，县里留下的越多。

分成合约有一个关键条件：比例稳定可预期。比例频繁变动，下级就会隐藏收入、减少投入。1994年分税制的核心贡献是把分成比例固定下来。

同一套行政架构，在不同分成规则下表现截然不同。分成规则一变，地方行为就跟着变。

{{< /principle >}}

{{< principle no="4" type="core" statement="管辖区竞争替代市场竞争——县与县争资源的效果等价于企业争客户" >}}

几千个县同时争投资、争项目、争税基。一个县开出的条件不够好，企业转身去隔壁。

竞争需要两个前提：决策者承担后果；决策者有足够的自主权。1994年之后的县级政府同时满足——财政收支自负盈亏，招商引资基本自主。

县际竞争不是设计出来的。它是分税制、土地制度和干部考核三个变量交叉后自发长出来的格局。没有人画过蓝图。

美国各州的税制竞争、欧盟成员国的监管竞争、城市之间的人才争夺——管辖区有自主权且承担后果，竞争就会出现。

{{< /principle >}}

## 合约条款怎么变，制度就怎么变

四条核心准则搭好了分析骨架。下面三条填入运作机制：制度怎么变、怎么试、怎么拆。

{{< principle no="5" type="support" statement="制度变迁发生在合约条款的边际调整上，不需要一次性体制转轨" >}}

家庭联产承包责任制没有废除集体所有制。它只改了一条合约条款：经营权从集体转到农户。

经济特区没有改变全国的贸易体制。它只在特定区域执行一组不同的税收和准入条款。

渐进式改革的微观机制在这里：每次改一两条合约条款，观测激励变化，有效就扩大范围。不需要先解决"市场经济还是计划经济"的定性问题。

{{< /principle >}}

{{< principle no="6" type="support" statement="合约实验可以分区进行——划一块地、定一组条款、观测结果、决定推广还是止损" >}}

深圳、珠海、汕头、厦门——四个经济特区就是四份不同的合约。跑通了，条款推广到沿海城市；跑出问题，损失锁定在试验田。

分区实验的关键是风险结构可控：实验区和非实验区之间有清晰的合约边界，失败不扩散。

企业同样适用。新业务先在一个区域试跑，用独立的激励条款和核算单元。验证有效再铺开。

{{< /principle >}}

{{< principle no="7" type="support" statement="分析经济现象，先找激励结构的微观载体——合约、分成规则、承包协议" >}}

GDP增速放缓，常见反应是谈"大环境"。但"大环境"是一系列具体合约安排的总效果。

操作方式：找到决定激励方向的具体合约——土地出让合同、税收分成规则、经营承包协议。看条款有没有变、变了以后激励指向哪里。

拿房地产下行举例。"楼市怎么了"太粗。拆成合约层面：土地出让收入下降→县级财政激励结构改变→招商行为转向。每一步可追踪、可验证。

{{< /principle >}}

## 从产权功能到管辖区竞争

产权看功能（第一条）打开了分析空间——不纠结所有制标签。交易费用决定合约形式（第二条）给出判断依据。层级分成（第三条）对齐纵向激励，管辖区竞争（第四条）对齐横向激励。

三条支撑准则解决"变化怎么发生"：边际调整（第五条）和分区实验（第六条）是路径，微观载体先行（第七条）是分析方法。

没有第一条，后面所有条都会卡在"产权不清晰"的标签上走不动。

## 合约需要执行保障

七条准则共享一个前提：合约条款能够被执行。

执行依赖两个条件：法律或行政体系提供最低限度的合约保障；违约成本高于守约收益。合同不被法院支持、政策承诺随时推翻、地方政府信用打折——合约的激励功能就会衰减。

产权的功能靠合约实现，合约的功能靠执行保障。链条断在任何一环，效率就打折扣。
