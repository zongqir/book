{{< sentence >}}
你的简历在HR手里被审视了多长时间，就说明了你有多大的被重视可能。
{{< /sentence >}}

在招聘过程中，一份简历在HR那里停留的时间很能说明问题。如果被快速否决，说明你不符合他们的某个隐含标准。如果被长时间审视，说明你勾起了他们的兴趣。

{{< sentence >}}
薪酬的不透明本身就是一种权力工具。
{{< /sentence >}}

如果你永远不知道同事拿多少钱，你就永远无法确定自己是否被公平对待。这种不确定性让人陷入内疚感和低估自己的状态。HR正是通过这种不透明来维持对员工的微妙控制。

{{< sentence >}}
被调整职位往往不是为了让你发挥更大作用，而是为了让你自愿离开。
{{< /sentence >}}

一个看起来"更有挑战性"的职位调整，可能是一个信号——你已经不适合现有的岗位。这个新位置可能充满了预设的困难，目的就是让你逐渐意识到自己不适合。

{{< sentence >}}
你能否进入某个会议室，就决定了你能否获得关键信息。
{{< /sentence >}}

在组织中，哪些人被邀请参加关键会议是精心选择的。被排除在重要决策会议之外，往往就是被权力圈子排除的信号。这个信号比任何明说的话都清楚。

{{< sentence >}}
晋升标准的灵活性，往往取决于某个人对现有权力的威胁程度。
{{< /sentence >}}

有些人能够以"不足够"的条件被晋升，因为他们被看好。有些人即使条件超群也被卡住，因为他们被认为是威胁。标准本身不重要，重要的是标准怎样被应用。

{{< sentence >}}
HR最大的权力不在于他们的话，而在于他们的沉默。
{{< /sentence >}}

当HR选择对某个人的不公平沉默以对，这个沉默本身就成了一种权力压制。被注意到的人，也许还有回旋的空间。被无视的人，才是真正陷入困境的。

{{< sentence >}}
你被分配到哪个团队，往往已经决定了你的职业前景。
{{< /sentence >}}

team assignment看起来很随意，但其实是精心计划的。好团队的人升职快。差团队的人再努力也看不到出头。HR通过控制这个分配，实际上是在控制每个人的职业轨迹。
