# 七条判断原则，把多元模型和反愚蠢纪律压成可调用准则

判断老出大错，往往不是少了一个更聪明的观点，而是少了几条会在关键时刻拦住自己的原则。《穷查理宝典》最值得长期带着走的，正是这些会不断约束冲动、虚荣和单一解释的判断准则。

{{< principle no="1" type="core" statement="长期判断的第一目标，不是显得更聪明，而是持续少做那些完全可以避免的蠢事。" >}}
很多人一上来就追求洞见和高明，芒格更在意的是先把那些反复会毁掉长期结果的错排掉。这个原则约束的误判很具体：把聪明表演当成判断质量，把偶尔做对当成体系可靠。少犯蠢不是保守，而是把长期复利先从明显漏洞里救出来。
{{< /principle >}}

{{< principle no="2" type="core" statement="任何判断都要尊重能力圈，知道自己不知道什么，比强装什么都懂更重要。" >}}
能力圈不是谦虚姿态，而是风险边界。它提醒你别把一时自信误当成真懂，也别因为旁人赚钱就把陌生问题硬拖进自己的判断区。常见误读是把能力圈读成永远不扩张，芒格更像是在说：可以学，可以扩，但别在没弄懂前先下注。
{{< /principle >}}

{{< principle no="3" type="core" statement="现实问题通常跨学科存在，单一模型很容易把复杂问题压扁。" >}}
多元模型真正约束的，是脑子里只有一把锤子时那种过度自信。业务、制度、人性、激励、周期往往同时起作用，只用一个学科的语言解释全部现实，结论常常会看上去利落，实际却漏掉关键变量。这个原则不是鼓励炫知识，而是要求交叉校正。
{{< /principle >}}

{{< principle no="4" type="support" statement="激励结构几乎总在强烈塑造人的行为，不看激励就很容易看错人和制度。" >}}
很多表面上冠冕堂皇的解释，一碰到激励结构就会变形。这个原则逼你少相信自我描述，多看谁因为什么得到好处、承担什么损失。最常见的误读，是把激励只理解成金钱，其实职位、面子、短期考核、组织氛围都在塑造行为。
{{< /principle >}}

{{< principle no="5" type="support" statement="反过来想往往比正面硬冲更能揭出真正危险的错法。" >}}
逆向思考的价值，不是故作深刻，而是先看失败路径。你先问这件事最可能怎么搞砸、哪些条件一旦缺位就别做，很多判断会立刻收紧。常见误读是把逆向当成唱反调；真正的逆向，是先用失败视角查漏，再决定值不值得正面推进。
{{< /principle >}}

{{< principle no="6" type="support" statement="真正高质量的机会不常出现，耐心等待本身就是判断力的一部分。" >}}
这个原则反对的是“手里有钱、有精力、有视角，就总得做点什么”的冲动。很多长期错误不来自看不见机会，而是来自低质量机会太多、自己又太急。耐心并不是消极，而是在好机会稀缺时不拿平庸行动安慰自己。
{{< /principle >}}

{{< principle no="7" type="support" statement="情绪稳定不是判断的附属品，它本身就是长期判断质量的关键条件。" >}}
模型再多，情绪一乱也会被拿去给冲动找理由。这个原则提醒你，判断不是纯脑力竞赛，还要看人在压力、贪婪、从众和挫败里能不能维持基本清醒。它约束的不是情绪存在本身，而是情绪总抢在判断前面接管动作。
{{< /principle >}}
