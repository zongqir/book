# 五条领导资格原则：第三卷怎样回答党凭什么领导中国

第三卷最怕被读成一般组织学。

它当然讲学习、作风、表达、领导方法，但这些在这里都不是普通内部问题。第三卷真正要回答的是：党凭什么领导中国。下面五条原则，就是这个问题的骨架。

{{< principle no="1" type="core" statement="党必须不断改造自己，不然即使路线大体正确，也没有资格领导中国" >}}
第三卷最硬的前提，不是“党当然应该领导”，而是领导资格必须在革命过程中不断重建。

为什么要整风？为什么要反对主观主义、宗派主义和党八股？因为一支党如果不能正视自己的问题，不能把自己从空话、僵化和圈子化里拉出来，它就会先失去和中国现实的真实联系。

这条原则保护的，不是内部整洁，而是政治资格。能不能领导，不只看你反对谁，也看你能不能改造自己。

常见误读是把整风理解成单纯纪律整顿。第三卷更深一层：不改造自己，党就会在最关键的时候变成它自己曾经反对过的那类空心力量。
{{< /principle >}}

{{< principle no="2" type="core" statement="学习必须从中国现实出发，不能把现成理论放到现实前面" >}}
《改造我们的学习》压出的，不是“多读书”或“少读书”。

它要求的是顺序：先有中国的问题，再有理论的进入；先看现实长什么样，再看理论怎么解释它。

这条原则之所以是核心原则，是因为第三卷很多别的判断都靠它站住。现实进不来，后面的整风会变假，群众路线会变姿态，全国政治方案也会飘。

常见误读是把它说成一句普通反教条口号。第三卷原来的力度更重：理论一旦跑到现实前面，党学得越多，反而可能离中国越远。
{{< /principle >}}

{{< principle no="3" type="support" statement="学风、党风、文风是一件事，三者一起坏，党就会同时失去判断、组织和表达能力" >}}
第三卷为什么反复把这三样连着说？因为它们本来就连在一起。

学风坏了，材料不扎实，判断会假。党风坏了，真问题进不来，组织会裂。文风坏了，哪怕偶尔有了相对正确的判断，也传不下去，变不成共同动作。

这条原则防的是把毛病切碎来看。把文风当成写作问题，把党风当成人际问题，把学风当成学习问题，都会低估第三卷的锋利。

第三卷真正指出的是：三者会互相放大，最后一起决定党能不能领导。
{{< /principle >}}

{{< principle no="4" type="support" statement="群众路线不是亲民姿态，而是认识和领导的共同方法" >}}
“从群众中来，到群众中去”后来太常见，反而容易失真。

第三卷里的意思很具体。群众中的经验是党认识中国的重要来源，但这些经验零散、分散，必须经过集中、提炼和比较，才能变成路线。路线形成以后，又必须回到群众中检验。

这条原则保护的，是党既不空、也不散。只靠上面想，党会脱离现实；只靠下面堆意见，组织又长不出共同办法。

常见误读是把群众路线读成道德要求。第三卷更硬：它首先是一种认识论，也是一种领导方法。
{{< /principle >}}

{{< principle no="5" type="support" statement="全国领导资格必须在胜利前就讲清，不是赢了以后自然会有" >}}
《论联合政府》把第三卷从党内问题一下推到了全国问题。

这条原则的意思是：党能不能领导全国，不是等胜利来了再说，而要在胜利前就向全国说明自己准备建立什么样的政治秩序，怎样处理不同力量，又凭什么代表更广大的人民。

这条原则让第三卷的整风、群众路线和全国政治方案连成了一件事。党内改造不是为了内部纯洁，而是为了面向全国；群众路线也不是地方经验，而是通向全国领导的一条路。

常见误读是把《论联合政府》当成末尾附加的政治宣言。其实它是第三卷整套原则的外化结果。
{{< /principle >}}

这五条合起来，第三卷的重心就清楚了。

党必须改造自己，学习必须从中国出发，学风党风文风不能分开看，群众路线是认识和领导的共同方法，全国领导资格也必须在胜利前就讲清。

这样再看第三卷，才不会把它压成一本普通组织手册。
