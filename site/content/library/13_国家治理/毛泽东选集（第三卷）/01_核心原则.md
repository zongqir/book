# 五条组织准则：从反对空话到把群众经验变成共同动作

第三卷里的原则，不再主要解决“局势怎么看”。

它们更像组织准则。方向已经有了，为什么组织还是会学偏、说偏、做偏？答案反复落在同一件事上：现实经验有没有进入组织，组织又有没有把它变成共同判断和共同动作。

下面两条是核心准则，后三条是支撑准则。

{{< principle no="1" type="core" statement="学习必须从现实问题出发，不从现成公式出发" >}}
材料没进来，结论再完整也是空的。

《改造我们的学习》真正约束的是一种常见错法：先拿着一套现成框架，再逼现实去配它。这样看起来学习很多，实际是在回避问题本身。

这条准则保护的不是“多调研”这么简单。它要求组织把真实问题当成学习入口，而不是把学习当成证明自己早就对了的过程。

{{< /principle >}}

{{< principle no="2" type="core" statement="组织真正的统一，靠的是原则讨论和可理解表达，不是表面一致" >}}
没有分歧，不等于没有问题。没人听得懂，也不等于路线已经传达。

第三卷把学风、党风、文风放在一起谈，背后是同一个判断：组织想形成真正统一，既要让真实分歧浮出来，也要把正确判断讲成别人能理解、能接受、能照着做的话。

这条准则防的是两头偏。一头是只讲正确，不管别人听不听得进去；另一头是为了维持和气，把原则讨论都压成沉默。

{{< /principle >}}

{{< principle no="3" type="support" statement="领导的本事，不是替群众想明白，而是把群众经验集中后再还回去" >}}
《关于领导方法的若干问题》最硬的一刀，是把“领导”从发命令改写成收集、集中、再返回的循环。

没有前面的收集，领导会空。没有中间的集中，组织会散。没有后面的返回，路线就只能停在会议纪要里。

这条准则提醒的是：领导不是代替群众，而是把零散经验压成共同办法。

{{< /principle >}}

{{< principle no="4" type="support" statement="表达能力是组织能力的一部分，不是附属装饰" >}}
《反对党八股》留下来的，不只是文风批评。

一句正确的话如果没人能听懂、记住、复述、照着做，它在组织里就是没完成任务。文风差，往往意味着问题没想透，或者根本不在乎能不能把人组织起来。

这条准则防的是把表达当包装。第三卷要求的，是把语言直接当成组织工具看。

{{< /principle >}}

{{< principle no="5" type="support" statement="真正大的胜利，必须在胜利前就准备好接管能力" >}}
《论联合政府》把很多人想留到“以后再说”的问题提前摆了出来。

只知道推翻什么，不知道建立什么，胜利后的局面就接不住。大局方案不是庆功时才需要的东西，而是长期组织、稳定联盟和赢得信任的前提。

这条准则防的是短视。越接近胜利，越要提前回答“之后怎么办”。

{{< /principle >}}
