{{< sentence >}}
文明的生存不是一种权利而是一种义务——每一代人都必须用行动重新赢得它。没有一个文明是因为"自然消亡"而结束的；它们都是因为不再能解决自己面临的问题而被替代。
{{< /sentence >}}

文明不会"自然老死"。它们消亡是因为它们的制度和习惯无法适应新的挑战——这意味着制度更新能力是文明存续的核心变量。

{{< sentence >}}
自由和平等是永恒的敌人。自由让强者上升、让不平等扩大；当不平等大到社会无法承受时，平等就通过暴力或立法把它压回去。然后自由再次开始扩大不平等。
{{< /sentence >}}

自由与平等的张力不是某个时代的特殊问题——它是人类政治的永恒周期。认清这个周期不是为了接受它而是为了在它到来时做出更好的应对。

{{< sentence >}}
道德的标准随着环境变化。农业社会和工业社会需要不同的道德体系——不是因为道德在"进步"而是因为不同的生产方式需要不同的行为规范。
{{< /sentence >}}

道德不是一成不变的天条而是人类社会为了合作而演化出的工具。工具需要适应使用环境。

{{< sentence >}}
历史对那些忘记历史的人施加的惩罚是让他们重演历史。但即使记住了历史的人也不一定能避免重演——因为每一代人都相信自己面临的情况是前所未有的。
{{< /sentence >}}

每一代人的傲慢都是一样的：相信自己处在历史从未面对过的全新处境中。杜兰特几千年的材料说明这种"前所未有"的感觉本身就是最古老的模式之一。

{{< sentence >}}
一个国家之所以能生存，不是因为它最强大，而是因为它最能适应变化。罗马取代希腊不是因为更有文化而是因为制度更能扩展。
{{< /sentence >}}

适应性而非绝对实力是文明存续的决定因素。
