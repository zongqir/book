{{< sentence >}}
吾敬李鸿章之才，吾惜李鸿章之识，吾悲李鸿章之遇。
{{< /sentence >}}

梁启超对李鸿章的总评浓缩在这三个动词里：敬他的才干，惜他的见识不够远，悲他的时代不允许他成功。三个层次的判断，一句话说完。

{{< sentence >}}
李鸿章是时势所造之英雄，非造时势之英雄。
{{< /sentence >}}

李鸿章的所有作为都是对外部压力的被动回应——太平天国来了就练兵，洋人来了就搞外交，甲午打了就去谈判。他从来没有主动定义过"中国应该走向哪里"。梁启超认为这是他和真正的伟人之间的根本差距。

{{< sentence >}}
不学无术，不知大体，是中国士大夫之通病，非独李鸿章为然也。
{{< /sentence >}}

梁启超在批评李鸿章的同时替他开脱了一半——这不是个人的缺陷，是整个士大夫阶层的结构性盲区。他们精通诗文和政治操作，但对现代世界的运转方式一无所知。

{{< sentence >}}
以一人而战一国，未有不败者也。
{{< /sentence >}}

甲午战争中李鸿章用北洋舰队（实质上是他个人的军事力量）对抗整个日本国家的军事动员体系。梁启超的判断是：不是李鸿章打不好，而是个人力量无法对抗国家体制。

{{< sentence >}}
知有兵事而不知有民政，知有外交而不知有内治，知有朝廷而不知有国民。
{{< /sentence >}}

梁启超对洋务运动局限性的终极概括。李鸿章知道要建军队、搞外交，但不知道现代国家需要民众参与、需要内部制度改革、需要国民意识。他做的所有事都是表层的，因为他看不到深层。

{{< sentence >}}
当戎马压境之时，为忍气吞声之外交，虽有苏张之辩，无所用之。
{{< /sentence >}}

李鸿章一生签了无数不平等条约。梁启超指出，问题不在于他的谈判技巧——当军事实力不存在时，再好的外交辞令也没有意义。
