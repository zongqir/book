{{< sentence no="1" quote="很多讨论之所以吵不清，不是因为分歧太大，而是因为根本不在同一层上说话。" >}}
{{< /sentence >}}

{{< sentence no="2" quote="现象不是机制，机制也不自动推出结论。" >}}
{{< /sentence >}}

{{< sentence no="3" quote="一旦语境换了，原来那套看似稳定的逻辑很可能也要跟着换。" >}}
