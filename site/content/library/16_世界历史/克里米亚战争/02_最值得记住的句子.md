{{< sentence >}}
克里米亚战争不是由某个疯狂的决策者发动的，而是由一群理性的决策者在信息不完整的条件下各自做出了看似合理的选择——这些选择加在一起变成了灾难。
{{< /sentence >}}

系统性灾难往往不是某个人的错，而是每个人都做了局部合理但全局有害的决定。

{{< sentence >}}
战地记者威廉·霍华德·拉塞尔的报道第一次让后方的公众看到了战争的真实面貌——不是英雄主义而是混乱、疾病和无谓的死亡。
{{< /sentence >}}

信息透明改变了战争的政治约束。当公众知道前线真正发生了什么，政府就不能再用"一切顺利"来搪塞。

{{< sentence >}}
轻骑兵冲锋是一个关于命令传递失败的经典案例——不是勇气不够而是沟通系统崩溃。发出命令的人、传递命令的人和执行命令的人理解的是三件不同的事。
{{< /sentence >}}

组织灾难经常不是因为没人做事，而是因为每个人做的不是同一件事。

{{< sentence >}}
俄国在克里米亚的失败暴露了一个帝国的核心矛盾：军事力量建立在农奴制基础上，而农奴制恰好是军事效率的最大敌人。
{{< /sentence >}}

一个系统的力量来源和它的致命弱点可能是同一个东西。

{{< sentence >}}
南丁格尔不只是一个护士，她是一个用数据说服官僚机构的改革者。她用统计图表证明士兵死于疾病而非战斗伤亡，迫使军方改变了整个后勤体系。
{{< /sentence >}}

改变系统的人不一定是权力最大的人，而是能用正确的证据在正确的时机说服正确的人的人。
