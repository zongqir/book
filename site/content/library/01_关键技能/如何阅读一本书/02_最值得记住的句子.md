# 七句话，把“读完”拉回“读懂”

{{< sentence no="1" quote="太多信息和太少信息一样，都会妨碍理解。" >}}
打开手机、打开浏览器、打开微信读书，信息从来不缺。缺的是消化。

下次觉得"我已经看了很多"却说不出要点的时候，停下来。问题大概率不在量，在于输入没变成理解。
{{< /sentence >}}

{{< sentence no="2" quote="阅读的目标不是读完，而是提升理解力。" >}}
翻到最后一页的满足感很强烈。但翻完和读懂是两件事。

每次读完一本书，试着合上它，用三句话复述作者的核心主张。说不出来，就是还没读懂。
{{< /sentence >}}

{{< sentence no="3" quote="主动阅读的核心，是带着问题读。" >}}
被动阅读像坐在传送带上等信息滑过。主动阅读是一直在问：作者想解决什么？凭什么这么说？我同意吗？

拿到新书，先写下三个你希望它回答的问题。读完回来看，答案有没有变。
{{< /sentence >}}

{{< sentence no="4" quote="还没检视就精读，是最常见的时间浪费。" >}}
看完目录、序言、结尾，花不了十分钟。但这十分钟能帮你判断一件事：接下来的十小时值不值得投。

尤其面对一摞待读清单的时候。先过一轮检视，砍掉不值得深读的，剩下的才值得认真拆。
{{< /sentence >}}

{{< sentence no="5" quote="抓不住作者的关键词，就进不了作者的思路。" >}}
每个作者有自己的核心概念。艾德勒自己的关键词就是"主动""层次""理解力"。

读任何一本论述类的书，先标出反复出现的三到五个词。搞清作者用它们指什么，后面的推理才跟得上。
{{< /sentence >}}

{{< sentence no="6" quote="在能准确复述之前，不要批评。" >}}
反对一个观点的前提是你确实听懂了它。很多争论的问题不在立场，在于双方都没准确理解对方。

下次想说"我不同意"之前，先试着用自己的话把对方的主张说一遍。说不准，就还没到评价的时候。
{{< /sentence >}}

{{< sentence no="7" quote="一个问题只靠一本书回答，永远是不够的。" >}}
单本书给你一个视角。把三五本书放进同一个问题框架里比较，共识和分歧才会浮出来。

做专题研究、写长文、做决策前，至少找两本以上持不同立场的书交叉读。靠单一来源下结论的风险太大。
{{< /sentence >}}
