{{< sentence >}}
希特勒不信任他的将军，他的将军不信任彼此。在一个建立在个人忠诚而非制度信任上的系统里，信息流动的效率取决于谁对谁说实话——而说实话的激励几乎为零。
{{< /sentence >}}

当组织的信息流动取决于个人关系而非制度设计时，坏消息会被系统性地过滤掉。

{{< sentence >}}
大屠杀不是战争的副产品，它是意识形态逻辑的终点——把一部分人类定义为非人，剩下的事情只是操作问题。
{{< /sentence >}}

语言先于行动。当一个群体被持续定义为"问题"而非"人"时，最终的暴力不是偶然的而是逻辑的必然。

{{< sentence >}}
德国的战时经济直到1943年才转入全面动员——在那之前希特勒一直相信闪电战能解决一切。这个延迟浪费了两年的工业潜力。
{{< /sentence >}}

对过去成功模式的路径依赖会让组织在需要转型时慢半拍。闪电战赢了法国，但不能赢苏联——承认这一点需要否定之前的成功。

{{< sentence >}}
纳粹体制的权力结构像一堆重叠的势力范围。没有清晰的职责边界，每个机构都在和其他机构争夺希特勒的注意力和资源。
{{< /sentence >}}

"向上竞争"的组织结构在短期可以刺激忠诚和干劲，但在需要协调的复杂任务面前会产出灾难性的资源浪费和内耗。

{{< sentence >}}
普通德国人在战争末期的心理状态不是反抗也不是支持，而是麻木。当灾难大到无法处理时，大多数人选择了缩进私人生活、只关心今天的面包和明天的轰炸。
{{< /sentence >}}

系统性压力到了极限之后人不会爆发而是关机。这解释了为什么极端环境下很少出现大规模自发反抗。
