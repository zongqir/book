# 创意黏力学里最该留下来的，是这些会在关键时候拽住你的提醒句

{{< sentence >}}
别人记不住，不一定是他不用心，可能是你没有给他抓手。
{{< /sentence >}}

这句最能打断“对方不认真听”的抱怨。

{{< sentence >}}
抽象说服不了人，具体才会留下。
{{< /sentence >}}

很多人输在明白得太抽象。

{{< sentence >}}
能讲清楚不够，别人还得能转述。
{{< /sentence >}}

这句把传播标准加上了。

{{< sentence >}}
可信不是靠喊权威，是靠让人看到为什么该信。
{{< /sentence >}}

权威能借力，但不能替你完成说服。

{{< sentence >}}
没有情绪和故事，很多正确观点只会停在头脑里。
{{< /sentence >}}

它们进不了行动。

## 这些句子什么时候最该翻出来

- 一开口就想解释很多的时候：先翻这页。
- 你觉得自己明明没说错，结果事情还是做坏的时候：先翻这页。
- 你知道问题在哪，却迟迟不知道从哪一句开始的时候：先翻这页。
- 对方的反应、组织的惯性或关系的压力开始把你带偏的时候：先翻这页。
