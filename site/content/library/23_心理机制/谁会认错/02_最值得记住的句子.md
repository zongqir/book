# 大脑不是在撒谎，是把事实改写了

## 辩护程序启动时你完全不知道

{{< sentence >}}
大多数人在大多数时候，都不会觉得自己在为自己辩护。这恰恰是自我辩护如此强大的原因。
{{< /sentence >}}

认知失调的核心产出。犯错之后的默认反应不是修正，而是加固。下次发现自己在为一个站不住的决定找更多理由时，停下来——辩护程序已经在跑了。

## 记忆不是录像机，是辩护的同谋

{{< sentence >}}
记忆不是忠实的记录装置，而是一个讲故事的装置，它讲的故事服务于自我。
{{< /sentence >}}

两个人吵完架，记住的是两个不同版本的"事实"。没有人在说谎——记忆在事后被辩护程序修改过了。用在家庭冲突、团队复盘、法律证词，任何依赖"我记得"的场景。

## 金字塔的第一步看起来无关紧要

{{< sentence >}}
金字塔上的第一步通常很小，小到几乎不会引起注意……但每走一步，下一步就变得更容易了。
{{< /sentence >}}

极化的起点不是极端立场。它是一个无关紧要的小决定。每为上一步辩护一次，下一步就自然了一些。选举里投一张票、会议上附和一句话、谈判中让一步——之后的辩护程序会把这个小决定变成身份的一部分。用来识别自己或团队是否已经在金字塔上了。

## 盲点里最大的一个：以为自己没有盲点

{{< sentence >}}
大脑生来就有盲点，而它最聪明的把戏之一，是让我们以为自己没有盲点。
{{< /sentence >}}

你能看到别人的自我辩护，看不到自己的。偏差盲点让每个人觉得"我看问题比较客观"。在自我评估、绩效反馈、跨部门争执中，先假设自己也有盲点，比假设自己客观更安全。

## 越核心的身份，越不能认错

{{< sentence >}}
当失调击中的是一个人自我认同的核心——我是好医生、我是公正的法官、我是称职的父母——辩护就会最猛烈。
{{< /sentence >}}

高身份绑定的错误是最难认的错。一个医生承认误诊，动摇的不只是这一个判断，还有"我是好医生"这个身份支柱。失调越强，辩护越猛，纠错的可能性越低。用来预判：谁在什么议题上最不可能让步。

## 承认错误是大脑最不想让你做的事

{{< sentence >}}
错误是发现的入口——但前提是我们愿意把它当成错误来看待，而不是当成别人的过错。
{{< /sentence >}}

全书的落点。认错不是道德表演，而是打断辩护程序的唯一出口。问题在于，大脑的全部计算力量都在阻止你走这条路。所以每一次认错都是逆着本能在走。
