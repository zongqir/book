{{< sentence quote="你现在的困境，往往是多年前的一个小决定累积的结果。" >}}
后悔不是对当时决定本身的否定，而是对长期后果的清算
{{< /sentence >}}

{{< sentence quote="一个看起来很优的选择，如果建立在放弃重要的东西上，长期就会变成悔恨。" >}}
最难的选择往往不是比较优劣，而是认清代价
{{< /sentence >}}

{{< sentence quote="你以为的暂时选择，往往会变成人生的定型。" >}}
没有'先这样试试'，每一步都在改变下一步的可能性
{{< /sentence >}}

{{< sentence quote="大多数人的后悔不在于做了什么，而在于没有想清楚就做了。" >}}
后悔的本质是决策质量差，不是结果不好
{{< /sentence >}}

{{< sentence quote="短期回报最大的选择，往往是长期代价最大的。" >}}
这就是为什么很多人年轻时"聪明"，却在中年碰到墙
{{< /sentence >}}

{{< sentence quote="改变人生比坚守错误的选择要简单，但需要更早的觉醒。" >}}
越早意识到错误，改变的代价越小
{{< /sentence >}}

{{< sentence quote="你的选择反映的是你的价值观，而不是你的理性。" >}}
同样的客观条件，不同价值观的人会做完全不同的选择
{{< /sentence >}}

{{< sentence quote="大多数人不是选择能力弱，而是没有想清楚自己真正在乎什么。" >}}
一旦明确了价值观，很多选择其实很容易
{{< /sentence >}}
