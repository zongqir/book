# 看报表前，先把这几句钉在桌上

## 先找证据链断点，再谈便宜不便宜

{{< sentence >}}
先拆资产质量，再信利润表上的漂亮数字。
{{< /sentence >}}

很多问题先写在资产负债表里，别让利润表先把你说服。

{{< sentence >}}
会计上能确认，不等于经济上真发生。
{{< /sentence >}}

适合在看到“确认收入”“公允价值变动”这类表述时拿出来。

{{< sentence >}}
利润要问来源，资产要问变现。
{{< /sentence >}}

这句能把研究从抽象数字拉回到真正能不能兑现。

{{< sentence >}}
低估值不是护城符，报表烂得够深照样会亏。
{{< /sentence >}}

便宜感最强的时候，最该回到这句。

{{< sentence >}}
研究不是找支持，而是先找证据链断点。
{{< /sentence >}}

只要你开始只收集顺耳的材料，就该停下来。

{{< sentence >}}
资产负债表常常比故事更早说真话。
{{< /sentence >}}

适合在市场还沉迷增长叙事时，用来提醒自己先看底盘。

{{< sentence >}}
一次性收益和常态盈利，不能混着估。
{{< /sentence >}}

估值前先回到这一句，能少掉很多看起来很便宜的错觉。

{{< sentence >}}
证据不厚，就别急着下结论。
{{< /sentence >}}

它不是拖延，而是在替你拦住研究还没站住的那一步。

## 下面这些动作前，先把格雷厄姆和多德请回来

- 看到漂亮利润时，你会不会先拆盈利质量
- 研究一家公司时，你还会不会先翻资产负债表
- 遇到低估值时，你会不会先查报表里最脆的地方
- 管理层给出乐观指引时，你会不会先找证据链断点
- 做估值时，你会不会主动换更保守的假设

这页最适合放在研究快要变成立场辩护的时候，用来逼自己回到报表和证据。
