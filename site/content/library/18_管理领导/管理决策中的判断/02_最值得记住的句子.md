{{< sentence >}}
第一个信息会深刻影响你对整个问题的判断。
{{< /sentence >}}

叫做"锚定偏差"。即使后来得到更多信息，第一个信息的影响力还是最大。决策时要主动寻求多个信息源。

{{< sentence >}}
你会倾向于只看支持自己观点的信息，忽视反对信息。
{{< /sentence >}}

叫做"确认偏差"。这是人的天性。补救办法就是强制自己去搜索反对意见。

{{< sentence >}}
高估自己的判断准确率是最常见的错误。
{{< /sentence >}}

叫做"过度自信"。很多人做决策时没有留足风险余地，结果一旦出问题就措手不及。

{{< sentence >}}
群体的判断会压制个人的独立思考。
{{< /sentence >}}

叫做"群体思维"。在团队中，一旦形成了多数意见，少数意见就很难被听到。

{{< sentence >}}
决策的实际结果往往和预期有巨大偏差。
{{< /sentence >}}

不是因为执行不力，而是因为判断本身出了问题。事后评估往往能发现问题在哪。

{{< sentence >}}
最好的决策者不是最聪慧的，而是最有系统的。
{{< /sentence >}}

系统能弥补个人的认知漏洞。一个有清晰决策流程的组织，往往比依靠个人聪慧的组织做得更好。
