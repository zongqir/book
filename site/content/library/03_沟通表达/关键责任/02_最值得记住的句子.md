# 关键责任里最该留下来的，是这些会在关键时候拽住你的提醒句

{{< sentence >}}
难的不是指出问题，是问题重复出现时你还敢不敢再谈。
{{< /sentence >}}

很多关系就在第二次沉默里坏掉。

{{< sentence >}}
先把承诺和结果之间的缺口说清，责任才有位置。
{{< /sentence >}}

没有缺口，后面都只是在发火。

{{< sentence >}}
重复出现的坏结果，别再按“偶发”处理。
{{< /sentence >}}

模式一旦形成，就已经不是一次失误了。

{{< sentence >}}
追责不是发泄，是把结果重新接回具体人和具体动作。
{{< /sentence >}}

不然就只剩“大家以后注意”。

{{< sentence >}}
问题谈到最后，如果没人具体接住下一步，等于没谈。
{{< /sentence >}}

责任对话最后一定要落到谁来改、改什么。

## 这些句子什么时候最该翻出来

- 一开口就想解释很多的时候：先翻这页。
- 你觉得自己明明没说错，结果事情还是做坏的时候：先翻这页。
- 你知道问题在哪，却迟迟不知道从哪一句开始的时候：先翻这页。
- 对方的反应、组织的惯性或关系的压力开始把你带偏的时候：先翻这页。
