{{< sentence >}}
成功不是做很多件事，是把一件事做对。
{{< /sentence >}}

看任何一个领域的顶尖人物——他们成功不是因为什么都做而是因为在一件事上做到了极致。多面手在每个领域都是业余的；专注者在一个领域是专家。

{{< sentence >}}
多任务处理是一个谎言。
{{< /sentence >}}

人脑不能同时处理两个需要注意力的任务——所谓的多任务只是在两个任务之间快速切换。每次切换都有认知成本：平均需要15分钟才能恢复到切换前的专注状态。一天切换十次就浪费了两个半小时。

{{< sentence >}}
纪律不是你需要的——你需要的是习惯。
{{< /sentence >}}

人们以为成功需要超人的自律。凯勒的判断：你只需要足够的纪律来养成一个习惯，然后习惯会替你工作。不要试图在所有事情上保持纪律——选一件事养成习惯就够了。

{{< sentence >}}
说"是"的时候你同时在对其他所有事情说"不"。
{{< /sentence >}}

每一个承诺都在消耗你的时间和注意力。答应做一件新事之前先问：这件事比我现在的"那一件事"更重要吗？如果不是——说不。

{{< sentence >}}
大事不是大的，是做了之后让其他事变小的。
{{< /sentence >}}

最重要的那一件事不一定是最大的任务——而是做了之后有连锁效应的任务。有时候一通关键电话比三天的加班更重要。
