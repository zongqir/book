## 打开出口

{{< sentence >}}
大部分人写不出东西，不是因为缺乏才华，而是因为内在批评者在你写出第一个句子之前就已经开始工作了。
{{< /sentence >}}

审查机制是有用的——在修改阶段。但如果它在产出阶段就介入，你连素材都没有，更别说修改。Brande 的核心发现就是这个错位：大多数写不出东西的人有一个过于活跃的内在审查者。

{{< sentence >}}
早晨醒来后立刻写。不想题目，不管质量，不回头看。写什么都行。
{{< /sentence >}}

这不是一个随意的建议，而是一个精确设计的训练。刚醒的时候，意识防线还没有完全建立，无意识最容易流出来。连续做几周这个练习，你在训练的不是写作技巧，是无意识出口的畅通程度。

{{< sentence >}}
如果你能每天写，不管质量如何，你就已经解决了写作最难的问题。
{{< /sentence >}}

Brande 把"能否坚持每天写"当成作家和非作家的分水岭。不是因为每天写能积累更多文字——而是因为每天写的习惯会逐渐削弱内在审查者的权力。你不再需要"等灵感来了再写"，因为你的无意识已经习惯了在固定时间打开出口。

## 区分两种工作

{{< sentence >}}
写作需要两种截然不同的心理状态，大部分失败来自于在错误的阶段使用了错误的状态。
{{< /sentence >}}

产出阶段需要无意识的自由流动——不加审查，不追求完美，允许混乱和低质量。修改阶段需要意识的精确判断——筛选、重组、精炼。两者的工具箱完全不同。混在一起用就像一边开车一边检修发动机。

{{< sentence >}}
学着像两个人一样工作：先像一个对一切都充满好奇的孩子，然后再像一个冷静的工匠。
{{< /sentence >}}

Brande 的"双重人格"比喻不是文学修辞，是一个操作指令。写初稿时禁止自己修改——哪怕看到明显的错误也继续往前写。修改时禁止自己产出新内容——只处理已有的材料。把两个角色在时间上物理隔离。

## 写作习惯的建立

{{< sentence >}}
写作不是等着灵感降临的事。它是一种可以训练的生理和心理习惯。
{{< /sentence >}}

Brande 反复强调的一个立场：写作和弹钢琴一样，是一种需要规律训练才能掌握的技能。灵感确实存在，但它更容易降临在一个每天都在写的人身上——因为他的无意识出口始终处于打开状态。等灵感再写的人，出口是关着的，灵感来了也流不出来。
