{{< sentence >}}
管理者的时间永远不属于自己。
{{< /sentence >}}

德鲁克发现管理者的最大困境不是能力不足，而是时间不受自己控制。下属找你、老板找你、客户找你、会议占满日程——你以为有八小时，实际可支配的可能只有两小时。承认这个现实是改善的起点。

{{< sentence >}}
有效性不是一种天赋，而是一种习惯。
{{< /sentence >}}

你不需要是天才才能成为有效的管理者。你需要的是练习五个习惯直到它们变成自动反应。这是德鲁克最重要的前提判断——它意味着有效性可以学、可以教、可以改善。

{{< sentence >}}
要事优先——而且一次只做一件要事。
{{< /sentence >}}

大多数管理者同时推进太多事情，每件都做一点但没有一件做到位。德鲁克的建议是极端的聚焦：一次只做一件最重要的事。不是因为其他事不重要——是因为你的有效时间太少，分散了就什么都做不好。

{{< sentence >}}
有效的管理者用人所长，而不是改人所短。
{{< /sentence >}}

你花大量时间试图补一个人的短板，不如把他放到能发挥长处的位置上。一个人的短板在他的长处面前是微不足道的——前提是你把他放对了位置。

{{< sentence >}}
决策不是找到正确答案，而是在正确的层次上界定问题。
{{< /sentence >}}

大部分管理者的决策错误不是选错了方案，而是在错误的层次上理解了问题。一个看起来是"要不要招人"的问题，可能真正需要决策的是"这个岗位是否还应该存在"。
