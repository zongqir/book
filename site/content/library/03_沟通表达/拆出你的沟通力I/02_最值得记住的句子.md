## 沟通的本质

{{< sentence >}}
沟通失败的原因，80% 来自意图不清，而不是技巧不足。
{{< /sentence >}}

当你有五个想法要传达，听众不知道哪个是核心；当你既想被认可又想指出对方错误，信号就自相矛盾；当你自己都没确定要说什么，再好的表达方式也救不了。问题通常不在怎样说，而在说什么。

{{< sentence >}}
你想要对方改变，他听到的是被评判。这不是表达问题，是意图问题。
{{< /sentence >}}

如果你的真实意图是"我想你变得更好"，但你的表达方式听起来像"你现在做错了"，对方就会启动防守而不是听你的建议。不是因为你没有用对词，而是因为你的意图和表达的姿态不一致。

{{< sentence >}}
最难的沟通，往往是你自己有矛盾的时候。
{{< /sentence >}}

既想要靠近对方，又想保持距离；既想让对方同意，又不愿意真的改变自己；既想坦诚相对，又想维护某个形象。这种内部矛盾会自动进入你的语调、措辞和沟通方式，对方会直观感受到这个矛盾。

## 改变沟通的起点

{{< sentence >}}
好的沟通，先是诚实地回答"我真正想要什么"。
{{< /sentence >}}

不是"我应该说什么听起来更聪明"或"我怎样说对方才不会生气"，而是在问自己一个更深的问题：我在这个对话里真正想要的是什么？被理解？改变对方？维护关系？获得信息？只要确认了这一点，你的沟通方向就有了。

{{< sentence >}}
当你停止试图控制对方的反应，沟通反而变得可能。
{{< /sentence >}}

很多人在沟通时的真实状态是在计算——说这句话对方会怎样反应，我要提前堵住他的反驳，我要引导他得出我期望的结论。这种试图控制的心态，会自动进入沟通，让对方感受到被操纵。
