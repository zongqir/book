# 那面墙不是现实砌的，是你自己搬砖搭的

## 关于看不见的墙

{{< sentence >}}
你的心智模式就是你脑子里的一面墙——你看不见它，但它决定了你能看到什么。
{{< /sentence >}}

心智模式不是观点，是过滤器。它不告诉你"这件事是什么"，它直接帮你决定"哪些事存在、哪些事不存在"。当你觉得自己没有选择时，可能不是选项少了，是过滤器太严了。

{{< sentence >}}
很多人不是被现实困住的，而是被自己关于现实的解释困住的。
{{< /sentence >}}

同样的处境，有人看到的是死胡同，有人看到的是岔路口。差异不在处境本身，在你用什么模式去解读它。每次卡住的时候问一句：这是事实，还是我的解释？

## 关于安全感

{{< sentence >}}
安全感不是来自拥有，而是来自确信自己有应对变化的能力。
{{< /sentence >}}

用房子、存款、编制来锚定安全感，等于把安全感绑在外部条件上。外部条件会变，安全感就跟着碎。真正稳定的安全感来自"就算条件变了，我也能处理"这种信心。

{{< sentence >}}
追求安全感的人，往往会选择那个最确定但最不想要的选项。
{{< /sentence >}}

当你做选择时，如果选了之后只是松了一口气但没有任何兴奋感，检查一下：你是在选你想要的，还是在逃避你害怕的。

## 关于借来的信念

{{< sentence >}}
你脑子里的很多信念不是你自己验证过的，是从父母、老师、朋友那里租来的。
{{< /sentence >}}

"35岁之前必须成家""稳定工作才是正经事""我不擅长和人打交道"——问自己：这些判断，哪一条是你亲自试过之后得出的结论？如果是租来的，你至少有权利退租。

{{< sentence >}}
不要用别人的地图找自己的路。
{{< /sentence >}}

别人的经验在别人的条件下成立。照搬别人的路径而不考虑自己的起点和方向，最多到达别人的终点——而那个终点可能不是你想去的地方。

## 关于职业和选择

{{< sentence >}}
兴趣、能力、价值观——三叶草缺了任何一片，职业体验就会出问题。
{{< /sentence >}}

觉得工作没劲但做得好：缺兴趣。觉得有劲但做不好：缺能力。做得好也有劲但总觉得哪里不对：价值观不匹配。诊断不适感先看是哪片叶子的问题。

{{< sentence >}}
成功不是证明你能做到，而是你本来就能做到，只是不敢。
{{< /sentence >}}

大部分人面前的障碍不是能力不够，是不敢尝试。不敢的背后通常是一条未验证的信念——"我做不到""我不配""我太晚了"。

## 什么场景拿出来翻

- 觉得没有选择时：翻心智模式那两句，检查是不是过滤器太严了。
- 做决定只想选最安全的：翻安全感那两句，分辨是选择还是逃避。
- 被"应该怎么做"压住时：翻租来信念那句，问自己有没有亲自验证过。
- 职业不满但说不清为什么：翻三叶草那句，逐片检查缺了什么。
- 什么都想好了就是不动手：翻最后一句，卡住你的通常不是能力，是一条关于自己的错误判断。
