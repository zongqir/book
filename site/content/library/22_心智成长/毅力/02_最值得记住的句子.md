{{< sentence >}}
大部分人把坚持不住归结为意志力不足。但意志力是资源，不是品格。资源会耗尽。
{{< /sentence >}}

用意志力扛一件事，和用存款撑一个月一样——总有花光的时候。兰德尔要说的不是"你要更努力"，而是"你得找到不靠存款就能过日子的办法"。

{{< sentence >}}
目标越模糊，放弃越容易。"变得更好"不是目标，"每天跑二十分钟"才是。
{{< /sentence >}}

模糊的目标没有反馈。你做了一天不知道算不算进步，不做一天也不知道算不算退步。兰德尔的标准是：如果你今天晚上没法判断"今天这个目标做到了没有"，目标本身就需要重写。

{{< sentence >}}
连续冲刺三周然后崩溃，不叫毅力。叫透支。
{{< /sentence >}}

透支和毅力看起来很像——都是在拼。区别在结局：毅力有节奏，能持续；透支没有节奏，以崩盘收场。判断自己是在坚持还是在透支，看一个信号：你有没有给恢复留出空间。

{{< sentence >}}
把跑鞋放在门口，比提醒自己"今天该跑步了"有效十倍。
{{< /sentence >}}

环境设计的价值在于它不依赖当天的状态。不管你今天心情好不好、动力足不足，只要跑鞋在门口，出门的概率就比跑鞋在柜子里高出一截。兰德尔的核心主张之一：降低启动成本，比增强意志力管用。

{{< sentence >}}
反馈是坚持的燃料。没有反馈的行动，再有意义也撑不久。
{{< /sentence >}}

写一本书如果只看"写完了没有"，你会在第三个月彻底失去动力。但如果每天看"今天写了多少字"，每个字都是反馈。反馈不需要是奖励，只需要是信号——告诉你，你在往前走。

{{< sentence >}}
最危险的时刻不是最难的那天，是刚刚取得一点进步之后。
{{< /sentence >}}

跑步计划第二周，体重开始下降。你觉得"差不多了"，开始放松。兰德尔观察到：大部分放弃不发生在最痛苦的阶段，发生在刚刚感觉好一点的阶段——因为痛苦的紧迫感消退了。
