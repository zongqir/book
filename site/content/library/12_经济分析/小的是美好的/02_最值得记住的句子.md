# 六句话让你重新审视"更大更多更快"的默认逻辑

## 规模与代价

{{< sentence >}}
任何组织，如果规模增大到一定程度，就需要投入越来越多的精力来维持自身的运转，而不是完成它原本应该完成的任务。
{{< /sentence >}}

大型组织最隐蔽的成本不在财务报表上。当一个团队从 10 人变成 500 人，沟通成本、协调成本、内部政治成本会吃掉越来越多的资源。下次有人说"做大就能降本"的时候，先问问维持这个"大"本身要花多少。

{{< sentence >}}
人太小了，不应该让大来统治；世界太大了，不应该让小来独占。关键问题不是'大还是小'，而是'什么尺度最合适'。
{{< /sentence >}}

Schumacher 不是简单地反大。他反对的是不问合适不合适就默认"越大越好"。一家餐厅和一家芯片代工厂需要的规模完全不同，硬套同一个逻辑才是问题。

## 效率的重新定义

{{< sentence >}}
一种经济学，如果只计算那些有价格的东西，必然会忽视那些真正重要但无法标价的东西——人的尊严、社区的完整性、自然环境的可持续性。
{{< /sentence >}}

GDP 统计不了一个社区的凝聚力，也统计不了一条河流的生态价值。当你说一个方案"更高效"的时候，想想被排除在计算之外的东西。效率的分母选得不对，结论就会系统性偏离。

{{< sentence >}}
从经济学家的角度看，佛教生活方式的奇妙之处在于：它以令人惊讶的少量消耗，获得了极为满意的结果。
{{< /sentence >}}

Schumacher 用佛教经济学做了一个思想实验：如果经济的目的不是最大化产出，而是用最小消耗达到人的满足，整个效率计算就翻转了。这不是要你去修行，而是提醒你：消费多不等于生活好，产出大不等于结果对。

## 技术与人的关系

{{< sentence >}}
中间技术的核心不是技术水平高低，而是技术是否让使用它的人保持了掌控力。
{{< /sentence >}}

一台你能理解、能维修、能根据本地需要调整的设备，比一台需要总部派专家来维护的精密仪器更有价值——至少对使用者来说如此。技术选择不只是功能比较，还是控制权的分配。

{{< sentence >}}
自然资源不是收入，而是资本。我们不是在花利息，而是在花本金。
{{< /sentence >}}

把石油、矿产、森林当成"收入"来消费，就像把存款本金当工资花。账面上很富裕，直到本金花完。Schumacher 五十年前的这个类比，在碳排放成为具体代价的今天，比他当年说的时候更容易理解。
