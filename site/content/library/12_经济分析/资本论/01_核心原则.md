{{< principle no="1" type="core" statement="核心原则：价值来自劳动，不来自交换" >}}
所有商品背后都有一个共同的度量标准——生产它们所耗费的人类劳动时间。价格在市场上波动，但价格围绕的那个中心，是劳动决定的。

这条原则不是说价格不重要，而是说：讨论"值多少钱"的时候，先问"花了多少劳动来生产"，再问"市场愿意付多少"。两者不总是一致的，差异本身就包含了重要信息。

应用方向：当你看到一种商品的价格远高于生产成本，不要只解释为"品牌溢价"或"需求旺盛"。多问一层——溢价部分到底对应了谁的什么付出？
{{< /principle >}}

{{< principle no="2" type="core" statement="核心原则：利润的来源是未被支付的劳动时间" >}}
工资看起来是劳动的报酬，但它支付的不是劳动创造的全部价值，而只是让工人能继续劳动的再生产成本。工人创造的价值和他拿到的工资之间的差额——剩余价值——才是利润的真正来源。

这条原则意味着：利润不是从市场上凭空赚来的，是从生产过程中抽取的。任何看似"双赢"的雇佣关系里，都存在一个结构性的不对称——工人创造的多于他拿到的。

应用方向：分析一个企业或行业的利润率时，除了看市场竞争力，还要看劳动报酬占产出的比例。如果利润率在上升，但工资占比在下降，剩余价值的抽取在加深。
{{< /principle >}}

{{< principle no="3" type="support" statement="支撑原则：竞争迫使积累，积累制造集中" >}}
资本家不扩大生产就会被淘汰。利润不是终点，而是下一轮投资的起点。这个循环的结果是：大资本吞并小资本，市场份额向少数玩家集中。

这条原则解释了为什么"自由竞争"最终会走向垄断或寡头。竞争不是保持分散的力量，竞争本身就在制造集中。
{{< /principle >}}

{{< principle no="4" type="support" statement="支撑原则：机器替代劳动不是意外，是资本积累的必然方向" >}}
每一轮技术升级都在提高"资本有机构成"——机器设备占总投入的比重上升，劳动力占比下降。短期内，技术创造新岗位可能弥补旧岗位的消失。长期看，劳动在生产中的份量持续缩小。

应用方向：讨论AI替代就业时，不要只看"会不会创造新工种"。马克思提醒的是结构性趋势——资本天然偏好用机器替代人，因为机器不会罢工、不会要求涨薪、可以24小时运转。
{{< /principle >}}

{{< principle no="5" type="support" statement="支撑原则：工人的谈判能力取决于&quot;后备军&quot;的规模" >}}
失业和半就业人口不是市场失灵的副产品。它们是资本积累过程的有机组成部分——一支随时可以替换在岗者的后备劳动力，让任何在岗者都不敢要价太高。

应用方向：判断劳动市场上工人是否有真正的议价权时，不要只看法律法规。看看这个岗位外面有多少人排着队等着进来。外面排的人越多，里面谈的人就越弱。
{{< /principle >}}
