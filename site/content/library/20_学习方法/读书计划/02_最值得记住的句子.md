{{< sentence >}}
在开始读一本书之前，先问问自己：我为什么要读这本书？我想从中获得什么？
{{< /sentence >}}

很多人开始读一本书时，已经忘记了自己为什么要读它。如果你没有清晰的目的，你的阅读就没有方向。同一本书，在不同的目的下，你应该用不同的阅读方式。

{{< sentence >}}
不是所有的书都值得深入阅读，但值得深入阅读的书必须得到深入的阅读。
{{< /sentence >}}

艾德勒强烈反对"应该读很多书"的想法。更重要的是在少数真正值得的书上花足够的时间和精力。一本被彻底理解的经典著作，比十本被快速扫过的新书的价值要大得多。

{{< sentence >}}
在开始读之前，了解一下作者想说什么。这样你在读的时候就知道要关注什么。
{{< /sentence >}}

很多人的阅读方式是被动的——他们跟随作者的思路走，哪里作者想讲，他们就听哪里。更主动的方式是先了解作者的大框架，然后在读的时候有目标地搜索和验证。

{{< sentence >}}
和作者进行对话。不是同意或反对，而是理解他为什么这么说，以及这个论点的问题在哪里。
{{< /sentence >}}

阅读不是单向的信息接收。真正的阅读是一种对话——你在思考"这个论点是否充分"、"是否有反例"、"他遗漏了什么"。这种对话会让你更深入地理解内容，也会刺激你的自己的思考。

{{< sentence >}}
一本书对你的影响，不在于你读了多少内容，而在于你真的理解了多少，以及那个理解是否改变了你。
{{< /sentence >}}

有些人读完一本书后能记住很多细节，但对核心思想没有理解。有些人只记住了几个关键概念，但这些概念改变了他对整个世界的理解。后者的阅读效果更高。

{{< sentence >}}
不同的书需要不同的阅读速度。试图用同样的速度读所有的书，结果是所有的书都没读好。
{{< /sentence >}}

一本小说可能一晚上就读完了。一本哲学著作可能需要几周。一篇有深度的论文可能需要你一字一句地读，甚至需要你查证其中的某些论点。不要被"应该快速阅读"的建议所约束。允许自己用适合每本书的速度来读。
