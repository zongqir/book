# 七条判断准则，把管理者从忙碌拉回有效

{{< principle no="1" type="core" statement="管理者先要面对时间真相，再谈工作安排。" >}}

忙碌不是有效的证据。德鲁克要求管理者先记录时间，先承认自己真实的时间是如何被切碎、被打断、被浪费的。看不见时间现实，后面任何计划都建立在幻觉上。

{{< /principle >}}

{{< principle no="2" type="core" statement="管理工作的起点不是任务清单，而是你要为谁创造什么结果。" >}}

一旦管理者把注意力放回贡献，很多伪忙会自动掉下去。你不是为了把自己塞满，而是为了让组织某个关键结果出现。

{{< /principle >}}

{{< principle no="3" type="support" statement="长处是配置原则，不是表扬语气。" >}}

德鲁克反复强调用长处，不是因为乐观，而是因为组织只有在长处上才能真正产出。围着短板分工，团队很快会陷入平均用力。

{{< /principle >}}

{{< principle no="4" type="support" statement="优先级真正靠放弃，而不是靠多线程硬撑。" >}}

要事优先的重点不在列出重要事项，而在主动放弃那些不重要却不断上门的事情。否则所谓优先级只是口号。

{{< /principle >}}

{{< principle no="5" type="support" statement="决策要把例外和规律分开，否则组织会被一次次临时应对拖散。" >}}

优秀决策不是更快拍板，而是先看眼前这件事到底是偶发状况，还是背后有一条规律值得被提炼成规则。

{{< /principle >}}

{{< principle no="6" type="support" statement="会议和沟通不是天然必要成本，管理者要持续清理它们对有效工作的侵占。" >}}

德鲁克没有把会议单独神圣化。站在他的框架里看，会议只是时间使用的一种形式。凡是占据大量时间、却不直接支撑关键贡献的会议，都该被视为管理失效信号。

{{< /principle >}}

{{< principle no="7" type="support" statement="有效不是一次性状态，而是需要反复训练和校准的职业纪律。" >}}

这条原则把整本书从“管理技巧书”拉回“自我训练书”。你不会因为看懂一次就永久有效。德鲁克真正强调的是持续练习：记录、修正、再记录、再修正。

{{< /principle >}}
