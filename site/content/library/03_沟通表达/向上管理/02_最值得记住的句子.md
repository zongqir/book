# 向上管理里最该留下来的，是这些会在关键时候拽住你的提醒句

{{< sentence >}}
向上沟通不是讨好，是别让关键问题总在最晚的时候才出现。
{{< /sentence >}}

这句能把人从“我是不是太会来事”里拉出来。

{{< sentence >}}
理解上级在怕什么，比一味解释自己做了什么更重要。
{{< /sentence >}}

向上汇报首先是对准对方的判断入口。

{{< sentence >}}
带着选项去汇报，比只带着情绪去汇报更容易得到支持。
{{< /sentence >}}

问题不是不能报，而是不能只报问题。

{{< sentence >}}
预期如果不提前管，最后就只能靠事后解释。
{{< /sentence >}}

很多误会都死在这里。

{{< sentence >}}
越重要的关系，越不能靠默契。
{{< /sentence >}}

向上管理本质上是在反默契。

## 这些句子什么时候最该翻出来

- 一开口就想解释很多的时候：先翻这页。
- 你觉得自己明明没说错，结果事情还是做坏的时候：先翻这页。
- 你知道问题在哪，却迟迟不知道从哪一句开始的时候：先翻这页。
- 对方的反应、组织的惯性或关系的压力开始把你带偏的时候：先翻这页。
