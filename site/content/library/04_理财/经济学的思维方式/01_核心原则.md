你不需要画供需曲线。海恩的经济学从一个日常动作开始：算代价。

每个选择都有代价，代价会传导。传导的终点，是一种没有人设计过的秩序。下面六条准则把这条推理链拆开——两条核心准则搭骨架，四条支撑准则填关节。

{{< principle no="1" type="core" statement="资源稀缺迫使选择，每个选择的真实代价是你放弃的最优替代方案" >}}

经济学的起点不是货币，是稀缺。时间有限、精力有限、预算有限。

稀缺意味着你必须选。选了 A 就放弃了 B。放弃的那个 B，才是 A 的真实代价——海恩把这叫机会成本。

机会成本不是事后回忆"我本来可以怎样"。它是决策前的标准动作：先把最优替代方案摆出来。看不到替代方案，选择就是盲的。

适用范围远超经济行为。选专业、选工作、选要不要加班——只要资源有限、选项互斥，它就生效。

{{< /principle >}}

{{< principle no="2" type="core" statement="稳定的社会秩序可以从分散决策中自发生长，不需要总设计师" >}}

超市货架上永远有你要的牛奶。没有人专门负责确保这件事。奶农、运输商、超市各自追求自己的利益，牛奶就到了你手边。

海恩反复用的一个概念是"自发秩序"。当规则清晰、产权明确时，大量个体各自做决策。汇总结果往往比中央计划更有效率。

这不是说所有自发结果都好。而是说，看到某种秩序时，第一反应不该是"谁设计了它"，而是"它是怎么长出来的"。理解了生长机制，才知道哪里该干预、哪里不该碰。

{{< /principle >}}

## 从个人选择到集体秩序

第一条管个人决策——面对选择，算清代价。第二条管集体结果——无数个人决策汇总后产生什么秩序。

从个人到集体的传导，由下面四条支撑准则负责。

{{< principle no="3" type="support" statement="决策发生在边际——重要的不是总量值不值，而是多一单位值不值" >}}

水比钻石重要，为什么水便宜钻石贵？

因为你不在"所有水"和"所有钻石"之间做选择。你只在"多喝一杯水"和"多买一颗钻石"之间做选择。多一杯水的价值极低，多一颗钻石的价值极高。

边际思维的操作要点：别问"这件事总共值不值"，问"再多做一点值不值"。

该不该加班？别算工作总体值不值。算今晚多干两小时，收益能不能盖过代价。该不该多招一个人？别算团队总产出。算多这一人能多带来多少。

边际分析把模糊的大判断拆成可操作的小判断。

{{< /principle >}}

{{< principle no="4" type="support" statement="人对激励做出反应——改变收益或代价的结构，行为就会跟着变" >}}

酒驾罚款提高，酒驾率下降。但如果同时放松了执法力度，被抓的概率降低，酒驾率可能反弹。

激励不是"奖好罚坏"那么简单。海恩强调的是完整的激励结构：预期收益和预期代价同时决定人们会不会做某件事。只改了其中一半，效果可能跟预期相反。

政策分析中常见的失误是只设计了激励的一侧。补贴某个行业，却没注意补贴改变了相邻行业的成本结构——意外后果就这么来了。

{{< /principle >}}

{{< principle no="5" type="support" statement="价格是传递稀缺信息的协调工具——它让素不相识的人做出彼此兼容的决策" >}}

没有人告诉咖啡农今年该种多少咖啡。价格上涨，多种；价格下跌，转种别的。价格把全球消费者的偏好翻译成农民可以执行的信号。

价格的力量在于压缩信息。你不需要知道咖啡涨价是因为巴西干旱还是韩国流行了咖啡文化。看到价格变了，就知道该调整行为。

扭曲价格——限价、补贴、管制——等于往信号里注入噪音。收到错误信号的人做出错误决策。错误决策累积起来，就是短缺或过剩。

{{< /principle >}}

{{< principle no="6" type="support" statement="比较优势驱动专业化——做自己相对成本最低的事，然后交换，双方都获益" >}}

一个律师打字比秘书快。她应该自己打字吗？

花一小时打字，就少一小时做法律咨询。咨询收入远高于打字节省的成本。请秘书打字，自己去接案子，两个人都过得更好。

这就是比较优势。判断标准只有一个：谁的机会成本更低。

比较优势解释了国家间贸易、公司内分工、家庭成员各司其职。它也解释了为什么"什么都自己做"几乎总是低效的——哪怕你什么都比别人强。

自愿交换能持续发生，根基在这里。只要两方的比较优势不同，交换就能为双方创造价值。

{{< /principle >}}

## 六条准则的推理链条

稀缺产生选择，选择在边际发生，边际决策受激励驱动。激励通过价格传递，价格引导专业化和交换，无数交换汇成自发秩序。

这是一条完整的链。断在任何一环，后面的判断都会出问题。

忽略机会成本，会高估收益。跳过边际分析，会在总量思维里打转。对激励结构视而不见，政策效果的预测就会失准。捂住价格信号，行政命令取代市场协调。放弃比较优势，等于选择什么都自己做。

## 海恩框架的隐含前提

六条准则有一个共同的隐含条件：参与者的成本和收益主要由自己承担。

当成本外溢给第三方时，价格信号就不完整。工厂排污、传染病防控、公共基础设施，都属于这类情况。自发秩序在这些场景下效率会打折。

准则的推理方向依然有效。但需要额外分析外部性的规模和干预的成本。

海恩本人不回避这些场景。他的立场是：即便需要干预，也要用经济学的方式想清楚干预的代价和替代方案。准则不告诉你"该不该干预"，但会逼你把干预的代价算清楚。
