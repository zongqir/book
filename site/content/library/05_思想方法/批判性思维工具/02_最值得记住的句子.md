这些句子的用法只有一个：在你觉得自己想清楚了的时候拿出来，看看是不是真想清楚了。

## 关于思维的默认质量

{{< sentence no="1" quote="每个人都会思考，但大部分思考如果不加干预，天然就是有偏见的、扭曲的、片面的、信息不足的。" source="《批判性思维工具》" >}}

多数人把"我有想法"等同于"我在思考"。保罗和埃尔德的起点是：思考是默认行为，但默认行为的质量很低。就像呼吸是默认行为，但默认呼吸方式不一定是最有效的。

调用场景：当你对自己的判断很有信心，却说不出自己检查过哪些环节。

{{< /sentence >}}

{{< sentence no="2" quote="你必须对你自己的推理过程进行推理。" source="《批判性思维工具》" >}}

思考和"对自己的思考进行思考"是两件不同的事。前者是一阶活动，后者是元认知。保罗和埃尔德的整套工具解决的就是后者——怎么对自己的推理做质检。

调用场景：当你意识到自己只是在推理，而没有检查推理本身。

{{< /sentence >}}

## 关于假设和盲区

{{< sentence no="3" quote="我们对自己最危险的假设往往毫不知情。" source="《批判性思维工具》" >}}

假设是八个思维元素中最隐蔽的一个。你可以把目的说清楚、把信息摆出来、把推理链写下来，但支撑这一切的前提假设可能从未被你审视过。

调用场景：你的论证链条完整，但结论还是让人不踏实——很可能问题在假设层，不在推理层。

{{< /sentence >}}

{{< sentence no="4" quote="判断一个人思维水平的最快方式，是看他能不能准确说出自己的观点所依赖的假设。" source="《批判性思维工具》" >}}

多数争论的分歧不在结论，在假设。两个人用同样的信息推出相反的结论，往往是因为底层假设不同。说清楚自己的假设，比反驳对方的结论更有效。

调用场景：和别人争了半天发现说不到一起去的时候。

{{< /sentence >}}

## 关于标准和质量

{{< sentence no="5" quote="清晰是思维的入口标准。如果一个表述不清晰，我们无法判断它是否准确，也无法判断它是否相关。" source="《批判性思维工具》" >}}

九条智力标准之间有优先级。清晰性排在第一位，不是因为它最重要，而是因为它是其他标准生效的前提。一句话如果连意思都不清楚，谈准确性和深度就是空转。

调用场景：讨论陷入僵局时，先检查双方的表述是否清晰——很多争论的根源是歧义，不是分歧。

{{< /sentence >}}

{{< sentence no="6" quote="仅仅因为一个论证与你的结论一致，并不意味着它是一个好论证。" source="《批判性思维工具》" >}}

自我中心性的核心表现之一：论证支持我的立场，我就觉得它合理。保罗和埃尔德要求的是反过来——用同样严格的标准检验支持你的论证和反对你的论证。

调用场景：当你发现自己只在挑对方论证的毛病，却从不检查支持自己立场的论证。

{{< /sentence >}}

## 关于智力特质

{{< sentence no="7" quote="智力谦逊不是缺乏自信，而是意识到自己的认知边界。" source="《批判性思维工具》" >}}

谦逊经常被误解为"觉得自己不行"。保罗和埃尔德定义的智力谦逊是一种精确的自我边界感——知道自己知道什么，更重要的是知道自己不知道什么。

调用场景：当你准备在不熟悉的领域发表强判断的时候。

{{< /sentence >}}
