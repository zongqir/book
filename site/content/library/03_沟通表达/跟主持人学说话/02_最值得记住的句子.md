# 八句话，把主持人的场面感拆成日常可用的提醒

{{< sentence >}}
别人说完话之后的那三秒，决定了对话会不会继续。
{{< /sentence >}}

接话的窗口期很短。三秒内如果没有人回应，说话的人就会觉得自己说错了什么。主持人从不让这三秒空着。

{{< sentence >}}
接话先接情绪，再接内容。顺序反了，对方会觉得你没在听。
{{< /sentence >}}

"最近累死了。""你试试番茄工作法。"——内容接了，情绪没接。"最近累死了。""一直这么赶？"——情绪接住了，对方自己会继续说下去。

{{< sentence >}}
会说话的人不是自己说得多，是让别人说得出来。
{{< /sentence >}}

递话能力。一个人在聚会里只管自己表达，哪怕说得再精彩，走的时候别人记住的是"他挺能说的"。一个人让每个人都聊了几句，走的时候别人记住的是"跟他在一起很舒服"。

{{< sentence >}}
递话的最高境界是对方觉得话题自然流到了他那里，而不是被你安排发言。
{{< /sentence >}}

"下面请小李分享一下"是安排。"小李上次做的那个项目好像也遇到了类似的情况？"是递话。区别在于对方是被邀请的还是被点名的。

{{< sentence >}}
冷场不是灾难，是你展示场面感的最好机会。
{{< /sentence >}}

所有人都怕冷场。但冷场时第一个开口的人，会被整个场面记住。不需要说什么聪明的话。"刚才那个话题挺有意思的，我还想接着聊"——简单一句，场面就活了。

{{< sentence >}}
控场不是抢话，是让话题有方向感。
{{< /sentence >}}

讨论跑偏时不需要打断别人。用一个问题把方向拉回来："这个点很有意思，不过我们刚才讨论的那个问题好像还没有结论，要不先把它收掉？"

{{< sentence >}}
救场的公式就两步：承认当下发生了什么，然后把方向引走。
{{< /sentence >}}

"这个问题确实不太好回答"——承认尴尬。"不过我们可以换一个角度来看"——引走方向。两步完成，全场呼吸恢复。

{{< sentence >}}
场面感不是演技，是你对在场每个人状态的持续关注。
{{< /sentence >}}

主持人看起来从容，是因为他一直在观察——谁想说话但插不上嘴？谁已经不耐烦了？谁需要被关注一下？表达能力的上限，不取决于你能说什么，取决于你能看到什么。
