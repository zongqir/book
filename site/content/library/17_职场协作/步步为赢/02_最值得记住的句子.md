{{< sentence >}}
能力强的人多的是，但位置好的人很少。
{{< /sentence >}}

职场竞争不缺能力强的人。真正稀缺的是位置好的人——那些坐在信息流中心、和关键决策人有关系、掌握关键资源的人。如果你还在为提升能力而焦虑，而那些位置好的人在轻松地影响决策，你已经输了。

{{< sentence >}}
你离关键决策人有多近，比你的能力有多强更重要。
{{< /sentence >}}

在一个组织里，有些决策是关键的。而关键决策往往由少数几个人做出。你离这些人有多近，决定了你的影响力有多大。能力再强，如果这些人不知道你、不信任你，你就没有机会。

{{< sentence >}}
信息不对称是职场竞争中最大的优势来源。
{{< /sentence >}}

有些人知道别人不知道的事，所以他们的决策更聪明。他们掌握的信息比其他人多，所以他们能做出更有利的选择。这个信息优势往往是决定性的。

{{< sentence >}}
被忽视往往比被反对更致命。
{{< /sentence >}}

你的想法被反对至少说明有人在听你的想法。如果你的想法被忽视，那说明你根本不在重要决策者的考虑范围内。在职场竞争中，被当成"空气"会比被当成"对手"更快地结束你的职业前景。

{{< sentence >}}
掌握资源的人不需要证明自己有多强，只需要证明自己掌握的资源有多值钱。
{{< /sentence >}}

有些人虽然个人能力平常，但因为他掌握预算、人员或关键的信息，他就有很大的权力。这个权力不是他赚来的，而是他掌握的资源给他的。

{{< sentence >}}
最聪明的职场人，不是在一个不对的位置上把能力发挥到极致，而是果断地离开这个位置。
{{< /sentence >}}

很多人在一个位置上深耕很多年，能力提升了，但位置没有改善。结果就是一个能力强但位置差的人，被一个能力平常但位置好的人打败了。真正聪明的职场人会及时识别"这个位置没有前景"，然后果断地跳到一个位置好的地方。

{{< sentence >}}
职场的阶跃式成长往往来自位置的改变，而不是能力的提升。
{{< /sentence >}}

有些人的职业生涯是线性的：每年的能力提升百分之十。但也有些人的职业生涯是阶跃式的：突然跳到一个新的位置，能力要求一样，但因为位置好，他的成就一下子增加了十倍。这个差异往往不是因为这个人比别人聪明，而是因为他懂得什么时候该改变位置。
