{{< sentence >}}
GDP 增长很快，但增长的果实没有流向普通人。
{{< /sentence >}}

不是经济没有增长，而是增长结构问题。地产商赚钱了，政府税收增加了，但工资增长跟不上。

{{< sentence >}}
房地产不是简单的消费，而是一个经济寻租的工具。
{{< /sentence >}}

土地升值、炒房差价、开发商利润——这些财富从哪来？从购房者的钱包来。

{{< sentence >}}
医疗和教育本来是公共服务，产业化以后变成了个人高消费。
{{< /sentence >}}

医疗成本和教育成本的暴涨，是政策导向的结果，不是"个人承受能力不足"。

{{< sentence >}}
中产的困境来自于支出端被全方位挤压。
{{< /sentence >}}

房贷、教育、医疗——一个都不能少。支出结构没有选择的空间。

{{< sentence >}}
宏观繁荣和微观困顿同时存在，说明增长模式有根本问题。
{{< /sentence >}}

这不是周期问题，是结构问题。短期刺激解决不了。

{{< sentence >}}
要改变生活压力，靠个人努力是不够的，还要看体制的改变。
{{< /sentence >}}

这不是鸡汤，而是郎咸平的现实评价。有些问题超出个人能力范围。
