{{< principle no="1" type="core" statement="群体有自己的价值观，它由群体的领导者和结构决定" >}}
一个群体的价值观不是民主选举出来的。而是由那个群体掌权的人定义的。

然后群体中的人会逐步内化这些价值观，最后以为那是"常识"或"道德"。
{{< /principle >}}

{{< principle no="2" type="core" statement="个人在群体中会失去自己的判断，取而代之是跟风" >}}
这不是说人变坏了。而是说，在群体中，人的大脑会进入一种不同的运作模式。

比理性和个人判断，群体认可和安全感对人的影响更大。
{{< /principle >}}

{{< principle no="3" type="support" statement="群体会排斥那些威胁到群体规则的人" >}}
如果你的言行威胁到群体的既得利益或群体认同，群体会联合起来排斥你。

这不需要显式的决定，而是自动发生的。
{{< /principle >}}

{{< principle no="4" type="support" statement="群体内部有等级，权力往往集中在顶端" >}}
宣称"人人平等"的群体，也往往有隐形的等级结构。

明白自己在群体中的位置，很重要。
{{< /principle >}}

{{< principle no="5" type="support" statement="群体会不断地进行&quot;我们和他们&quot;的分类" >}}
群体的凝聚力往往来自对"敌人"或"外人"的共同认识。

没有清晰的"他们"，群体就容易分裂。
{{< /principle >}}
