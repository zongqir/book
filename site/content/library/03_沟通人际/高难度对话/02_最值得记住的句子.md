## 关于对方、关于你、关于这场对话

{{< sentence no="1" quote="你没办法控制别人怎么对你，但你能选择自己怎么回应。" >}}
全书的地基。一旦你把注意力从"他怎么能这样"移到"我接下来做什么"，对话的走向就开始改变。不是忍让，是把主动权从对方手里拿回来。
{{< /sentence >}}

{{< sentence no="2" quote="别人的行为是他们的选择，你的反应是你的选择。" >}}
上一句的操作版。挑剔的人扔过来一句刺，你不是必须接住——你可以让那句话落在地上。
{{< /sentence >}}

{{< sentence no="3" quote="你不需要赢得每一场对话，你需要的是不输掉这段关系。" >}}
最容易判断错的地方。和难搞的人说话时特别容易滑进"谁对谁错"的轨道。一旦进了这条轨道，你赢了论点也输了合作空间。
{{< /sentence >}}

## 进入对话前的校准

{{< sentence no="4" quote="如果你期待对方会改变，你就把结果交给了你控制不了的变量。" >}}
对话前最该调整的是预期。带着"这次他一定会理解"的期待进去，出来时一定失望。带着"我把该说的说清楚"的预期进去，不管对方什么反应你都能收场。
{{< /sentence >}}

{{< sentence no="5" quote="情绪油箱快见底时别硬聊。" >}}
能量管理是贝克特尔最实际的建议之一。你在疲惫、焦虑或刚被别的事气到的时候，处理冲突的能力断崖式下降。这时候开口不是勇敢，是给自己挖坑。
{{< /sentence >}}

## 对话进行中的操作锚点

{{< sentence no="6" quote="安全感消失的瞬间，对方听到的就不再是你说的内容了。" >}}
你说的道理再正确，对方一旦觉得不安全——被攻击、被否定、被审判——他的耳朵就关上了。剩下的对话全是防御反应。先把安全感建回来，然后再说事。
{{< /sentence >}}

{{< sentence no="7" quote="改变不了难搞的人，但可以改变和他对话的条件。" >}}
换时间、换地点、换话题入口、换一个人在场。条件变了，同一个难搞的人的反应可能完全不同。比磨话术有效率得多。
{{< /sentence >}}

{{< sentence no="8" quote="设定边界不是攻击对方，是保护对话能继续下去。" >}}
很多人不敢设边界，怕显得不近人情。但不设边界的结果是对话越来越偏、情绪越来越大，最后不了了之。提前说清"我们今天聊什么、不聊什么"，反而是在帮双方。
{{< /sentence >}}

## 什么时候拿出来用

对话前——检查自己的情绪和预期。油箱不够就推迟；预期太高就降级。

对话中——感觉自己要发火或者想赢的时候，停一秒，问自己："我是在争对错，还是在解决问题？"

对话后——复盘时不问"他为什么又这样"，问"下次我可以换什么条件"。

和长期打交道的难搞同事、家人、客户之间的持续沟通，反复会用到这组提醒。不是用一次就管用，是每次用都帮你偏少一点。
