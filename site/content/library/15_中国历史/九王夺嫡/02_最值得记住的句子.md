{{< sentence >}}
太子被废不是因为太子无能，而是因为康熙活得太长——一个准备继位三十年的人，和一个当了三十年皇帝不愿放手的人，注定会发生冲突。
{{< /sentence >}}

权力交接的时间窗口如果被无限拉长，继承者的焦虑和在位者的猜忌会同步升级，最终互相摧毁。

{{< sentence >}}
在没有明确规则的竞争中，最先暴露野心的人最先被淘汰。
{{< /sentence >}}

大阿哥、八阿哥的失败都有一个共同特征：过早让康熙看到了他们的夺位意图。在不确定谁是裁判的游戏里，低调是一种生存策略。

{{< sentence >}}
联盟的价值取决于退出成本。能随时散伙的联盟，关键时刻一定会散伙。
{{< /sentence >}}

八爷党看起来势力最大，但成员之间的利益绑定不够深。当康熙施压时，联盟迅速瓦解。

{{< sentence >}}
胤禛赢在两件事：让康熙放心和在关键时刻执行力最强。
{{< /sentence >}}

不是最有才华、不是最有人脉、不是最受欢迎——而是让权力持有者觉得"交给他最安全"。

{{< sentence >}}
康熙最后的选择不是"谁最好"而是"谁最不危险"。一个六十多岁的老皇帝，优先考虑的是稳定而不是理想。
{{< /sentence >}}

权力交接中的决策标准往往不是"最优"而是"最稳"——尤其当决策者已经老了、累了、受够了不确定性。
