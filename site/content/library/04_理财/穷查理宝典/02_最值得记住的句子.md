# 决策一复杂，就把这几句翻出来

## 多元模型和反蠢原则，该怎么接上手

{{< sentence no="1" quote="先别做蠢事，很多投资收益就是这么留下来的。" >}}
适合在你觉得必须立刻做点什么的时候拿出来。
{{< /sentence >}}

{{< sentence no="2" quote="激励怎么设计，结果往往就怎么歪。" >}}
这句能把研究从表层数字拉回到人为什么会那样行动。
{{< /sentence >}}

{{< sentence no="3" quote="只用一种模型看世界，迟早会把自己骗舒服。" >}}
观点越顺耳，越要换一个模型再看一遍。
{{< /sentence >}}

{{< sentence no="4" quote="反过来想，常常比往前冲更值钱。" >}}
当你只剩买入理由时，这句能替你补回另一半。
{{< /sentence >}}

{{< sentence no="5" quote="证据不够厚时，聪明的动作通常是少动。" >}}
它拦住的不是机会，而是自信过头。
{{< /sentence >}}

{{< sentence no="6" quote="复杂不等于高级，简单也不等于浅。" >}}
别拿一堆概念堆出判断的幻觉。
{{< /sentence >}}

{{< sentence no="7" quote="少数高把握机会值得重手，其余时候先别忙。" >}}
真正的集中，不是到处找重仓理由。
{{< /sentence >}}

{{< sentence no="8" quote="知道偏差名称不算本事，能在自己身上认出来才算。" >}}
复盘错误时最该把这句放在最前面。
{{< /sentence >}}

## 这些时刻最需要查理把你往后拽一步

- 看到一个很诱人的观点时，你会不会先问自己最可能在哪犯蠢
- 研究公司时，你还会不会先查激励怎么设
- 听到一个漂亮解释时，你会不会主动换一套模型再看一遍
- 想下重注时，你会不会先确认自己是不是只是在模仿自信
- 复盘错误时，你会不会先从自己身上的偏差找原因

真正的调用方式，是在你最想证明自己聪明的时候，先把错误和偏差翻出来。
