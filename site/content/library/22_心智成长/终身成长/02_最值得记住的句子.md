# 关于失败、努力和天赋——德韦克说得最准的几句话

{{< sentence >}}在固定型思维模式中，失败从一个行为（我失败了）转变成了一种身份（我是一个失败者）。{{< /sentence >}}

这句话拆穿了固定型思维最核心的陷阱。一次失败如果被解读成身份判决，你的整个应对系统就会从"解决问题"切换到"保护自尊"。而保护自尊的方式通常是回避——不再尝试你可能失败的事。

{{< sentence >}}成长型思维模式并不意味着所有事都能改变。有些人的天赋确实不同。成长型思维关注的是：无论起点在哪里，你都可以通过努力和策略有所发展。{{< /sentence >}}

纠正一种常见误读——成长型思维不是"人人都能成为天才"的鸡汤。它承认起点不同，但拒绝用起点来定义天花板。"你有没有天赋"和"你能不能变得更好"是两个独立的问题。

{{< sentence >}}当你感觉到"聪明"的标签受到威胁时，你已经在固定型思维模式里了。{{< /sentence >}}

识别信号。任何时候你在某件事上犹豫，而犹豫的原因不是"这件事是否值得做"，而是"如果做不好会不会显得很蠢"——固定型模式正在替你做决定。

{{< sentence >}}表扬孩子的努力和策略，而不是他们的天赋和智力。{{< /sentence >}}

德韦克最著名的实验结论。适用范围远超育儿：你怎么评价自己的成功，决定了你怎么应对下一次失败。如果你的自我肯定总是"我就是擅长这个"，你在给自己设陷阱。

{{< sentence >}}在固定型思维模式中，努力是一个危险信号——它意味着你不够聪明。{{< /sentence >}}

解释了为什么有些人宁可不努力也不愿承认自己认真了还是不够好。"我没认真"比"我认真了还是失败了"安全得多——前者保住了"其实我行"的可能性。

{{< sentence >}}每一次你面对挑战时选择退缩，你的固定型思维模式就赢了一次，而你的成长就少了一次。{{< /sentence >}}

两种模式是累积效应。不是某一次选择决定了你的人生轨迹，而是大量的微选择——挑不挑战、问不问问题、试不试新方法——堆积起来形成了完全不同的能力曲线。

{{< sentence >}}真正的自信不是相信自己永远不会失败，而是相信自己有能力从失败中学习。{{< /sentence >}}

重新定义自信。固定型的自信建立在"我不会失败"上，所以特别脆弱。成长型的自信建立在"失败了我也能处理"上，能承受的冲击量级完全不同。
