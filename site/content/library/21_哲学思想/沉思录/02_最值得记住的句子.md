# 给自己的紧急提醒，不是给别人的格言

有些话不是格言，是深夜给自己的紧急提醒。《沉思录》里的句子大多重复出现，措辞不同，指向同一件事。下面十句按日常调用频率排列。每句后面写的是：它纠正什么误判，在什么场景下拿出来用。

以下为基于公共解释资源的价值转述，非逐字原文。

{{< sentence >}}
"你有权力控制你的思想——而非外在事件。认识到这一点，你就能找到力量。"
{{< /sentence >}}

控制二分法的最简版本。调用场景：当你发现自己在反复纠结一件你根本改变不了的事情时——别人的态度、已经发生的结果、无法预测的未来。这句话提醒你把注意力撤回到你能管的地方：你此刻的判断。

{{< sentence >}}
"晨起时对自己说：今天我将遇到好管闲事的人、忘恩负义的人、傲慢的人、欺诈的人、嫉妒的人。"
{{< /sentence >}}

这句话的力量在"提前说"。大多数情绪崩溃来自意外感。如果早上已经告诉自己会遇到麻烦，下午真遇到时反应就不同。调用场景：每天早上出门或开始工作前的那几秒钟。

{{< sentence >}}
"障碍本身就是道路。"
{{< /sentence >}}

遇到阻碍时的默认反应通常是抱怨和回避。这句话提供了第三个选项：障碍本身就是你练习判断力和韧性的材料。调用场景：项目出了意外、计划被打断、遇到了预期之外的困难。

{{< sentence >}}
"死亡并不可怕。可怕的是关于死亡的判断。"
{{< /sentence >}}

这句话的适用范围远超死亡本身。把"死亡"替换成任何让你恐惧的事——失败、被拒绝、失去工作——逻辑依然成立。恐惧的来源通常是你对事件的想象和判断，不是事件本身。调用场景：焦虑升起时，先检查"我到底在怕什么"。

{{< sentence >}}
"你很快就会忘记一切；一切也会很快忘记你。"
{{< /sentence >}}

宇宙视角的最苦涩版本。调用场景：当你被名声、评价、面子困住时。你在意的那些人的看法，几年后谁还记得？这句话帮你在虚荣和真正重要的事之间做选择。用起来有些冷，但确实管用。

{{< sentence >}}
"忿怒带来的伤害，远大于引发忿怒的事件本身。"
{{< /sentence >}}

愤怒的真实代价：它消耗你的时间、精力和判断力，通常远超触发它的那件小事。调用场景：快要发火的时候，问自己一个问题——这件事值得我拿接下来两小时的心情去交换吗？

{{< sentence >}}
"不要像被迫一样去做公共事务。也不要像自怜者一样。"
{{< /sentence >}}

奥勒留写给自己的工作态度提醒。做事时有两种常见的心态陷阱：觉得是被逼的，或者觉得自己太辛苦。两种都会侵蚀行动质量。调用场景：当你发现自己在一边工作一边抱怨"为什么是我"的时候。

{{< sentence >}}
"你所拥有的不过是当下这一刻。"
{{< /sentence >}}

对抗两种判断失真的工具：对过去的懊悔和对未来的焦虑。两者都在消耗你此刻的判断力。调用场景：当你发现自己在反刍昨天的错误，或者预演明天的灾难时，把注意力拉回手上正在做的事。

{{< sentence >}}
"对宇宙有益的一切，对你也有益。"
{{< /sentence >}}

斯多葛式的接受逻辑。发生在你身上的事是宇宙整体运转的一部分。你可以不喜欢它，但抗拒它不会改变它，只会增加你的内耗。调用场景：面对无法改变的结果——裁员、生病、天气、他人的选择——时，放下"为什么是我"的追问。

{{< sentence >}}
"砍掉你的枝干、剥去你的皮，如果你有智慧，这些都不能阻止你继续生长。"
{{< /sentence >}}

韧性的最小定义：外部条件被剥夺之后，你是否还能继续做该做的事。调用场景：遭遇重大损失或挫折之后，用来检验自己还剩什么是不可剥夺的——你的判断、你的意图、你的选择。
