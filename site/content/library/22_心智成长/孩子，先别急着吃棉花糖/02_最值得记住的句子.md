## 关于选择的本质

{{< sentence >}}
每一次拒绝眼前的诱惑，你都在投票选择你想成为的人。
{{< /sentence >}}

不是在拒绝快乐，是在选择你想要的人生版本。

{{< sentence >}}
后悔的不会是"我没有及时满足自己"，而是"我浪费了能建立更好人生的时间"。
{{< /sentence >}}

## 关于长期视角

{{< sentence >}}
现在的克制和未来的收获有很直接的因果关系。很多人拖延的原因，就是这个因果关系不够直观。
{{< /sentence >}}

看不清因果，就坚持不了。

{{< sentence >}}
如果你不能忍受现在的小不舒服，就得忍受未来的大失败。没有第三种选择。
{{< /sentence >}}

## 关于诱惑和环境

{{< sentence >}}
战胜诱惑最好的方法，不是增强意志力，而是让自己远离诱惑。
{{< /sentence >}}

聪明人改变环境，而不是和环境较劲。

{{< sentence >}}
你的自律程度，往往取决于你的朋友圈。和什么人在一起，你就会变成什么样的人。
{{< /sentence >}}

## 关于习惯的力量

{{< sentence >}}
一个在小事上能延迟满足的人，面对大事时的选择空间会大得多。
{{< /sentence >}}

小的延迟满足是在锻炼这个肌肉。

{{< sentence >}}
你越早开始培养延迟满足能力，这个能力就能在你身上发挥更久、更大的作用。
{{< /sentence >}}

## 关于人生的杠杆点

{{< sentence >}}
能和眼前的诱惑说不的人，就掌握了改变自己人生轨迹的最大杠杆。
{{< /sentence >}}

这比任何外部机遇都更有决定性。
