{{< sentence >}}
纳粹不需要大多数人支持他们做的每一件事。他们只需要大多数人不反对就够了。
{{< /sentence >}}

沉默的多数不是中立的——他们是极权体制的必要基础设施。没有反对就是事实上的许可。

{{< sentence >}}
希特勒的每一步权力扩张都有法律依据——国会纵火法令、授权法案、总统与总理合并。合法不等于正当。
{{< /sentence >}}

用合法程序做不正当的事情，这比直接违法更危险——因为它让人们失去了"这是违法的"这个最简单的抵抗理由。

{{< sentence >}}
经济复苏是纳粹获得广泛支持的最强武器。失业率下降让大多数德国人愿意忽略人权侵犯。
{{< /sentence >}}

物质改善可以换取对政治自由的放弃。当人们的生活在变好时，"不要谈政治"变成了一种受欢迎的态度。

{{< sentence >}}
纳粹对犹太人的迫害是渐进的。1933年的抵制犹太商店和1938年的水晶之夜之间隔了五年——每一年都比前一年多一点点。
{{< /sentence >}}

渐进式升级让每个步骤看起来都只是"上一步的自然延伸"。回头看是清晰的阶梯，身处其中时每一步都"没那么严重"。

{{< sentence >}}
最容易被纳粹改造的不是底层而是精英——法官、教授、企业家、军官。他们有最多的理由配合体制：社会地位、经济利益、职业前途。
{{< /sentence >}}

精英阶层的配合比底层的狂热更重要。底层的暴力可以被管控，精英的服从让整个体制运转起来。
