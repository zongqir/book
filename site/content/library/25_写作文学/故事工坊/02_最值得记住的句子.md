## 关于动手与拖延的分水岭

{{< sentence >}}
你还没准备好的感觉不会消失——它只会在你开始写的那一刻变成别的东西。
{{< /sentence >}}

拿来治"再准备一下再开始"这个循环。准备感不会自然到达一个阈值然后启动写作；动手本身才是让准备感转化为行动力的开关。

{{< sentence >}}
初稿不是作品的低配版，是作品的原材料。
{{< /sentence >}}

修改"初稿应该尽量写好"这个预期。初稿的功能是提供可加工的素材，不是直接呈现最终质量。对初稿要求太高，结果是什么都产不出来。

## 关于练习暴露问题的机制

{{< sentence >}}
你以为自己懂了三幕结构，直到你试着用它写一个故事。
{{< /sentence >}}

任何写作概念在被用来构建一个具体作品之前，理解都是虚的。练习不是验证你会了，是暴露你还差什么。

{{< sentence >}}
写不出对话的人通常不是缺技巧，是不知道这个角色到底想要什么。
{{< /sentence >}}

对话卡壳时先别查对话技巧——回去确认角色的核心欲望。角色的目的清晰了，对话自然有方向。

## 关于完成的真实含义

{{< sentence >}}
场景是砖，故事是房子。会砌砖不等于会盖房子。
{{< /sentence >}}

单场景写作和完整故事之间的跨越不是量的增加，是能力类型的切换。从局部到整体的编织需要单独练习。

{{< sentence >}}
最有效的练习是那种让你在中途想放弃的练习。
{{< /sentence >}}

做起来舒服的练习只是在重复已有能力。让你卡住的才是正在拓展你边界的。
