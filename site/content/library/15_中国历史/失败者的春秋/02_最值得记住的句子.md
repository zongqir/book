{{< sentence >}}
在春秋时代，道德是政治的包装纸，撕掉包装纸，里面都是利益交换。
{{< /sentence >}}

道德话语在春秋争霸中从未缺席，但它的作用不是约束行为，而是为行为辩护。当晋国要灭掉某个小国时，总能找到"该国不义"的理由。

{{< sentence >}}
小国的悲剧不在于弱小，而在于总觉得自己可以在大国博弈中保持中立。
{{< /sentence >}}

郑国、卫国这些小国经常试图左右逢源，在晋楚争霸中两不得罪。结果往往是两边都不信任，成为第一个被牺牲的对象。中立是奢侈品，弱者承担不起。

{{< sentence >}}
失败的政治家和成功的政治家，差别往往不在能力，而在运气和站队时机。
{{< /sentence >}}

春秋政坛上，能力出众却站错队的大有人在。政治成败的决定因素，有时就是在关键时刻选对了靠山。个人英雄主义在权力游戏中作用有限。

{{< sentence >}}
当一个制度开始崩坏时，最先受伤的总是那些最相信这个制度的人。
{{< /sentence >}}

春秋晚期，礼制逐步瓦解，但总有一些贵族还在坚持旧规矩。他们的下场往往最惨，因为对手已经不按规则出牌，而他们还在自我约束。

{{< sentence >}}
权力斗争中，今天的盟友就是明天的敌人，今天的敌人也可能是明天的盟友。
{{< /sentence >}}

春秋时期的政治联盟变化极快，利益重组导致关系重组。固守某种联盟关系的国家和个人，往往会被新的政治组合抛弃。

{{< sentence >}}
历史记住的是成功者的名字，但成功者的手段往往来自失败者的经验教训。
{{< /sentence >}}

每一个春秋霸主的成功策略，背后都有无数失败者的试错成本。失败者的价值不在于他们取得了什么，而在于他们证明了什么路走不通。

{{< sentence >}}
在政治博弈中，最危险的不是强敌，而是队友的背叛。
{{< /sentence >}}

春秋历史上最惨烈的失败，往往来自内部倒戈。外敌再强，至少方向明确；队友翻脸，防不胜防，杀伤力更大。
