# 关系要先看位置，再看热度；先看非冗余，再看亲密度

《结构洞》最有穿透力的地方，在于它把人从“关系很多的人”重新定义成“站在某个位置上的人”。

一旦视角变了，很多关于人脉的老习惯都会失效。下面五条原则，正是这本书最值得留下来的方向感。

{{< principle no="1" type="core" statement="关系的价值先看非冗余，再看数量和热度" >}}
同样二十个联系人，彼此越重复，网络价值越低。

你和联系人共享的圈子、行业、信息源越相似，这段关系越容易只带来重复内容。它当然仍然可以提供信任和陪伴，但对机会、信息和新判断的增量会变小。

所以关系不是先看多不多，而是先看它有没有把你带到一个原本接触不到的地方。
{{< /principle >}}

{{< principle no="2" type="core" statement="站在两个互不连通群体之间，优势来自信息差，也来自协调权" >}}
结构洞真正值钱的，不只是你知道得更早。

当两边彼此不熟、缺少直接通路时，站在中间的人还会多出解释、筛选、排序和撮合的空间。别人不只依赖你传递信息，也依赖你告诉他们这条信息该怎么理解。

这就是为什么同样拥有信息，桥梁位置的人往往能获得更高回报。不是因为他更热闹，而是因为位置给了他更多主动权。
{{< /principle >}}

{{< principle no="3" type="support" statement="桥梁位不是靠加更多人得到，而是靠稳定出入两边且都说得上话得到" >}}
很多人误以为多加几个异行业联系人，就算开始占结构洞。

不是。真正的桥梁位要求你在两边都能持续出入。你知道两边各自的语言、节奏和关切，也能让两边都愿意把你当作可信的通路。

没有这层可进可出的能力，联系人再多，也只是分散的名片，不是位置。
{{< /principle >}}

{{< principle no="4" type="support" statement="闭合网络保信任，结构洞保机会，别拿一种机制替另一种" >}}
伯特最容易被误读成“只要桥梁，不要团体”。

其实不是。紧密闭合的关系网能提供信任、执行默契和稳定支持。结构洞则更擅长带来新信息、新连接和竞争优势。两种机制解决的是不同问题。

只追结构洞，你会缺少落地和托底。只守闭合网络，你会越来越难看到新机会。
{{< /principle >}}

{{< principle no="5" type="support" statement="位置优势会被填平，必须持续移动到新的断点上" >}}
结构洞不是永久资产。

一旦两边自己接上了，你原先的中间位置价值就会下降。原来只有你知道的路径，后来别人也知道；原来必须通过你协调的事，后来双方能直接对话。

所以真正要守的不是某一个固定洞，而是持续发现和占住新断点的能力。位置优势本质上是动态的。
{{< /principle >}}

五条放在一起看，方向已经很清楚：别再只算你认识多少人，要先看这些连接有没有重复；别只图热闹，要看你是不是占住了别人还没接上的那道缝。
