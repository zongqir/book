{{< sentence >}}
一个经理人的产出，等于他所管辖和影响的组织的产出。
{{< /sentence >}}

不是你自己干了多少活，而是你的团队产出了多少成果。经理人的价值不在个人贡献，在于通过管理动作放大团队整体产出。

{{< sentence >}}
管理杠杆率：同样一小时的管理活动，对产出的影响可以相差十倍。
{{< /sentence >}}

花一小时做绩效评估可能影响一个人未来半年的表现。花一小时回复不重要的邮件对产出几乎为零。经理人应该把时间集中在高杠杆活动上。

{{< sentence >}}
一对一面谈是经理人最重要的管理工具，频率由下属决定，内容由下属主导。
{{< /sentence >}}

一对一不是汇报会。主角是下属，不是你。你的角色是提问和倾听，不是布置任务。频率取决于下属的成熟度——新人每周一次，老手每两周一次。

{{< sentence >}}
开会不是浪费时间，开没有目的的会才是浪费时间。
{{< /sentence >}}

格鲁夫把会议分成两种：过程导向会议（定期、结构固定）和任务导向会议（临时、围绕具体决策）。分清类型，会议效率就上来了。

{{< sentence >}}
绩效评估的目的不是打分，而是改善下属的未来表现。
{{< /sentence >}}

如果你写完绩效评估对方没有改变任何行为，这次评估就白做了。评估的每一句话都应该指向一个具体的行为改变。

{{< sentence >}}
培训是经理人能做的最高杠杆率活动之一。
{{< /sentence >}}

花四个小时准备一次培训，面对十个人讲两小时。如果每人的工作效率只提升1%，这四小时的投入回报率超过任何单一管理动作。
