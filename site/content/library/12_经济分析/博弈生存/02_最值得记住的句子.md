## 识别局面

{{< sentence >}}
你以为在做选择，其实在被局面选择。
{{< /sentence >}}

当一个人反复纠结"我该不该答应"，往往说明局面结构已经限定了他的选项。先看清局，再看选项。

{{< sentence >}}
一次性博弈里，背叛是理性的；重复博弈里，信任才是理性的。
{{< /sentence >}}

判断要不要守信，先判断你和对方还会不会再见面。这不是道德问题，是博弈次数决定的策略问题。

## 动作选择

{{< sentence >}}
威胁只在可信时才有效，承诺只在可验证时才成立。
{{< /sentence >}}

空头威胁和空头承诺在博弈论里是等价的——都不会改变对方行为。关键不是你说了什么，而是对方是否相信你会执行。

{{< sentence >}}
在信息不对称的局里，先出手的人暴露信息，后出手的人利用信息。
{{< /sentence >}}

先手未必有利。谁先亮牌，谁就给对方送了筹码。在不确定对方底牌时，等待本身是一种有价值的策略。

{{< sentence >}}
合作不是因为善良，而是因为未来的影子足够长。
{{< /sentence >}}

重复博弈中合作的前提是双方都预期这段关系会持续。一旦有一方觉得"反正以后不会再打交道"，合作立刻瓦解。

## 退出与切换

{{< sentence >}}
当你发现自己在一个零和局里，最好的策略可能是换一个局。
{{< /sentence >}}

零和博弈的特征是总收益固定。在这种结构里，你多得的每一分都是对方少的。如果局面不可能变成正和，退出去找另一个局，往往比在里面死磕更划算。

{{< sentence >}}
最危险的博弈不是对手强，而是你不知道自己在什么局里。
{{< /sentence >}}

信息劣势比力量劣势更致命。一个清楚局面结构的弱者，可能比一个稀里糊涂的强者活得更久。
