## 感知与情绪

{{< sentence >}}
你对一件事的感知，不等于那件事本身。
{{< /sentence >}}

你看见同事开会时皱眉，判断"他对我的方案不满意"。但皱眉可能是胃不舒服。你的感知跳过了所有其他解释，直接落在最符合你担心的那一种。

改变的判断：从"我看到的就是事实"到"我看到的是我的解读版本"。

用得上的场景：每次你对别人的表情、语调或沉默产生确定判断时，先暂停，列出至少两种其他可能性。
{{< sentence >}}
在刺激和反应之间，有一个你可以选择的空间。
{{< /sentence >}}

伴侣说了一句让你不舒服的话。你的身体已经准备好反击或冷战。但在冲动和行动之间，存在一小段间隔——如果你能意识到它。

改变的判断：从"我控制不住反应"到"反应不是自动的，只是太快了"。

用得上的场景：生气、委屈、被冒犯时。练习方式是物理性的：深呼吸一次，或者把要说的话先在脑子里过一遍。两秒钟就够。
## 倾听与语言

{{< sentence >}}
倾听不是等对方说完，是确认你理解的跟对方想说的是同一件事。
{{< /sentence >}}

你以为你听懂了；对方以为你没听懂。这中间差的不是耳朵，是一步确认动作。"你的意思是不是……"看起来简单，但大部分人在对话中从来不说。

改变的判断：从"我认真听了就算听了"到"对方觉得被听见才算听了"。

用得上的场景：对方倾诉、抱怨或解释一件你本能觉得"没什么大不了"的事情时。
{{< sentence >}}
&quot;你总是这样&quot;和&quot;你昨天做了这件事&quot;——对方听到的是两句完全不同的话。
{{< /sentence >}}

"你总是"是定性，把一个行为变成对人的整体判断。对方接收到的不是反馈，是攻击。换成描述具体行为——时间、地点、动作——对方才有可能回应，而不是防御。

改变的判断：从"我说的是事实"到"我说的方式决定了对方能不能接住"。

用得上的场景：每次想用"总是""从来不""每次都"开头时，把它换成上一次的具体事实。
## 冲突与关系

{{< sentence >}}
冲突不是关系的敌人；回避冲突才是。
{{< /sentence >}}

多数人害怕冲突，把冲突等同于伤害。但回避冲突积累的不满，比冲突本身更能腐蚀关系。损害关系的不是"吵了一架"，是"有话不说，慢慢不想说了"。

改变的判断：从"吵架说明关系出了问题"到"不敢吵才说明关系出了问题"。

用得上的场景：你发现自己在一段关系里开始回避某些话题时。那不是成熟，是撤退信号。
{{< sentence >}}
每次不回应，都是一次微小的否定。
{{< /sentence >}}

对方发了条消息，你看了没回。对方说了件开心的事，你"嗯"了一声继续看手机。单独一次无所谓；积累到一定数量，对方就开始不说了。

改变的判断：从"没回应不代表不在乎"到"在不在乎不由你定义，由对方的感受定义"。

用得上的场景：亲密关系和长期友谊中。关系变淡很少因为大事件，多数因为日常回应的消失。
