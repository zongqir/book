{{< sentence >}}
团队的第一次会议决定了团队文化的基调。
{{< /sentence >}}

新团队的第一次会议不只是认识一下——它设定了沟通方式、决策风格和期望标准。如果第一次会议就允许迟到、跑题和没有结论，这些行为会变成默认规范。

{{< sentence >}}
一对一不是汇报会，是为下属服务的时间。
{{< /sentence >}}

很多管理者把一对一变成了进度检查——"你做了什么？遇到什么问题？"更好的一对一是让下属主导议程：他想讨论什么？他需要什么支持？他的职业发展需要什么？

{{< sentence >}}
团队冲突不是问题。回避冲突才是。
{{< /sentence >}}

健康的团队有大量的争论——关于方向、优先级、解决方案。不健康的团队要么没有争论（表面和谐），要么争论变成了人身攻击（恶性冲突）。管理者的工作是让争论聚焦在事而不是人上。

{{< sentence >}}
清晰的角色定义减少80%的团队摩擦。
{{< /sentence >}}

大多数团队冲突的根源不是性格不合——是角色不清。谁负责决策？谁负责执行？谁需要被咨询？谁需要被通知？把这些明确了，大多数"人际问题"就消失了。

{{< sentence >}}
反馈要具体、及时、聚焦行为而非人格。
{{< /sentence >}}

"你做得不够好"不是反馈——是评价。反馈是："上周的客户演示中你跳过了技术方案部分，导致客户没有建立信心——下次建议花五分钟走一遍技术方案。"具体到行为、影响和建议。
