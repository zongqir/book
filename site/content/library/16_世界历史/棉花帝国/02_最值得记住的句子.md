{{< sentence >}}资本主义最初并非建立在自由劳动、自由土地和自由市场的基础上，而是建立在奴役、掠夺土地和强制市场开放的基础上。{{< /sentence >}}

这句话打破了关于资本主义起源的主流神话。贝克特指出，我们今天所理解的"自由市场经济"实际是暴力扩张的结果，而非其起点。

{{< sentence >}}棉花连接了奴隶与工厂工人，连接了种植园主与工业资本家，连接了美洲与欧洲、非洲与亚洲。{{< /sentence >}}

单一商品如何成为全球化的载体。贝克特用棉花展示了早期全球化的真实形态——不是平等的贸易关系，而是基于权力不对等的资源配置。

{{< sentence >}}战争资本主义通过占有他人的土地和劳动来积累财富，工业资本主义则通过提高生产力来积累财富。{{< /sentence >}}

两种资本主义形态的核心区别。前者依靠直接暴力，后者依靠市场机制和技术进步。但贝克特强调，后者的出现以前者的积累为前提。

{{< sentence >}}当美国南部的奴隶制被废除后，资本家们转向了印度、埃及和其他殖民地，在那里建立新的强制劳动体系。{{< /sentence >}}

暴力并未消失，而是转移阵地。这句话揭示了资本主义全球扩张的连续性——当一个地区的强制劳动体系瓦解，资本就会在其他地区重建类似体系。

{{< sentence >}}棉花工业的兴起不仅改变了经济结构，更重塑了国家权力、社会关系和全球政治格局。{{< /sentence >}}

经济变迁的政治后果。贝克特不把经济史当作纯粹的经济问题，而是展示了经济力量如何重塑政治制度和社会结构。

{{< sentence >}}工业革命的成功建立在全球范围内的不平等交换之上——原料生产地区的贫困是制造业地区繁荣的条件。{{< /sentence >}}

发展与不发展的辩证关系。这句话解释了为什么工业化没有在全球均匀展开，而是造成了持续至今的中心-边缘结构。

{{< sentence >}}到19世纪末，棉花帝国已经创造了一个新的世界——一个由资本、技术和国家权力主导的世界。{{< /sentence >}}

历史转折的总结性判断。贝克特用"棉花帝国"这个概念概括了现代世界体系的形成过程——从分散的地方经济转向统一的全球市场。
