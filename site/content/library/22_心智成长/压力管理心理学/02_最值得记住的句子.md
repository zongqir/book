## 压力不在事件上，在评估上

{{< sentence >}}
压力不是事件的属性，也不是人的属性，而是特定环境和特定个体相遇时产生的。
{{< /sentence >}}

把"压力大"归因为性格或环境，都只看到一半。同样的 deadline，有人兴奋有人崩溃。差别在两者相遇时的评估结果。

{{< sentence >}}
你感受到的压力强度，取决于感知到的要求和感知到的资源之间的比值。
{{< /sentence >}}

要求不变，资源感知翻倍，压力就减半。管理压力的两个抓手——降低感知要求，或者提升感知资源——从这个比值推出来。

{{< sentence >}}
初级评估决定你是否启动压力反应，次级评估决定你怎么反应。两步都可以出错。
{{< /sentence >}}

威胁评估一旦启动，身体就进入战斗或逃跑模式。但启动条件可能是错的——你把一封普通邮件读成了最后通牒。在次级评估上低估自己的能力，也同样常见。

## 策略选对了方向才有效

{{< sentence >}}
问题聚焦的应对改变处境，情绪聚焦的应对调节感受。用错方向，越努力越疲惫。
{{< /sentence >}}

能改的事情拿来"接受"，该接受的事情拿来"解决"。减压失败最常见的错配就在这里。

{{< sentence >}}
应对灵活性比任何单一应对策略都重要。
{{< /sentence >}}

没有万能策略。能在不同情境里切换策略的人，长期压力水平显著更低。固守一种方式，再有效也会在某些情境里失灵。

## 你以为有的支持，未必在关键时刻可用

{{< sentence >}}
社会支持只在你感知它可用时才起缓冲作用。
{{< /sentence >}}

身边有人不等于有支持。如果你判断"求助说明我无能"，再多支持网络也不会被激活。

{{< sentence >}}
慢性压力不是一次大事件造成的，而是无数次微小评估的累积。
{{< /sentence >}}

每天几十次"我能应付吗"的微判断，叠加成持续紧绷。管理慢性压力，要管评估频率，不只是单次强度。
