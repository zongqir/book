## 情绪压过纪律的瞬间

{{< sentence >}}
在期货市场，让你赔钱的不是行情，是你自己。
{{< /sentence >}}

每次亏损复盘到最后，真正的原因几乎都不是判断错误，而是在判断正确时没有执行，或者在判断错误时拒绝承认。这句话的价值不在正确，在于它需要在每次开仓前被重新确认。

{{< sentence >}}
交易中最难的事情不是预测市场，而是控制自己。
{{< /sentence >}}

预测能力和执行能力是两件事。很多交易者花了90%的时间研究行情方向，剩下10%才想起自己有没有纪律去执行。青泽十年的教训浓缩成一个发现：方向看对了照样亏钱，因为手跟不上脑子。

## 仓位失控的代价

{{< sentence >}}
在期货交易中，重仓是万恶之源。
{{< /sentence >}}

重仓不一定立刻亏钱，但它把容错空间压到了零。一个正常波动就能触发止损，一个隔夜跳空就能清空账户。青泽自己最惨的几次，都发生在仓位加到"再扛一下"的时候。

{{< sentence >}}
赚了钱之后，最危险的事情不是亏回去，而是开始相信自己无所不能。
{{< /sentence >}}

连续盈利带来的不是能力提升，而是风险感知下降。加仓更随意，止损更犹豫，品种选择更大胆——这个链条青泽走了不止一次，每次结尾都是同一个字：崩。

## 止损是生存的底线

{{< sentence >}}
止损是交易者唯一的生命线。
{{< /sentence >}}

不止损的交易者不是勇敢，是还没被市场教育够。青泽见过太多人扛单扛到爆仓，也包括他自己。止损执行得快不快，直接决定了你能在市场里待多久。

## 对抗人性的训练

{{< sentence >}}
市场永远是对的，你的意见不重要。
{{< /sentence >}}

调用场景非常具体：当你持有一个亏损头寸，心里在想"市场错了，迟早会回来"的时候——这句话就该出现。市场不需要你同意，它只需要你的钱。

{{< sentence >}}
投机像山岳一样古老，华尔街没有新鲜事。
{{< /sentence >}}

青泽引用利弗莫尔的话来提醒自己：所有你正在经历的冲动、恐惧和贪婪，一百年前有人已经经历过了。这句话不是哲学感慨，是在你觉得"这次不一样"的时候最好的冷水。

## 场景速查

- **刚赚了一笔想加码**：重读"赚了钱之后最危险的事情"那条
- **扛着亏损不肯走**：重读止损生命线和"市场永远是对的"
- **觉得自己看懂了行情**：重读"控制自己比预测市场更难"
- **换了新品种想试手气**：重读"投机像山岳一样古老"
