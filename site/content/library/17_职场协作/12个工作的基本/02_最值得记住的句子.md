{{< sentence >}}
与其寻找"适合自己的工作"，不如先把眼前的工作做到能被信任。
{{< /sentence >}}

大部分人在职业早期花太多时间纠结方向，太少时间打磨手头的事。信任不是你选了对的岗位就会自动获得的——信任来自交付质量，和你在哪个岗位无关。

{{< sentence >}}
工作中的乐趣不是入职前就能预判的，是做到一定程度之后才会出现的。
{{< /sentence >}}

"我对这份工作没有热情"通常不是判断，是借口。你还没做到让自己有成就感的程度，当然感受不到乐趣。乐趣是能力的副产品，不是选择的奖励。

{{< sentence >}}
习惯决定了你的下限，技能只决定你的上限。
{{< /sentence >}}

精力全压在学新技能上，却忽略守时、回复、跟进这些基本习惯——技能可以学，习惯一旦养坏要改的成本高得多。

{{< sentence >}}
人脉不是认识多少人，是有多少人愿意在关键时刻帮你说一句话。
{{< /sentence >}}

加了微信不叫人脉，合作过一次、对方下次还愿意合作才叫。人脉的本质是重复博弈中的信用积累。

{{< sentence >}}
沟通能力不是"会说话"，是能让对方准确理解你想表达的内容。
{{< /sentence >}}

表达流利和沟通有效是两回事。说了一堆话对方还是不知道你要什么——问题不在口才，在于你自己没有想清楚。

{{< sentence >}}
能力的积累有阶段性，跳过前一个阶段练后面的能力，看起来走了捷径，实际上把坑留在了后面。
{{< /sentence >}}

入职三年就急着拓人脉的人，往往在专业能力上留了短板。别人愿意跟你交换资源，前提是你有资源可以交换。

{{< sentence >}}
选择能力不是在关键时刻突然拥有的，是在之前无数个小决定中练出来的。
{{< /sentence >}}

跳槽、转行这种大选择之所以难，不是因为选项太多，是因为你在日常工作中没有练过"评估选项、承担后果"这套动作。

{{< sentence >}}
持续学习不是不断报课，是能在工作中识别出自己缺什么、然后有目标地补。
{{< /sentence >}}

到处上课但工作没有明显改善的人，问题不在学习量，在于学习没有对准缺口。先诊断再治疗，这个顺序不能反。

{{< sentence >}}
信任一旦失去，恢复的成本是建立时的数倍。
{{< /sentence >}}

一次失约、一次推责、一次隐瞒，就能抹掉几个月积累的信用。守住信任比拓展信任重要。

{{< sentence >}}
健康和时间管理不是等到出问题才需要关注的"高级话题"，只是被大多数人排在了最后。
{{< /sentence >}}

大久保幸夫把健康和时间管理放在靠后的阶段，不是说它们不重要——是说在职业早期，你不太可能认真对待它们。等到身体出警报或者时间完全失控时，这两项才从"知道该做"变成"必须做"。
