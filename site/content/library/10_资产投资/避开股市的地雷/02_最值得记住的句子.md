## 关于止损和决策纪律

{{< sentence >}}
你觉得股票还会反弹，所以你不卖。这就是为什么你会亏50%。
{{< /sentence >}}

止损最难的不是在第一次下跌时执行，而是在你有理由相信"还会涨回来"的时刻执行。但如果基本面确实变了，"相信反弹"就是在用希望替代分析。

{{< sentence >}}
下跌本身不是风险。被迫下跌才是。
{{< /sentence >}}

股票下跌分两种：一种是市场波动，你选择等；一种是基本面破裂，你必须走。区分这两种的唯一办法是事前就定好规则，而不是事后再看着K线图判断。

## 关于听消息和赌博心理

{{< sentence >}}
如果你投资的原因是"某某人说这只股票会涨"，那你不是在投资，你在赌。
{{< /sentence >}}

消息、推荐、内幕——这些东西的价值在股市里被严重高估。而且你能听到的"内幕"，机构早就知道了。

{{< sentence >}}
在你理解一只股票之前，无论它的消息多好，你都不该买。
{{< /sentence >}}

理解一只股票意味着你能说出三个理由解释为什么现在的价格相对其基本面是合理的。如果你只能说"人气好"或"涨幅大"，那就不是理解。

## 关于账户心理和止盈

{{< sentence >}}
浮盈时你觉得自己很聪明，这正是你最容易犯错的时刻。
{{< /sentence >}}

高光时刻往往是过度自信的开始。赚到的钱比亏的钱更容易让人冒险——因为花的是"不是本金"的钱。

{{< sentence >}}
不知道什么时候卖，比不知道什么时候买更致命。
{{< /sentence >}}

很多人有止损意识，但几乎没人有止盈计划。结果是：小赚变大亏。在买入时就定好目标价格，达到就走，这比看起来简单得多。
