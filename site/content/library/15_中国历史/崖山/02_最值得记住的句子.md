{{< sentence >}}
崖山之前的每一步撤退都在缩小选择空间。到最后一步时，已经没有"选择"可言——只剩下"战"和"死"两个选项。
{{< /sentence >}}

不可逆的连续退让最终会把你推到一个没有退路的角落。每一步退让看起来是"保存实力"，但累积起来是"丧失选项"。

{{< sentence >}}
忠诚在平时是一种美德，在末路是一种惯性。分辨它到底是信念还是惯性，需要看这个人在有选择的时候怎么做。
{{< /sentence >}}

有退路时还选择留下的人是真忠诚。没有退路时的"忠诚"可能只是无处可去。

{{< sentence >}}
十万人殉国不是十万个独立决策——集体行动一旦启动，个人判断会被群体压力覆盖。
{{< /sentence >}}

在极端情境下，个人判断让位于群体动力学。不要把集体行为简单理解为每个个体意志的汇总。

{{< sentence >}}
南宋的灭亡不是军事失败那么简单——财政崩溃在军事崩溃之前已经发生。没有钱的军队，打不了仗也撤不了退。
{{< /sentence >}}

军事失败往往是财政失败的延时表现。当你看到军事崩溃时，财务危机已经发生了很久。

{{< sentence >}}
投降者不一定是懦夫，殉国者不一定是英雄。在末路上做出不同选择的人，面对的约束条件往往不同。
{{< /sentence >}}

用单一的道德标签去评判处于极端困境中的人，不是理解而是偷懒。
