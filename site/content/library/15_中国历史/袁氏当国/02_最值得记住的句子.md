{{< sentence >}}
袁世凯是一个绝顶聪明的人，但他的聪明用在了错误的方向上。
{{< /sentence >}}

唐德刚对袁的评价始终保持这种双面结构——不否认能力，但指出方向错误。这比简单地骂他"窃国大盗"要有穿透力得多。

{{< sentence >}}
中国的帝制不是被推翻的，是自己倒塌的。
{{< /sentence >}}

辛亥革命的实际过程远比后来的叙事更混乱。清帝退位更像是一场各方交易的结果，而不是革命力量碾压的结果。

{{< sentence >}}
共和政体在中国的第一次实验，被几个完全不懂共和为何物的人操控着。
{{< /sentence >}}

袁世凯不懂共和，但宋教仁、孙中山也未必完全理解议会政治需要什么样的社会基础。这句话的锋利之处在于它指向所有人，而不只是袁。

{{< sentence >}}
称帝这件事，不是袁世凯一个人疯了，而是一群人一起疯了。
{{< /sentence >}}

洪宪帝制不是独裁者的独角戏。杨度、严复这样的知识分子也在劝进名单上。唐德刚的观察是：环境可以让一群聪明人同时丧失判断力。

{{< sentence >}}
袁世凯一辈子最大的本事就是看人下菜碟，但称帝这件事没有人能给他下菜碟。
{{< /sentence >}}

袁的核心技能是读懂对手的意图并做出反应。但称帝是一个需要自己判断方向的决策——他的技能包里没有这一项。
