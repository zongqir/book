{{< sentence >}}
管理解决的首先不是效率问题，而是取舍问题。
{{< /sentence >}}

资源永远不够，注意力永远有限。管理者最值钱的地方，不是多做一点，而是决定先做什么、不做什么。

{{< sentence >}}
一个团队跑偏，先看目标是不是模糊，再看执行够不够用力。
{{< /sentence >}}

目标含糊时，努力只会放大混乱。很多执行问题，源头其实是上游判断没讲清。

{{< sentence >}}
坏激励会把好人也推向坏结果。
{{< /sentence >}}

别急着把问题归到态度。先看这套考核、分工和反馈，是不是在奖励错误行为。

{{< sentence >}}
报表是管理视角，不是业务现实本身。
{{< /sentence >}}

数字会让人产生掌控感，但数字只呈现被定义过的部分。关键场景还是要回到一线去看。

{{< sentence >}}
管理者最稀缺的资源不是时间，是清醒判断。
{{< /sentence >}}

把一天填满不难，难的是把有限精力留给真正值得亲自判断的事。很多琐事不是要你做，是要你别被它们拖走。

{{< sentence >}}
没有被持续反馈修正的管理动作，最后都会变成惯性。
{{< /sentence >}}

一开始有效的方法，放到新的团队和阶段未必还有效。管理不是背答案，是不断校正。
