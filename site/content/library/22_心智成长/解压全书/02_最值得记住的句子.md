{{< sentence quote="压力本身不是问题，问题在于慢性压力下你的身体无法恢复。" >}}
短期压力能让你警惕，长期压力就变成了伤害
{{< /sentence >}}

{{< sentence quote="你对事件的解读方式比事件本身更决定了你的压力水平。" >}}
同样的事件，不同的认知会产生完全不同的生理反应
{{< /sentence >}}

{{< sentence quote="预期和不确定性是压力的两大根源，比事件本身的严重性更重要。" >}}
明确的坏消息往往比不确定的好消息压力更小
{{< /sentence >}}

{{< sentence quote="失控感是压力反应的关键触发器。" >}}
同样的压力，感觉自己能控制的人压力水平明显更低
{{< /sentence >}}

{{< sentence quote="社会支持能直接改变压力对身体的伤害程度。" >}}
有人倾听和陪伴，压力激素的影响就会被明显削弱
{{< /sentence >}}

{{< sentence quote="长期压力会改变大脑的结构，特别是影响学习和记忆的区域。" >}}
这不只是心理问题，而是真实的神经生物学改变
{{< /sentence >}}

{{< sentence quote="压力管理最有效的方法往往不是放松，而是改变对压力的认知。" >}}
同样的冥想，相信它有效和不相信的人效果差异巨大
{{< /sentence >}}

{{< sentence quote="体育活动可能是对抗压力最有效的生物学手段。" >}}
比冥想、呼吸练习等心理手段的科学证据都要强
{{< /sentence >}}
