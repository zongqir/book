## 关于广告的基础认识

{{< sentence >}}
广告不能创造需求，只能激发已存在的需求。
{{< /sentence >}}

如果消费者根本不在乎你这个品类，你的广告有多创意都白搭。广告的前提是：这个人心里其实想要或需要这样的东西。

{{< sentence >}}
消费者不记得广告，只记得它给他的感觉和品牌关联。
{{< /sentence >}}

你花500万做的宏大campaign，消费者可能几天就忘了。但那个反复出现的logo、那句反复听到的slogan、那个建立在他脑子里的品牌气质，会留下来。

## 关于信息和认知

{{< sentence >}}
你认为重要的卖点，往往不是消费者关心的。
{{< /sentence >}}

企业喜欢强调产品参数、工艺、成分。但消费者的脑子里装的是"用了这个我会变成什么样"，而不是"这个产品怎么做的"。

{{< sentence >}}
在消费者心里建立一个清晰的位置，比在市场上做得最大更有价值。
{{< /sentence >}}

你可以不是最大的品牌，但你要在某个特定的消费者心智中占据一个特定的位置。"最健康的""最便宜的""最有品味的"，任何一个都比"什么都很不错"更容易被记住。

## 关于促销的真相

{{< sentence >}}
如果没有人因为你的促销而试过你，那这个促销的目的就没有实现。
{{< /sentence >}}

促销不能只是"折扣"。好的促销设计能消除消费者的尝试障碍，让他们愿意主动去试。

{{< sentence >}}
促销的最终目的是改变购买习惯，而不是在这一次多卖一点。
{{< /sentence >}}

如果一个促销活动过了就回到原来的销售量，那说明这个促销没有建立任何长期的购买习惯改变。

## 关于广告效果评估

{{< sentence >}}
曝光量和销售量无关，转化率才是。
{{< /sentence >}}

你的广告被1000万人看到，但转化率只有0.01%，不如被10万个精准消费者看到、转化率2%来得有效。

{{< sentence >}}
最难衡量的效果，往往是最有价值的效果。
{{< /sentence >}}

品牌认知度提升、消费者信任度改变，这些虚的东西反而比直接销售额更能长期决定品牌前景。但它们最难测量，所以经常被忽视。
