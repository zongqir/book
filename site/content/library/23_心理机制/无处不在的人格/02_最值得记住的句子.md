## 关于人格的根源

{{< sentence >}}
没有哪个人是天生这样的，都是被生活逼成这样的。
{{< /sentence >}}

性格被视为天生的，但实际上是一个人在面对生活威胁时的逐步自我改造。一个"固执"的人往往是因为早期有过被强制改变的经历，所以学会了坚守。

{{< sentence >}}
人格是一个防御系统，防的是那些曾经伤害过他的东西。
{{< /sentence >}}

一个人为什么总是很警惕？因为他学到了，放松警惕会有代价。一个人为什么总是想控制一切？因为失控曾经让他很恐怖。这不是"人格缺陷"，是一个人的生存智慧。

## 关于人格和行为的关系

{{< sentence >}}
你看到的人格特征，往往是一个人对某个特定环境的适应。换个环境，他可能会完全不同。
{{< /sentence >}}

同一个人，在权威面前可能很顺从，在他在乎的人面前可能很控制。这不是虚伪，而是一个人在用不同的防御机制应对不同的威胁。

{{< sentence >}}
人格不是一个人怎么行动，而是一个人怎么理解威胁。
{{< /sentence >}}

两个人面对同样的情况，一个会看到"机会"，一个会看到"风险"。这个看法的不同，决定了他们之后的行为。而这个看法，往往来自早期的经历。

## 关于改变人格

{{< sentence >}}
人格改变不是靠意志力，而是靠环境的改变和经验的累积。
{{< /sentence >}}

一个总是被否定的人，不能通过"我要自信"而变得自信。他需要经历足够多的被接纳，慢慢地，他对自己的理解才会改变。

{{< sentence >}}
理解你的人格从哪里来，是改变它的前提。
{{< /sentence >}}

如果你只是恨自己"太敏感""太退缩""太爱控制"，那你永远改变不了。只有当你理解了这些特征对你曾经的意义，你才有可能去重新选择。

## 关于人格和他人

{{< sentence >}}
最难接纳的人，往往是用你无法理解的防御机制来应对这个世界的人。
{{< /sentence >}}

你不能理解为什么他那么粘人、那么爱哭、那么爱发脾气，因为你的防御机制不是这样的。但对他来说，这是他唯一知道的活着的方式。

{{< sentence >}}
改变一个人不能靠指责他的人格，只能靠让他感到足够安全。
{{< /sentence >}}

当一个人感到真的被接纳时，他的防御才会慢慢放下。在那之前，所有的指责和劝说都只会强化他的防御。
