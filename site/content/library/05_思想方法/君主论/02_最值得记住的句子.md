让人不舒服的判断往往最有用。下面七句从《君主论》中筛出，每一句都在纠正某个人们舍不得放下的错觉。按纠偏力度排序。

## 关于善良的代价

{{< sentence >}}
"一个人如果在一切事情上都想发善心，那么在许多不善良的人当中他势必毁灭。"
{{< /sentence >}}

不是劝你做坏人。提醒的是：如果环境里有不守规则的参与者，你的善良就是一个系统漏洞。策略必须考虑对手的实际行为，而不是你期望的行为。

{{< sentence >}}
"人们冒犯一个自己爱戴的人，比冒犯一个自己畏惧的人更少顾忌。"
{{< /sentence >}}

爱戴建立在对方的心情上，畏惧建立在对后果的计算上。前者随时可以翻脸，后者有成本约束。管理中经常有人困惑"我对他们那么好，为什么不领情"——答案就在这里。

## 关于判断的起点

{{< sentence >}}
"所有武装的先知都胜利了，没有武装的先知都毁灭了。"
{{< /sentence >}}

萨沃纳罗拉在佛罗伦萨靠布道获得巨大影响力。民众热情一退潮，他没有强制手段维持秩序，迅速被推翻并处死。摩西有刀剑做后盾，律法才站得住。好想法如果没有执行力保护，只是别人的素材。

{{< sentence >}}
"谁促使他人强大，谁就自取灭亡。"
{{< /sentence >}}

路易十二扶持教皇亚历山大六世扩大势力，帮助威尼斯扩张领土。这两个被他养大的力量转头成了他在意大利最大的障碍。在商业合作和组织联盟中，帮对方变强之前先问一句：强大之后，他还需要你吗？

## 关于时机

{{< sentence >}}
"伤害人的事应当一下子干完，恩惠应该一点一点地施与。"
{{< /sentence >}}

组织变革中最常犯的错误是持续小规模调整。每次一点不舒服，持续几个月，所有人都在焦虑中等待下一刀。一次到位反而让人能开始适应。好消息慢慢放，每次给一点期待。

{{< sentence >}}
"疾病开始时容易治疗，难于识别；等它变得容易识别时，已经难于治疗了。"
{{< /sentence >}}

所有延迟处理问题的理由——"再看看""还不确定""也许会自己好"——都在帮助问题长大。等问题大到所有人都看得见，处理成本已经翻了几倍。早期行动看起来小题大做，实际成本最低。

## 关于准备

{{< sentence >}}
"命运犹如暴涨的河流，人在平时不修堤坝，洪水来了就只有溃败。"
{{< /sentence >}}

这句话约束一种常见的侥幸心理：现在一切正常，不用做准备。准备只能在不需要的时候完成。等需要的时候再动手，时间窗口已经关了。
