# 先把人从“眼前强弱”和“共同敌人”里拽出来

{{< sentence >}}
眼前谁强，不等于最后谁赢。
{{< /sentence >}}

第二卷最先要你改掉的，就是把此刻对比直接当终局的冲动。

{{< sentence >}}
真正要看的不是力量大小，而是力量怎么变。
{{< /sentence >}}

这句比上一句更硬。它逼你把时间带进分析，而不是停在横截面里。

{{< sentence >}}
阶段判断一错，后面的打法越完整，错得越大。
{{< /sentence >}}

第二卷真正重的地方，不只是结论，而是阶段。什么时候该守、该熬、该反攻，不能混。

{{< sentence >}}
统一战线要做，但不能把自己做没了。
{{< /sentence >}}

这句专门纠正“为了合作，什么都能让”的想法。第二卷很清楚，合作不是自我取消。

{{< sentence >}}
交出判断权换来的，不是团结，是慢性消失。
{{< /sentence >}}

这句最适合在合作越来越顺、自己却越来越不像自己的时候翻出来。

{{< sentence >}}
只会说反对什么，撑不起一场长期合作。
{{< /sentence >}}

第二卷为什么一定要讲新民主主义，这句就够解释。没有“建成什么”，联盟迟早只剩临时同仇敌忾。

{{< sentence >}}
民族战争如果只按军事来想，一半已经看丢了。
{{< /sentence >}}

这句专门提醒你：第二卷一直在把战争、政治和国家重新绑在一起。

{{< sentence >}}
表面和平不等于真实让位，台面上的话要和台下的力量一起看。
{{< /sentence >}}

这句其实已经把第四卷提前预示出来了。第二卷后段开始，单看说法已经不够。

## 这些句子什么时候最该翻出来

- 一看到强弱悬殊就想下结论时：先看第 1、2 句。
- 情绪高涨，想提早决战时：先看第 3 句。
- 合作关系越来越顺，自己却越来越弱时：先看第 4、5 句。
- 大家只会讲共同敌人，没人回答以后怎么办时：先看第 6、7 句。
- 只盯表面和谈、表面姿态时：先看第 8 句。

第二卷真正配留下来的，不是鼓气的话，而是这些能逼你把时间、联盟和国家一起看清的话。
