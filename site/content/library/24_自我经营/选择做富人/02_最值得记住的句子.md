{{< sentence >}}
财富差异的起点不是收入，是支出控制。
{{< /sentence >}}

同等收入的人，为什么有人会富有？因为他们的支出增长速度比收入慢。可支配资本持续存在，复利就有了基础。

{{< sentence >}}
先减少支出，再期待增加收入，顺序反过来会一直失败。
{{< /sentence >}}

工资涨了，生活水平也涨了。这样永远没有存款。正确顺序是：先建立"储蓄优先"的支出结构，然后每一次加薪都推动储蓄增长，而不是消费增长。

{{< sentence >}}
富人的奢侈消费来自资本积累之后，不是之前。
{{< /sentence >}}

在财富积累阶段，富人往往比你想象的更节制。名牌和奢侈品来的很晚，而不是来自"借贷透支"。

{{< sentence >}}
小额频繁消费比大额消费对财富破坏更大。
{{< /sentence >}}

看不见的支出漏洞在哪里？反复购买小商品。每次五块十块，看起来微不足道，一年累计会是一笔巨款。而且这类消费完全没有长期规划，最浪费。

{{< sentence >}}
投资的真谛是时间复利，不是一次翻倍。
{{< /sentence >}}

富人投资的目标通常不是"赚快钱"，而是"保证稳定增长"。这看起来保守，但三十年复利的力量会打败任何一次赌博。

{{< sentence >}}
快速致富的欲望会摧毁大部分小额积蓄。
{{< /sentence >}}

一旦有了一点钱，立刻被"股票翻倍""加密致富"的故事吸引。结果往往是本金消失。这个陷阱最容易掉进去，也最容易摧毁长期计划。

{{< sentence >}}
财富不是命运，而是选择的累积结果。
{{< /sentence >}}

你现在的贫富状态是过去选择的结果。未来的状态取决于从今天开始的选择。没有不可改变的贫富。
