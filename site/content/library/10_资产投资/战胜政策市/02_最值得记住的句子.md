## 政策转向的信号

{{< sentence >}}
政策底往往早于市场底，等你确认市场见底时，股价已经涨了一大段。
{{< /sentence >}}

政策放松的第一个信号出现时，市场通常还在跌。大多数人的反应是"等稳一稳再说"。但历史数据显示，从政策底到市场底的时间差通常是1到3个月。等到市场真正企稳，最便宜的筹码已经被抢走了。

{{< sentence >}}
利好出尽是利空，利空出尽是利好。
{{< /sentence >}}

当市场对好消息不再反应时，说明好消息已经被价格充分消化了，接下来稍有风吹草动就是下跌。反过来，当市场对坏消息麻木时，最坏的阶段可能已经过去了。

## 政策周期的节奏感

{{< sentence >}}
在中国股市，每一轮大行情的起落都和政策周期高度吻合。
{{< /sentence >}}

不是偶然的巧合，而是制度安排的结果。A股市场的参与者结构、发行制度、资金来源都受到政策的直接影响。把政策周期从投资分析里去掉，就像预报天气时去掉风向——剩下的信息不够做判断。

{{< sentence >}}
政策放松初期买入，政策收紧初期卖出——这个策略的胜率比任何技术指标都高。
{{< /sentence >}}

核心不在具体的买卖时点，在于把政策周期当作仓位管理的主要参考。政策放松时加仓、政策收紧时减仓——即使你判断不了个股好坏，这个框架也能避开大多数系统性风险。

## 政策信号的解读方法

{{< sentence >}}
不要只看政策说了什么，要看政策什么时候说的、用什么力度说的。
{{< /sentence >}}

同样一句"要保持流动性合理充裕"，出现在经济下行初期和出现在下行末期，含义完全不同。政策表态的措辞变化、频率变化和时间窗口，比政策内容本身更有信息量。

{{< sentence >}}
央行连续三次降准，信号已经够清晰了。如果你还在等第四次确认，你等的不是确认，是勇气。
{{< /sentence >}}

大部分投资者不是看不到政策信号，而是不敢在信号出现时行动。政策信号的价值不在于发现，在于你愿不愿意在不确定中做出决策。

## 场景速查

- **央行突然降息但市场还在跌**：大概率是政策底提前于市场底，不要等"再确认一下"
- **利好政策频出但市场不涨了**：利好出尽，考虑减仓
- **舆论一边倒地悲观**：检查政策面有没有已经出现放松信号
- **市场大涨但找不到政策催化剂**：可能是惯性上涨，要谨慎
