{{< sentence >}}
不是你做了什么，而是你让谁知道了你做了什么。
{{< /sentence >}}

能力很重要，但能力如果没人知道就没有价值。职场达人往往很善于在适当的时候、对适当的人说出自己的成就。这不是吹牛，而是有效的沟通。

{{< sentence >}}
专业和专业之间的差异来自于细节处理的差异。
{{< /sentence >}}

很多人做事都"差不多"，但职场达人会在细节上花更多的心思。同样的报告，职场达人的版本会更清晰、更有说服力。这种细致赢得了专业的声誉。

{{< sentence >}}
人脉最大的价值不在于你现在能得到什么，而在于别人什么时候想到你。
{{< /sentence >}}

良好的人脉关系意味着当机会出现时，有人会想到你。这个"被想到"往往决定了你是否有机会。

{{< sentence >}}
坚守原则和灵活变通不矛盾。你可以坚守价值观，同时灵活处理方式。
{{< /sentence >}}

职场达人往往有明确的原则，但在执行上很灵活。他们不会因为一时的政治压力就放弃原则，但也会找到实现原则的最有效方式。

{{< sentence >}}
你的价值不是通过频繁跳槽来提升，而是通过在某个地方做深做透来提升。
{{< /sentence >}}

频繁跳槽往往被看作不稳定。职场达人反而是通过在一个地方的长期积累来建立自己的专业形象。当然，也要知道什么时候该跳。
