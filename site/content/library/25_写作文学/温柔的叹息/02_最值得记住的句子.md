## 关于无法说出口的在意

{{< sentence >}}
有些事情你不说，不是因为不想说，是因为说出来就没办法当作不在意了。
{{< /sentence >}}

小说里一个角色在离别前沉默了很久。沉默不是冷漠——是最后一道防线。说出来意味着承认，承认意味着不能再装作无所谓。

{{< sentence >}}
最残忍的不是分开，是分开之后发现日子照过。
{{< /sentence >}}

生活没有因为某个人的离开而崩塌，这件事本身比离开更让人难过。它证明了你的痛苦在世界的运转面前微不足道。

## 关于犹豫和选择

{{< sentence >}}
犹豫不是软弱，是你在用最后的力气对抗一个你已经知道答案的问题。
{{< /sentence >}}

小说里最安静也最让人难受的段落，都发生在角色明知道该怎么做却迟迟不动的时刻。那个停顿里塞满了不甘心。

{{< sentence >}}
温柔是一种非常贵的选择。贵在你明知道会受伤，还是选择不硬起来。
{{< /sentence >}}

不是所有的柔软都是无力。有时候选择温柔需要比选择强硬更大的勇气——因为温柔意味着你不给自己留退路。

## 关于记忆和遗忘

{{< sentence >}}
你以为你已经忘了，直到你路过那条街的时候下意识放慢了脚步。
{{< /sentence >}}

身体的记忆比大脑的决定诚实。理智说"已经过去了"，脚步说"没有"。这种不一致本身就是小说最好的素材。

{{< sentence >}}
叹息是说不出来的那句话最后变成的声音。
{{< /sentence >}}

标题的注脚。所有无法表达的、被压下去的、放弃了的想说的话，最终压缩成一声叹息——轻到几乎听不见，重到读完整本书都忘不掉。
