{{< sentence author="谌旭彬" book="秦制两千年" >}}
秦制的核心，是皇权、官僚、编户齐民三者构成的稳定三角形，这个三角形一旦确立，就具有自我复制和自我修复的能力。
{{< /sentence >}}

这个三角形的稳定性来自相互制衡：皇权需要官僚执行命令，官僚需要编户齐民提供税收，编户齐民需要皇权保护免受豪强侵害。任何一方试图打破平衡，另外两方都会联合对抗。

{{< sentence author="谌旭彬" book="秦制两千年" >}}
郡县制的本质不是行政区划，而是中央政府绕过地方豪强，直接控制基层资源的制度设计。
{{< /sentence >}}

这解释了为什么郡县制能延续两千年：它解决的不是管理效率问题，而是权力分配问题。只要中央政府存在，就需要这套绕过中间层的直达机制。

{{< sentence author="谌旭彬" book="秦制两千年" >}}
文书行政的威力不在于提高效率，而在于让权力运行可追溯、可考核、可复制。
{{< /sentence >}}

现代组织管理的核心仍然是这套逻辑：把决策过程文档化，把执行结果数据化，把责任归属明确化。区别只是从竹简变成了数字化系统。

{{< sentence author="谌旭彬" book="秦制两千年" >}}
每次改朝换代，新统治者都以为自己在创新，实际上都在重新发明秦制的轮子。
{{< /sentence >}}

制度的生命力比创制者更强。理解这一点，能避免在组织变革中重复前人已经踩过的坑，也能理解为什么某些"新"模式其实是旧制度的变体。

{{< sentence author="谌旭彬" book="秦制两千年" >}}
秦制的崩溃往往不是因为制度本身失效，而是因为执行者偏离了制度的核心逻辑。
{{< /sentence >}}

这个判断对分析任何组织都有效：当一个体系开始出问题时，先检查是制度设计有缺陷，还是执行过程中有人改变了游戏规则。大多数情况下，问题出在后者。
