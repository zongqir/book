# 先别忙着庆祝，先看你会不会把胜利自己做丢

{{< sentence >}}
桌上能谈，不等于手上就能停。
{{< /sentence >}}

这句专门用来挡“局势缓和了，可以先松口气”的冲动。

{{< sentence >}}
胜势一冒头，最怕的不是犹豫，而是处处都想赢。
{{< /sentence >}}

第四卷真正重的地方，不在鼓人更勇，而在逼人更收。

{{< sentence >}}
优势不压到一点上，就会很快薄得像没有优势。
{{< /sentence >}}

这句把第四卷的推进逻辑压得最短，也最硬。

{{< sentence >}}
打下来不算难，难的是立刻接出下一步。
{{< /sentence >}}

第四卷最不迷信单点胜利，它真正盯的是连续性。

{{< sentence >}}
翻旧账之前，先把人分层，不然多数人会先被你推走。
{{< /sentence >}}

这句最适合在利益要动、局势又还没稳的时候翻出来。

{{< sentence >}}
进城先立规矩，别先忙着开新口子。
{{< /sentence >}}

这句把《党委会的工作方法》最值钱的地方压出来了。

{{< sentence >}}
旧秩序倒得越快，新秩序越不能等赢完再想。
{{< /sentence >}}

这句把第四卷最后那层国家问题提前顶了出来。

{{< sentence >}}
战争胜利如果接不到国家形式上，前面的胜利仍然会悬着。
{{< /sentence >}}

第四卷真正和前三卷拉开距离的地方，就在这句里。

## 这些句子什么时候最该翻出来

- 看见表面和谈、表面缓和时：先看第 1 句。
- 手里刚有一点优势、什么都想一起做时：先看第 2、3 句。
- 拿下一步以后，不知道下一步怎么接时：先看第 4 句。
- 一碰利益重排就想一把推平时：先看第 5 句。
- 接手范围一下变大，内部开始乱时：先看第 6 句。
- 所有人都只盯最后一程，没人说之后怎么运行时：先看第 7、8 句。

第四卷最该留下来的，不是胜利语气，而是这些能让你在最容易出错的时候先收住自己的话。
