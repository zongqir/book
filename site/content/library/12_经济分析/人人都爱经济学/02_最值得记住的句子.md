## 关于选择与成本

{{< sentence >}}
任何选择都有机会成本。问的不是"这个值吗"，而是"比起其他选择，这个值吗"。
{{< /sentence >}}

看起来再好的机会，如果放弃了更好的机会去做它，那就不值。

{{< sentence >}}
人对隐形成本往往视而不见。真正的成本不是你支付的钱，而是你放弃的最好的替代方案。
{{< /sentence >}}

## 关于激励与行为

{{< sentence >}}
改变一个激励，行为会立刻改变。政策经常失效，不是因为政策不对，而是激励结构没有真的改变人的动机。
{{< /sentence >}}

口头号召往往不如改变激励有效。

{{< sentence >}}
人对直接激励很敏感，但往往看不到间接激励。一项政策的直接后果和间接后果往往完全相反。
{{< /sentence >}}

## 关于资源与分配

{{< sentence >}}
"免费"的东西往往最贵，因为它的分配成本高。要么拼关系，要么排长队，要么要有名额限制，要么有隐形的价格。
{{< /sentence >}}

从来没有真正的免费，只是成本的转移。

{{< sentence >}}
权力最终也是稀缺资源。谁有权分配，谁就有了价值。这也是为什么政治这么重要。
{{< /sentence >}}

## 关于市场与效率

{{< sentence >}}
市场能高效地配置资源的前提是：完全信息、大量买卖家、自由进出。缺少任何一个，市场就会失灵。
{{< /sentence >}}

理解市场什么时候有效，什么时候需要政府干预。

{{< sentence >}}
人们讨厌不公平，但公平成本很高。所以社会往往选择"有效"而不是"公平"。理解这个权衡，你就理解了很多制度设计。
{{< /sentence >}}

## 关于信息与决策

{{< sentence >}}
信息不对称的地方，就是有人能赚到不该赚的钱。保险、二手车、人才招聘，本质都是在处理信息不对称。
{{< /sentence >}}

理解信息不对称，你就能看见很多商机和陷阱。

{{< sentence >}}
没有人知道未来。所以所有的选择本质上都是在不确定下的赌博。好的决策不是预测对了，而是赌的时候把概率算清楚。
{{< /sentence >}}
