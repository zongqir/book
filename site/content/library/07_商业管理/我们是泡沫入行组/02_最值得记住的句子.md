银行走廊里管用的规则从来不写在手册上。池井户润把它们藏在角色的对话和叙述者的旁白里。

{{< sentence no="1" quote="以牙还牙，加倍奉还。" >}}
半泽直树的标志性宣言。听起来像报复，实际上是一条行动原则——反击的力度必须超过对方施加的压力，否则组织系统会把你的反抗当噪音过滤掉。前提是手里有足够的证据。
{{< /sentence >}}

{{< sentence no="2" quote="银行不会记得你做对了什么，但会精确记得你出过什么错。" >}}
组织的记忆机制是不对称的。业绩好的时候归功于团队和体制；出了问题的时候责任精确到个人。在这种系统里，"不犯错"比"做对的事"更接近生存策略。
{{< /sentence >}}

{{< sentence no="3" quote="泡沫期进来的人，就是用来被消耗的。" >}}
没人当面说出这句话，但人事部门的行为逻辑一直在执行它。扩招期进来的人是组织膨胀的产物；收缩期到来，膨胀的部分最先被切。"消耗"二字是关键——被淘汰的人不是犯了错，是组织正常运转的代谢物。
{{< /sentence >}}

{{< sentence no="4" quote="你以为自己在选择职业，其实是经济周期在替你选。" >}}
入行那年日经指数在什么位置，比你的毕业成绩更能预测你十年后的处境。宏观周期面前，个人选择的自由度远比想象中小。
{{< /sentence >}}

{{< sentence no="5" quote="忠诚到最后，只是方便他们在需要的时候找到你。" >}}
忠诚让你变得可预测——总在岗位上、总能配合、不会突然离开。对组织来说，可预测意味着可控。需要牺牲品的时候，可控的人最好找。
{{< /sentence >}}

{{< sentence no="6" quote="同期里的朋友，也是同期里的对手。" >}}
同期制度把同一年入行的人绑在一起评比。信息共享和利益竞争同时发生。半泽和渡真利的互助之所以成立，恰恰因为两人不在同一条晋升赛道上。赛道一旦重叠，关系会迅速变质。
{{< /sentence >}}

{{< sentence no="7" quote="流程不是用来保护你的，是用来在出事的时候定位你的。" >}}
银行的审批流程、签字确认、风控节点——每一步都在记录"谁碰过这件事"。流程的主要功能是在错误发生后精确定位责任人。防止错误只是附带效果。
{{< /sentence >}}
