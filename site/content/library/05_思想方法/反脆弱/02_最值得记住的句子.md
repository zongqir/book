塔勒布的格言有刺。它们不是为了好听，是为了在你准备"做点什么"的时刻拦住你，或者在你准备什么都不做的时刻推你一把。下面七句按纠偏力度排序。

## 判断波动的方向

{{< sentence >}}
"风能吹灭蜡烛，却能让火越烧越旺。你要成为火，渴望风的到来。"
{{< /sentence >}}

同样的冲击，不同的系统反应相反。遇到变化时，先别急着判断"这是好事还是坏事"。先判断自己现在是蜡烛还是火。蜡烛需要保护，火需要燃料。搞反了，保护会闷死火，放任会吹灭蜡烛。

{{< sentence >}}
"如果你的策略每一步都需要预测正确，你的策略就是脆弱的。"
{{< /sentence >}}

规划做得越精密、每一步卡得越紧，对"一切按计划进行"的依赖就越深。一旦某一步偏了，整条链断裂。判断一个方案好不好，不是看它在理想条件下跑得多漂亮，而是看某个环节出错时它还能不能站住。

## 减法比加法更有力

{{< sentence >}}
"最强大的智慧在于知道什么该去掉——减法认识论。"
{{< /sentence >}}

人的本能是加东西。出了问题加流程，感觉空虚加活动，不够安全加保险。塔勒布反复强调：去掉有害的东西，比加上好的东西更可靠。戒烟的收益比任何保健品都大。砍掉一个毒性客户的收益比拉十个新客户都大。遇到问题时，先问"有什么可以不做"。

{{< sentence >}}
"大自然的冗余不是浪费。两个肾、额外的骨密度、多出来的肺容量——都是留给极端情况的。"
{{< /sentence >}}

效率思维把冗余视为浪费。在正常情况下确实如此。但正常情况不是永远。一个肾出了问题，另一个顶上。系统在极端压力下还能运转，靠的就是平时"多出来"的那部分。砍掉冗余去追求效率，等于在晴天卖掉雨伞。

## 该信谁

{{< sentence >}}
"不要问一个人怎么看，问他的仓位是什么。"
{{< /sentence >}}

分析师预测市场涨，但自己没有持仓。医生建议做手术，但如果是他自己的孩子，他会选保守治疗。顾问建议激进扩张，但他的薪酬不取决于扩张结果。判断建议是否可信，看给建议的人是不是在承担后果。

## 别和时间对赌

{{< sentence >}}
"一本书活了两千年，它再活两千年的概率远高于一本刚出版的书。"
{{< /sentence >}}

新技术、新概念、新工具，存活率远低于你的预期。旧事物之所以还在，是因为它经受住了淘汰压力。选工具、选方法、选知识，林迪效应是最省力的筛选器：存活时间越长，预期剩余寿命越长。追新之前，先查查旧的为什么还没被淘汰。

{{< sentence >}}
"现代化不是用新的替代旧的，是给旧的加上选择权。"
{{< /sentence >}}

真正持久的进步不是推翻旧系统，而是在保留旧系统的基础上增加新选项。筷子存在了几千年，刀叉也存在了几百年。两者共存，选择权在使用者手里。一个"新方法全面取代旧方法"的叙事，通常比它承诺的更脆弱。

## 场景映射

遇到以下场景时，把对应的句子调出来：

- **准备增加流程、规则、工具时**：先问"有什么可以去掉"——减法认识论
- **评估一个计划或方案时**：检查"如果某一步出错，整个方案是否还能站住"——脆弱的策略
- **面对变化或冲击时**：先判断自己是蜡烛还是火——再决定是防御还是拥抱
- **听取某人的建议时**：看他的仓位，不看他的表态——皮肤参与博弈
- **选择工具、技术、方法时**：查存活时间——林迪效应
- **评估一个系统的"效率"时**：检查冗余是否被砍光——冗余不是浪费
- **听到"革命性""颠覆性"时**：检查旧的为什么还没被淘汰——新不等于好
