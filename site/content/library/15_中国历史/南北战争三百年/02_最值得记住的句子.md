{{< sentence >}}
战争不是政治的延续，战争是技术的较量。
{{< /sentence >}}

政治意志再强烈，技术劣势无法弥补。南朝士族再有文化优越感，面对北朝骑兵冲击时依然溃败。

{{< sentence >}}
马镫的普及比任何制度改革都更深刻地改变了战场格局。
{{< /sentence >}}

一个简单的金属环，让骑兵从移动射击平台变成重装冲击兵器。步兵方阵的统治地位从此终结。

{{< sentence >}}
城墙的高度决定了政权的寿命。
{{< /sentence >}}

攻城技术每进步一步，防守方就必须加高城墙、增厚城基。落后一代的城防体系，意味着政权随时可能覆灭。

{{< sentence >}}
北朝的优势不在勇猛，在于将草原战术与中原技术完美结合。
{{< /sentence >}}

单纯的游牧骑兵无法攻坚，纯粹的农业步兵缺乏机动。北朝政权掌握了两套技术，南朝只有一套。

{{< sentence >}}
长江天险对步兵战争有效，对骑兵时代的意义被高估了。
{{< /sentence >}}

骑兵优势在于快速突破和战略机动，不在于渡江作战。真正阻止北朝南下的是后勤补给，不是长江本身。

{{< sentence >}}
三百年南北对峙的根本原因是技术扩散的不平衡。
{{< /sentence >}}

军事技术从草原向农区传播需要时间，从北方向南方扩散需要载体。技术差距存在一天，统一就推迟一天。
