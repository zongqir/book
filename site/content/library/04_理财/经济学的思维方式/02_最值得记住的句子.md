你对经济问题的大部分直觉，海恩都会翻一遍。不是因为直觉蠢，是因为直觉只看到了一半。

下面十句话按认知翻转的层次排列。前面拆个人决策里的盲区，后面拆市场和政策层面的盲区。

## 你付的钱不是代价，放弃的那个才是

{{< sentence no="1" quote="选择的真实代价不是你花了多少钱，而是你为此放弃的最佳替代方案。" >}}

花三百块吃一顿饭，账单上写着三百。但代价不在账单上——是你本来可以拿这三百块和这两个小时去做的最好的事。

机会成本是海恩整个框架的起跑线。少了这一步，后面所有判断都在歪着走。

{{< /sentence >}}

{{< sentence no="2" quote="已经花掉的钱不应该影响你下一步的决定。" >}}

电影开演二十分钟，确定难看。走还是不走？

"票都买了"不构成继续看的理由。票钱已经回不来了——经济学叫沉没成本。继续看,额外赔进去的是你接下来的时间。遇到"反正已经投了这么多"的念头时,拿这句话拦一下。

{{< /sentence >}}

## 多一杯水和多一颗钻石，答案完全反过来

{{< sentence no="3" quote="值不值，永远是在问'多一单位值不值'，不是在问'总共值不值'。" >}}

"教育很重要"——没人反对。但"再多拨一百亿给教育"是另一个问题。那一百亿也可以修路、降税、加强医疗。答案取决于边际上的比较：多出来这笔钱放在哪里产出最大。

边际分析把模糊的"重不重要"翻译成可以比较的"多一点值不值"。

{{< /sentence >}}

{{< sentence no="4" quote="水比钻石重要得多，但多一杯水远不如多一颗钻石值钱。" >}}

经济学史上最有名的悖论，答案只有一层：你从来不在"所有水"和"所有钻石"之间做选择。选择永远发生在边际上——多一杯还是少一杯。

搞清这条，"为什么救命药那么贵"和"为什么空气免费"就不再矛盾。

{{< /sentence >}}

## 改了代价结构，行为就跟着转向

{{< sentence no="5" quote="人对激励做出反应——但反应的方向不一定是你设计的那个。" >}}

强制系安全带，驾驶员觉得更安全，开得更猛。行人伤亡率没降，反而上升。

激励从来不是单向开关。改变了收益和代价的一侧，行为就调整——调整方向未必顺着意图走。

{{< /sentence >}}

{{< sentence no="6" quote="价格是把全世界的供需信息压缩成一个数字。管制它，就是往信号系统里灌噪音。" >}}

汽油涨价，你不需要知道是中东减产还是炼厂检修。价格变动本身就是信号：让消费者少用、让生产者多产。

限价之后信号断了。消费者不知道该节省，生产者不知道该增产。短缺、排队、黑市——不是巧合，是通信系统被掐断后的必然反应。

{{< /sentence >}}

{{< sentence no="7" quote="判断一项政策，不要看它打算做什么，看它改变了谁的激励结构。" >}}

租金管制打算保护租客。结果房东停止维修，新建出租房减少，现有租客不敢搬——管制把还没找到房子的人挡在了门外。

好意图加上坏激励，效果经常比不干预还差。海恩反复用这类案例只提醒一件事：意图不等于结果。

{{< /sentence >}}

## 没有总指挥，但货架上永远有牛奶

{{< sentence no="8" quote="即使你什么都比对方强，和对方合作仍然对你有利——只要你们的相对成本不同。" >}}

律师打字比秘书快。她应该自己打字吗？

打字一小时，少做一小时法律咨询。咨询收入远高于省下的打字成本。比较优势只看一个指标：谁的机会成本更低。哪怕绝对能力全面占优，合作仍然划算。

{{< /sentence >}}

{{< sentence no="9" quote="没有人安排货架上永远有牛奶——数百万人各自做决定，牛奶就到了你手边。" >}}

奶农、运输商、超市各自追求自己的利益。没有中央计划局发指令。但每天早上，牛奶准时出现在便利店的冷柜里。

海恩管这叫自发秩序。规则清晰、产权明确的时候，分散决策的汇总结果往往比集中指挥更稳定。自发结果未必都好——关键在于弄清它怎么长出来的。搞清了生长机制，才知道哪里该碰、哪里别碰。

{{< /sentence >}}

{{< sentence no="10" quote="产权不是保护有钱人的特权——它是让所有人都敢做长期投入的前提。" >}}

种一棵树，果实随时可能被没收。你还会种吗？

产权让投入的收益归自己，损失也归自己。有了这条保障，人才愿意放弃今天的消费，投入一个不确定的未来。没有产权保障的地方，再多资源和人才也推不动长期积累。

{{< /sentence >}}

## 遇到经济争论时的追问清单

讨论某项选择值不值——先调第 1、2 句。机会成本和沉没成本，拆开账面数字背后的真实代价。

争论某件事该不该多做——先调第 3、4 句。边际分析把"重不重要"翻译成"多一点值不值"。

评估某项政策好不好——先调第 5、6、7 句。追问激励变了什么、价格信号断没断、谁在暗处承担代价。

讨论分工、贸易、市场秩序——先调第 8、9、10 句。比较优势、自发秩序和产权，三块拼在一起才看得清合作怎么发生。
