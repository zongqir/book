{{< sentence >}}
你的方案没问题，但你跳过了你的直属领导——这就是问题。
{{< /sentence >}}

在国企，流程不只是效率工具——它是组织权力分配的表达方式。你绕过直属领导去找更上面的人，内容对不对已经不重要了。你打乱的是信息分发链条：你的领导不知道这件事，下次开会被问到会尴尬。尴尬会变成对你的不信任。

{{< sentence >}}
会前没有人跟你通过气的议题，在会上提出来大概率会被搁置。
{{< /sentence >}}

国企的会议功能是"确认"，不是"讨论"。真正的讨论发生在会前的非正式沟通里。你带着一个大家没提前消化过的方案走进会议室，等于在逼所有人当场表态——没人愿意这样做。

{{< sentence >}}
中午一起吃过几次饭，直接影响你跨部门协调的配合速度。
{{< /sentence >}}

这不是腐败，是信任的积累方式。在国企这种不完全依靠市场激励的组织里，人和人之间的协作意愿很大程度上建立在"熟悉度"之上。你跟隔壁部门的人从来不打交道，需要他帮忙的时候他排你在最后面——这不是他针对你，是你没有在日常建立信任通道。

{{< sentence >}}
领导开会时点你的名表扬你，不一定是好事。
{{< /sentence >}}

在国企里，被当众表扬有时候意味着你被"架"起来了——领导用表扬给你加了一个标签，后面如果出了问题，大家第一个想到的就是"不是说他最行吗"。学会低调地把活做了，比被高调地表扬更安全。

{{< sentence >}}
"大家有什么意见？"不是在征求意见——是在确认没有反对意见。
{{< /sentence >}}

这个句式在国企会议里的真实含义是："方案已经定了，现在是最后一个可以提出异议的窗口。如果你没有必须说的话，请沉默——你的沉默会被记录为同意。"

{{< sentence >}}
写报告的时候，"基本完成"和"完成"差别很大。
{{< /sentence >}}

在国企的文字系统里，措辞精度直接关联责任分配。"基本完成"给自己留了余地，"完成"是把话说死了。一旦后面出了问题，"完成"两个字就成了追责依据。蒋子龙说，在国企学会用词的轻重，比学会写漂亮文章更重要。

{{< sentence >}}
新人最安全的策略是：多做事，少表态。
{{< /sentence >}}

你还没搞清楚哪个领导和哪个领导的关系，就在会上表态支持某个方案——你可能无意中站了队。在弄清楚组织里的人际地图之前，用行动证明你能干活就够了。观点可以以后再表达，站错队很难纠正。
