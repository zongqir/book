{{< sentence >}}
忙碌是最不会被质疑的逃避方式。
{{< /sentence >}}

别人说你"太闲了"你会不舒服，说你"太忙了"你反而有点得意。忙碌自带正当性——没有人会批评一个忙碌的人。但正是这种正当性，让它成了最隐蔽的回避策略。你用忙碌回避的那个东西，藏在"忙"的正当性后面，安全得很。

{{< sentence >}}
如果你停下来会慌，那让你一直跑的就不是热情——是恐惧。
{{< /sentence >}}

区分很简单：热情驱动的行动，做完了会满足。恐惧驱动的行动，做完了只会松一口气，然后立刻找下一件事。如果你的"休息"总是伴随着焦躁，驱动力的性质就已经暴露了。

{{< sentence >}}
你不等于你做的事。但你的神经系统可能已经把这两样焊在一起了。
{{< /sentence >}}

"我忙所以我有价值"——这个等式不需要你同意，只需要你的身体在闲下来的时候紧张就够了。理性上知道"人的价值不取决于产出"和身体上能接受空闲，完全是两件事。

{{< sentence >}}
不敢止步的人最缺的不是休息技巧，是对自己存在本身的安全感。
{{< /sentence >}}

冥想、度假、断联——这些休息方式对不敢止步的人来说不是放松，是煎熬。因为问题不在身体累不累，在于"什么都不做的我还值得存在吗"。这个问题不解决，换什么休息方式都是隔靴搔痒。

{{< sentence >}}
看见驱动力的真面目是第一步。不是为了批判它，是为了不再被它劫持。
{{< /sentence >}}

赵昂没有要求你立刻停下来。要求的是：下次你又冲去加班的时候，暂停三秒，问自己一个问题——"我是想做，还是不做会慌？"能回答这个问题，就已经从自动模式切换到了手动模式。
