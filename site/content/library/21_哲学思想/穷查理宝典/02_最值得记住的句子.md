# 十句话拆开"聪明人为什么还是做蠢事"这个死结

芒格说话不绕弯。他的表达密度高、刺痛感强，最有用的不是金句效果，是你做决策时能直接调出来拦住自己的那种提醒力。下面十句按杠杆排——排在前面的，越容易在判断卡住时帮你翻转视角。

## 思维模型与判断结构

{{< sentence >}}
"拿着锤子的人，看什么都像钉子。"
{{< /sentence >}}

芒格说的不只是工具偏差。他指的是一种更深的结构性盲区：你受过什么训练，就倾向于用那套框架解释一切。经济学家看到激励，心理学家看到动机，工程师看到系统。每个人都觉得自己看到了全貌。这句话该在你准备用"唯一正确框架"分析问题的时候跳出来。

{{< sentence >}}
"在手里只有锤子的人看来，每个问题都像钉子。这是一种特别容易酿成灾难的思维方式。"
{{< /sentence >}}

上一句的加强版。芒格加了"酿成灾难"四个字，把它从思维提醒升级为风险警告。单一模型不只是效率低，它能制造系统性错判。

## 逆向排除

{{< sentence >}}
"如果我知道我会死在哪里，那我就永远不去那个地方。"
{{< /sentence >}}

逆向思考的原型句。它听起来像玩笑，但操作起来非常认真：做任何决策前先列出所有能让你彻底失败的路径，然后逐条避开。正面思考容易被乐观劫持，反过来想更容易看到真实风险。

{{< sentence >}}
"反过来想，总是反过来想。"
{{< /sentence >}}

两个字的决策纪律。在你兴奋地论证"为什么这件事会成功"的时候，停下来，先论证"为什么这件事会失败"。如果失败论证比成功论证更有说服力，别动手。

## 认知偏差与心理纪律

{{< sentence >}}
"我这辈子遇到的聪明人，没有一个不是每天都在阅读的。一个都没有。"
{{< /sentence >}}

这不是读书推荐。芒格指的是持续学习是判断质量的基础设施，不是锦上添花。停止学习的聪明人，靠的是存量认知，而存量认知在变化的环境里会快速贬值。

{{< sentence >}}
"在商界和科学界，有条非常有用的古老守则，它分两步：(1) 找到一个简单的、基本的道理；(2) 非常严格地按照这个道理去行事。"
{{< /sentence >}}

对复杂性崇拜的直接回击。芒格不信复杂模型。他相信找到几条经得住检验的基本道理，然后执行的纪律强过策略的精巧。

## 能力圈与安全边际

{{< sentence >}}
"知道自己不知道什么，比聪明更有用。"
{{< /sentence >}}

能力圈原则的心理基础。高估自己知道的范围比无知更危险——无知至少会让你小心，而自以为知道会让你大胆冲进灾难。

{{< sentence >}}
"你不需要成为每个领域的专家。你需要知道你懂什么，你不懂什么，以及那条边界在哪里。"
{{< /sentence >}}

能力圈的操作定义。"我在这个领域有判断力"和"我不在这个领域有判断力"之间的那条线，比两边的面积都重要。

## 耐心与复利

{{< sentence >}}
"大钱不是在买卖中赚到的，而是在等待中赚到的。"
{{< /sentence >}}

对交易频率崇拜的直接否定。芒格和巴菲特的投资回报，绝大多数来自长期持有少数几家公司。频繁操作在数学上就是在打断复利。这句话适用于一切需要时间积累才能见效的事。

{{< sentence >}}
"人们算计太多，思考太少。"
{{< /sentence >}}

一句话戳穿"数据驱动"的盲区。精确计算一个错误的前提，不如模糊思考一个正确的方向。这句话在你沉迷于电子表格但忘了抬头看大局的时候最该响起来。
