{{< sentence >}}
头一百小时，靠兴趣驱动。
{{< /sentence >}}

刚开始学习的时候，你靠热情走。选一个你真的感兴趣的领域，允许自己在这个阶段探索和游走。这一百小时的目标不是掌握，而是确认"这是我真的想要做的事"。

{{< sentence >}}
第一百到三百小时，靠结构学习。
{{< /sentence >}}

兴趣的热潮会过去。你需要建立一个学习的框架——什么是必须学的，什么可以后期补充，什么完全不需要。没有这个框架，你会在细节里迷失。

{{< sentence >}}
第三百到七百小时，靠实践反复。
{{< /sentence >}}

这是最关键的阶段。光读书和听课不够，你需要不断地做、不断地反馈、不断地修正。这个阶段往往是最辛苦的，因为你会发现自己的不足。

{{< sentence >}}
第七百到一千小时，靠总结迭代。
{{< /sentence >}}

到了这个阶段，你开始整理自己学到的东西。形成自己的方法论、自己的理解。这不是被动地接收知识，而是主动地创造知识。

{{< sentence >}}
超过一千小时，你就从学生变成了实践者。
{{< /sentence >}}

一千小时之后，学习就不再是你的中心任务。你开始用学到的东西去做真实的项目。在这个过程中继续学习，但学习变成了辅助而不是主要活动。
