# 八句朴素判断，把管理从玄学拉回动作

## 管事还是管人

{{< sentence >}}
管理就是让下属明白什么是最重要的。
{{< /sentence >}}

管理者最常见的假动作是"什么都重要"。陈春花的判断很直接：下属不知道优先级，管理者就没在管理。

{{< sentence >}}
管理不是管人，是管事。
{{< /sentence >}}

一说到管理就想到"带人""激励""领导力"——这把管理浪漫化了。管理的第一件事是把事情理顺、分清楚、跟到底。人的问题，大部分是事的问题没解决。

## 计划不只是目标

{{< sentence >}}
计划管理不是关于目标的，而是关于资源的。
{{< /sentence >}}

很多团队的计划只有目标没有资源匹配。目标定得再漂亮，不回答"人从哪来、钱从哪来、时间怎么分配"，就只是口号。

{{< sentence >}}
计划就是为实现目标而寻找资源的一系列行动。
{{< /sentence >}}

注意"一系列行动"四个字。计划不是一页 PPT，是动作序列——谁做什么、什么时候做、资源怎么调、偏了怎么纠。

## 组织到底在解决什么问题

{{< sentence >}}
组织存在的理由就是让本不能胜任的人可以胜任。
{{< /sentence >}}

不要等到找来完美的人才开始干活。组织的价值在于通过分工、流程和协同，让普通人也能产出合格结果。等人等到天荒地老，不如把流程先跑起来。

## 领导不是天赋

{{< sentence >}}
没有不好的士兵，只有不好的将军。
{{< /sentence >}}

团队绩效差，管理者的第一反应常常是"人不行"。陈春花把镜头翻回来：下属做不到，首先检查的是管理者有没有给方向、给资源、给反馈。

{{< sentence >}}
好的管理就是能够让一线员工得到资源并有权力运用这些资源。
{{< /sentence >}}

管理者垄断资源和决策权，一线员工空有责任没有手段。陈春花反复提的一个判断：管理要往下授权，而不是往上集权。

## 管理不能靠信任兜底

{{< sentence >}}
管理所做的就是让人没有机会犯错。
{{< /sentence >}}

听起来不近人情。但陈春花的意思很实在：好的管理靠制度、靠流程、靠检查节点把犯错空间压小。信任是奢侈品，纪律是基本面。

## 场景调用

| 场景 | 调用哪句 |
|---|---|
| 团队不知道先做什么 | 管理就是让下属明白什么是最重要的 |
| 计划做了但执行断裂 | 计划就是为目标寻找资源的一系列行动 |
| 跨部门推不动 | 组织存在的理由是让本不能胜任的人可以胜任 |
| 下属能力不足想换人 | 没有不好的士兵，只有不好的将军 |
| 一线执行没活力 | 让一线员工得到资源并有权力运用 |
| 出了错想追责 | 管理所做的就是让人没有机会犯错 |
