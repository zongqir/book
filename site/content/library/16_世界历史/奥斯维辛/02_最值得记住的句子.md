{{< sentence >}}
奥斯维辛不是某一天突然建成的。它是经过反复试验、逐步升级、技术优化后才变成高效杀人工厂的。
{{< /sentence >}}

从歧视到隔离到驱逐到屠杀，每一步都比前一步只多一点点。没有一个环节看起来是"不可接受的跳跃"——但终点是毒气室。

{{< sentence >}}
大部分参与者不需要被强迫。服从、惯性和对职业角色的认同就足以驱动他们完成分配到的工作。
{{< /sentence >}}

"我只是在执行命令"不是借口，但它描述了一个真实的心理机制——当任务被充分分解后，每个人只看到自己负责的那一小块，道德判断被稀释到消失。

{{< sentence >}}
受害者之间也没有团结。极端匮乏的环境下，人与人的关系首先是竞争——争夺食物、空间和生存的微小机会。
{{< /sentence >}}

不要美化苦难。极端环境暴露的不只是加害者的残忍，也包括受害者的脆弱和人性的阴暗面。

{{< sentence >}}
旁观者不是无辜的。集中营周围的居民知道那里在发生什么，但选择了不看、不问、不管。
{{< /sentence >}}

系统性犯罪需要三方配合：执行者的服从、受害者的无力、旁观者的沉默。少一方都无法持续。

{{< sentence >}}
战后很多参与者终生没有感到内疚。他们不是压抑了良心——他们真的认为自己只是在做本职工作。
{{< /sentence >}}

这比"他们心里知道错了但不敢说"更可怕。如果连内疚都没有，说明制度化暴力确实能重塑参与者的道德感知。
