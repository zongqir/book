# 五条接管准则：从连续打赢到把新秩序立住

第四卷里的原则，带着很强的时间压力。

局势已经在变，机会也已经出来，可旧秩序还没退干净，新秩序也没完全站稳。这个时候最怕乱发力、乱结盟、乱接盘。下面两条是核心准则，后三条是支撑准则。

{{< principle no="1" type="core" statement="大势转折时，先盯决定全局的敌手，不要四面开火" >}}
局势越乱，越要收主次。

第四卷反复压的是这一刀。主要敌手没打穿，就先别把次要矛盾同时抬高。什么都想一起解决，最后通常是哪一头都没解决。

这条准则约束的是躁进。机会一来，人最容易把积压已久的所有问题一起翻出来。第四卷要的却是先封住枝节，把最决定局面的那一头先处理掉。

{{< /principle >}}

{{< principle no="2" type="core" statement="总体优势不够时，要在局部形成压倒性优势，并把它连续接下去" >}}
优势如果被均匀分掉，很快就不像优势了。

第四卷的用兵和推进判断，都围着这个原则打转。不是到处露头，而是在关键点压到够厚，打完一处立刻接下一处，让局部胜利一段段改写整体局面。

这条准则防的是平均主义式进攻。手里明明有机会，却因为什么都舍不得放，最后把胜势摊薄。

{{< /principle >}}

{{< principle no="3" type="support" statement="重排利益时，要先分清打击对象、争取对象和必须稳住的多数" >}}
第四卷处理土地和新区工作时，最怕的不是动作不够狠，而是分类不清。

该打谁，争取谁，稳住谁，如果三层混成一锅，局面马上会乱。能团结的人被推走，能运转的秩序被打断，胜利反而会漏掉。

这条准则保护的是接管后的可持续性。利益当然要动，但不能把多数人一起推到对面。

{{< /principle >}}

{{< principle no="4" type="support" statement="接手新局面时，先立集体决断、纪律和议事法，不靠个人兴奋往前冲" >}}
第四卷越往后，越强调党委会和纪律。

这不是为了把人管死，而是为了防止胜利后的扩张把内部也冲散。谁拍板，怎么讨论，信息怎么回来，事情先后怎么排，这些不立，接管很快就会变成各自为战。

这条准则提醒的是：大局变大以后，英雄式推进会越来越不够用。

{{< /principle >}}

{{< principle no="5" type="support" statement="胜利前就要回答新秩序靠什么运行，不能等赢完再补课" >}}
第四卷后段的判断很明确。打下来，不等于坐得稳。

靠谁来组织，怎样处理旧势力，怎样安顿中间层，国家形式靠什么站住，这些都必须在胜利前先有答案。否则前线赢得越快，后面越容易乱。

这条准则防的是“先赢后说”的拖延。越接近胜利，越不能把建制问题推给未来。

{{< /principle >}}
