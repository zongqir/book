## 关于政策与企业的关系

{{< sentence >}}
在中国做企业，你的对手往往不是竞争者，而是政策的不确定性。
{{< /sentence >}}

整本书里最密集的失败模式不是"产品不行"或"市场判断错"，而是"政策变了你没来得及调"。年广久、李经纬、戴国芳、唐万新——每一个故事都在确认这句话。场景：做任何超过三年的商业计划之前，先评估政策环境的稳定性。

{{< sentence >}}
在灰色地带赚到的钱，终归要用灰色的方式偿还。
{{< /sentence >}}

价格双轨制时期靠批文倒卖赚到的利润、产权不清时期靠模糊边界做大的企业，在政策收紧时最先被清算。灰色收入不是不可以赚，但要清楚地知道——这笔钱的保质期取决于政策窗口开多久。

## 关于时机和节奏

{{< sentence >}}
中国商业史上几乎每一个重大机会，窗口都只开了两三年。
{{< /sentence >}}

1992年南巡后的下海潮，2001年入世后的出口爆发，2003年之后的房地产窗口——回头看都是确定性的大机会。但窗口开着的时候，大部分人在犹豫。窗口关了之后，机会成本陡升。

{{< sentence >}}
活下来比什么都重要。活到政策重新打开的那一天，你就又有机会。
{{< /sentence >}}

1989年之后大量民营企业关门。1992年南巡讲话一出，活下来的那批人几乎全部重新起飞。死掉的不是因为不优秀，而是因为没挺过政策收缩期。经济下行期最重要的策略不是寻找新机会，而是保现金流、控杠杆、等窗口。

## 关于企业家与体制

{{< sentence >}}
在中国，最成功的企业家不是最有创新力的，而是最懂得处理政商关系的。
{{< /sentence >}}

柳传志、任正非、王石、鲁冠球——三十年站到最后的企业家，共同特征不是技术最强或市场最敏锐，而是在政商之间保持了微妙的平衡。既不过度亲近政府（避免被卷入政治风险），也不完全远离（确保获得必要的政策资源）。

{{< sentence >}}
每一代企业家都以为自己是在创造历史，其实他们都是在回应历史给出的约束条件。
{{< /sentence >}}

八十年代的约束是"什么能做什么不能做还不清楚"。九十年代的约束是"国有还是民营，产权怎么算"。两千年代的约束是"全球化的红利还能吃多久"。企业家的空间始终被这些约束界定。看清约束比看到机会更重要。

## 场景映射

- 评估一个长期商业计划的可行性：调用"对手是政策不确定性"
- 经济收缩期的企业决策：调用"活下来比什么都重要"
- 犹豫要不要抓住某个新兴机会：调用"窗口只开两三年"
- 分析某个企业家的成功原因：调用"最懂处理政商关系"
- 在灰色地带看到了暴利机会：调用"灰色地带的钱终归灰色偿还"
