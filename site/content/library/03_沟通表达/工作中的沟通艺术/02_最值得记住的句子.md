## 角色和场景判断

{{< sentence >}}
沟通效果首先取决于你对场景的判断，而不是你的表达能力。
{{< /sentence >}}

措辞再好，用错了场景也白费。你在全员会议上提了一个只该和直属领导私下讨论的问题——不管你说得多漂亮，结果就是让领导尴尬。先判断场景，再组织语言。

{{< sentence >}}
职场里的每一句话都有一个隐含的角色前缀。
{{< /sentence >}}

同样说"我觉得时间不够"，从项目经理嘴里说出来是预警，从实习生嘴里说出来是抱怨。你以为你在就事论事，但听的人先看的是你的角色。

## 渠道和信息层次

{{< sentence >}}
选错渠道比说错话更难补救。
{{< /sentence >}}

批评用邮件发，白纸黑字永远存在。表扬用即时消息说，三秒钟就被刷走了。渠道承载的不只是信息，还有信息的分量和寿命。

{{< sentence >}}
信息的详细程度应该匹配对方的决策需要，而不是你掌握的全部内容。
{{< /sentence >}}

给老板汇报，他需要结论和关键风险；给执行团队沟通，他们需要步骤和时间节点。你塞了一堆对方用不上的信息，不是详尽，是添乱。

## 倾听不只是理解

{{< sentence >}}
在职场里，倾听是一种权力行为——你听谁说话、听多久，传递的是地位信号。
{{< /sentence >}}

会议中你打断了谁、认真听完了谁，所有人都在看。你以为你在表达自己的判断，其实你在对全场广播你怎么排序在场的人。

{{< sentence >}}
回应对方之前先确认你听到的和对方想表达的是同一件事。
{{< /sentence >}}

"你是说截止日期改了？"比"我知道了"有用十倍。特别是在邮件沟通里，一次确认能省掉三轮来回。

## 冲突策略选择

{{< sentence >}}
冲突处理策略没有最好的，只有和当前利害关系匹配的。
{{< /sentence >}}

不是所有冲突都值得花时间协作解决。鸡毛蒜皮的分歧果断迁就或回避，省下来的精力用在真正重要的博弈上。

{{< sentence >}}
带着"赢"的目标进入职场冲突，不管结果如何你都已经输了一半。
{{< /sentence >}}

职场关系是长期博弈。你在一次争论中让对方丢了面子，赢了这个回合，但你在未来需要他配合的时候，成本会翻倍回来。

## 什么时候翻这些句子

- 要发一封重要邮件时，停一秒想：渠道对不对、信息量对不对
- 会议上准备开口时，先确认你当前的角色
- 和同事产生分歧时，先判断这件事值不值得认真处理
- 发现对方态度突变时，检查一下是不是你的表达力度和角色不匹配
