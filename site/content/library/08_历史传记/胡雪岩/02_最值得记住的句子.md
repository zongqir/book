商场上没有人能独自走远。你抬别人，别人才抬你。胡雪岩经营关系的起点就是这六个字——先让对方觉得被托起来了，回报自然会来。

在任何需要长期合作的场景里，先想能帮对方什么，再想自己要什么。
{{< sentence no="2" quote="有势才有利。做生意，先借势。" >}}
{{< /sentence >}}

利润不是凭空产生的，它跟着权力走。胡雪岩的每一笔大生意背后都有一个正在上升的政治势力。

判断一个商业机会时，先看它背后的权力结构是否稳固。
{{< sentence no="3" quote="八个坛子七个盖，盖来盖去不穿帮。" >}}
{{< /sentence >}}

资金调度的本质：永远在腾挪。胡雪岩同时运转钱庄、丝行、药号和官方采购，靠资金在不同业务间快速流转。

提醒你关注的不是利润率，而是现金流断裂的风险。坛子越多，某天盖不上的概率越大。
{{< sentence no="4" quote="前半夜想想自己，后半夜想想别人。" >}}
{{< /sentence >}}

先算清自己的利益，再去理解对方的处境。胡雪岩谈判时从不假装无私，但他总能找到双方都能接受的方案。

适用于所有需要在利己和利他之间找平衡的谈判。
{{< sentence no="5" quote="银子用得光，交情用不光。" >}}
{{< /sentence >}}

钱花完了可以再赚，人情用完了就没了。胡雪岩在经营中反复做一个选择：宁可短期亏钱，也要维护关系。

在关系型商业环境里，信用比现金更值钱。
{{< sentence no="6" quote="做人无非是个真字。" >}}
{{< /sentence >}}

胡雪岩的"真"不是天真，是一种策略性的坦诚。在对方都在算计的时候，你比别人多真一点，反而占据信任优势。

但注意边界：这种坦诚建立在你已经想清楚底线的前提上。不是什么都说，是关键时刻让对方觉得你可信。
{{< sentence no="7" quote="冤家宜解不宜结。" >}}
{{< /sentence >}}

胡雪岩几乎不主动树敌。遇到冲突，第一反应是找利益交汇点化解。在官商网络里，今天的对手可能明天就变成你需要的中间人。

在任何长期博弈中，少一个敌人比多一个朋友更重要。
{{< sentence no="8" quote="靠山，靠山，靠的是山。山倒了，什么都没了。" >}}
{{< /sentence >}}

高阳用小说后半段证明了这句话。胡雪岩的全部商业版图建立在左宗棠的政治地位上。左宗棠一失势，帝国即刻崩塌。

每当你发现核心业务依赖一个你无法控制的外部力量时，这句话就该响起来。
{{< sentence no="9" quote="满天打算盘。" >}}
{{< /sentence >}}

不止算自己的账，还要算对方的账、第三方的账、官场的账。胡雪岩做任何决策前，先在脑子里把所有相关方的利益和顾虑过一遍。

单线思考是商业决策的大敌。尤其在涉及权力关系的场景里，至少要同时算三方的账。
{{< sentence no="10" quote="天下没有白占的便宜。" >}}
{{< /sentence >}}

每笔收益背后都有代价，有些代价当时看不见。胡雪岩从政府采购中获取暴利的同时，也把自己绑上了政治风险的战车。

做决策时，永远要问：这笔便宜的隐含成本是什么？谁会在将来向我收这笔账？
