## 大转型不赶工

{{< sentence no="1" quote="中国的政治社会制度，从帝制转到民治，大约需要两百年。" >}}
{{< /sentence >}}

这个时间估计在学界一直有争议。但这句话的价值在于它把"转型"从一个事件变成了一个过程。一旦接受转型是过程，对"为什么还没成功"的焦虑就会降温。
{{< sentence no="2" quote="历史三峡，进了峡口就没有回头路。你可以在里面翻船，但不可能逆流而上。" >}}
{{< /sentence >}}

杀伤力大。它同时否定了两种幻想——复古的幻想和一步到位的幻想。
{{< sentence no="3" quote="我们这一代人，是在三峡中间的人。" >}}
{{< /sentence >}}

写这话时大概在1990年代。意思很明确：离出峡口还远，别急着庆祝。
## 帝制怎么死的

{{< sentence no="4" quote="清朝不是被推翻的，是自己倒的。辛亥革命不过给了最后一推。" >}}
{{< /sentence >}}

多数通俗叙事把辛亥写成"推翻满清的伟大革命"。唐德刚的判断更冷：你推什么推，它自己站不住了。
{{< sentence no="5" quote="科举制度考了一千多年，最后考出来的是一批不会跟洋人打交道、也不会跟老百姓打交道的人。" >}}
{{< /sentence >}}

这句话毒。它指向的是一个人才选拔系统的全面失效。
{{< sentence no="6" quote="李鸿章是大清最好的裱糊匠。但裱糊匠再好，房子烂了就是烂了。" >}}
{{< /sentence >}}

"裱糊匠"后来几乎成了李鸿章的标签。这个词既肯定能力，又划清了个人能力的上限。
## 改不了和砸不动

{{< sentence no="7" quote="戊戌变法要是成功了，中国至少提前二十年走上君主立宪。但它不可能成功。" >}}
{{< /sentence >}}

逻辑链条：不是康有为无能，是整个官僚系统没有执行能力来接住改革方案。
{{< sentence no="8" quote="孙中山会革命，不会建设。这不怪他——革命和建设需要的是两套完全不同的本事。" >}}
{{< /sentence >}}

这句话得罪了不少人。但论据扎实：辛亥之后十几年乱局，恰恰说明推翻容易、建设难。
{{< sentence no="9" quote="袁世凯当了总统还要当皇帝，不是他疯了，是整个社会还没学会怎么过没有皇帝的日子。" >}}
{{< /sentence >}}

最后半句才是重点。帝制思维不是一个人的病，是几千万人的惯性。
