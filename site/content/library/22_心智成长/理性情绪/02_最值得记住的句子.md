## 关于失败和不完美

{{< sentence >}}
一件事做得不完美，不代表我是一个失败的人。
{{< /sentence >}}

打破"要么完美，要么就意味着失败"的二元论。大多数实际的努力都在中间某处。

{{< sentence >}}
他人的批评针对的是这件事，不是我这个人。
{{< /sentence >}}

分离"行为和自我"。即使有人说"你这样不好"，也不等于"你这个人不好"。

## 关于他人的评价

{{< sentence >}}
我无法控制别人的想法，我只能控制我对他们想法的反应。
{{< /sentence >}}

把焦点从"他怎么看我"转向"我怎么看待他的看法"。这才是真正在你控制范围内的。

{{< sentence >}}
不是所有人都会喜欢我，这很正常，也很健康。
{{< /sentence >}}

从"被所有人接纳"的幻想中走出来。承认这个现实后，不被某个人接纳就不再是灾难。

## 关于焦虑和不确定性

{{< sentence >}}
我在想象最坏的情况，不是在预测未来。
{{< /sentence >}}

焦虑的人常把"可能发生"和"一定发生"混淆。识别出这一点，焦虑本身就会淡化。

{{< sentence >}}
如果真的发生了，我也能处理——因为我过去应对过难题。
{{< /sentence >}}

用过去的经验打破对未来的灾难化想象。你的生存能力往往比你认为的要强。

## 关于情绪本身

{{< sentence >}}
有这个情绪不代表我应该按照情绪去做。
{{< /sentence >}}

拆开"感受情绪"和"被情绪控制"两件事。你可以感到愤怒，同时选择不做愤怒驱动的事。

{{< sentence >}}
这个情绪会过去，它不是永久的。
{{< /sentence >}}

情绪往往看起来压倒一切，但时间和转移注意力会自然削弱它。记住这一点，能减缓情绪的迫切感。
