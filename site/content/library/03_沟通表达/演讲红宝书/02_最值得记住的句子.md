# 十句提醒把演讲从硬撑拉回可控

## 准备和听众判断是第一道防线

{{< sentence >}}
台上翻车的原因，多数藏在台下。
{{< /sentence >}}

紧张、卡壳、跑偏这些临场症状，往往只是准备阶段失误的集中暴露。

把注意力从"我怎么表现"移到"我怎么准备"，改进才有抓手。

{{< sentence >}}
先问听完后对方要做什么，再决定讲什么。
{{< /sentence >}}

准备顺序反了，内容再多也找不到主线。

目标、听众、核心结论——三件事没定，不碰内容。

{{< sentence >}}
你以为在替听众着想，其实一直在按自己的理解讲。
{{< /sentence >}}

大段铺背景、堆术语、按自己认知排序，都是"对象感缺失"的表现。

换位判断是一个需要单独做的准备动作，不只是态度问题。

## 结构、开场和结尾决定上限

{{< sentence >}}
说得复杂，通常不是内容高级，是结构还没压清。
{{< /sentence >}}

你自己没压出主线，听众不会在现场替你整理。

一场演讲保住两到四个主点就够，多了反而全丢。

{{< sentence >}}
开场的任务只有一个：争取注意力。
{{< /sentence >}}

前三十秒花在感谢和寒暄上，后面再好的内容也要加倍才能拉回来。

开场的任务只有一个：让人愿意继续听。

{{< sentence >}}
结尾的任务：把判断钉住。
{{< /sentence >}}

很多表达中间不差，最后发散，整场印象就塌了。

听众最后带走什么，取决于你有没有在结尾收住重点。

## 临场控制和复盘是长期提升的关键

{{< sentence >}}
怯场不能靠硬扛，只能靠准备把干扰降下来。
{{< /sentence >}}

越把紧张当性格缺陷，越改不动。

熟悉结构、预演开头、缩短句子、走一遍问答——堵住紧张能放大的空位。

{{< sentence >}}
问答不是附加题，是演讲的正式环节。
{{< /sentence >}}

只准备"我要讲什么"远远不够。

真正难的场合，多数发生在提问、质疑和打断里。提前列问题、练回拉主线，比多背三页稿子管用。

{{< sentence >}}
即兴发言和正式演讲不是一回事，准备逻辑不能混。
{{< /sentence >}}

信息汇报、目标型演讲、即兴回应，各有各的准备重心。

把所有场合用同一套模板对付，结构失衡和目标错位都会来。

{{< sentence >}}
一场演讲值不值，看它有没有推动判断和行动。
{{< /sentence >}}

讲完了对方既没有更清楚的判断，也没有更明确的下一步，这场表达多半只完成了信息经过。

演讲的终点是影响，不是说完。

## 准备、上台、复盘各查哪句

- 准备前：先查第 1、2、3 句。目标定了没有，听众想清了没有，内容是按谁的理解排的。
- 写提纲时：查第 4 句。主线压出来没有，主点是不是太多。
- 练开场结尾：查第 5、6 句。开头有没有让人愿意听，结尾有没有收住判断。
- 上台前：查第 7、8 句。怯场有没有用准备降级，问答有没有提前预判。
- 复盘时：查第 9、10 句。这次到底哪一环出了问题，下次只改一个最关键的。
