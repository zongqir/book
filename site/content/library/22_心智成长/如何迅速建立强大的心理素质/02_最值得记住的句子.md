{{< sentence >}}
面试前手心出汗不是胆子小——是交感神经过度激活。每种"心理素质差"的表现背后都有具体的神经回路。
{{< /sentence >}}

把"我心理素质差"拆成具体的生理机制，恐惧感就从"我这个人有问题"变成了"我这个回路需要训练"。前者让人绝望，后者让人有事可做。

{{< sentence >}}
大脑可以被训练改变。这不是鸡汤，是神经可塑性的基本原理。
{{< /sentence >}}

神经通路用得越多越强，不用就弱化。每一次在压力面前选择新的反应模式，都在物理层面重塑你的大脑。雅顿反复回到这个点：改变不是"想通了"，是"练够了"。

{{< sentence >}}
压力耐受、情绪调节、认知灵活性、社交信心——四个维度各管各的，哪个弱补哪个。
{{< /sentence >}}

不需要同时练四项。汇报时脑子空白，练认知灵活性。被批评后情绪崩盘，练情绪调节。面试前手抖，练压力耐受。像去健身房一样，哪块肌肉弱就练哪块。

{{< sentence >}}
情绪标记是调节情绪最快的动作。给情绪一个名字，前额叶就能开始介入了。
{{< /sentence >}}

心理学研究反复确认："我现在很愤怒"这六个字本身就能让杏仁核的激活程度降低。不是因为说了就不愤怒了，是因为"命名"这个动作启动了前额叶的理性加工通道。

{{< sentence >}}
渐进式暴露的逻辑很简单：你害怕什么，就小剂量地接触什么。重复足够多次之后，神经系统会更新它的威胁评估。
{{< /sentence >}}

不是让你跳进深水区。是让你先在浅水区站一会儿，等心跳恢复了，再往前走一步。每一步都在告诉杏仁核：这里没有你以为的那种危险。

{{< sentence >}}
"迅速"不是一周见效。指的是方向明确、路径清晰、不在错误的方向上浪费时间。
{{< /sentence >}}

雅顿用这句话管理预期。心理素质的提升需要重复训练和时间积累。"迅速"的含义是你从第一天就知道该练什么、怎么练，不走弯路。不是跳过积累过程本身。
