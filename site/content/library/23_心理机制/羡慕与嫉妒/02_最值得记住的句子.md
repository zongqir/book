{{< sentence >}}
羡慕先暴露的不是别人有什么，而是你在乎什么。
{{< /sentence >}}

情绪一冒出来，别急着羞愧。先看它指向的对象是什么，那里面通常藏着你没说出口的欲望排序。

{{< sentence >}}
嫉妒最怕的不是别人更好，而是自己可能被替代。
{{< /sentence >}}

一旦比较牵连到位置、关系或稀缺资源，情绪就会立刻变尖。它关心的不只是差距，还关心失去。

{{< sentence >}}
离你越近的人，越容易触发比较。
{{< /sentence >}}

陌生人的成功常常只是新闻，同辈、同事、朋友的成功更像威胁。因为参照系近，差距会被放大成“本来我也该有”。

{{< sentence >}}
很多羡慕表面冲着结果，底下其实冲着身份。
{{< /sentence >}}

你未必真想要对方那份工作、那段关系、那种生活。你更想要的，往往是那份结果背后代表的位置和确认。

{{< sentence >}}
压住比较没有用，看懂比较才有用。
{{< /sentence >}}

一味劝自己“别想了”通常只会让情绪换个地方冒出来。把它说清楚，才可能把它从暗处拉到明处。

{{< sentence >}}
比较最好的出口，不是自责，也不是贬低对方，而是把情绪改造成行动线索。
{{< /sentence >}}

它至少能帮你回答一个问题：你接下来要补能力、调目标，还是承认这并不是你真正想过的生活。
