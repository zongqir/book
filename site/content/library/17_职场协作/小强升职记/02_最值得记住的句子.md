{{< sentence >}}
忙不等于有产出。
{{< /sentence >}}

每天加班到深夜的人不一定比准时下班的人产出多。区别在于：你是在做重要的事还是在应对紧急的事。大多数"忙"都是后者——被别人的紧急事情推着走。

{{< sentence >}}
收集一切，大脑是用来思考的不是用来记事的。
{{< /sentence >}}

GTD的第一步：把脑子里所有待办事项全部写下来——不管大小。你的大脑不擅长记住十几个待办事项——它一边记一边焦虑。写下来之后大脑就解放了，可以专注思考。

{{< sentence >}}
两分钟能做完的事立刻做。
{{< /sentence >}}

一封邮件、一个签字、一个确认——不超过两分钟就能处理的事不要放进待办清单。放进去的成本（记录、追踪、回想上下文）比直接做还高。

{{< sentence >}}
先吃掉那只青蛙。
{{< /sentence >}}

每天开始工作时先做最困难、最重要的任务——不是最紧急的。如果你把"青蛙"留到下午，一整天都在逃避它，注意力被消耗但事情没做。

{{< sentence >}}
每周回顾比每天计划更重要。
{{< /sentence >}}

每天的计划让你知道今天做什么。每周回顾让你知道这周做的事是不是在推动真正重要的目标。没有周回顾，你可能高效地做了一周不重要的事。
