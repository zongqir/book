柏拉图构想理想国，亚里士多德分类政体类型。马基雅维利做了一件不同的事：他只看实际发生了什么。哪个统治者活下来了，他做了什么；哪个统治者垮了，他犯了什么错。下面六条准则就是从这些案例中沉淀出来的判断规则。

{{< principle no="1" type="core" statement="看人实际怎么做，不看人应该怎么做。" >}}
所有政治错误中最昂贵的一种，是按照人"应该"有的样子来制定策略。第十五章说得很直接：一个人如果在一切事情上都想发善心，在许多不善良的人当中迟早完蛋。

这条准则不是犬儒主义，是一个观察方法。预设人会按道德标准行事，你的策略就建立在一个经常失效的前提上。预设人会按利益行事，预判准确率高得多。

调用场景：做人事安排、制定激励方案、评估合作方可靠性时，把"他应该会这么做"替换成"他有什么动机这么做"。
{{< /principle >}}

{{< principle no="2" type="core" statement="权力的唯一度量是结果，不是意图。" >}}
仁慈但丢了国家的君主和残酷但保住秩序的君主，马基雅维利毫不犹豫选后者。意图保护不了任何人。

切萨雷·波吉亚在罗马涅的统治残酷但有效——他留下了秩序。佛罗伦萨人因为怕"残酷"的名声，不敢处理皮斯托亚的内乱，结果整座城市陷入混乱。仁慈造成的破坏比残酷更大。

这条准则约束一种根深蒂固的偏好：按意图评价行为。马基雅维利要求按结果评价。
{{< /principle >}}

{{< principle no="3" type="support" statement="被畏惧比被爱戴安全得多。" >}}
如果两者只能选一个，选被畏惧。理由冷静到刺人：爱戴取决于对方的心情，畏惧取决于你的行动。依赖不在自己控制范围内的东西来维持权力，是最脆弱的结构。

边界写得很清楚：被畏惧可以，被憎恨不行。一旦触发憎恨——侵占财产、夺人妻女——畏惧就不再起保护作用，反而变成叛乱的燃料。
{{< /principle >}}

{{< principle no="4" type="support" statement="必须伤害时一次用尽，施恩时细水长流。" >}}
残酷如果集中使用然后立刻转向安抚，效果好于反复小剂量伤害。前者让人害怕但很快适应，后者让人永远不安。

阿加托克利斯靠屠杀夺取叙拉古政权，之后迅速稳定局面。他的残酷被一次用完，后续只有治理。而持续施加小恶的统治者，每天都在制造新的敌人。

施恩的逻辑正好反过来。好处要一点一点给，让人始终有新期待。一次性把好处用完，之后就没有筹码了。
{{< /principle >}}

{{< principle no="5" type="support" statement="依靠自己的军队，不依靠雇佣兵和外援。" >}}
雇佣兵没有忠诚可言。和平时抢你的钱，战时保不了你的命。援军更危险：他们忠于派遣者。如果打赢了，你就成了附庸。

迁移版本：核心能力和关键资源不能外包。外包出去的东西，控制权就不在你手里了。失去控制权那天，对方没有任何义务继续支持你。

意大利城邦一百年来依赖雇佣军，结果是每一次外部入侵都毫无抵抗力。法国、西班牙的军队在意大利来去自如，城邦只能在几个强权之间摇摆求生。
{{< /principle >}}

{{< principle no="6" type="support" statement="命运占一半，但你只能管另一半。" >}}
马基雅维利承认运气的力量。洪水来了，什么堤坝都挡不住。但他紧接着补了一句：洪水不来的时候，你该修堤坝。

运气不可控，准备可控。一个好的统治者在太平时期做的事——建立制度、储备资源、培养忠诚——决定了他在危机中能不能撑过去。等危机来了才开始准备，时间窗口已经关闭。

调用场景：判断一件事"是运气差"还是"准备不足"时，先问——事情发生之前，你做了什么来应对这种可能？
{{< /principle >}}
