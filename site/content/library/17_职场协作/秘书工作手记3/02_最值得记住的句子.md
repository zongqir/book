{{< sentence >}}
领导没说完的话，才是你最该听的话。
{{< /sentence >}}

明确说出来的指令，任何人都能执行。但领导说到一半停下来、换了个话题、或者用了"你看着办"——这些中断和模糊，往往才是需要你判断的地方。训练自己对"没说完"的敏感度，比训练执行速度更重要。

{{< sentence >}}
准备工作不是做给领导看的，是让你在突发情况下不慌。
{{< /sentence >}}

会前准备了八份材料，可能只用到两份。但你准备了，遇到临时追问的时候你能三秒内递上答案。准备的目的不是展示勤奋，是降低你自己在高压场景下的反应延迟。

{{< sentence >}}
座次安排背后是权力信号，不是礼貌问题。
{{< /sentence >}}

谁坐主位、谁挨着谁、谁被安排在角落——这些不是随便排的。每一个座位安排都在传递"这次会议里谁的意见权重最高"。看懂座次，你就看懂了半个会议的决策结构。

{{< sentence >}}
会议纪要不是记录，是确认。
{{< /sentence >}}

新手以为会议纪要就是把大家说的话记下来。老手知道，纪要的作用是把"谁答应了什么"白纸黑字地固定住。写纪要的时候，比"记全"更重要的是"确认清楚"——谁承担什么、截止时间是什么、标准是什么。

{{< sentence >}}
打电话之前先想好最后一句话。
{{< /sentence >}}

很多人打电话前想的是"我要说什么"。李绍飞建议反过来——先想好这通电话结束时你希望对方记住的那一句话是什么，然后倒推整个通话的结构。

{{< sentence >}}
递材料的时候多说一句"变化在第三页"，省掉对方五分钟的翻找时间。
{{< /sentence >}}

省心感来自这种细节。你交上去的材料可能有二十页，领导可能只关心其中三行。主动标出重点位置，不是越权，是帮对方把注意力放在该放的地方。

{{< sentence >}}
领导批评你的时候，问题通常不是他说出来的那件事。
{{< /sentence >}}

表面上说你"这个报告格式不对"。但如果只是格式问题，他不会专门叫你过来说。可能是这份报告让他在会上丢了面子，可能是最近你的状态让他不满意、借这件事提醒你。学会在批评里听出没说的那层意思。

{{< sentence >}}
主动汇报不是邀功，是让领导不需要来问你。
{{< /sentence >}}

领导主动来问你进展，说明他已经开始担心了。在他问之前主动说一句"这件事目前到这一步了，预计后天完成"——你花十秒钟，省他一天的惦记。
