# 八句话重排你对钱、时间和注意力的优先级

## 三种资源的排序决定所有选择

{{< sentence >}}
注意力 > 时间 > 金钱。
{{< /sentence >}}

全书最底层的一句判断。排序反过来的人会为了省钱浪费时间，为了省时间浪费注意力。接受这个排序，所有关于"值不值得"的决策都会跟着变。

{{< sentence >}}
你拥有的最宝贵的财富是你的注意力。
{{< /sentence >}}

钱可以再赚，时间不可再生，但注意力连时间都能浪费掉。注意力放在哪里，你的世界就长成什么形状。

{{< sentence >}}
付费就是捡便宜。
{{< /sentence >}}

反直觉，但逻辑成立。花钱买回注意力和时间，比省钱丢掉注意力划算。你犹豫"该不该花这笔钱买工具或课程"时，调出这句。

## 时间的卖法决定天花板

{{< sentence >}}
所谓"个人财富自由"，是指某个人再也不用为了满足生活必需而出售自己的时间了。
{{< /sentence >}}

把财富自由从一个模糊的幻想变成一个可计算的门槛——被动收入 ≥ 生活成本。定义清楚了，路径才能被规划。

{{< sentence >}}
给自己打工的人，其实是把同一份时间出售了两次。
{{< /sentence >}}

上班时间同时在给老板和自己积累价值——前提是你在做的事本身能产生可复用的技能或作品。如果只是在消耗时间，那你只卖了一次。

{{< sentence >}}
你居然以为你能管理时间？
{{< /sentence >}}

时间自己流走，谁也拦不住。你能管理的只有自己的注意力——放在什么事上，用什么优先级去做。"时间管理"这个词本身就在误导方向。

## 耐心是一种投资策略

{{< sentence >}}
耐心是一切成长的刚需。
{{< /sentence >}}

复利的前提是不中途抽走本金。技能学习、写作积累、投资增值——任何值得长期持有的东西，前期都看不出回报。耐心不是你的性格特征，是你的投资决策。

那些在一个方向上持续投入超过三年的人，能力增长曲线已经拐了。在第一年退场的人看不到这个拐点。

{{< sentence >}}
活在未来，而不是活在当下。
{{< /sentence >}}

"活在当下"是消费型心态，"活在未来"是投资型心态。区别在于：做当下决策时，参照系是眼前的舒适还是两年后的你。

这句话在你面对"现在享受 vs 延迟满足"的选择时最有用。它不是让你否定当下，而是让你用未来的视角给当下的选择做校准。

{{< sentence >}}
七年就是一辈子。
{{< /sentence >}}

一个人在七年里持续投入一件事，他在这件事上的竞争力已经指数级领先。七年不是一个精确数字，是一个提醒：你愿意在一件事上待多久，决定了你能走到哪。
