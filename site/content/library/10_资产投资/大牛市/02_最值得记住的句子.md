# 关于牛市周期的几个关键提醒

## 牛市是周期的一部分，不是新常态

{{< sentence >}}
牛市的钱很好赚，但赚来的往往是虚假的自信。
{{< /sentence >}}

在牛市里赚钱，往往反映的是市场周期而不是你的本事。

{{< sentence >}}
周期终会反转，看不到转折信号的时候最危险。
{{< /sentence >}}

一旦反转信号出现，往往已经太晚了。

{{< sentence >}}
在牛市中最值钱的，不是选股的本事，而是识别周期位置的能力。
{{< /sentence >}}

不同位置采取不同策略，才是保护自己的方式。

## 关于牛市中的高收益

{{< sentence >}}
高收益往往意味着高风险被掩盖了，而不是风险消失了。
{{< /sentence >}}

盈利掩盖的风险，往往在周期反转时集中爆发。

{{< sentence >}}
所有人都赚钱的时候，要特别小心，说明市场可能已经定价过度了。
{{< /sentence >}}

全民狂欢往往就是泡沫的标志。

## 关于风险识别

{{< sentence >}}
风险在熊市中才显露，但它其实在牛市中就已经积累了。
{{< /sentence >}}

看不到风险不等于风险不存在。

{{< sentence >}}
牛市中最不值钱的，就是安全边际。
{{< /sentence >}}

人们都在追求收益，忘记了为风险留出空间。
