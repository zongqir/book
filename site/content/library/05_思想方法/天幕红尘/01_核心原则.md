# 见路不走——从条件出发的判断纪律

大多数判断失误不是因为智力不够，是因为锚点选错了。看到一条被验证过的路，默认就该走——这个假设本身才是问题。叶子农反复演示的五条原则，拆的就是这个假设和它背后的认知惯性。

{{< principle no="1" type="core" statement="判断的锚点是条件，不是路径" >}}

路径是别人在别的条件下走出来的结果。条件变了，路径就失效了。但人的本能是看到路就想走，因为走路比判断条件省力。

叶子农判断苏联会解体，不是因为他有政治倾向，是因为经济结构、民族矛盾、制度刚性这些条件摆在那里，指向一个不可逆的结论。他没有参考任何人的预测，只看条件本身。

这条原则的操作含义是：每次做判断之前，先问"我在看条件还是在找路"。如果答案是后者，停下来。
{{< /principle >}}

{{< principle no="2" type="core" statement="实事求是是能力，不是态度" >}}

所有人都说自己实事求是。但"事"和"是"之间隔着一整套认知纪律：你得能把预设放下，把情绪摘掉，把已经投入的沉没成本忘掉，只看眼前的条件指向什么。

叶子农能做到这一点，不是因为他比别人聪明，是因为他愿意为此付出代价——包括和所有人不一样的孤立。

实事求是的难度不在"求"，在"实"：你愿不愿意接受条件告诉你的、你不想听的答案。
{{< /principle >}}

{{< principle no="3" type="support" statement="因果是条件的函数，不是意愿的函数" >}}

想要某个结果不等于能得到某个结果。结果由条件决定，意愿只能影响你在条件范围内做什么，不能改变条件本身的指向。

老九想让面馆生意变好，光想没用。叶子农让他看自己的条件——手艺、位置、客群——然后从条件出发找可行的做法。意愿提供了动力，但方向是条件给的。
{{< /principle >}}

{{< principle no="4" type="support" statement="不能用别人的条件推自己的结论" >}}

这条原则挡住的是最常见的判断错误：看到别人成功了，复制他的做法。问题是他的条件和你的条件不一样。他的位置、资源、时机、竞争环境，没有一样是你能直接搬过来的。

见路不走不是反对学习。它反对的是不检查条件就照搬。学习的正确姿势是：看懂别人为什么在那些条件下选了那条路，然后回到自己的条件，重新判断。
{{< /principle >}}

{{< principle no="5" type="support" statement="认知的上限不是智力，是你愿不愿意放下已有判断" >}}

叶子农不是小说里最聪明的人。但他是唯一一个在判断被事实否定时，愿意立刻放下旧判断的人。

大部分人做不到这一点——不是因为看不见新条件，是因为放不下旧结论。沉没成本、面子、习惯、安全感——这些东西把旧判断焊死在脑子里。

见路不走要求你随时可以拆掉它们。这不是智力问题，是纪律问题。
{{< /principle >}}
