{{< sentence >}}
投降不是一个时刻而是一个过程。放下武器只是第一步，放下信仰需要更长的时间——有些人终生没有完成。
{{< /sentence >}}

军事投降可以在一天内完成。心理投降需要数年。价值体系的重建需要一代人。

{{< sentence >}}
黑市是战后日本最真实的经济。官方的配给体系养不活人，只有黑市能提供生存必需品。法律和生存之间，人们选择了生存。
{{< /sentence >}}

当正式制度无法满足基本需求时，非正式制度会自动填补空白。黑市不是道德问题而是制度失效的症状。

{{< sentence >}}
美国人用民主的名义实施了审查——禁止批评占领军的言论。用不民主的方式推行民主，这个矛盾从来没有被解决。
{{< /sentence >}}

外部强制的改革天然携带这个矛盾：改革的内容是自由的，但改革的方式是强制的。接受改革的人心里清楚这个矛盾。

{{< sentence >}}
天皇的"人间宣言"让整个国家的信仰体系在一夜之间失效。当最高权威说"我不是神"的时候，建立在他神性上的所有制度和价值都需要重新找理由。
{{< /sentence >}}

权威的自我否定比外部批评更具破坏力——也更具建设性。旧秩序最彻底的终结者往往是旧秩序自己。

{{< sentence >}}
日本人没有简单地变成美国人想要他们变成的样子。他们接受了新制度的外壳，但在里面填充了自己的内容。
{{< /sentence >}}

被强加的制度会被接受者重新解释。最终运转的制度既不是施加者设计的样子也不是接受者原有的样子——而是两者碰撞后的混合产物。
