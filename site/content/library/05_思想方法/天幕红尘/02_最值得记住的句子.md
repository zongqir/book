# 四个字拆掉路径依赖

{{< sentence quote="见路不走。" >}}
{{< /sentence >}}

四个字，没有修饰。它说的不是"不走路"，是"不因为路在那里就走"。判断的依据是条件，不是路径的存在。每次准备复制一个被验证过的做法时，这四个字值得默念一遍。

{{< sentence quote="不是我有什么能耐，是条件在那摆着。" >}}
{{< /sentence >}}

把功劳从个人能力上卸下来，放回条件。叶子农预判苏联解体不是因为他比苏联问题专家更懂政治，是因为他看的是经济结构和制度刚性这些不以人的意志为转移的条件。这句话最该在你得意于自己"看得准"的时候想起来——大概率不是你准，是条件明显。

{{< sentence quote="实事求是不是一句空话，是一种能力。" >}}
{{< /sentence >}}

所有人都说自己实事求是。区别在于，大部分人说的"实事求是"是"用事实支持我已有的判断"，叶子农说的是"让事实替我做判断"。前者是选择性使用证据，后者是放弃对结论的控制权。能力差距就在这里。

{{< sentence quote="你不知道自己是按照愿望在判断还是按照条件在判断。" >}}
{{< /sentence >}}

最危险的认知状态不是判断错了，是不知道自己的判断锚点在哪。愿望和条件混在一起时，你以为自己在分析，其实在自我说服。这句话适合在做完一个让自己满意的判断后拿出来用——满意本身就是警报信号。

{{< sentence quote="条件的可能产生了结果的必然。" >}}
{{< /sentence >}}

因果关系不是线性的，是条件驱动的。同一个行动，条件不同，结果不同。这句话拆掉的是"做了 A 就会得到 B"这种线性因果思维。每次觉得"只要这样做就一定行"的时候，问一句：条件支持吗？

{{< sentence quote="生存法则很简单，忍人所不忍，能人所不能。" >}}
{{< /sentence >}}

不是励志口号。它描述的是一个结构性事实：条件论要求你做大多数人不愿意做的事——放下已有判断、接受不舒服的结论、在所有人觉得你错的时候继续按条件行事。做到这些的门槛不是智力，是忍耐力。

{{< sentence quote="想，不是你的权力。实事求是，是你的权力。" >}}
{{< /sentence >}}

"想"在这里指的是基于愿望的推演——你想要什么结果，就往那个方向推。叶子农说这不算你的权力，因为推演结果不由你决定。你能决定的只有一件事：是不是老老实实从条件出发。

这句话把"自由意志"的范围画得非常小，但也非常实在。

## 调用场景

这七句话覆盖三类判断时刻：

准备复制别人做法的时候——"见路不走"和"不能用别人的条件推自己的结论"拦住你。

对自己的判断感到满意的时候——"你不知道自己是按照愿望在判断还是按照条件在判断"给你降温。

觉得"只要这样做就一定行"的时候——"条件的可能产生了结果的必然"提醒你查一遍条件。
