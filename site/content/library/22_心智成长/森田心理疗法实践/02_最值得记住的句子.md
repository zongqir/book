# 带着症状生活，比消灭症状更有效

## 焦虑循环的拆解开关

{{< sentence >}}
顺其自然，为所当为。
{{< /sentence >}}

八个字压住了整套操作指令。焦虑来了不推开，该干什么干什么。它不要求你变勇敢，不要求你想通——只要求你别停下手上的事。

{{< sentence >}}
症状固着的原因不是症状本身，而是对症状的注意和抵抗。
{{< /sentence >}}

精神交互作用的一句话版本。你之所以一直焦虑，不是焦虑太强，是你的注意力一直盯着它。每一次"我要控制它"的念头，都在给循环续电。

{{< sentence >}}
不安常在。
{{< /sentence >}}

人永远不会完全没有不安。追求"彻底平静"本身就是新的焦虑来源。承认不安是生活的底色，它反而退到了背景里。

## 行动和情绪的真实关系

{{< sentence >}}
行动的方向由目的决定，不由情绪决定。
{{< /sentence >}}

"目的本位"是森田疗法的操作原则。判断今天该做什么的标准是目的，不是"我现在心情怎么样"。情绪只是天气，目的才是方向。

{{< sentence >}}
恐惧和欲望是同一枚硬币的两面。
{{< /sentence >}}

你怕社交出丑，是因为在乎别人的看法。怕失败，是因为想成功。恐惧的强度反映欲望的强度。消灭恐惧等于消灭动力——这笔账算不过来。

{{< sentence >}}
情绪是自然现象，会自己来，也会自己走。
{{< /sentence >}}

情绪的规律和天气一样：你不能命令它晴天，但你也不需要。等一等，它自己会变。你要做的只是别在下雨的时候停下所有事情。

## 对抗的代价

{{< sentence >}}
越想睡着，越睡不着。
{{< /sentence >}}

精神交互作用最直观的版本。你盯着"我有没有睡着"这个问题，大脑就保持清醒来回答你。放弃监控，睡眠自己回来。这个道理可以平移到几乎所有焦虑症状上。

{{< sentence >}}
烦恼是活着的证据。
{{< /sentence >}}

烦恼消失的那天，生活也停了。森田疗法不承诺无忧无虑，它承诺的是另一件事——带着烦恼，生活照样能过得扎实。
