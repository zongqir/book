# 思考的框架：最值得记住的句子

## 模型组合与决策校准

{{< sentence >}}你思考的质量，取决于你脑子里装着的模型。{{< /sentence >}}

整句话的力量在"取决于"三个字。不是"受影响"，不是"有关系"——是直接决定。模型少，判断就窄；模型多但不会组合，判断就乱。帕里什要你先承认一件事：你的判断力上限，被你的模型库存卡着。

{{< sentence >}}两个模型给出矛盾答案的时候，才是真正需要思考的时候。{{< /sentence >}}

多数人遇到矛盾会选自己更熟悉的那个结论。帕里什说反了——矛盾暴露的是盲区，是你平时看不到的信息终于浮出水面。下次两个分析框架打架，别急着挑一个赢家，先搞清楚它们为什么打架。

{{< sentence >}}地图不是疆域。{{< /sentence >}}

阿尔弗雷德·科日布斯基的原话，帕里什把它放在全书最前面。任何模型都是对现实的简化。你依赖的那张地图越精确，你就越容易忘记它省略了什么。遇到模型预测失败时，先查地图漏了什么，别急着怀疑现实出了问题。

## 反转与二阶思考

{{< sentence >}}反过来想，永远反过来想。{{< /sentence >}}

这句来自查理·芒格，帕里什把它放进核心工具箱。多数人习惯正向推演"怎么才能成功"；反转思维先问"怎样一定会失败"，再逐条排除。不是悲观，是用排除法逼出更稳的方案。在你觉得计划完美的时候，拿这句话出来用一次。

{{< sentence >}}一阶思考看到结果，二阶思考看到结果的结果。{{< /sentence >}}

涨价能增加短期收入——这是一阶。涨价导致老客户流失、竞争对手趁虚而入——这是二阶。帕里什反复强调：一阶思考人人都会做，区别在于谁愿意多追一步。绝大部分决策灾难，都能在二阶就被提前看到。

## 能力圈与概率判断

{{< sentence >}}能力圈最重要的边界不是你擅长什么，是你知道自己不擅长什么。{{< /sentence >}}

沃伦·巴菲特在投资上反复讲这个道理，帕里什把它扩展到所有决策场景。一个人最贵的错误，通常发生在他以为自己懂但其实不懂的地方。画出能力圈不是为了自我限制——是为了在圈外保持警觉。

{{< sentence >}}概率思维的敌人不是不确定性，是假装确定。{{< /sentence >}}

面对复杂决策时说"我觉得大概率会怎样"，比说"一定会怎样"看起来没底气。但帕里什指出：真正危险的是后者。假装确定会让你跳过对冲、忽略替代方案、拒绝更新判断。承认不确定，反而能做出更稳的决策。

{{< sentence >}}最危险的不是错误答案，是错误的问题。{{< /sentence >}}

你花三天分析一个方案的可行性，最后发现方案本身就不该被提出来。帕里什把"先确认问题对不对"放在所有模型之前。用对了模型解错了问题，比用错模型更浪费时间。做任何深度分析之前，先花五分钟确认你在回答正确的问题。

## 场景速查

决策前拍板犹豫 → 拉出"两个模型矛盾才该思考"，检查是不是只用了一个角度。

计划看起来完美 → 用反转思维问"怎样会失败"。

做出非常确定的判断 → 用"假装确定是最大敌人"提醒自己补一个反面假设。

进入不熟悉的领域 → 用能力圈边界检查自己是不是在圈外冒充圈内。

分析做了很久却没有进展 → 先用"最危险的是错误问题"确认方向。
