## 关于小说的底线标准

{{< sentence >}}
小说在读者脑中制造的是一场生动的、连续的梦——作者的全部工作就是维持这场梦不被打断。
{{< /sentence >}}

加德纳最核心的判断。一旦接受这个标准，你写的每一个句子都有了明确的检验方式：读者读到这里时，梦还在不在？如果你自己朗读时感到"这里卡了一下"，大概率梦断了。

{{< sentence >}}
过度修饰是自恋的症状——读者被弹出故事，开始注意你的文笔而不是人物的处境。
{{< /sentence >}}

初学者最常见的错觉是"写得漂亮等于写得好"。加德纳的判断正好相反：让读者注意到文笔本身，说明文笔在干扰梦境。这句话能在你写出自我感觉特别好的段落时提个醒。

## 关于人物和道德

{{< sentence >}}
道德小说不是宣讲正确观点的小说，而是认真对待人类处境、不回避复杂性的小说。
{{< /sentence >}}

消除了"道德小说=说教"的误解。你可以写任何黑暗的主题，只要你不是在利用人物来兜售某个预设结论。在你发现笔下人物变成了某个观点的传声筒时，想想这句话。

{{< sentence >}}
人物不是作者的棋子。当你让人物做出不符合他自身逻辑的事情，只是为了推进情节，读者会立刻感觉到假。
{{< /sentence >}}

情节和人物的优先级之争，加德纳的答案是人物优先。你规划好了一个情节转折，但人物到了那个位置不可能做出那个选择——这时候应该改情节，不是硬让人物服从。

## 关于写作过程

{{< sentence >}}
每一种技术错误都是梦断裂的症状：视角混乱、节奏失控、细节失真、时间线跳跃——它们的共同后果是读者被弹出故事。
{{< /sentence >}}

把分散的"写作禁忌"统一到一个判断框架下。你不需要记住所有规则，只需要不断问：读者的梦在这里还连着吗？

{{< sentence >}}
作者的思考过程不应该出现在正文里。当你在叙述中解释'我为什么这样写'或者'读者应该注意到'，你已经跳出了故事在做旁白。
{{< /sentence >}}

对元叙述的彻底否定。如果你需要在故事中告诉读者"这里很重要"，说明叙事本身没有把重要性传达出来。改叙事，不加注释。

{{< sentence >}}
才华不能替代练习。最有天赋的写作者如果不反复修改，成品也不会比勤奋的中等才能者好多少。
{{< /sentence >}}

加德纳对天才论的拒绝。修改不是对才华不足的补偿，修改本身就是写作的核心过程。如果你觉得初稿就应该很好，需要重新理解写作是怎么回事。
