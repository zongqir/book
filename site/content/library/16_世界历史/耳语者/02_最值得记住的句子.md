{{< sentence >}}
在斯大林时代，沉默不是一种选择而是一种生存技能。学会不说话的人活下来了——但他们也丧失了说真话的能力，这种能力在恐惧消退之后很长时间都没有恢复。
{{< /sentence >}}

沉默是可以被训练的。一旦沉默变成习惯，它会在恐惧消失之后继续存在——几代人之后还在影响一个社会表达真实想法的能力。

{{< sentence >}}
孩子们在学校被教育要爱斯大林胜过爱父母。有些孩子真的揭发了自己的父母——不是因为他们邪恶而是因为他们太年幼，分不清忠诚和背叛。
{{< /sentence >}}

当国家要求孩子对它的忠诚超过对家庭的忠诚时，家庭作为社会基本单元就已经被击碎了。

{{< sentence >}}
很多人在被捕之前就已经在精神上准备好了。他们收拾了一个小包裹放在门口，每天晚上等着敲门声。恐惧不需要真的被逮捕就能完成它的工作。
{{< /sentence >}}

恐惧的效率远高于实际的暴力——一次逮捕可以让整栋楼安静十年。极权体制依赖的不是无限的暴力资源而是恐惧的放大效应。

{{< sentence >}}
被平反的人回到家发现没有人愿意听他们的故事。邻居不想知道——因为知道了就意味着自己当年的沉默是有罪的。遗忘是一种集体的自我保护。
{{< /sentence >}}

集体遗忘不是因为人们健忘而是因为记忆对旁观者来说太沉重。承认幸存者的经历等于承认自己的冷漠。

{{< sentence >}}
有些家庭在三代人之后才第一次谈论祖辈发生过什么。沉默可以穿透血缘传递下去——孩子不知道为什么家里从来不谈论某些话题，但他们继承了这种禁忌。
{{< /sentence >}}

创伤的代际传递不需要语言——沉默本身就是一种传递机制。
