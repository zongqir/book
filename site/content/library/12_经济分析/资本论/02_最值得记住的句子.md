## 拆开利润的来源

{{< sentence >}}
劳动力的价值和劳动力在劳动过程中创造的价值，是两个不同的量。
{{< /sentence >}}

整部《资本论》最底层的一句。工资支付的是"劳动力的再生产成本"——让工人明天还能来上班的费用。工人在工厂里创造的价值远大于这个成本。差额就是利润的来源。搞清这一句，才能理解后面所有关于剥削的讨论。

{{< sentence >}}
资本来到世间，从头到脚，每个毛孔都滴着血和肮脏的东西。
{{< /sentence >}}

不是修辞渲染。马克思指的是"原始积累"——资本主义起步阶段，资本的来源不是什么勤劳致富，而是暴力圈地、殖民掠夺、奴隶贸易。讨论"合法财产权"时，这句提醒你追问一层：最初那笔原始资本是怎么来的？

## 拆开商品的面纱

{{< sentence >}}
商品拜物教使人与人之间的社会关系，采取了物与物之间关系的虚幻形式。
{{< /sentence >}}

你在超市里看到的是商品和价格标签。你看不到的是：这件商品背后有多少人劳动了多少小时，在什么条件下。价格把所有这些关系都压缩成一个数字，让你误以为自己只是在和东西打交道。实际上你在和一整套生产关系打交道。

{{< sentence >}}
一切商品对它们的占有者是非使用价值，对它们的非占有者是使用价值。
{{< /sentence >}}

生产者不是为了自己使用而生产——他生产是为了交换。卖家手里的商品对卖家没有使用价值，只有交换价值。买家的钱对买家来说只有在换成商品后才有意义。这句话说的是：市场经济里，每个人都在为别人生产，但不是因为利他，而是因为交换。

## 拆开积累的逻辑

{{< sentence >}}
资本的积累就是无产阶级的增长。
{{< /sentence >}}

资本规模越大，雇佣的绝对人数可能增加，但劳动在总产出中的份额在下降。机器替代劳动，剩下的工人变得更依赖资本，失去独立生存的能力。财富在一极积累，贫困在另一极积累。

讨论"经济增长让所有人受益"时，先检查：增长的收益到底分给了谁？

{{< sentence >}}
竞争迫使每个资本家不断扩大生产，而扩大生产的唯一手段就是进行积累。
{{< /sentence >}}

资本家不积累不是因为贪婪，是因为不积累就会被竞争淘汰。这是系统压力，不是道德选择。场景：看到企业疯狂扩张、烧钱圈地时，不要只归因于"老板野心大"——竞争结构本身在逼迫每一个参与者加速。

{{< sentence >}}
工人出卖的不是劳动，而是劳动力。
{{< /sentence >}}

劳动是一个过程，不是一种商品。工人出卖的是他在一定时间内工作的能力。这个区分不是文字游戏——它解释了为什么工人即便自愿签约，在结构上仍然处于不利位置：他卖的是能力，但能力创造的价值被别人拿走了。

## 场景映射

- 讨论工资是否公平：调用"劳动力的价值和劳动创造的价值是两个量"
- 看到企业利润大增但员工待遇没涨：调用"积累就是无产阶级的增长"
- 分析平台经济为什么压低骑手收入：调用"工人出卖的不是劳动，而是劳动力"
- 讨论消费主义和商品崇拜：调用"商品拜物教"
- 审视财富的历史来源：调用"资本来到世间滴着血和肮脏的东西"
