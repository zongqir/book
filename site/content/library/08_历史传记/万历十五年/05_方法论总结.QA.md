# 为什么不写方法论总结

## 当前为什么不留

黄仁宇的"大历史观"更接近一种观察视角，而不是一套可以独立提取、独立传授的方法论。

方法论总结的前提是：存在一组可复制的步骤或框架，读者学会之后能在自己的场景里独立操作。"大历史观"不满足这个条件。

"大历史观"的核心是：从长时段看制度演变，关注技术管理能力而非道德判断。这更像一种观察习惯或分析偏好，而不是一套有明确步骤的方法。

## 主要证据缺口

黄仁宇在《万历十五年》中使用了"大历史观"，但没有系统性地教读者如何运用它。

他在其他著作（如《中国大历史》《赫逊河畔谈中国历史》）中有更多的方法论阐述，但那些内容不属于《万历十五年》本身。如果硬写方法论总结，就必须大量引入其他文本，偏离了"基于这本书"的原则。

在本书范围内，能提取的"方法"有限：

- 选一个时间切面做剖析
- 用人物平行叙事替代时间线叙事
- 关注制度运行而非个人功过
- 从技术管理角度评估制度

这些更接近写作技法和分析习惯，而不是可传授的方法论。

## 硬写最容易写假在哪里

最大的风险是把"大历史观"包装成一套操作流程。

"第一步选时间切面，第二步找代表人物，第三步分析制度缺陷"——这样的流程看似清晰，但黄仁宇的分析能力来自他几十年的史学训练和独特的问题意识。把结果倒推成流程，是在制造方法论幻觉。

另一个风险是把观察习惯和正式方法论混为一谈。"从长时段看问题"是一种视角，不是一种方法。教人"要从长时段看"，不等于教会了人怎么看。

## 什么条件下值得重开

如果后续获得以下材料或完成以下工作，可以考虑重开：

1. 系统阅读黄仁宇其他著作，提取出"大历史观"的完整方法论陈述
2. 找到黄仁宇自述研究方法的访谈或论文
3. 完成"大历史观与其他史学方法的比较"专题，有足够的材料支撑方法论提取
4. 用户明确要求把"大历史观"作为一个独立方法论来总结，且接受这需要跨越本书范围

在此之前，08_论证链已经把黄仁宇在本书中的论证方式拆得足够清楚。读者如果想学他怎么分析问题，论证链比一篇方法论总结更诚实。
