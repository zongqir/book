# 看公司时，先把这几句钉在桌上

## 最该反复调用的提醒

{{< sentence no="1" quote="先问这家公司怎么赚钱，再问它值多少钱。" >}}
估值不能替代商业理解。

连钱从哪里来、为什么能继续来都说不清，数字再便宜也很难真便宜。
{{< /sentence >}}

{{< sentence no="2" quote="股票不是代码，它是一门生意的一部分。" >}}
只盯股价，最容易把波动当机会。

把对象换成企业以后，问题会自动变成行业、客户、竞争和资本回报。
{{< /sentence >}}

{{< sentence no="3" quote="高增长不等于高质量，能留住利润才更要紧。" >}}
很多亮眼公司最后输在竞争太容易跟进。

护城河不够深，增长越快，后来回落时反而越痛。
{{< /sentence >}}

{{< sentence no="4" quote="好公司也可能是坏投资，只要你买得太贵。" >}}
企业质量和买入价格是两回事。

前者决定能不能长期赚钱，后者决定你能不能把这份赚钱能力拿到手里。
{{< /sentence >}}

{{< sentence no="5" quote="护城河不是故事，它要落在利润不容易被抢走这件事上。" >}}
品牌、规模、网络效应这些词都可能被滥用。

最后还是要回到一个问题：竞争对手进来以后，利润会不会被打穿。
{{< /sentence >}}

{{< sentence no="6" quote="管理层最值得看的，不是讲话，而是钱怎么分。" >}}
回购、分红、并购、扩产、举债，每一笔资本配置都在暴露判断。

会讲故事，不等于会替股东做长期有利的决定。
{{< /sentence >}}

{{< sentence no="7" quote="看不懂，不是研究不够深，很多时候就是不该碰。" >}}
投资里最贵的冲动之一，就是不懂也要参与。

承认边界，往往比多抓一个机会更能保护回报。
{{< /sentence >}}

{{< sentence no="8" quote="估值不是找精确答案，而是判断自己有没有留错的余地。" >}}
模型算得再细，也盖不住未来本来就会变。

安全边际的作用，不是让你显得谨慎，而是替你承认不确定性。
{{< /sentence >}}

{{< sentence no="9" quote="便宜股最先要问的，不是有多便宜，而是为什么会便宜。" >}}
低估值有时是机会，有时是问题已经写在价格里。

如果原因来自生意恶化、行业变差或资产质量虚，便宜本身就不是理由。
{{< /sentence >}}

## 这些句子最适合在哪些场景拿出来

- 看到热门赛道和高增长故事，很想先上车时，先回到第 1、3、5 句。
- 看到低 PE、低 PB，就觉得捡到便宜时，先回到第 4、8、9 句。
- 管理层叙事很强，资本配置却越来越激进时，先回到第 6 句。
- 遇到结构复杂、自己其实没搞懂的公司时，先回到第 7 句。
- 建模以后越来越自信时，先回到第 8 句。

## 真正的用法不是背下来，而是拿它们卡住冲动

看公司最怕一路顺着自己的乐观往前走。

更稳的做法，是每研究完一家公司就回来问几句：钱怎么赚，利润能不能守住，管理层怎么花钱，便宜到底便宜在哪，自己是不是真看懂了。只要这组问题开始变成习惯，投资动作会慢下来，失误也通常会跟着少一些。
