{{< sentence >}}联合政府的提出，与其说是制度设计，不如说是政治策略{{< /sentence >}}

双方都知道"联合政府"听起来很美好，但各自盘算的是如何在这个框架下最大化自己的利益。制度只是工具，权力分配才是核心。

{{< sentence >}}国民党的联合政府方案，本质上是在野党参政而非分享权力{{< /sentence >}}

蒋介石愿意给共产党一些部长位置，但军队指挥权、财政控制权这些关键权力绝不松手。这种"参政"更像是政治装饰。

{{< sentence >}}共产党接受联合政府谈判，是希望通过合法途径获得战略缓冲空间{{< /sentence >}}

毛泽东很清楚军事准备和政治谈判要同时进行。谈判桌上争取到的每一点合法性，都能为后续的军事行动创造更好的环境。

{{< sentence >}}双方的分歧不在于要不要民主，而在于谁来定义民主{{< /sentence >}}

国民党说的民主是在现有框架内的有限开放，共产党要的民主是对现有权力结构的根本改造。同一个词汇背后是完全不同的政治图景。

{{< sentence >}}当政治谈判无法解决根本分歧时，军事准备就成了最终选项{{< /sentence >}}

从1945年下半年开始，双方一边谈判一边调兵遣将。政治解决的可能性越小，军事解决的准备就越充分。

{{< sentence >}}联合政府的失败证明了权力无法通过纯粹的制度设计来分配{{< /sentence >}}

任何政治安排最终都要落到具体的权力关系上。没有实力基础的制度承诺，在关键时刻都不可靠。

{{< sentence >}}历史的吊诡在于，双方都低估了对方的决心和能力{{< /sentence >}}

国民党以为共产党最终会接受次要地位，共产党以为国民党在军事和政治压力下会做出实质性让步。这种误判加速了谈判的破裂。
