# 销售中最值得记住的几句话

## 销售的本质不是说服，而是解决

{{< sentence >}}
最有效的销售，往往来自于客户的信任而非技巧。
{{< /sentence >}}

客户能感知出来你是在帮他还是在骗他，这感知是最强的销售工具。

{{< sentence >}}
销售不是说服别人要什么，而是帮别人看清他们真正需要什么。
{{< /sentence >}}

这个转变一旦发生，整个销售过程都会改变。

{{< sentence >}}
短期成交和长期关系往往是对立的，真正的赢家选择长期关系。
{{< /sentence >}}

快速成交很容易，难的是建立客户对你的长期信任。

## 关于客户理解的关键洞察

{{< sentence >}}
你对客户问题的理解深度，决定了销售的说服力。
{{< /sentence >}}

客户最尊敬的销售人员，往往是那些比他更懂他的问题的人。

{{< sentence >}}
客户不买产品，客户买的是对他问题的解决方案。
{{< /sentence >}}

同样的产品在不同客户手里有不同价值，关键是怎样对应他的问题。

{{< sentence >}}
听比说更重要，理解比推销更重要。
{{< /sentence >}}

很多销售人员话太多，听得太少，这是客户不信任的主要原因。

## 销售关系的建立原则

{{< sentence >}}
诚实的销售人员，往往卖得最多。
{{< /sentence >}}

因为他们建立的不是一次性成交关系，而是长期信任关系。

{{< sentence >}}
有时候最好的销售决定是不卖。
{{< /sentence >}}

客户不需要你的产品时告诉他不需要，这样他才会在真正需要时相信你。

{{< sentence >}}
销售的革命，就是从竞争关系变成协作关系。
{{< /sentence >}}

客户和销售不再是对立的，而是在一起解决问题。
