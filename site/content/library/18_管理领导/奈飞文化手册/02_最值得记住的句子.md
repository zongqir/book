{{< sentence >}}我们要的不是快乐的员工，要的是高绩效的员工。{{< /sentence >}}
这句话击中了所有"以人为本"管理理念的软肋。麦考德发现，追求员工满意度和追求业务结果之间经常是冲突的。真正的责任是帮助员工在工作中获得成就感，而不是让他们在工作中感到舒适。高绩效本身就是最大的激励。

{{< sentence >}}如果你不能开除某个人，你就不应该雇用他。{{< /sentence >}}
这是对雇佣决策最直接的检验标准。大部分组织的问题不是招人太慢，而是解雇太慢。每一个雇佣决策的背后，都应该有一个清晰的标准：什么情况下这个人必须离开。这个标准比任何职位描述都重要。

{{< sentence >}}我们不是一个家庭，我们是一支球队。{{< /sentence >}}
家庭成员是无条件接纳的，球队成员是基于贡献选择的。这个比喻区分了情感管理和绩效管理的边界。球队文化意味着每个位置都有明确的职责，每个人都要为结果负责，不合适的人会被替换，而不是被"教育"或"培养"。

{{< sentence >}}最大的不忠诚是让一个不胜任的人继续呆在不适合的岗位上。{{< /sentence >}}
这颠覆了传统的忠诚度概念。真正的忠诚是对组织目标的忠诚，不是对个人关系的忠诚。让不胜任的人留在岗位上，既耽误了他的职业发展，也拖累了整个团队的效率。及时的人员调整是对所有人负责的表现。

{{< sentence >}}绝对诚实不是残忍，而是最大的善意。{{< /sentence >}}
诚实反馈是帮助人成长的唯一方式。那些被包装过的、委婉的反馈看起来很体贴，实际上剥夺了员工改进的机会。成年人有能力处理真实的评价，他们需要的是准确的信息，不是情感安慰。

{{< sentence >}}按市场最高价付薪，其他激励都是多余的。{{< /sentence >}}
当薪酬足够有竞争力时，员工会为了保持这个薪酬水平而努力工作。复杂的激励制度往往在解决薪酬不够导致的问题。与其设计各种激励方案，不如直接给到市场最高价，让员工为了维持这个待遇而自我激励。

{{< sentence >}}每个员工都应该知道如何让公司成功，以及自己在其中的作用。{{< /sentence >}}
透明的信息共享是高绩效文化的基础。员工不是执行命令的机器，是需要理解全局的决策参与者。当每个人都清楚公司的战略目标和自己的贡献方式时，就不需要复杂的管理流程来协调行动。

这些句子的威力在于它们不是管理技巧，而是管理哲学的根本转换。麦考德用奈飞的成功证明了这些看似"无情"的原则实际上是对所有人最大的尊重：把每个人当作有判断力、有责任心的成年人来对待。
