## 关于判断力的陷阱

{{< sentence >}}
在群体中，即使你坚持自己的判断，你的想法往往已经被群体无形地塑造过了。
{{< /sentence >}}

真正独立的判断很难。大多数人以为的"我的想法"，其实是"群体允许我有的想法"。

{{< sentence >}}
当群体的判断和你的判断冲突时，相信群体的人会比相信自己的人多。
{{< /sentence >}}

## 关于安全感的陷阱

{{< sentence >}}
人在群体中，宁可放弃做自己，也要换取安全感和归属感。
{{< /sentence >}}

这不是懦弱，是人的本性。

{{< sentence >}}
一个群体对你的接纳程度，往往和你有多"像"它有关。
{{< /sentence >}}

## 关于身份的陷阱

{{< sentence >}}
一旦你接受了一个身份（比如"这个公司的员工""这个宗教的信徒"），你就开始用这个身份来解释自己的一切行为。
{{< /sentence >}}

身份既给了你归属感，也限制了你的想象空间。

{{< sentence >}}
群体会不断地强化你的身份认同。最后你分不清，哪些是真的你，哪些是被强化的身份。
{{< /sentence >}}
