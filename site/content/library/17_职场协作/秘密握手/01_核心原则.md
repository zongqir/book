# 非正式权力网络运作的五条判断准则

多数人在组织里碰壁时，第一反应是"我能力不够"或"公司不公平"。里尔登的研究指向第三种可能：你在用错误的规则体系理解自己所处的环境。下面五条准则纠正的就是这类识别错误。前两条锁定基本框架，后三条处理具体场景里的判断。

{{< principle no="1" type="core" statement="每个组织都有两套规则，非正式规则才决定资源分配和晋升结果" >}}

正式规则写在员工手册和流程制度里。非正式规则从来不落纸面，但每个资深成员都心知肚明。谁的意见在会前已经锁定了方向？谁的反对必须提前化解？哪些议题上会只是走程序？这些信息构成组织运行的实际操作系统。只看正式规则做判断的人，相当于用旅游地图在丛林里导航。

常见误读是把非正式规则等同于"潜规则"或"暗箱操作"。里尔登的区分更精确：非正式规则是组织行为学的研究对象。它们有规律可循，可以被观察和学习，和腐败是两件事。

{{< /principle >}}

{{< principle no="2" type="core" statement="政治智商是可训练的识别-判断-行动能力，不是天生的性格特质" >}}

里尔登用"政治智商"指一组可拆解的能力：识别组织中的非正式权力结构、判断当前场景适用哪种行动方式、在行动后读取反馈并调整。这组能力不依赖外向或能言善道。内向的人、技术出身的人同样可以具备高政治智商。关键区别不在性格，在于你是否主动观察和练习。

把政治智商当天赋的危害在于：它给了所有人一个放弃学习的借口。"我不擅长这个"变成了终身免修证。

{{< /principle >}}

{{< principle no="3" type="support" statement="政治风格决定判断盲区——知道自己的默认风格，就知道自己最可能在哪翻车" >}}

四种政治风格——纯粹主义者、团队合作者、街头斗士、策略家——不是性格分类，是行为倾向的描述。

纯粹主义者在需要妥协的场景里容易僵住。团队合作者在需要坚持的场景里容易退让。街头斗士在需要耐心的场景里容易冲动。策略家如果切换太频繁，容易被周围人判定为不可信赖。

用法不是"变成策略家"，是"知道盲区在哪，提前设防"。

{{< /principle >}}

{{< principle no="4" type="support" statement="进入核心圈靠信号交换——业绩是资格，入场凭的是另一套识别系统" >}}

里尔登观察到一个稳定的模式：进入组织核心圈的人，不一定业绩最好，但一定最早学会了内圈的信号系统。

信号包括：在合适的时机向合适的人表达支持、在非正式场合建立私人关系、用恰当的方式展示对组织优先级的理解。这些动作看起来微小，加在一起构成一套成员身份识别机制。

被排除在外的人通常不是被针对。他们等着业绩自动说话，但业绩说的是能力，信号说的是归属。

{{< /principle >}}

{{< principle no="5" type="support" statement="印象管理是组织沟通的正式通道，回避它等于放弃一半发言权" >}}

"我不想搞印象管理，我只想靠实力说话"——在里尔登的框架里，这是一个典型误判。

印象管理不是表演。你在会上选择先说什么后说什么、邮件用什么语气、走廊遇到上级时选择聊什么——这些都是印象管理。你不做选择，别人就替你做。结果往往不是你希望的。

里尔登的建议不是"学会演戏"，而是"主动控制信息的流向和节奏"。你不需要说假话，但你需要选择在什么场合说什么真话。

{{< /principle >}}

## 五条准则之间的判断逻辑

第一条确认赛场：你是不是还在只看正式规则。第二条确认能力观：你愿不愿意把政治智商当一项需要练习的技能。通过这两关之后，第三条帮你定位盲区，第四条指出入场路径，第五条提供日常操作通道。

五条的使用顺序就是诊断顺序。跳过前面直接练后面的技巧，效果会打折扣——因为你还没看清赛场在哪。
