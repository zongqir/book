{{< sentence >}}
执行是一种学科，是战略的一个内在组成部分。
{{< /sentence >}}

大多数领导者把执行当成"下面的事"——我负责想清楚方向，你们负责做到。博西迪和查兰说不对。执行不是战略之后的事，而是战略之中的事。一个不考虑执行可能性的战略不是好战略。

{{< sentence >}}
领导者必须亲力亲为做三件事：选人、定战略、管运营。
{{< /sentence >}}

这三件事不能委派。你可以委派日常事务，但人员配置、战略方向和运营细节的最终判断权必须留在领导者手里。因为只有你能把这三个流程串起来——人能不能支撑战略？战略能不能落成运营？运营有没有对的人来执行？

{{< sentence >}}
脱离现实是执行失败的首要原因。
{{< /sentence >}}

领导者在会议室里讨论目标时很兴奋，但他们对一线现实的了解远不如一线员工。你设定的增长目标和市场容量对不上。你的产品路线图和工程团队的能力对不上。战略要基于现实，不是基于愿景。

{{< sentence >}}
人员流程是三大流程中最重要的一个——选错人，什么流程都救不了。
{{< /sentence >}}

你的战略可以调整，运营可以优化，但如果关键位置上坐着不对的人，一切努力都是打折的。人员决策不是HR的事——是领导者最重要的决策。

{{< sentence >}}
跟进是执行力最薄弱的环节。
{{< /sentence >}}

开会定了计划，散会各回各家。三个月后开下一次会发现什么都没变。为什么？因为没有跟进。决策不跟进等于没决策。
