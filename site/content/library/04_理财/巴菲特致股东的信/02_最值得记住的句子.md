# 一看公司就想下单时，先把这几句拿出来

## 先把股票重新看成企业所有权

{{< sentence >}}
买股票前，先问自己愿不愿意买下整个企业。
{{< /sentence >}}

这句最能把研究从走势和情绪拉回到生意本身。

{{< sentence >}}
好生意先看能不能长期赚，别先看明年会不会更热。
{{< /sentence >}}

热度最强的时候，最该回到这一句。

{{< sentence >}}
管理层最硬的作业不是会说，而是会分钱。
{{< /sentence >}}

回购、并购、分红和扩张，都会比口号更快暴露判断。

{{< sentence >}}
好公司和好价格是两回事。
{{< /sentence >}}

喜欢企业的时候更要拿它压住自己。

{{< sentence >}}
集中不是态度，是理解够深后的结果。
{{< /sentence >}}

别把别人能做的仓位表达，偷换成自己也该做。

{{< sentence >}}
市场给机会时要耐心，市场给诱惑时也要耐心。
{{< /sentence >}}

真正的耐心，不只是等便宜，也包括忍住不追。

{{< sentence >}}
不懂的生意，再便宜也不该靠近。
{{< /sentence >}}

能力圈不是保守姿态，而是少犯大错的地板。

{{< sentence >}}
复利最怕频繁把好资产换成热资产。
{{< /sentence >}}

这句最适合放在每次想追新故事的时候。

## 这些时候最该翻回巴菲特

- 看公司时，你会不会先想如果整家买下还愿不愿意持有
- 研究管理层时，你会不会先看资本怎么分配
- 遇到好公司时，你还能不能等价格
- 调仓时，你是不是在用所有者视角，而不是在追热
- 新机会出现时，你会不会主动回到能力圈

真用到位时，你会先问生意、资本配置和价格，最后才问市场今天怎么走。
