{{< sentence >}}
以色列-巴勒斯坦冲突不是一个等待解决的问题——对某些利益方来说，维持现状本身就是"解决方案"。只有理解了谁从现状中获益，才能理解为什么和平进程反复失败。
{{< /sentence >}}

当一个问题长期不被解决时——先问：谁从问题的持续存在中获益？如果有强大的利益方从现状中得到好处，他们不会有动力推动改变。

{{< sentence >}}
中东的独裁者不是自己维持了权力——他们背后有一整套国际支持体系。切断这套体系比推翻一个独裁者更难也更重要。
{{< /sentence >}}

本地问题的根源可能在本地之外。只看本地政治就像只看水面上的冰山。

{{< sentence >}}
"中东需要民主化"这个判断本身没错，但谁来民主化、按谁的标准、以什么速度——这些问题的答案比"要不要民主化"更决定结果。
{{< /sentence >}}

方向正确不等于执行可行。细节不是技术问题而是政治问题——每一个"怎么做"的选择都牵动不同的利益。

{{< sentence >}}
伊拉克战争展示了"出口民主"的悖论：用武力带来的秩序不是民主——它是占领。被占领者眼中的"解放"和占领者眼中的"解放"是完全不同的东西。
{{< /sentence >}}

你认为自己在帮助别人不等于别人认为你在帮助他们。意图不能替代效果的评估。

{{< sentence >}}
媒体呈现的中东是一个由危机和暴力定义的地方。看不见的部分——日常生活、文化生产、普通人的选择——被系统性地遮蔽了。
{{< /sentence >}}

对任何地区或组织的印象如果只来自危机报道，你看到的不是全貌而是偏差。
