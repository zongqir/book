{{< sentence >}}
影响力不是天赋，是策略。成功的行为改变者同时使用了多个来源的力量。
{{< /sentence >}}

有些人看起来特别有影响力，不是因为他们有魅力，而是因为他们——有意识或无意识地——同时激活了多个影响力来源。只用一种策略（只靠奖励、只靠说服）几乎注定失败。

{{< sentence >}}
找到关键行为。不要试图改变一切，找到那一两个能撬动整个系统的行为。
{{< /sentence >}}

行为改变最常见的失败模式是"什么都想改"。格雷尼强调：大多数复杂问题可以归结到两三个关键行为上。找到它们，集中所有力量在它们上面。

{{< sentence >}}
让不想做的事变得想做——这是个人动机问题。让做不到的事变得做得到——这是个人能力问题。两个都要解决。
{{< /sentence >}}

"为什么人们不做你希望他们做的事？"答案永远是两个之一：不想做（动机），或做不到（能力）。只激励不赋能，人会受挫。只赋能不激励，人会敷衍。

{{< sentence >}}
利用同伴压力而不是管理权威来推动改变。
{{< /sentence >}}

一个人对行为改变的最大影响者不是老板，而是身边的同事。如果团队里有两三个人开始用新方式工作，其他人跟上的概率远高于老板发一封邮件要求改变。

{{< sentence >}}
改变环境比改变人更有效。
{{< /sentence >}}

很多行为问题不是人的问题，是环境的问题。把健康食物放在餐厅最显眼的位置，不健康食物的消费就会下降——不需要任何说教。组织行为同理：与其要求人们遵守流程，不如把流程嵌入工具。

{{< sentence >}}
如果你只用两到三个影响力来源，你的成功概率大概是十分之一。用到四个以上，概率跃升到十分之四。
{{< /sentence >}}

这是格雷尼团队的实证发现。行为改变不是线性的——多激活一个来源不是多10%的成功率，而是阶梯式跳升。这就是为什么单一策略总是失败。
