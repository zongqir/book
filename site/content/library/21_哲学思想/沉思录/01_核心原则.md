# 你管得住的只有判断，但这就够了

奥勒留反复在说同一件事：你管得住的只有判断。《沉思录》里的原则不是推演出来的逻辑链条，而是一个人在每天的压力中反复给自己的提醒。同一条原则在不同卷里以不同措辞出现，因为提醒需要重复才有效。

以下七条从这些反复提醒中提取。排列按它们在日常判断中被调用的优先级。

{{< principle no="1" type="core" statement="只关心你能控制的事——判断、意图和行动" >}}
这是《沉思录》所有其他原则的地基。奥勒留把世界分成两半：你能控制的和你不能控制的。你能控制的只有三样：对事物的判断、行动的意图、当下的反应。天气、别人的想法、战争的结果、疾病——都在另一半。

这条原则约束的误判是：把精力花在不可控的事情上，然后因为改变不了而痛苦。

常见的误读是把它当成"什么都不管"。奥勒留管了很多——他治理帝国、指挥战争。他放下的是对结果的执念，守住的是对判断和行动质量的要求。

边界在于：当"我控制不了"变成逃避行动的借口时，这条原则就被误用了。
{{< /principle >}}

{{< principle no="2" type="support" statement="痛苦来自你对事件的判断，不来自事件本身" >}}
同事批评你，你感到愤怒。但愤怒不是批评的必然后果。批评是外在事件，愤怒是你的判断产物——你判断"他在冒犯我"，然后才有了愤怒。

奥勒留用这条原则做实时修正。每次情绪升起时，先问一个问题：这个情绪是事件造成的，还是我的判断造成的？

如果是判断造成的，判断可以修改。

这条原则在面对真实伤害时需要谨慎。身体疼痛是真实的，丧亲之痛是真实的。奥勒留处理的主要是日常情绪放大——把小事判断成大事的倾向。
{{< /principle >}}

{{< principle no="3" type="support" statement="记住你会死——用终局校准当下" >}}
死亡在《沉思录》里出现的频率极高。奥勒留不是在恐惧死亡，而是把它当校准工具。

想到终将一死，很多纠结就松了。你还要为那封邮件生气一整天吗？你还要为三年前的事耿耿于怀吗？

死亡提醒做的事情很具体：帮你在当下分辨轻重。把注意力从琐碎的情绪纠缠中拉回来，集中在真正值得做的事情上。

使用边界：对有死亡焦虑或抑郁倾向的人，这个工具可能加重负担。
{{< /principle >}}

{{< principle no="4" type="support" statement="从宇宙的尺度看，你的烦恼几乎不存在" >}}
奥勒留经常把视角拉到宇宙层面。地球在宇宙中是一个点，你的一生在时间长河中是一瞬，你此刻的烦恼在你一生中只是一个微小的波动。

这个视角切换不是让你觉得自己不重要。它做的事情是：把当前困境从"天大的事"缩放到"有限的事"。

当你被一个问题困住时，试着把镜头拉远：一年后这件事还重要吗？十年后呢？一百年后有谁记得？

这种缩放能让你更冷静地判断眼前的困境到底值得投入多少精力。
{{< /principle >}}

{{< principle no="5" type="support" statement="你只拥有当下这一刻" >}}
过去已经发生，不可更改。未来还没来，不可预知。你能触碰的只有此刻。

奥勒留用这条原则对抗两种倾向：对过去的懊悔和对未来的焦虑。两者都在消耗当下的判断力。

操作方式很简单：注意力回到手上正在做的事。不是"活在当下"的口号，是"你的判断只能作用于此刻"的事实。

当下专注是控制二分法的时间维度延伸。你能控制的只有判断，而你能判断的只有现在。
{{< /principle >}}

{{< principle no="6" type="support" statement="你是理性存在者，对共同体有责任" >}}
奥勒留从不把自省当作退缩的理由。他反复提醒自己：人是社会性的理性存在者，为共同体服务是天性的一部分。

这条原则约束的误判是：把斯多葛式内在自由理解成"只管自己"。

奥勒留在前线指挥战争的同时写下这些反省。他的自省从来不是为了逃离职责，而是为了更好地承担职责。管好自己的判断，是为了在公共事务中做出更好的决定。

当内省变成逃避行动的舒适区时，这条原则就被忘掉了。
{{< /principle >}}

{{< principle no="7" type="support" statement="提前预期人性的缺陷，遇到时就不会崩溃" >}}
早上起来告诉自己：今天会遇到好管闲事的人、忘恩负义的人、傲慢的人。

这不是犬儒。这是基于人性现实的心理准备。

奥勒留的逻辑是：如果你每天早上都假设所有人会表现得体，那么遇到不得体的行为时就会愤怒和失望。但如果你接受"有些人就是会这样"，遇到时就能更冷静地回应。

这条原则和第二条配合使用：预期人性缺陷（原则七），然后在情绪出现时检查自己的判断（原则二）。
{{< /principle >}}
