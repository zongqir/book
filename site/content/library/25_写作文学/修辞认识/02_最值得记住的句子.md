## 关于说服力的来源

{{< sentence >}}
说服力最强的句子往往不是论证最严密的句子，而是结构安排最巧的句子。
{{< /sentence >}}

下次被一段话打动时先暂停，问自己：是论据让你信了，还是论据出场的顺序让你信了？大量日常说服靠的是排列组合，不是证据质量。

{{< sentence >}}
修辞不负责让话变漂亮，负责让话变有力。
{{< /sentence >}}

把修辞和文采解绑。最有力的修辞可以用极朴素的词完成——关键在于词的位置和句子之间的配合，不在单个词的华丽程度。

## 关于语言的隐性组织

{{< sentence >}}
问句不是用来提问的，是用来让对方在心里默认一个方向的。
{{< /sentence >}}

反问句、设问句在修辞中的真实功能是预设答案方向。意识到这一点之后，再遇到密集反问就不会自动跟着走了。

{{< sentence >}}
并列不是平等的——放在最后一项的，永远比前面的重。
{{< /sentence >}}

"自由、平等、博爱"——顺序不是随机的。并列结构里的位序本身就是权重分配。写作时、分析时都值得注意。

## 关于防御性阅读

{{< sentence >}}
能被一段话感动，又能说清这段话在用什么手段感动你——这才叫修辞意识。
{{< /sentence >}}

修辞意识不是让你变成冷漠的分析机器。它让你同时保持两种能力：被打动的能力，和看穿打动机制的能力。

{{< sentence >}}
一篇文章里最危险的句子，是你读完觉得理所当然的那一句。
{{< /sentence >}}

"理所当然"的感觉恰恰说明这句话的修辞做得太好了——好到你不觉得自己被说服了。越是"显然如此"的判断，越值得拆开看看结构。
