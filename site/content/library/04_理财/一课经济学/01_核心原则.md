多数经济学书给你一堆概念。黑兹利特只给一条原则，然后反复用它拆真实政策，直到你形成条件反射。

下面是从全书提炼出的判断准则。核心原则只有一条，其余四条是它在不同维度上的展开。忘掉具体案例没关系，只要这五条还在，判断力就在。

{{< principle no="1" type="core" statement="判断任何经济政策，必须追踪它对所有群体在所有时间段的所有后果——不只是受益者的眼前效果" >}}

黑兹利特认为经济学的全部艺术就在这一句：不只看某一群体的眼前效果，要看所有群体的长远效果。

坏经济学家只盯着政策的直接受益者，把看得见的好处当作全部收益。好经济学家多走一步：谁在暗处承担代价？这个代价什么时候会显现？

这条原则的力量在于它是通用筛子。不需要记住每个政策的具体分析，只需要对任何政策主张问同一个问题：完整后果是什么？

{{< /principle >}}

{{< principle no="2" type="support" statement="资源总量有限——花在 A 上的钱就是没花在 B 上的钱，不存在凭空创造的需求" >}}

破窗谬误是全书的开场案例。小流氓砸了面包店的窗户，有人说这是好事——玻璃店因此有了生意，经济被"刺激"了。

但被遮蔽的事实是：面包店主本来要用那笔钱买一套新西装。窗户碎了，玻璃店多了一笔生意，裁缝店少了一笔。社会总财富没有增加，只是被重新分配了。

这条原则一旦内化，你会自动对"灾难刺激经济""战争带动就业""消费拉动增长"这类说法保持警觉。

{{< /principle >}}

{{< principle no="3" type="support" statement="价格是资源稀缺程度的信号——管制价格等于掐断信号，让资源配置失去方向" >}}

价格不是商家随便标的数字。它压缩了供需双方的大量信息，告诉生产者该多生产什么、少生产什么，告诉消费者什么值得多花钱、什么该省着用。

价格管制打断了这条信息链。限价看起来保护了消费者，实际上取消了生产者增加供给的动力。供给减少，短缺出现，排队或黑市取代了价格，配置效率下降。

放开价格不是"对穷人残忍"，是让信号系统正常运转——有了信号，资源才能流向最需要它的地方。

{{< /principle >}}

{{< principle no="4" type="support" statement="财富来自生产，不来自消费和货币——印钞票不创造财富，只稀释已有财富的购买力" >}}

刺激消费、扩大需求、增发货币——这些政策听起来能让经济繁荣。但黑兹利特指出，货币只是交换媒介。

多印一倍的钞票，商品总量不会跟着翻倍。多出来的钱追逐同样多的商品，结果是价格上涨。先拿到新钞票的人（政府和银行）获益，后拿到的人（普通工薪阶层）承受物价上涨的代价。

通胀不是"经济过热"的副作用，是一种隐性税——从固定收入者和储蓄者手里悄悄转移财富。

{{< /principle >}}

{{< principle no="5" type="support" statement="政府每花一块钱，都是从私人部门征来的——政策不创造资源，只决定资源由谁使用" >}}

公共工程创造了就业，这看得见。但资金来自税收，那些钱如果留在纳税人手里，会被花在别的地方，创造另一批就业——这看不见。

政府支出与私人支出之间不是"多出来的"与"本来就有的"的关系，而是"A 用途"与"B 用途"的替代关系。

判断公共支出是否值得，不能只看它产出了什么，还要问它替代掉了什么。如果替代掉的那个用途产出更高，公共支出就是净损失。

{{< /principle >}}

## 五条原则之间的结构

第一条是根，后面四条是根在不同地面上长出来的枝。

资源有限（第二条）解释了为什么必须追踪完整后果——因为用在这里的钱不可能同时用在那里。价格是信号（第三条）解释了市场如何自动分配有限资源。财富来自生产（第四条）拦住了"印钱就能解决问题"的幻觉。政府不创造资源（第五条）拦住了"政府花钱就能刺激经济"的幻觉。

四条支撑原则缺了任何一条，核心原则的"追踪完整后果"就会在那个维度上留下盲区。

## 原则的使用边界

黑兹利特的分析框架有一个隐含前提：市场参与者的行为后果由市场价格充分反映。

当这个前提不成立时——比如污染的代价由第三方承担、公共卫生的收益无法被个人独占——单靠"追踪市场后果"不够，还需要考虑市场之外的后果。这些场景下，原则的方向依然对，但需要补充分析工具。

具体边界在 06 适用边界与失效条件中展开。
