{{< sentence >}}
硬实力让你能做这个工作。软实力让你能做好这个工作。
{{< /sentence >}}

有编程技能让你能找到工程师的工作。但软实力——和人的协作、学习新技术、管理自己的时间——决定了你是否能做好这个工作。

{{< sentence >}}
没有软实力，硬实力也很难发挥。
{{< /sentence >}}

一个优秀的工程师，如果不会和人沟通，他的想法无法被理解。如果不会学习，他的技能会逐渐落伍。如果不会管理情绪，他在面对挑战时会崩溃。

{{< sentence >}}
软实力在职业中期变得更关键。
{{< /sentence >}}

早期，硬实力很重要。你需要证明自己能做这个工作。但越往上走，软实力变得越关键。管理者的成败往往取决于他的沟通、领导力、战略思维——这些都是软实力。

{{< sentence >}}
沟通是最被忽视但最有价值的投资。
{{< /sentence >}}

很多人投入时间学硬技能。但投入时间改进沟通的人很少。结果是有能力的人因为无法有效表达而被看低，而会表达的人反而获得了更多认可。

{{< sentence >}}
情绪管理的能力决定了你在压力下的表现。
{{< /sentence >}}

职场中压力总是存在的。能否在压力下保持清醒的头脑和效率，决定了你能走多远。
