# 先看结构再做选择——波特的八句底层判断

## 关于行业结构

{{< sentence >}}
行业的利润水平不取决于产品好不好，取决于行业的竞争结构。
{{< /sentence >}}

好产品在坏结构的行业里也赚不到钱。餐饮行业出品再好，平均利润率就是5%——因为进入门槛低、替代品多、客户转换成本几乎为零。在做"怎么赢"之前，先问"这个行业允不允许赢家赚到钱"。

{{< sentence >}}
竞争不只发生在现有对手之间。供应商、买方、潜在进入者和替代品都在瓜分你的利润。
{{< /sentence >}}

只盯着同行看，会漏掉利润被吞掉的真正原因。你和对手打了个平手，但利润比去年少了——可能不是对手的问题，是供应商涨价了、客户的议价能力变强了、或者出现了一个全新的替代方案。

{{< sentence >}}
进入壁垒低的行业，利润必然会被新进入者稀释到平均水平。
{{< /sentence >}}

如果任何人花十万块就能开一家和你一样的店，你今天的高利润就是明天新对手蜂拥而入的邀请函。壁垒不是你想建就能建的——规模经济、专利、品牌认知、转换成本——但如果没有壁垒，利润不可能持久。

## 关于战略选择

{{< sentence >}}
最危险的战略位置是"夹在中间"——既不是成本最低，也没有明显差异化。
{{< /sentence >}}

成本领先的企业靠效率赚钱。差异化的企业靠独特性赚溢价。夹在中间的企业两头不靠：打不了价格战，也撑不起溢价。利润最薄的企业几乎都在这个位置。

{{< sentence >}}
差异化的标准不是你觉得自己有多不同，而是客户愿不愿意为这个不同多付钱。
{{< /sentence >}}

很多企业自认为差异化——"我们的设计更好""我们的服务更周到"。但如果客户在你和对手之间选择时，主要看的还是价格，那你的差异化就没有成功。差异化必须转化为客户的支付意愿，否则只是自我感觉。

{{< sentence >}}
聚焦不是小打小闹，是在一个细分市场里建立无法替代的位置。
{{< /sentence >}}

聚焦战略不是"因为做不大所以只做一小块"。它是一个主动选择：在一个足够小的市场里，做到这个市场里的成本最低或差异化最强。小市场的好处是大玩家往往不值得投入资源来抢。

## 关于价值链

{{< sentence >}}
竞争优势不来自企业整体，来自价值链上的具体环节。
{{< /sentence >}}

"我们的优势是团队"说了等于没说。优势必须落到价值链的具体位置——是采购更便宜、生产效率更高、物流更快、还是售后服务让客户更依赖你？找到那一两个环节，集中资源做到对手无法复制。

{{< sentence >}}
竞争对手能模仿你的产品，但很难模仿你的整条价值链。
{{< /sentence >}}

单个功能或服务容易被抄。但一整条价值链的配合方式——采购、生产、物流、销售之间的协同——抄起来成本极高。持久的竞争优势往往不在某一个点上，在整条链的咬合方式上。

## 什么场景拿出来用

进入一个新行业前——先做五力分析，看行业结构允不允许赚钱。

发现利润在下降但不知道为什么——逐条检查五种力量，哪一种在变强。

团队在讨论"我们的竞争优势是什么"——把回答落到价值链的具体环节上。

战略定位模糊、什么都想做——检查是不是"夹在中间"了。
