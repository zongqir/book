{{< sentence >}}
每一条线上只写一个关键词。
{{< /sentence >}}

思维导图和随手画联想图的分水岭就在这里。多个词写在一条线上，后面的联想被锁死——大脑只能顺着这串词往下走。一个词单独放在一条线上，它向四面八方都是开放的。大多数人画导图时最先犯的错就是往一条分支上塞太多字。

{{< sentence >}}
传统笔记的每一个特征都在压制记忆：单色、线性、纯文字、无节奏变化。
{{< /sentence >}}

这句话值得在每次打开笔记本时默念一次。用单色笔写满一页，对大脑来说几乎等于一面灰墙。没有颜色差异、没有图像锚点、没有空间层级，大脑能抓住的线索太少，只能靠重复硬记。换成彩色、有图、有层级的格式，不是为了好看，是为了给大脑更多抓手。

{{< sentence >}}
大脑不是按线性方式工作的，它是放射性的。
{{< /sentence >}}

这是博赞整套方法的地基。随便想一个词——"咖啡"——你脑子里冒出来的不是百科定义，而是气味、颜色、某个下午、某家店。这些联想从中心词向四周散开，每个联想又能触发新的联想。思维导图只是把这个内部过程画到纸上。

{{< sentence >}}
一张图像抵得上一千个词。
{{< /sentence >}}

博赞引这句老话不是在抒情。大脑处理图像的速度和通道数量远超文字。你在导图中心画一个苹果的图案，比写"苹果"两个字多激活好几个脑区。记忆强度不是内容决定的，是编码格式决定的。

{{< sentence >}}
关键词是思维导图的货币单位。
{{< /sentence >}}

导图不是缩略版笔记。缩略版笔记是把句子删短，导图是只保留那些能触发整块联想的词。"货币单位"这个比喻很准——每个关键词都应该是一个可以兑换出大量信息的节点，不是一个需要上下文才能理解的碎片。

{{< sentence >}}
颜色是大脑最快的分类信号。
{{< /sentence >}}

同一张导图上用三种以上颜色，大脑在扫视时就已经完成了初步分类。你不需要刻意去想"这部分属于什么"，颜色已经替你标好了。博赞要求中心图像至少三种颜色、不同主干用不同颜色，底层原因都在这里。

{{< sentence >}}
线条的粗细变化本身就在传递层级信息。
{{< /sentence >}}

靠近中心的主干粗，越往外越细。大脑不需要编号、不需要缩进，光凭线条粗细就能判断"这是主题""这是细节"。比数字标层级省掉了一层翻译。

{{< sentence >}}
思维导图不是笔记的升级版，是一种完全不同的编码方式。
{{< /sentence >}}

很多人把思维导图当成"更好看的笔记"，画出来还是线性结构加了点颜色。博赞反复纠正这个认知：导图和笔记的关系不是"好"和"差"的关系，是两种完全不同的信息存取方式。混在一起用，两边的优势都打折。

## 这些句子在什么场景下会被调用

记录场景——开会、听课、读书——准备动笔前，先问自己一个问题：我这次要用线性格式还是放射性格式？这个选择本身比记录内容更影响后续能不能调用。

复习场景——翻旧笔记时发现"写了很多但什么都想不起来"。问题大概率不在内容质量，在编码格式。回头用导图重新编码一遍，效果会完全不同。

画图场景——导图画到一半觉得乱了。大概率是某条分支上塞了太多字，或者没用颜色区分层级。回到"每条线一个词、不同分支不同色"这两条，通常就能理清。
