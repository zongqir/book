# 市场越一致，越要把这几句拿出来

## 先看风险和周期位置，再谈出手

{{< sentence >}}
先问风险，再问收益。
{{< /sentence >}}

任何一个机会一旦只剩收益叙事，就该把这句拿出来。

{{< sentence >}}
一级思维看故事，二级思维看故事里谁已经信了多少。
{{< /sentence >}}

真正值钱的，不是你觉得对不对，而是价格已经反映到哪一步。

{{< sentence >}}
市场越一致，越要先看自己是不是站在拥挤那边。
{{< /sentence >}}

舒服的共识，往往不是舒服的赔率。

{{< sentence >}}
周期位置会改写同一个资产的风险回报比。
{{< /sentence >}}

别把静态看法拿到不同环境里硬套。

{{< sentence >}}
激进和保守都该是环境判断，不该是性格表演。
{{< /sentence >}}

仓位不是人设，它该跟环境一起变。

{{< sentence >}}
安全边际不是永远低价，而是价格里没有把最好结果提前用完。
{{< /sentence >}}

别把便宜的表面，误当成足够的保护。

{{< sentence >}}
最危险的时候，常常不是看起来最吓人的时候，而是大家都觉得没事的时候。
{{< /sentence >}}

风险在顺风阶段最容易被低估。

{{< sentence >}}
真正会伤人的，往往是高估了自己承受风险的能力。
{{< /sentence >}}

环境越顺，越要把这句放在桌上。

## 这些场面最适合拿马克斯压一压自己

- 市场很一致时，你会不会先问价格里已经塞了多少乐观
- 决定加仓或减仓时，你会不会先看周期位置
- 想表达逆向观点时，你会不会先确认不是为了显得聪明
- 风险看起来很低时，你会不会反而更谨慎
- 做长期计划时，你还会不会区分环境判断和性格偏好

最有用的时候，往往不是你看空或看多，而是你得先判断自己是不是站在太舒服的位置。
