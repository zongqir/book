## 独处的价值

{{< sentence >}}
一个人只有在独处时才能完全成为自己。不爱独处的人，也不会爱自由。
{{< /sentence >}}

独处和孤独不是同一件事。独处是主动选择和自己的精神世界在一起。孤独是被动地感觉没人在乎。前者是充盈的，后者是匮乏的。叔本华谈论的始终是前者。

{{< sentence >}}
社交的本质是互相迁就。迁就得越多，自我保留得越少。
{{< /sentence >}}

每一次社交都需要调低一部分自我来适应群体。偶尔的调低是礼貌，长期的调低是消耗。如果你在社交之后经常感觉疲惫而非充实，不是社交能力不够，是你在为迁就付出超额的精力。

## 庸俗的本质

{{< sentence >}}
庸俗不是缺乏品味，是缺乏独处的能力。
{{< /sentence >}}

叔本华定义的庸俗不是粗俗。一个人可以很有教养、很有品味，但如果完全无法忍受和自己待着，需要不断从外部获取刺激和认可来维持内心的平衡——在叔本华的标准里，这就是庸俗。

{{< sentence >}}
无聊是内在空洞最诚实的信号。一个人越害怕无聊，他的精神越依赖外部供给。
{{< /sentence >}}

无聊感不是时间太多的结果，是精神资源太少的症状。一个内在丰富的人独处时不会无聊——有太多东西要想、要读、要消化。无聊的人不是没事做，是没有内部世界可以进入。

## 平衡

{{< sentence >}}
适度的独处是精神的充电。过度的独处是对人性的违背。
{{< /sentence >}}

叔本华虽然推崇独处，但也承认人是社会性动物。关键不是完全不社交，而是有能力选择——你能独处而不恐慌，也能社交而不迷失。丧失选择权的两个方向都是陷阱：完全离不开人，和完全无法面对人。

{{< sentence >}}
和优秀的人相处是社交的最高形式，和书相处是独处的最高形式。
{{< /sentence >}}

叔本华并不反对所有社交。他反对的是那种"为了不独处而社交"的惯性。如果社交对象能刺激你的思考、扩展你的视野，那社交本身就是精神生活的一部分。标准不是"社交还是独处"，而是"这段时间让我更丰富了还是更空虚了"。
