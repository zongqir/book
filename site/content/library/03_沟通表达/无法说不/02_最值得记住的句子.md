{{< sentence >}}
每次你说"是"的时候，你也在对其他事说"否"。
{{< /sentence >}}

答应一个额外的任务，意味着你减少了在其他事上的时间。每一个"好"都是一个隐形的"不好"。清楚地意识到这一点，你就会更谨慎地说"是"。

{{< sentence >}}
拒绝不是冷漠，而是尊重。
{{< /sentence >}}

如果你无法拒绝，你就无法认真对待你同意的承诺。被迫的"是"往往导致质量差的执行。真正的尊重是：我只承诺我能做好的事。

{{< sentence >}}
那些被你伤害最深的人往往是被你无限退让伤害的。
{{< /sentence >}}

当你无限地为别人退让时，你失去了自己。你变得怨恨、疲惫、没有热情。这个改变最后会影响你和别人的关系。拒绝反而是保护关系。

{{< sentence >}}
好好拒绝比勉强答应更有道德。
{{< /sentence >}}

如果你无法做好某件事，但还是答应了，这对申请者是不公平的。他们得到的是你的勉强，而不是你的全力。清晰的拒绝让他们知道真实的情况，可以做出更好的选择。

{{< sentence >}}
学会拒绝的人往往比容易答应的人更被尊重。
{{< /sentence >}}

有清晰边界的人看起来更有自信、更有主见。人们反而更愿意和这样的人打交道。容易答应的人往往被认为没有立场。
