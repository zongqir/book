## 关于判断他人的默认程序

{{< sentence >}}
我们不是先观察一个人，再形成判断；而是先激活一个类别，再用类别去"看"这个人。
{{< /sentence >}}

这是社会认知最基础也最容易被忽略的事实。你以为自己在观察，其实图式已经替你筛选了注意力。招聘、相亲、团队评估——凡是涉及快速评价他人的场景，都值得用这句话自检。

{{< sentence >}}
行为者在视觉场景中最突出，所以我们天然把行为归因到人身上，而不是情境。
{{< /sentence >}}

基本归因错误的认知根源。同事犯了错，你的注意力锁定在他身上，不在他面对的系统压力上。下次想说"他就是这种人"之前，试着把背景拉回视野中心。

{{< sentence >}}
负面信息在印象形成中的权重天然高于正面信息——一次失误比十次出色表现更容易留下来。
{{< /sentence >}}

消极偏差不是个性缺陷，是进化遗留的加工倾向。在人事评估中，这意味着你对某人的负面记忆可能被不成比例地放大了。

## 关于刻板印象的自动性

{{< sentence >}}
即使你明确反对某种偏见，在认知负荷高的条件下，类别信息仍然会优先于个体信息被提取。
{{< /sentence >}}

"我不歧视"是态度声明，不是认知事实。意志不能覆盖自动加工——你需要的是注意力资源和刻意的个体化加工。这句话在多元团队管理和面试评分中最该被记住。

{{< sentence >}}
图式一旦激活，就会主动寻找确认信息，同时降低对矛盾信息的敏感度。
{{< /sentence >}}

确认偏差在社会判断中的版本。你觉得某人"不靠谱"，之后他做的每件事都会被你往"不靠谱"方向解读。打破循环的方式不是"保持开放心态"，而是刻意去找反例。

## 关于自我认知的盲区

{{< sentence >}}
人对自己行为原因的解释，往往是事后建构的合理化叙事，而不是真实的因果描述。
{{< /sentence >}}

内省不等于自我了解。你以为你知道自己为什么做了那个决定，但认知研究反复证明：人对自己动机的报告经常是错的。这句话能让你在做复盘时少一点自信，多一点怀疑。
