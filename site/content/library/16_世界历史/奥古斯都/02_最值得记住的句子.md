{{< sentence >}}
屋大维的第一个政治决策是接受凯撒的遗产——包括凯撒的名字、凯撒的钱和凯撒的所有敌人。
{{< /sentence >}}

继承权力意味着同时继承了权力带来的所有风险。这是一个十八岁少年做出的判断——他选择了最大的赌注而不是安全地退到一边。

{{< sentence >}}
凯撒的失败在于他让人看到了他的独裁。屋大维的成功在于他让人看不到。
{{< /sentence >}}

凯撒直接称独裁官，被元老院刺杀。屋大维吸取教训：权力可以集中，但不能展示集中。保留共和制外壳是政治安全的必要条件。

{{< sentence >}}
每一次清洗完成后，屋大维都会立即转向和解——不是因为仁慈，而是因为永远杀下去没有尽头。
{{< /sentence >}}

知道什么时候停止使用暴力和知道什么时候开始使用暴力一样重要。暴力可以清除威胁，但只有和解才能建立秩序。

{{< sentence >}}
屋大维把权力分散到多个正式头衔中——没有一个头衔看起来是专制的，但加在一起就是绝对权力。
{{< /sentence >}}

这是制度设计的天才：每一个单独的权力看起来都合理且有限——保民官权、行省统帅权、大祭司——但组合起来就是无人可以挑战的垄断。

{{< sentence >}}
奥古斯都最后的胜利不是赢得战争而是赢得了和平——他创建的制度在他死后又运转了四百年。
{{< /sentence >}}

战场上的胜利是暂时的。制度的胜利才是持久的。屋大维的真正成就不是打赢了内战，而是内战结束后建立了一套所有人都愿意维持的体制。
