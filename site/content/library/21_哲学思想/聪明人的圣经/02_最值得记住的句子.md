## 看清欲望和恐惧的原始力量

{{< sentence >}}
大多数人的行为，不是由道德驱动，而是由欲望和恐惧驱动——他们只是非常善于为自己的行为编织道德理由。
{{< /sentence >}}

一个人为了升职而背后伤害竞争对手，然后告诉自己"这是为了公司的长期利益"。一个人为了保持财富而反对转移支付，然后说"这是为了经济效率"。他们真的相信自己的理由——但底层的驱动力其实很简单。

{{< sentence >}}
如果一个选择对某人有利，他就几乎不可能被说服放弃它，无论你用什么论证。
{{< /sentence >}}

这就是为什么道德说教往往无效。因为你在试图用"道德"来说服一个人放弃对他有利的东西。你说"这在道德上是错的"，他听到的其实是"你想让我放弃对我有利的东西"。他的防守会自动启动。

{{< sentence >}}
大多数人更害怕改变，而不是害怕继续承受痛苦。
{{< /sentence >}}

这解释了为什么有人在不好的关系里停留、为什么有人在不满意的工作上坚持、为什么有人容忍被欺骗。不是因为他们喜欢痛苦，而是因为已知的痛苦比未知的改变更能接受。

## 看清自我欺骗的聪明

{{< sentence >}}
自我欺骗是一门艺术。人能同时相信两个自相矛盾的东西：为了坚守某个形象，同时也为了获得某个利益。
{{< /sentence >}}

一个有权的人说"我关心弱势群体"，同时做的每个决策都让自己的优势更大。他真的关心吗？他会说是的。但他更关心的其实是自己的优势。他的大脑很聪明地让他两个都相信。

{{< sentence >}}
最聪明的人，是那个能看清自己的真实动机的人，而不是那个道德最高尚的人。
{{< /sentence >}}

这听起来可能很冷酷。但想想看，一个能诚实地说"我这样做是因为对我有利"的人，往往能做出更清晰、更有效的决策。而那个坚持"我是出于道德"的人，往往被自己的虚伪所限制。

## 关于权力和道德的真相

{{< sentence >}}
道德往往被权力使用——有权的人用道德来说服无权的人接受不公平。
{{< /sentence >}}

"为了集体利益""为了更大的好"——这些道德语言往往来自那些从不公平中获益最多的人。当你听到这样的语言时，问自己：谁在从这个"道德"的主张中获利？
