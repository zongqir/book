# 七句话，让你在对方说完之后知道从哪里拆

{{< sentence >}}
不要反驳对方的结论，反驳他得出结论的过程。
{{< /sentence >}}

直接否定结论会变成立场对抗。指出推理过程里的漏洞，对方自己也会动摇。攻击路径比攻击终点更有效。

{{< sentence >}}
对方说"都是这样"的时候，你只需要找到一个不是这样的。
{{< /sentence >}}

以偏概全是最常见的逻辑漏洞。一个反例就能让全称判断失效。你不需要证明他完全错了，只需要证明他说得太绝对了。

{{< sentence >}}
听到"要么A要么B"时，先想第三个选项。
{{< /sentence >}}

虚假二选一是一种控制话语方向的手段。多数复杂问题不只有两个选项。跳出对方设定的选择框架，反驳就已经完成了一半。

{{< sentence >}}
对方换了一个词继续论证时，停下来，问他这两个词是不是同一件事。
{{< /sentence >}}

偷换概念的过程通常很隐蔽。对方从"效率"开始说，说着说着变成了"速度"。如果你不盯着关键词，他就用一个不同的东西偷偷替换了论证基础。

{{< sentence >}}
"某某专家也这么说"不是论证。再大的名字也不能替你做推理。
{{< /sentence >}}

诉诸权威是用身份的重量替代逻辑的重量。专家的结论可能是对的，但"专家这么说"不等于"因此它是对的"。反驳时只需要问：专家给出的理由是什么？

{{< sentence >}}
在你不确定该怎么反驳时，先把对方的论证拆成三段：他的结论、他的理由、他的假设。
{{< /sentence >}}

这是最基本的反驳准备动作。多数时候，漏洞不在结论里，而在理由或隐含假设里。拆完之后，你会发现至少有一环没那么稳。

{{< sentence >}}
反驳的目标不是让对方闭嘴，是让在场所有人都看到他论证里的问题。
{{< /sentence >}}

争论有两个听众——你的对手和旁边的第三方。对手可能永远不会承认你说得对，但旁观者能看到你的反驳是否成立。好的反驳服务于第三方的判断。
