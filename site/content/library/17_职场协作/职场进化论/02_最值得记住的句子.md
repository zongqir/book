{{< sentence >}}
你最擅长的那项技能，可能正是让你最难转身的那个包袱。
{{< /sentence >}}

越精通一件事，你在它上面投入的时间和身份认同越多。当这件事的市场价值开始下降的时候，最痛苦的不是不会别的——而是你不愿意放下这个。承认一项苦练出来的能力正在贬值，是适应的第一步，也是最难的一步。

{{< sentence >}}
行业在变的时候，焦虑的人在观望，适应的人已经在试。
{{< /sentence >}}

观望本身不是错。但观望超过三个月还没有任何行动，那就不是在"等信号"——是在用观望掩盖害怕失败。适应不需要看清全局才动手。你不知道整个行业会怎么变，但你可以先花两周学一个新工具，看看自己学不学得会。

{{< sentence >}}
你上一次主动学一个没人要求你学的东西，是什么时候？
{{< /sentence >}}

如果想不起来，说明你的"变异率"太低了。在稳定环境里，你可以靠已有技能吃很久。但环境一变，你的储备里没有新东西可以调用——就像一个物种在环境突变时没有遗传多样性可以选择，只能等着被淘汰。

{{< sentence >}}
五年经验和一年经验重复五次不是一回事。
{{< /sentence >}}

有些人工作了五年，每一年都在做不同的事、解决不同的问题、积累不同的能力。有些人工作了五年，做的事跟第一年差不多。前者有五年的进化记录，后者只有一年的存活记录复制了四次。

{{< sentence >}}
不要在一棵树上把所有鸡蛋都挂满。
{{< /sentence >}}

你的收入来源、技能依赖、行业认知、人际网络，如果全部绑定在一家公司或一个行业上，那就是把所有赌注压在"这个环境不会变"上。适应性冗余不是浪费——它是保险。平时看不出用，环境变化的时候就是你的退路。

{{< sentence >}}
简历上最有价值的不是你做过什么，而是你从没做过到做成之间花了多久。
{{< /sentence >}}

这句话衡量的是学习速度。在一个快速变化的市场里，你会什么不如你多快能学会什么。一个用三个月从零掌握一项新技能的人，比一个在同一项技能上磨了十年的人更能适应变化。

{{< sentence >}}
被淘汰的不是最弱的，是最不愿意变的。
{{< /sentence >}}

职场里的"淘汰"通常不是因为你能力差。是因为环境变了，你的能力没跟着变。那些被优化掉的人里面，有不少曾经是团队里最强的——只是他们强在旧环境需要的那种能力上。
