{{< sentence >}}在你考虑任何具体证据之前，先问一句：没有这些证据时，概率是多少？{{< /sentence >}}

基准率是所有概率判断的起点。拿到一条新信息就开始评估，等于在没有坐标系的地方画点。体检报告、投资建议、招聘评价——任何判断如果跳过了"这件事本来的发生率是多少"，后面的推理都在空中建楼。

{{< sentence >}}强证据让你大幅更新，弱证据只微调——但多数人分不清两者的区别。{{< /sentence >}}

贝叶斯更新的关键不在公式，在于区分证据强度。一篇同行评审的随机对照实验和一条朋友转发的文章，应该触发不同幅度的判断调整。如果你无论看到什么新信息都大幅转向，你不是在更新——你是在被信息流牵着跑。

{{< sentence >}}如果你对自己的判断从不标概率，你就永远发现不了自己过度自信。{{< /sentence >}}

"我觉得这件事会成"和"我觉得 80% 概率会成"是两个截然不同的判断。前者无法校准——你不知道自己错了多少。标了数字之后，三个月回头统计：标 80% 的事有多少真发生了？如果只有一半——你的置信度需要大幅修正。

{{< sentence >}}承认不确定性不是软弱，是对现实最准确的描述。{{< /sentence >}}

"我不确定"在多数文化里被视为能力不足的信号。但概率思维翻转了这个判断：能说出"我有 60% 的把握"的人，比说"我很确定"的人诚实得多，也更可能长期做出更好的判断。确定性是幻觉；概率是对幻觉的校准。

{{< sentence >}}好决定不是选概率最高的选项，而是选期望值最大的——概率乘以后果。{{< /sentence >}}

90% 概率赚一块钱，不如 20% 概率赚一百块。多数人做决定时只看"哪个更可能"，忽略后果的量级。期望值思维强制你同时考虑两件事：这件事多大概率发生？发生了之后影响有多大？两个数字的乘积才是行动依据。

{{< sentence >}}极端表现之后，回归平均几乎是确定的——无论你做不做任何干预。{{< /sentence >}}

连续三年业绩第一的基金经理，第四年大概率会回落。考试发挥特别差的学生，下次大概率会好一些。这不是运气转向，是统计规律。一旦理解回归均值，你会停止在极端结果上过度归因——教练骂完球员表现变好，不是骂的功劳；是回归均值的功劳。

{{< sentence >}}没有消息不是"没有信息"——它本身就是一条有力的证据。{{< /sentence >}}

医学检查结果迟迟没出来，你以为信息量是零。但如果严重情况通常会被加急通报，"没有消息"就意味着严重情况的概率在下降。沉默本身携带信息。在竞标、求职、投资判断里，"对方没有回复"往往比"对方说了什么"更值得分析。

{{< sentence >}}校准不是一次性修正，而是持续运行的反馈循环——你预测，现实回应，你调整。{{< /sentence >}}

概率思维不是学完一套原则就够了。它要求你建立一个持续运转的系统：每次做判断时标注置信度，事后追踪结果，发现偏差时修正。和肌肉训练一样——停练就退化。校准精度是用出来的，不是读出来的。
