# 八句提醒，把抢答习惯拉回提问能力

这些句子不是拿来显得会思考的，而是拿来拦住自己抢答的。

你一发现讨论越来越快、立场越来越硬、问题越来越糊，就该把它们翻出来。

{{< sentence no="1" quote="先别急着答，先问自己: 现在到底在回答哪个问题。" >}}
这是总开关。问题没认准，后面的赞成、反对、补充和反驳都会跑偏。

它适合放在开会、写评论、做决策之前。先把题目认出来，再决定值不值得继续说。
{{< /sentence >}}

{{< sentence no="2" quote="问题一旦含糊，分歧就会越吵越大，因为每个人答的都不是同一件事。" >}}
很多争论表面上针锋相对，实际只是题型混了。有人在说事实，有人在谈价值，有人在催行动。

当你感觉讨论越来越热却毫无推进时，这句最该被调出来。
{{< /sentence >}}

{{< sentence no="3" quote="把立场藏进问法里，得到的只会是站队，不是理解。" >}}
这句专门拦那种“你难道不觉得……”“是不是只有……才会……”的问法。它看着像提问，实际已经把答案铺好了。

无论你在采访、带团队还是和朋友谈事，这句都能帮你挡住伪问题。
{{< /sentence >}}

{{< sentence no="4" quote="分不清事实、解释、价值和行动，就别指望结论会稳定。" >}}
很多混乱不是难，而是混。四种问题挤在一句里，任何答案都会显得又对又不对。

它适合用在阅读文章、复盘项目和做个人选择的时候，先把层次拆开。
{{< /sentence >}}

{{< sentence no="5" quote="一个问题值不值得回答，先看它有没有把标准说清。" >}}
你到底在用什么尺度判断好坏、可行、重要、值得。标准不亮出来，讨论只会在语气上打转。

当两边都觉得自己很有道理，可谁也说不服谁时，通常就是这句该出来的时候。
{{< /sentence >}}

{{< sentence no="6" quote="没有替代问题，很多确定感只是第一反应的回声。" >}}
问题只要换一种问法，很多理所当然的结论就会松动。替代问题不是抬杠，而是给思考留第二扇门。

它适合在你对一个结论感到特别顺的时候，强迫自己再问一遍还能不能别样理解。
{{< /sentence >}}

{{< sentence no="7" quote="证据经不起追问，再漂亮的问题也撑不起判断。" >}}
有些问题问得很聪明，最后却只通向空心判断，因为证据根本没跟上。

这句提醒你，提问不是脱离材料的脑力表演。问得再好，也要回到证据上。
{{< /sentence >}}

{{< sentence no="8" quote="真正好的提问，会先让自己暴露盲点，而不是先让别人难堪。" >}}
这句把提问从攻击姿态里拉回来。问题的价值不在让谁哑口无言，而在帮你看见原来没看见的地方。

当你发现自己提问越来越像进攻，这句就该顶在最前面。
{{< /sentence >}}

## 这些提醒该怎么调出来

- `开会卡住时`：先看第 1、2、4 句。先统一问题，再分题型。
- `看新闻或刷信息流时`：先看第 3、5、7 句。别让带倾向的问法替你做判断。
- `做复盘时`：先看第 4、6、7 句。把事实、解释和改法拆开，再补反证。
- `做个人选择时`：先看第 1、5、6 句。先问自己到底在选什么，再看标准是不是说清了。
- `准备追问别人时`：先看第 3、8 句。问题要打开理解，不要只是布置陷阱。

把它们记成一组顺序会更好用: 先认题，再分型；先亮标准，再找反证；先查证据，再落结论。
