{{< sentence >}}过渡就是一段脆弱期，在这段时间里，你缺乏既定的关系网络、对新环境的深度了解，以及已被证实的可信度{{< /sentence >}}

每个新任管理者都会经历这个脆弱窗口，关键是如何最快速度建立这三项基础资产。

{{< sentence >}}你的成功取决于你提升组织获得成果能力的程度，而不是你个人成果的程度{{< /sentence >}}

这句话直接打破了个人贡献者思维。管理者的价值通过团队成果体现，个人再优秀也无法替代系统能力建设。

{{< sentence >}}在你能够真正改变一个系统之前，首先要理解这个系统存在的原因{{< /sentence >}}

急于变革是新任管理者的常见陷阱。现状虽然不完美，但通常有其合理性，盲目推翻会破坏隐性平衡。

{{< sentence >}}90天计划不是关于你要完成什么，而是关于你要成为什么样的领导者{{< /sentence >}}

过渡期的核心任务是角色转换和领导力建立，具体项目完成是手段，不是目的。

{{< sentence >}}早期胜利必须在你能够掌控的领域内，同时对关键利益相关者有意义{{< /sentence >}}

早期胜利的两个硬标准：可控性和相关性。缺乏任何一个都会适得其反。

{{< sentence >}}如果你不能在前30天内识别出关键的非正式影响者，你就很难在90天内建立必要的联盟{{< /sentence >}}

正式组织架构图只显示权力结构，真正的影响力网络往往隐藏在非正式关系中。

{{< sentence >}}过渡失败的最大原因不是能力不足，而是无法快速适应新环境的复杂性{{< /sentence >}}

技能通常可以迁移，但环境复杂性每次都是全新挑战。适应能力比专业能力更重要。

这些句子构成了过渡期成功的核心原则体系。每条原则都指向一个关键认知：过渡不是延续，而是重新开始。新环境需要新策略，过往成功经验可能成为包袱。掌握这些原则，能帮助新任管理者避开最常见的过渡陷阱。
