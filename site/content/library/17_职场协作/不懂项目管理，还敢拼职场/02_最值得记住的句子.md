{{< sentence >}}
没有明确的目标，就不是项目，只是工作。
{{< /sentence >}}

项目的第一个特征是有明确的目标。如果你说不清楚这个项目要达成什么，说明还没有真正开始。

{{< sentence >}}
成本意识决定了你的决策质量。
{{< /sentence >}}

每个决策都有成本。有效的项目管理，就是在满足目标的前提下，最小化成本。

{{< sentence >}}
没有优先级的项目，就是失控的项目。
{{< /sentence >}}

有限的资源，无限的需求。如何分配资源？关键是优先级。没有清晰的优先级，资源就会被浪费在不重要的事上。

{{< sentence >}}
风险往往来自于不确定性。识别和管理风险，是优秀项目管理的标志。
{{< /sentence >}}

每个项目都有风险。关键是能否提前识别出来，并制定应对方案。

{{< sentence >}}
进度管理的关键是定期的复盘和调整。
{{< /sentence >}}

项目总会偏离计划。关键是能否及时发现并调整。定期的进度检查和调整，是保持项目在轨道上的方式。
