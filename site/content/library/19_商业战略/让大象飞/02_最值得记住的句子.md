{{< sentence >}}
创新不是一个部门，是一种能力。
{{< /sentence >}}

把创新外包给"创新实验室"是大公司最常犯的错误。实验室做出了原型但主业务不接——最后实验室变成了展示窗口而不是产品入口。创新能力需要嵌入核心业务团队。

{{< sentence >}}
你的第一个想法几乎一定是错的。
{{< /sentence >}}

几乎所有成功的创业公司都经历了重大转型——从最初的想法转向完全不同的方向。Instagram最初是签到应用，Slack最初是游戏公司。重要的不是第一个想法对不对，而是你多快能发现它是错的。

{{< sentence >}}
大公司杀死创新的方式不是说"不"，而是说"等一等"。
{{< /sentence >}}

很少有公司会直接否定创新想法。更常见的做法是"先做个调研""等下一个季度""纳入明年的规划"——每一次推迟都在消耗团队的动力和市场窗口。

{{< sentence >}}
小团队打败大团队，因为小团队做决策不需要开会。
{{< /sentence >}}

创新团队保持5-7人。超过这个数字，沟通成本急剧上升，共识变得越来越难。亚马逊的"两个披萨团队"规则——如果两个披萨喂不饱一个团队，团队太大了。

{{< sentence >}}
失败的成本在下降，不试的成本在上升。
{{< /sentence >}}

云计算、开源工具、低代码平台让做一个实验的成本极低。但不做实验的成本——错过市场、被对手超越、组织僵化——在不断上升。
