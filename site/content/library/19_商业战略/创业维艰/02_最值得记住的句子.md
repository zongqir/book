{{< sentence >}}
创业中不会有好时光，只有两种时光：要么很糟糕，要么前景不明。
{{< /sentence >}}

这句话直接打破了创业的理想化想象。霍洛维茨认为，如果你在等待"好时光"来临，说明你还没理解创业的本质。创业就是持续应对危机，那些看似顺利的阶段，其实只是危机之间的间隔。接受这个现实，才能建立正确的心理预期。

{{< sentence >}}
困难时期区分出平庸的CEO和伟大的CEO。任何人都能在顺境中管理公司，只有真正的领导者才能在逆境中脱颖而出。
{{< /sentence >}}

和平时期的管理技能——激励团队、制定战略、优化流程——在危机中可能完全无效。战时CEO需要不同的技能：快速决策、承担责任、直面现实。这种区分帮你判断自己是否具备在极限情况下领导公司的能力。

{{< sentence >}}
作为CEO，你必须设法让公司在没钱的情况下继续运转，让人们在不被支付薪水的情况下继续工作，让客户在产品有缺陷的情况下继续购买。
{{< /sentence >}}

这句话描述了创业中的"不可能任务"。常规商业逻辑是：有资源才能运转，有产品才能销售，有薪水才能留人。但创业经常打破这些逻辑。CEO的核心能力就是在资源不足时仍能推动公司前进。

{{< sentence >}}
在决定裁员时，你必须考虑的不是员工的感受，而是剩下员工的未来。
{{< /sentence >}}

这是霍洛维茨关于困难决策的典型观点。裁员痛苦，但如果不裁员导致公司倒闭，所有员工都会失业。战时决策的标准不是避免痛苦，而是选择伤害最小的方案。这种思维方式适用于所有"两害相权取其轻"的情况。

{{< sentence >}}
公司文化不是你写在墙上的价值观，而是你在最困难时刻做出的决策。
{{< /sentence >}}

理念和现实的差距在危机中暴露得最明显。员工不会记住你说过什么，而是记住在关键时刻你怎么选择。裁员时是否诚实？业绩不佳时是否推卸责任？资金紧张时是否先削减自己的薪水？这些决策才真正定义了公司文化。

{{< sentence >}}
没有秘诀，只有拼搏。
{{< /sentence >}}

霍洛维茨用这句话总结创业的本质。不存在能够避免困难的捷径，不存在一劳永逸的解决方案。创业就是持续面对从未遇过的问题，然后想办法解决。这种朴素的观点，反而比任何技巧都更接近创业的真实状态。

{{< sentence >}}
关键不在于避免失败，而在于从失败中恢复的速度。
{{< /sentence >}}

这句话重新定义了失败。在快速变化的创业环境中，失败不可避免。真正的竞争优势在于学习速度和调整能力。公司死不死，往往不取决于犯了多少错误，而取决于能否快速从错误中学习并改正。

这些句子共同构成了霍洛维茨的核心观点：创业是一场持久战，考验的不是避免困难的能力，而是在困难中保持理性、承担责任、快速学习的能力。
