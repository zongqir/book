{{< sentence >}}
你不是"容易生气的人"。你是一个在某些情境下选择了生气的人。
{{< /sentence >}}

"我就是脾气不好"是一个身份标签，一旦贴上就不用改了。但戴尔把它拆开：你上周二没生气，上周四生了气——说明生气是你在特定情境下的选择，不是你的固定属性。选择可以换，属性换不了。

{{< sentence >}}
内疚不能改变过去，它唯一的功能是瘫痪现在。
{{< /sentence >}}

内疚让你觉得自己在"负责任"。但仔细看——你在为一件已经发生、无法改变的事消耗当下的精力。这不是负责任，是用自我惩罚代替行动。真正的负责是改变下一次的做法，不是反复咀嚼上一次的错误。

{{< sentence >}}
担忧是用想象中的未来折磨真实的现在。
{{< /sentence >}}

你担心的事情有多少真正发生了？大部分没有。但每一次担忧都消耗了真实的时间和精力。担忧不是"未雨绸缪"——未雨绸缪有行动计划，担忧只有反复播放的灾难影片。

{{< sentence >}}
需要所有人都喜欢你，是放弃自己判断力的最快方式。
{{< /sentence >}}

讨好的本质不是善良，是恐惧。害怕被拒绝、被不喜欢、被孤立。为了避免这种恐惧，你让别人的期望替代了你自己的标准。结果是你越来越不知道自己要什么——因为你一直在忙着满足别人要什么。

{{< sentence >}}
"应该"是别人塞给你的剧本，不是你自己选的生活。
{{< /sentence >}}

"你应该结婚。""你应该稳定。""你应该感恩。"每一个"应该"背后都有一个你没有审查过的预设。戴尔的建议不是反叛所有"应该"，而是逐个检查：这个标准是我认同的，还是我从来没想过就接受了的？

{{< sentence >}}
对未知的恐惧，本质上是对"不能控制"的恐惧。
{{< /sentence >}}

不敢换工作、不敢搬城市、不敢开始一段新关系——这些表面看是害怕失败，底层其实是害怕"不知道会怎样"。但你现在知道的那些"确定"也是假的——你只是对不确定性选择了忽略而已。

{{< sentence >}}
停止为不在场的人管理情绪——你不欠任何人一种固定的反应。
{{< /sentence >}}

"如果我不接这个电话，他会不高兴。""如果我拒绝了，她会失望。"你在代替别人预演情绪，然后为这个预演调整自己的行为。但你预演的那些反应，有多少是真实发生的？你在管理一个想象中的别人。
