{{< sentence >}}
秦之所以亡，不在于法严，而在于法不能自行。
{{< /sentence >}}

吕思勉不认为秦朝因为暴政而亡。他的判断更精确：秦的制度在设计上是完整的，但执行层面出了问题——法令从咸阳发出，到基层已经面目全非。制度的失败不总是设计的问题，更多是执行的问题。

{{< sentence >}}
郡县制的本质不是分割地方权力，而是让中央直接触达每一寸土地。
{{< /sentence >}}

封建制下，国王管不到诸侯的领地。郡县制把所有中间层打掉，中央直接任命每一级官员。这不是行政区划的改变，是权力结构的根本性重组。

{{< sentence >}}
汉承秦制，这四个字里有整个汉朝政治的密码。
{{< /sentence >}}

刘邦打天下时骂秦始皇，坐天下后用秦始皇的制度。吕思勉指出，汉朝几乎完整继承了秦的行政体系——郡县制、律令制、官僚选拔制度——只是在表面上恢复了一些封建元素作为安抚。

{{< sentence >}}
察举制的意义在于打开了一条让非贵族进入权力中心的通道。
{{< /sentence >}}

汉武帝推行察举制，地方官推荐人才给中央。这条通道虽然经常被豪族垄断，但它在制度层面确立了"能力可以替代血统"的原则。这个原则后来演化成了科举制。

{{< sentence >}}
编户齐民使国家与个人之间不再有缓冲层。
{{< /sentence >}}

秦汉的户籍制度把每个人登记在册。个人直接对国家承担赋税和兵役义务，中间没有封建领主、没有庄园、没有行会。这是一种极高效也极沉重的管理方式。
