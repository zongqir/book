# 逛商场、看产品、翻财报时，先把这几句记住

## 先从生活线索里找，再拿数字验

{{< sentence no="1" quote="能在生活里先看见变化，是个人投资者少数真的优势。" >}}
别急着羡慕别人信息更快，先确认自己有没有把眼前的变化看进研究。
{{< /sentence >}}

{{< sentence no="2" quote="熟悉产品不是结论，只是研究的起点。" >}}
这句能防止你把消费体验直接偷换成投资判断。
{{< /sentence >}}

{{< sentence no="3" quote="先分清公司是哪一类，再决定该用哪把尺。" >}}
不同类型的公司，用同一种标准最容易判断走形。
{{< /sentence >}}

{{< sentence no="4" quote="高增长要和负债、现金流一起看。" >}}
故事最顺的时候，最该把这句拿出来。
{{< /sentence >}}

{{< sentence no="5" quote="看得懂的故事，必须翻成能核对的数字。" >}}
只要数字一落地就站不住，故事就还不能用。
{{< /sentence >}}

{{< sentence no="6" quote="好公司也会买贵，热闹门店也会看错。" >}}
热门和高频出现，并不自动等于好回报。
{{< /sentence >}}

{{< sentence no="7" quote="分批买入是在给判断留修正空间。" >}}
尤其适合用在自己刚从生活线索走到投资结论的时候。
{{< /sentence >}}

{{< sentence no="8" quote="一旦原始逻辑变了，就别拿习惯替它续命。" >}}
持有久了以后，这句会比买入前更重要。
{{< /sentence >}}

## 下面这些时刻最该把彼得·林奇翻出来

- 生活里看到一家火热公司时，你会不会先把它分到正确类别
- 发现线索后，你还会不会去核对财务和负债
- 看到高增长时，你会不会先问增长是怎么来的
- 买入前，你会不会先写出最容易失效的地方
- 持有后，你还会不会跟踪原始逻辑有没有变

它是拿来把熟悉感翻成判断，不是拿来给顺眼故事背书。
