{{< sentence >}}
把产品放在第一位，其他的——收入、利润、市场份额——都会跟着来。
{{< /sentence >}}

Google的战略逻辑不是先找到商业模式再开发产品，而是先做出最好的产品再想怎么赚钱。搜索做到最好，广告收入自然出现。这条逻辑不是所有公司都能用——你得有足够的资本撑到产品变现。

{{< sentence >}}
招聘是管理者能做的最重要的事。
{{< /sentence >}}

Google在招聘上投入的时间和资源远超大多数公司。一个差的招聘决定造成的损害远大于一个好的管理决定带来的收益。先把对的人招进来，管理的难度会大幅下降。

{{< sentence >}}
创意精英不需要被管理，需要被释放。
{{< /sentence >}}

施密特造了"创意精英"这个词——那些既有技术深度又有商业直觉还有创造力的人。传统管理把他们当执行者用是最大的浪费。给他们问题而不是答案，给他们空间而不是流程。

{{< sentence >}}
默认公开信息，而不是默认保密。
{{< /sentence >}}

Google的做法是除非有明确理由保密，否则信息默认对全公司公开。这和大多数组织相反——大多数组织默认保密，只在"需要知道"时才分享。信息透明让决策质量提高、政治行为减少。

{{< sentence >}}
用数据做决策，但不要让数据替你做决策。
{{< /sentence >}}

Google以数据驱动著称，但施密特强调数据是输入不是答案。数据能告诉你"是什么"，不能告诉你"应该怎么做"。判断力仍然是管理者最核心的能力。

{{< sentence >}}
如果你想要创新，就要允许失败。不允许失败的公司不可能创新。
{{< /sentence >}}

Google的很多产品失败了——Google Wave、Google Plus。但这些失败是创新体系的必然产物。问题不是"怎么避免失败"，而是"怎么让失败的成本可控、让成功的收益最大化"。
