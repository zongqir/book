# 一听到高增长，就先把这几句拿出来

## 成长故事要过管理层和组织这两关

{{< sentence >}}
成长股最难的不是找增长，而是确认增长能不能继续长得好。
{{< /sentence >}}

真正难的是持续扩张质量，不是某一年的漂亮数字。

{{< sentence >}}
道听途说不值钱，交叉验证才值钱。
{{< /sentence >}}

调研一多，越要拿这句防自己把八卦当证据。

{{< sentence >}}
产品、销售和研发，最后都会写进报表。
{{< /sentence >}}

别把经营质量和财务结果拆成两张皮。

{{< sentence >}}
管理层坦不坦诚，迟早会体现在问题暴露时。
{{< /sentence >}}

真正该观察的，是它怎么面对坏消息。

{{< sentence >}}
真正的成长股，扩张和回报要能同时成立。
{{< /sentence >}}

增长如果一直靠牺牲回报换，后面会越来越贵。

{{< sentence >}}
高增长不该自动等于高估值，也不该自动等于高质量。
{{< /sentence >}}

赛道热的时候尤其要把这句拿出来。

{{< sentence >}}
太早卖掉持续复利的公司，代价常常比一时回撤更大。
{{< /sentence >}}

成长股里最常见的亏，不总是买错，也可能是拿不住。

{{< sentence >}}
费了很大劲调研，也不等于可以无视边界。
{{< /sentence >}}

调研越深，越要保持对不确定性的敬畏。

## 调研做到一半时，最该回来查这页

- 听到高增长故事时，你会不会先去做交叉验证
- 研究公司时，你还会不会看研发、销售和管理层是不是同向发力
- 看管理层时，你会不会留意它怎样谈问题
- 持有后，你还能不能分清逻辑破坏和正常波动
- 想卖出时，你会不会先问是不是过早放弃了继续复利的资产

真正的用法，是在故事很迷人时逼自己回到交叉验证、组织质量和长期回报。
