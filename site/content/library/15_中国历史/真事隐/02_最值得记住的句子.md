{{< sentence >}}
历史不只是过去发生的事，还包括后人选择怎么讲述这些事。
{{< /sentence >}}

这句话看似简单，但它动摇了"读历史就是了解事实"的假设。事实是一回事，叙事是另一回事，而我们接触到的几乎都是叙事。

{{< sentence >}}
每一种历史叙事都是一种筛选——被记住的和被遗忘的加在一起才是完整的历史。
{{< /sentence >}}

被遗忘的部分不是因为不重要，可能是因为不适合某种叙事框架。柯文提醒：遗忘本身是有结构的，不是随机的。

{{< sentence >}}
当政治需要和历史记忆发生冲突时，改变的通常是记忆。
{{< /sentence >}}

集体记忆不是由事件本身决定的，而是由"谁控制了讲述权"决定的。同样的事件在不同的政治需要下会被讲成完全不同的故事。

{{< sentence >}}
"真事隐"不是说真事被消灭了，而是说真事被另一种讲法覆盖了。
{{< /sentence >}}

书名"真事隐"本身就是方法论声明：真实的事件没有消失，它被压在了某种叙事下面。研究者的任务是把那层叙事揭开。

{{< sentence >}}
历史学家最大的偏见不是立场偏见，而是叙事偏见——不自觉地把复杂的过去整理成一个有头有尾的好故事。
{{< /sentence >}}

整理本身就是扭曲。真实的历史过程往往杂乱无序、多线并行、没有清晰的因果链。把它写成"从A到B"的线性故事，已经是一种选择性建构。
