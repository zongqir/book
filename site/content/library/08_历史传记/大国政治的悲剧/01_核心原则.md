# 无政府体系下大国行为的五条判断准则

米尔斯海默的五个假设组合在一起，不只是解释国际政治的工具。它们指向一种更一般的判断方式：在没有裁判的博弈里，参与者的行为由结构决定，不由善意决定。

{{< principle no="1" type="core" statement="在没有最高权威的体系里，每个参与者都必须假设最坏情况来保障自身安全。" >}}

国际体系没有"世界警察"。联合国安理会不是法院，它没有强制执行判决的能力。当一个大国受到威胁时，没有人有义务来救它。

这意味着每个大国都必须自助。你可以和别人签安全协议，但协议的执行力取决于签约方的实力和意愿——而这两样东西随时会变。

常见误读是把这条原则理解成"大国都是坏人"。不是。是体系不提供可靠的第三方保护，所以自保成了理性选择。把这条原则搬到商业竞争里：在没有可执行的行业规则时，假设竞争对手会利用一切合法手段挤压你，比假设大家会自觉守规矩更安全。

{{< /principle >}}

{{< principle no="2" type="core" statement="意图不可知，能力可观测——判断威胁时优先看能力，不优先看对方说了什么。" >}}

你永远无法确定另一个大国的真实意图。外交声明可以是真的，也可以是策略。今天的善意领导人明天可能下台，换上一个完全不同的人。

米尔斯海默把这一点推到极致：既然意图不可知，那么唯一可靠的判断依据是对方的能力。一个拥有强大军事力量的邻国，不管声称多爱和平，都是潜在威胁——因为能力在那里，意图随时可能变。

这不是偏执。在信息不对称的环境里，高估威胁的代价通常低于低估威胁的代价。一个公司如果只看竞争对手的公开声明，而不跟踪对方的产能、专利储备和人才布局，迟早会被打个措手不及。

{{< /principle >}}

{{< principle no="3" type="support" statement="权力是相对的，不是绝对的——一方变强意味着另一方变弱。" >}}

安全不是绝对量。你有多安全，取决于你和潜在对手之间的实力差距。你变强了但对手变得更强，你其实更不安全了。

这就解释了为什么大国对邻国的经济增长格外敏感。经济增长本身是中性的，但在无政府体系里，经济增长意味着潜在军事能力的增长，而军事能力的增长改变了力量对比。

这条原则约束的误判是：以为合作能让所有参与者同时变强而不改变相对位置。在安全领域，相对收益比绝对收益重要。

{{< /principle >}}

{{< principle no="4" type="support" statement="地理是权力的硬约束——大片水体能阻止力量投射，地区霸权比全球霸权现实。" >}}

米尔斯海默把地理因素拉回了权力分析的核心。大片水体——大洋——极大限制了一个大国向远方投射军事力量的能力。这就是为什么美国可以成为西半球的地区霸主，但没法直接控制欧亚大陆。

由此推出的行为预测是：大国的首要目标是成为自己所在地区的霸主，同时阻止其他地区出现另一个霸主。这就是"离岸平衡"的逻辑——你不需要亲自占领对方的地盘，你只需要确保那个地区的力量保持平衡，没有一家独大。

忽略地理因素去谈权力，容易高估远距离威胁，低估近距离威胁。

{{< /principle >}}

{{< principle no="5" type="support" statement="大国的行为模式跨时代稳定——结构压力不会因为技术进步或政治制度变化而消失。" >}}

拿破仑时代的法国、威廉二世的德国、冷战时期的苏联、今天的大国——面对的结构压力本质上一样。技术变了，武器变了，通讯变了，但无政府状态没变，意图不可知没变，权力的相对性没变。

这条原则约束的是一种常见乐观：以为民主制度的扩散、全球化的深入或核武器的存在已经从根本上改变了大国政治的逻辑。米尔斯海默不否认这些因素的影响，但他认为它们改变的是博弈的形式，不是博弈的底层结构。

判断一个新现象是否"改变了游戏规则"时，先问：它改变了无政府状态吗？它让意图变得可知了吗？如果两个都没有，游戏规则的底层逻辑还在。

{{< /principle >}}
