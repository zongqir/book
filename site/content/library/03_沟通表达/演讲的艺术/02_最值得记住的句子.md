{{< sentence >}}
听众不是来见证你的准备，而是来拿走他们需要的东西。
{{< /sentence >}}

一旦把舞台理解成“证明我行不行”，注意力就会全跑到自己身上。演讲真正的起点，是先想听众听完要带走什么。

{{< sentence >}}
一场演讲最好只让听众牢牢记住一个核心意思。
{{< /sentence >}}

你讲得越多，听众带走的越少。先保住一个主轴，再决定哪些材料值得围上去。

{{< sentence >}}
结构比措辞更先决定一场演讲能不能被听懂。
{{< /sentence >}}

顺序乱了，再漂亮的句子也救不回来。听众跟不上结构时，不会提醒你，他们只会默默掉线。

{{< sentence >}}
紧张不是故障，是上台前的正常能量。
{{< /sentence >}}

问题不在紧张本身，在你把紧张误读成“我快要出错了”。把它当成准备启动的信号，状态反而更稳。

{{< sentence >}}
例子不是装饰，它负责把抽象观点落到听众脑子里。
{{< /sentence >}}

没有例子，观点只会在台上成立。例子一落地，听众才知道这件事和自己有什么关系。

{{< sentence >}}
好的开头不是炫技，是尽快让听众知道为什么要继续听。
{{< /sentence >}}

开头拖太久，后面再好也很难拉回来。前几十秒最重要的任务，是给出问题、利益或张力。

{{< sentence >}}
现场失误时，先保节奏，不要急着保完美。
{{< /sentence >}}

忘词、卡顿、翻页出错都不是致命点。真正致命的是你当场慌掉，把全部注意力从听众那里抽走。
