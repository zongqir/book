{{< sentence >}}
任何报告都要遵循"空雨伞"逻辑：看到天空乌云密布（事实），判断要下雨（解释），所以要带伞（建议）
{{< /sentence >}}

这是麦肯锡最基础的思维结构。很多人写报告时只堆砌事实，或者直接给建议，中间缺少解释环节。有了这个三段式框架，任何复杂问题的汇报都变得清晰。

{{< sentence >}}
每次发言前先在脑中倒计时30秒，确保第一句话就能让对方明白你要说什么
{{< /sentence >}}

这就是著名的"30秒电梯原则"。麦肯锡要求顾问能在30秒内把核心观点说清楚，因为高管的时间极其宝贵。这个习惯会彻底改变你的表达效率。

{{< sentence >}}
用SCRAP笔记法记录会议：Situation（背景）、Complication（问题）、Resolution（解决方案）、Action（行动）、Polite ending（礼貌结尾）
{{< /sentence >}}

麦肯锡顾问的笔记不是流水账，而是有固定结构。这个框架确保你记录的信息都是可执行的，开完会就知道下一步该做什么。

{{< sentence >}}
邮件的标题必须包含具体的行动要求，比如"请在周三前确认会议时间"而不是"关于会议安排"
{{< /sentence >}}

这个小技巧能让你的邮件回复率提高一倍。标题就是邮件的"30秒原则"，收件人扫一眼就知道需要做什么。

{{< sentence >}}
工作日的前两个小时是黄金时间，只用来处理最重要且最难的任务，其他时间处理邮件和例行工作
{{< /sentence >}}

麦肯锡顾问的时间分配原则：把最清醒的时间留给最重要的工作。大部分人的做法恰恰相反，一上班就处理邮件，等到下午才开始正经工作，那时注意力已经分散了。

{{< sentence >}}
每个PowerPoint页面只传达一个核心信息，标题就是这页的结论
{{< /sentence >}}

麦肯锡的PPT制作标准。他们的每一页幻灯片都能独立成立，标题不是"市场分析"而是"市场增长率达到15%，超出预期"。这样的表达方式让听众立刻知道重点。

{{< sentence >}}
和客户开会前，准备三个版本的汇报：5分钟版、15分钟版、45分钟版，根据现场情况选择
{{< /sentence >}}

这体现了麦肯锡对不确定性的应对方式。他们不会因为会议时间突然压缩就慌乱，而是提前准备多个版本，确保无论什么情况都能把核心观点传达清楚。
