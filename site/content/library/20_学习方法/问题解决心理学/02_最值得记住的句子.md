以下六条判断句从认知心理学的问题解决研究中提炼而来。每条指向一种特定的认知陷阱。记住它们不是为了引用，是为了卡住的时候能快速自我诊断。

## 你手里拿着锤子，就会把所有东西看成钉子

{{< sentence >}}
对物体常规功能的固有认知，会阻止人发现同一物体的其他用途。
{{< /sentence >}}

功能固着的核心。蜡烛实验里没人想拆图钉盒当烛台——不是看不见盒子，是脑子里已经给它定了性。

碰到"材料都在手边就是想不出来"的场景时，问自己：我对这些东西的定义，是题目规定的，还是自己加的？把每个工具、每份资源的"默认用途"写出来，然后逐条问"如果不是这个用途呢"。
## 问题的画法决定了你能找到哪些答案

{{< sentence >}}
问题的表征方式决定了搜索空间的边界。
{{< /sentence >}}

同一个任务换个说法，解出率可以翻几倍。这不是文字游戏，是认知结构在起作用。你怎么理解问题，就决定了大脑在哪个范围里搜索。画错了，正确答案就在搜索范围之外。

下次卡住时，在纸上重新写一遍"我到底在解什么"。如果写出来的版本跟脑子里想的不一样，多半是表征在悄悄限制你。
## 连续做对五道题，第六道反而更容易做错

{{< sentence >}}
连续成功建立的解题路径会变成无意识的默认反应，即使新问题需要完全不同的方法。
{{< /sentence >}}

Einstellung 效应。卢钦斯的水壶实验证明，连续用同一方法做对之后，被试自动套用同样的路径——哪怕眼前有更简单的解法。

成功经验变成了思维惯性。

对策只需一个动作：解完一道题后问自己"有没有另一种做法"。如果想不到，可能已经被定势锁住了。
## 那条线可以画到格子外面

{{< sentence >}}
自我施加的约束是问题解决中最隐蔽的障碍。
{{< /sentence >}}

九点问题里，没人说线不能画出方阵边界。但绝大多数人默认了这条规则。

自我设限之所以难识别，是因为你把自己加的假设当成了题目给的条件。两者混在一起，你根本没意识到有一堵墙是可以拆的。

方法：把你认为"不可改变"的条件列出来，逐条检查——这条到底是外部硬约束，还是我自己假定的？
## 走开不是放弃，是让大脑换一种搜索模式

{{< sentence >}}
酝酿期的作用不是休息，而是降低当前表征路径的激活强度，让其他表征方式有机会浮出来。
{{< /sentence >}}

卡了很久之后出去走一圈，回来突然想通了。认知心理学的解释是：持续思考会强化当前表征路径，暂停让这条路径的激活度下降，其他可能的路径才有空间冒出来。

关键前提：暂停之前你已经认真想过一轮。没有充分输入的暂停只是走神。
## 专家记住的不是更多碎片，是不同的拼法

{{< sentence >}}
专家和新手的差异不在信息量，而在信息的组织方式。
{{< /sentence >}}

象棋实验：大师五秒看一盘棋能复原二十几个子，但如果棋子是乱摆的，大师和新手一样记不住。

大师记的不是位置，是有意义的结构——一组棋子打包成一个"块"。新手看见散装零件，大师看见组装好的模块。

这意味着"多练自然就会"只说对了一半。练习要朝"打包信息、识别结构"的方向走，不只是增加题量。
## 你解决的不是问题本身，而是你理解的那个版本

{{< sentence >}}
人永远只在自己建构的问题空间里搜索答案。
{{< /sentence >}}

所有认知陷阱的根源都可以追溯到这一条。你不是在解决"问题"——你是在解决"你理解的那个问题"。两者之间的偏差，就是你卡住的原因。

这条判断的操作价值很直接：当你卡住时，第一反应不应该是"再用力想"，而是"我理解的那个版本，跟原题一样吗"。写下来，对照，找偏差。偏差本身就是突破口。
