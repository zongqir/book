如果一件事所有人都说能做到，那它就不是壁垒。京东选择自建物流正是因为没人愿意做——不愿意做就是壁垒。
{{< /sentence >}}

壁垒来自别人不愿意付出的成本。容易做的事所有人都会做——然后就没有壁垒了。

{{< sentence >}}
用户不关心你是平台模式还是自营模式——他只关心东西什么时候到、有没有问题、退换方不方便。商业模式是你的事，用户体验是用户的事。
{{< /sentence >}}

模式之争是行业内部的辩论。用户只看结果不看过程。

{{< sentence >}}
京东亏了十年。每一年投资人都问什么时候盈利。刘强东的回答是：物流网络建成的时候。他赌的不是今年的利润而是十年后的基础设施。
{{< /sentence >}}

十年亏损需要的不只是钱——还有对判断的信心和对压力的忍耐力。

{{< sentence >}}
中关村卖假货的商家利润更高。刘强东选择了正品低价——短期看是傻，长期看是唯一可持续的路。口碑不能靠广告建只能靠不作假建。
{{< /sentence >}}

信任是最贵的资产。建起来需要十年但摧毁只需要一次。

{{< sentence >}}
管理快递员的关键不是罚款而是让他觉得这是自己的事业。京东快递员的收入、五险一金和职业发展路径比行业平均水平高一截。
{{< /sentence >}}

一线员工的对待方式决定了他面对客户时的态度——这和海底捞的逻辑一样。
