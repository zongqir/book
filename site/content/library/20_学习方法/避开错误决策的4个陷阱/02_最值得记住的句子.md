# 避开错误决策的4个陷阱：最值得记住的句子

## 识别陷阱的八句诊断话

{{< sentence >}}当你只看到"要不要做"的时候，你已经掉进了窄框架。{{< /sentence >}}

窄框架最隐蔽的地方在于，它让你以为自己在做选择，其实你只是在对一个方案做"是或否"的判断。有效的选择至少需要两个以上互相独立的选项。

{{< sentence >}}确认偏差不是你在撒谎，是你的搜索引擎出了问题。{{< /sentence >}}

你不是故意只看有利证据。你的注意力自动放大支持现有判断的信息，自动过滤矛盾信号。问题不在动机，在机制。

{{< sentence >}}如果你必须说服自己"这次不一样"，大概率这次没什么不一样。{{< /sentence >}}

每次为一个偏离常规的决定找理由时，先停下来查一下基准率。"这次不一样"是过度自信最常见的伪装。

{{< sentence >}}做决定之前先问：我愿不愿意拿这个决定打赌？{{< /sentence >}}

打赌不是为了计算赔率，是为了把模糊的"我觉得"变成具体的"我有多确定"。很多人一听到"打赌"就后退——后退本身就是信号。

{{< sentence >}}十分钟后你会怎么想？十个月后呢？十年后呢？{{< /sentence >}}

短期情绪会劫持判断。10-10-10 框架把你从当下的情绪漩涡里拉出来，让三个时间尺度上的自己同时投票。

{{< sentence >}}如果你最信任的朋友来问你同样的问题，你会给什么建议？{{< /sentence >}}

给别人建议时，你不会被自己的情绪干扰，也不会被沉没成本绑架。这句话的价值不在"旁观者清"，在于它能瞬间切换你的视角。

{{< sentence >}}为最好的结果做计划，同时为最坏的结果做准备。{{< /sentence >}}

过度自信的问题不是你乐观，是你只为一种结果做了准备。预设失败场景不是悲观——是让你在事情偏离时有路可退。

{{< sentence >}}好的决策流程不能保证好结果，但能大幅降低系统性犯错。{{< /sentence >}}

决策质量和结果质量是两件事。WRAP 不承诺你做对每一个决定，但能让你不再反复栽在同一类陷阱里。

## 调用场景速查

- 面对"做还是不做"的选择时——调出窄框架那句，先找第三个选项
- 发现自己只在搜集支持性证据时——调出确认偏差那句，主动找反面信息
- 纠结到失眠、情绪快要替你做决定时——调出 10-10-10 那句，拉开时间距离
- 对自己的判断特别有信心时——调出"打赌"那句，测试信心是否经得起下注
- 决策已做、开始执行时——调出"为最坏结果做准备"那句，补一个预案
