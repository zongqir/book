# 读完三个月后，你的判断习惯变了吗

## 直觉判断出现时，你会停一下吗

场景：面试进行了三分钟，你已经"知道"这个人行不行了。

读之前的反应：直接按直觉走，后面的时间用来确认。

读之后应该出现的变化：能意识到"我的判断已经形成了，但它是系统1给的"。会问自己——这个判断是什么时候形成的，是在看完所有信息之后还是之前？会在直觉旁边放一个独立判断再做比较。

自检：最近一次在重要判断中真的暂停了吗？还是知道应该暂停，但实际上没有？

## 数字判断中，你能找到锚在哪里吗

场景：谈判中对方先报了一个价格。你开始从那个价格往下还。

读之前的反应：觉得自己在"理性还价"。

读之后应该出现的变化：能意识到自己的调整起点就是对方的报价——你在对方的锚上做调整，调整幅度通常不够。会在听到对方报价之前先形成自己的独立估算。或者在听到锚定数字后，强迫自己从反方向做一次估算。

自检：最近一次做数字估算时，你能说出你的起点是从哪来的吗？如果说不出来，你还在锚定里。

## 同一件事换种说法，你的选择还一样吗

场景：医生告诉你"手术存活率 90%"。你感觉还行。换一个医生说"手术死亡率 10%"。你开始犹豫。

读之前的反应：跟着措辞走，觉得两种说法"感觉不同"。

读之后应该出现的变化：遇到二选一或风险评估时，会主动把描述翻转。"95% 通过率"→"5% 失败率"。如果翻转后判断变了，会回到原始数据用绝对数值做决定。

自检：最近一个月做过框架翻转吗？哪怕一次？

## 听完一个精彩故事，你会去查基率吗

场景：朋友说他投了一个创业项目赚了十倍。你开始考虑是不是也该投。

读之前的反应：被故事打动，觉得"这个方向确实有机会"。

读之后应该出现的变化：在被故事打动之后，会去查——这类投资的平均回报率是多少？创业项目的整体成功率是多少？朋友这个案例在总体中处于什么位置？

自检：你能说出上一次"先查基率再听故事"是什么时候吗？

## 在得失之间做选择时，你校正了损失权重吗

场景：一笔投资已经亏了 30%。你不想卖，觉得"再等等说不定会回来"。

读之前的反应：继续持有。理由是"现在卖就真亏了"。

读之后应该出现的变化：能识别出这是损失厌恶和沉没成本在同时工作。会问自己——如果我现在手上没有这笔投资，拿着等量的现金，我会在这个价格买入吗？如果不会，那"继续持有"和"在这个价格买入"是同一个决定。

自检：最近一次面对"要不要止损"时，你做了这个替换测试吗？

## 信息只有一面时，你的确定感会打折吗

场景：同事推荐了一个供应商，只说了优点。你觉得不错。

读之前的反应：信息越少故事越圆，确定感越强。直接推进。

读之后应该出现的变化：能意识到 WYSIATI 在工作——你看到的信息越完整、越连贯，越需要怀疑。会主动去找反面信息：这个供应商有客户投诉过吗？同行怎么评价？换过几次主要客户？

自检：最近一次只听到正面评价时，你主动去找过反面信息吗？

## 做了某事之后情况好转，你会先排除回归均值吗

场景：团队连续两周表现很差。你开了一次严厉的复盘会。下一周表现回升了。你觉得"严格管理果然有效"。

读之前的反应：归因给自己的管理动作。

读之后应该出现的变化：会先问——连续两周的差表现是不是已经偏离了正常水平？如果是，回升是不是不做任何事也会发生的统计回归？只有排除了回归均值之后，才把功劳归给管理动作。

自检：你上一次归因"做了 X 之后 Y 变好了"时，考虑过回归均值吗？

## 一张自检速查表

| 判断场景 | 读之前的默认反应 | 读之后该有的新反应 | 行为真的变了吗 |
|---|---|---|---|
| 直觉已经有了答案 | 直接跟着走 | 暂停，独立判断对比 | |
| 数字估算 | 从第一个数字开始调 | 先形成独立估算 | |
| 两种说法感觉不同 | 跟着措辞走 | 翻转框架再判断 | |
| 被故事打动 | 直接行动 | 先查基率 | |
| 不想接受损失 | 继续持有 | 做替换测试 | |
| 只听到一面信息 | 确定感增强 | 主动找反面 | |
| 做了X后Y变好 | 归因给X | 先排除回归均值 | |

最后一列留空。它不是给别人看的——是你自己填的。
