## 结论先到手，理由慢慢找——顺序一反就变辩护

{{< sentence no="1" quote="在你决定赞同还是反对之前，先找出支撑结论的理由。" >}}
多数人听完一段话，直觉反应是赞同或反对。但赞同和反对都不是分析。分析从一个更笨的问题开始：凭什么？

把结论和理由拆开看，你经常会发现两者之间的连接比想象中松。理由可能只撑住了结论的一个角落，剩下靠情绪、权威或惯性补齐。
{{< /sentence >}}

{{< sentence no="2" quote="一个论证的质量，取决于它愿意暴露多少前提给你检验。" >}}
好论证和坏论证的差别，不在结论听起来多确定。差别在于它有没有把地基摊给你看。

越是把前提藏着的论证，越需要你替它翻。
{{< /sentence >}}

## 假设不浮出水面，论证就像建在暗桩上

{{< sentence no="3" quote="价值观假设是一条隐形的优先级排序，决定论证往哪个方向倒。" >}}
一个人主张严格监管，背后默认安全比自由更重要。另一个人主张放松管制，背后默认效率比公平更优先。

两个人争的不是事实，是没摊开的优先级。把这条优先级翻出来，争论才可能变成对话。
{{< /sentence >}}

{{< sentence no="4" quote="描述性假设关于世界怎样运作——不说出来，你就会当成已知条件跳过。" >}}
"减税能刺激投资"，背后假设企业拿到多余的钱会投入生产。"严刑能降低犯罪率"，背后假设犯罪者会理性计算代价。

这些关于世界运作方式的信念没有被论证，却在替论证承重。
{{< /sentence >}}

## 证据摆在那里不等于证据够用

{{< sentence no="5" quote="个人经验是最生动的证据，也是代表性最差的证据。" >}}
"我爷爷抽了一辈子烟活到九十"——生动、具体、有画面感。但一个案例能说明可能性，说明不了规律。

每种证据有自己的天花板。个人经验、权威引用、案例、研究数据、类比，Browne 和 Keeley 给每种都画了可信度边界。你不需要变成统计专家，但要知道手里这条证据最多能撑多大结论。
{{< /sentence >}}

{{< sentence no="6" quote="统计数据不会说谎，但使用它的人会替它选择说什么。" >}}
平均数、样本量、基数、时间窗口——改变其中任何一个，同一组数据可以支撑完全相反的结论。

遇到数字，先问三件事：谁选的这组数字？用的什么口径？换一种算法结论还站不站得住？
{{< /sentence >}}

## 只看到一个解释就停下来，是最常见的判断事故

{{< sentence no="7" quote="每当你看到一个因果声明，先问：有没有替代原因能解释同样的结果。" >}}
喝咖啡的人更长寿——可能是咖啡有益，也可能是喝咖啡的人整体生活习惯更健康。某培训后业绩上升——可能是培训有效，也可能同期市场回暖。

替代原因不是抬杠。它在确认你手里的解释是不是唯一合理的那个。
{{< /sentence >}}

{{< sentence no="8" quote="谬误之所以有效，是因为它们长得像好推理。" >}}
人身攻击、稻草人、虚假两难、滑坡论证、诉诸情感——这些推理错误之所以常见，是因为它们在日常对话里听起来很顺。

识别谬误不需要背清单。核心只有一条：这段推理跳过了什么？从理由到结论之间有一步被偷掉了，多半就是谬误在起作用。
{{< /sentence >}}
