{{< sentence >}}
波托西曾是世界上最富有的城市。它的白银建造了西班牙的宫殿和教堂——留给波托西的只有矿坑和贫困。财富的流向就是权力的流向。
{{< /sentence >}}

资源产出地和财富积累地可以完全不同。一个地方的"繁荣"可能建立在另一个地方的"榨干"之上。

{{< sentence >}}
拉丁美洲的欠发达不是发展的前一个阶段。它是发达国家发展的另一面——同一个硬币的两面。一方的富裕和另一方的贫穷由同一套机制制造。
{{< /sentence >}}

"你们还需要发展"这个说法掩盖了一个事实：你的"发展"可能正在制造别人的"欠发达"。

{{< sentence >}}
每一次拉丁美洲的出口繁荣都以同样的方式结束——价格下跌、资源枯竭、利润流出、留下一片废墟。从白银到糖到橡胶到石油，模式从未改变。
{{< /sentence >}}

依赖单一资源出口的经济结构是一个陷阱——繁荣时看起来像机会，枯竭时才发现是牢笼。

{{< sentence >}}
美国推翻危地马拉民选政府不是因为共产主义威胁。是因为联合果品公司的利益受到了威胁。地缘政治的语言经常是经济利益的伪装。
{{< /sentence >}}

当一个强大的外部力量声称出于"原则"干预你的事务时，先查查有没有具体的经济利益在驱动这个"原则"。

{{< sentence >}}
拉丁美洲的精英和外部掠夺者不是对立的。他们是同盟——本地精英从中间抽取佣金，外部掠夺者获得资源。真正被牺牲的是底层的劳动者和土地。
{{< /sentence >}}

掠夺通常需要本地代理人。外部力量不可能在没有内部合作者的情况下持续抽取资源。
