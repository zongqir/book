## 价值链与利润分布

{{< sentence >}}
同样的商品，设计的利润率往往是代工利润率的十倍以上。
{{< /sentence >}}

一件苹果手机，苹果的设计和品牌获得的利润，比富士康的代工利润大得多。这不是因为苹果工程师更勤劳，而是因为他们在价值链的高端。

{{< sentence >}}
一个国家的富有程度，取决于它在全球价值链中的位置，而不是居民的勤劳程度。
{{< /sentence >}}

日本人工资很高，因为日本企业掌握了高端产业。孟加拉国的工人一样很勤劳，但工资很低，因为只能做最低端的代工。

## 低端陷阱

{{< sentence >}}
陷入低端价值链后，很难单靠努力向上爬。因为成本优势是你仅有的竞争力。
{{< /sentence >}}

一旦工资上升，你的唯一优势就没有了。企业会转向更便宜的地方。

{{< sentence >}}
要升级，需要的不只是个人努力，还需要产业升级和战略转变。
{{< /sentence >}}

这需要国家和企业的配合，而不仅仅是个人的努力。

## 全球化的矛盾

{{< sentence >}}
全球化带来了效率提升，但也导致了收入差异的扩大。
{{< /sentence >}}

发达国家的高端企业获利最大。最贫困的国家的劳动者薪资最低。中间层被压缩。

{{< sentence >}}
对个人来说，提升自己在价值链上的位置，往往比"更勤劳"更重要。
{{< /sentence >}}

学习新技能、掌握高端产业的知识、往价值链上游移动，这些比单纯的努力更能提升收入。
