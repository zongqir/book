{{< sentence >}}
增长创造复杂性，复杂性是增长的沉默杀手。
{{< /sentence >}}

公司变大之后自然会增加层级、流程和审批。每一个都有合理的理由——但它们加起来的效果是让组织变慢、变迟钝。这是增长的悖论：你越成功就越容易被成功的副产品杀死。

{{< sentence >}}
大多数公司不是被竞争对手打败的，而是被自己内部的复杂性拖死的。
{{< /sentence >}}

祖克和艾伦的研究发现，增长失速的首要原因不是外部竞争而是内部问题——决策变慢、客户声音被层级过滤掉、每个人都在走流程而不是在解决问题。

{{< sentence >}}
创始人精神不需要创始人——它是一种可以被组织学习和保持的状态。
{{< /sentence >}}

很多人以为创始人精神只有创始人才有。但祖克和艾伦发现，持续增长的公司都找到了制度化保持创始人精神的方法——不是靠个人魅力而是靠组织设计。

{{< sentence >}}
一线执着意味着管理层必须持续接触真实的客户和业务细节。
{{< /sentence >}}

CEO最后一次和客户直接交谈是什么时候？如果超过一个月——一线信息已经被过滤了至少两层。你看到的"客户反馈"是被包装过的数据而不是真实的声音。

{{< sentence >}}
主人翁心态的对立面不是懒惰，而是"这不是我的事"。
{{< /sentence >}}

大公司里最常听到的一句话："这不归我管。"不是因为员工懒——是因为组织设计让每个人只对自己那一小块负责。主人翁心态意味着即使不是你的KPI，看到问题也会去解决。
