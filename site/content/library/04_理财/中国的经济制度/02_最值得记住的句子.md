经济增长的解释有很多种。张五常只用两样东西：产权界定和合约条款。

这组判断围绕同一个追问——谁拥有什么权利，谁和谁签了什么合约，条款如何决定行为。每一句都是分析工具，可以迁移到别的制度、别的组织、别的国家。

## 使用权先把活干了，所有权可以慢慢来

{{< sentence no="1" quote="（大意）产权的功能不要求法律上写明'私有'；使用权、收入权和转让权在合约中界定清楚，效率就能出来。" >}}

产权争论中最常见的死胡同：所有权归谁？

这个问题在很多转型经济体里没有答案，或者有答案但执行不了。换一个问法：谁有权使用？收入归谁？能不能转让？三项在合约里写清楚，激励就到位了。不需要先解决"谁是主人"。

这个切换把意识形态争论变成了技术问题：三项权利各自界定到什么程度？

调用场景：讨论国企改革时，把焦点从"该不该私有化"移到"使用权和收入权有没有界定清楚"。

{{< /sentence >}}

{{< sentence no="2" quote="（大意）联产承包就是在不动所有权的前提下，把使用权界定到能激励生产的程度。" >}}

土地还是集体的。但谁种、种什么、收成归谁，合约写得明白。干多少和收多少之间的关系被重新绑定，农民的行为立刻就变了。

调用场景：分析"产权不清晰但生产力突然上升"的案例时，先检查使用权和收入权是不是在合约层面被重新界定了。

{{< /sentence >}}

{{< sentence no="3" quote="（大意）土地使用权的有偿出让，把一块不能动的资产变成了地方政府最强的竞争筹码。" >}}

土地不能搬走，但使用权可以交易。地方政府拿使用权吸引投资、换基础设施、换税源。

前提只有一个：使用权已经从所有权中分离出来，成为独立的合约标的。分离不到位，这个机制就启动不了。

调用场景：讨论土地财政时，追问的起点不是"卖地好不好"，而是"使用权与所有权的分离边界画到了哪里"。

{{< /sentence >}}

## 合约形式不是选出来的，是交易费用逼出来的

{{< sentence no="4" quote="（大意）选择什么合约安排，不取决于什么最'合理'，取决于哪种交易费用最低。" >}}

同一个行业，有的地方用计件、有的用月薪、有的用分成。每种形式对应一组监督成本、度量成本和违约成本。哪种组合最便宜，哪种就会被选中。

这条判断把"应该"变成了"是"。不再问一个制度安排应该怎样，而是问它在当前交易费用约束下会怎样。

调用场景：分析一个组织为什么采用某种薪酬或治理结构时，先算交易费用，别急着判断"应该怎么做"。

{{< /sentence >}}

{{< sentence no="5" quote="（大意）层层分成比固定税额更能适应波动——收入好时多分，收入差时少分，双方都不会退出。" >}}

固定税额有一个硬伤：收入下降时，交不出的一方要么违约，要么破产。分成合约把波动风险分摊给所有参与者，合约本身更稳定。

中国从中央到地方的财政分成，底层逻辑就是这个。地方政府在经济下行时还能运转，不至于因为交不出固定数额而崩盘。

调用场景：设计收入分配机制时——合伙分成、加盟费、平台抽成——用这条判断固定费还是比例分成更稳。

{{< /sentence >}}

{{< sentence no="6" quote="（大意）制度变迁不需要顶层设计一步到位；合约条款逐条调整，累积起来就是系统性变化。" >}}

"大爆炸"和"渐进"的争论持续了几十年。从合约角度看，这个分类本身就粗糙。

实际发生的变化，很多既不来自顶层设计，也不来自刻意的渐进推进。一个条款改了，效果好，就改下一个。没有蓝图，但有方向。

调用场景：复盘制度变革时，不要只找"谁设计了这个"。先看最早改动的那条合约条款是什么，改完之后哪些相邻条款跟着动了。

{{< /sentence >}}

## 县是竞争的基本单位

{{< sentence no="7" quote="（大意）县与县之间争投资、争项目、争税源，竞争压力不亚于企业之间的市场竞争。" >}}

教科书里的竞争主体是企业。但在中国经济的实际运行中，最激烈的竞争常常发生在县与县之间。一个县税率太高、审批太慢、基础设施太差，投资就流向隔壁。

这种压力迫使每个县持续优化自己的"合约条件"——和企业改善产品没有本质区别。

竞争单位的发现比竞争机制的设计更重要。找对了单位，机制会自己长出来。

调用场景：分析地方经济差异时，不只看资源禀赋。看这个县面对的竞争压力有多大——邻居在干什么，投资者有没有替代选择。

{{< /sentence >}}

{{< sentence no="8" quote="（大意）从中央到镇，每一层都按比例分成；拿到的越多，留下的越多，每一层都有把蛋糕做大的动力。" >}}

层级组织里激励最难解决。上面怕下面偷懒，下面怕干多了好处全被拿走。

分成合约的解法很直接：每一层的收入和管辖范围内的产出挂钩，留成比例事先约定。干得好，自己留得多。激励来自合约条款本身，和道德号召无关。

调用场景：分析多层级组织（总部—区域—门店，平台—城市—骑手）的激励失灵时，检查收入分配是固定的还是按比例的。

{{< /sentence >}}

{{< sentence no="9" quote="（大意）合约可以在一个县试、一个省试；试成了推广，试砸了叫停，不必全国一起冒险。" >}}

很多制度创新死于"无法试错"——全面推开，或者一动不动。

合约的天然优势就在这里：局部签订、局部执行、局部观察。地理上可以隔离，失败成本可以封顶。经济特区、开发区、自贸区，都是这个逻辑的具体应用。

调用场景：推动组织内部制度变革时，先找一个可以签"局部合约"的试验田。不要等完美方案出来再全面铺开。

{{< /sentence >}}
