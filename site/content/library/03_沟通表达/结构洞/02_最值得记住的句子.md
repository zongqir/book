# 一想到人脉就往数量上冲时，先把这六句翻出来

{{< sentence >}}
关系最贵的部分，通常不是熟，而是不重复。
{{< /sentence >}}

如果你周围的人彼此高度相似，他们会带来很多确认，很少带来增量。非冗余，才是网络真正开始值钱的地方。

{{< sentence >}}
站在两个圈子之间，比站在一个圈子中心更容易先看见机会。
{{< /sentence >}}

团块中心有影响力，桥梁位置有信息差。两种都重要，但它们提供的优势不是同一种。

{{< sentence >}}
结构洞带来的第一份收益，往往不是资源，而是选择权。
{{< /sentence >}}

你先知道、先理解、先看到能不能接起来，于是你比别人更早拥有决定要不要推进的空间。

{{< sentence >}}
真正的桥梁位，不是认识两边的人，而是两边都愿意通过你。
{{< /sentence >}}

没有可信通行权，中间位置只是一种想象。可进可出，才叫位置。

{{< sentence >}}
闭合网络给你托底，结构洞给你上升空间，少了哪边都不完整。
{{< /sentence >}}

只守熟人圈，机会会越来越重复。只追桥梁位，信任和执行会越来越空。两种机制要分清，不要互相替代。

{{< sentence >}}
结构洞一旦被别人补上，原来的优势就开始折旧。
{{< /sentence >}}

所以别把某个位置当终身资产。真正可复利的是持续发现新断点的能力。
