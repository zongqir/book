{{< sentence >}}
"巴黎烧了吗？"——希特勒在电话里反复追问这个问题。他要的不是军事目标的摧毁而是一座文明象征的毁灭。战争到了最后阶段，破坏本身成了目的。
{{< /sentence >}}

当破坏不再服务于军事目标而成为纯粹的报复时——命令的合法性已经碎了。执行者是否还有服从的义务？

{{< sentence >}}
肖尔铁茨是一个一辈子服从命令的军人。让他不服从的不是勇气——而是面对卢浮宫和巴黎圣母院时意识到有些东西比自己的军人荣誉更重要。
{{< /sentence >}}

不服从一个命令需要另一种更高的忠诚来支撑。肖尔铁茨找到的支撑点不是政治信念而是文明本身。

{{< sentence >}}
炸药已经安装好了。按钮已经准备好了。在历史的那个时刻，巴黎的命运取决于一个人的手指——而不是任何大趋势或制度力量。
{{< /sentence >}}

大多数时候历史是趋势和制度的产物。但在某些压缩到极致的时刻，一个人的选择权重可以超过一切结构性因素。

{{< sentence >}}
巴黎的地下抵抗组织在盟军到达之前自行起义——不是因为时机成熟而是因为他们无法再忍受等待。情感的时间表不服从军事的时间表。
{{< /sentence >}}

人的忍耐有限。计划再完美如果忽略了人的情绪极限就会被提前打断。

{{< sentence >}}
勒克莱尔的法国装甲师被要求等盟军统一部署再进入巴黎。他没等——因为他知道巴黎人等不起。有时候不服从上级命令是唯一正确的选择。
{{< /sentence >}}

执行者有时比指挥者更了解现场——当现场信息和总部指令矛盾时，在场的人承担的风险和责任更大。
