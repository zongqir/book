## 视角与距离

{{< sentence >}}
选择叙事视角不是选择谁来讲故事，是选择读者被允许知道多少。
{{< /sentence >}}

第一人称限制了信息量但增加了亲密感。全知视角提供了全局但牺牲了悬念。选视角的核心问题不是"哪个更好"，而是"这个故事的张力来源于读者知道得多还是知道得少"。

{{< sentence >}}
叙事距离决定了读者是站在角色身边还是站在远处看。距离一变，同一个场景的情绪温度完全不同。
{{< /sentence >}}

同一个死亡场景，近距离叙述让你心碎，远距离叙述让你沉思。新手写作者的常见问题不是选错了距离，而是在一个段落里距离忽远忽近——读者的情绪被颠来倒去，最终什么感觉都没留下。

## 节奏与结构

{{< sentence >}}
节奏快不是字数少，是信息密度高。节奏慢不是拖沓，是给读者消化的时间。
{{< /sentence >}}

三页纸的追车戏可以很快——如果每一句都在推进动作。半页的对话可以很慢——如果两个人在互相试探而什么都不说明。判断节奏的标准是信息推进的速度，不是文字的长短。

{{< sentence >}}
场景转换是节奏的换挡器。连续三个同节奏的场景，读者就开始走神。
{{< /sentence >}}

紧张的打斗之后接一段安静的独白。密集的对话之后接一段景物描写。换挡不是打断，是让读者的注意力重新聚焦。

## 细节与对话

{{< sentence >}}
好的细节不是描述环境，是泄露人物。
{{< /sentence >}}

"桌上有一杯凉掉的咖啡"——这不是在写环境。这是在说一个人太专注于某件事而忘了喝咖啡。细节的功能不是填充画面，是用最小的篇幅传递最大的信息量。

{{< sentence >}}
对话不是两个人轮流发表观点，是两个人在说话的同时做别的事情——试探、隐瞒、暗示、逃避。
{{< /sentence >}}

真实的对话里，人说出来的往往不是他真正想说的。文字表面在聊天气，潜台词在确认关系。好的对话有两层：读者能听到的，和读者能感觉到但没被说出来的。
