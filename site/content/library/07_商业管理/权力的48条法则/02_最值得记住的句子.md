格林的写法本身就是箴言体。每条法则的标题就是一句可以直接调用的判断。下面八句按提醒价值排列——排在前面的，是最容易被忽视、代价也最大的。

## 信号与自我暴露

{{< sentence >}}
永远不要盖过主子的光芒。让在你之上的人永远感到自己高人一等。
{{< /sentence >}}

富凯用一场晚宴毁掉了自己。多数人的本能是在上级面前展现才能，争取认可。格林说这恰好搞反了：让上级感到被威胁，你展现的能力越强，你的处境越危险。

{{< sentence >}}
少说多听。你说得越多，越容易说出蠢话。
{{< /sentence >}}

沉默在权力场里是一种资源。话多的人不断暴露自己的判断、倾向和弱点。而沉默的人保留了解读空间，让别人猜不透。

## 依赖与不可替代性

{{< sentence >}}
让别人来为你工作，但永远记得把功劳归于自己。
{{< /sentence >}}

这条法则刺耳，但格林给出的理由很务实：在权力场里，被看见的人获得权力，干活的人获得疲劳。如果你只会埋头做事等别人来认可，你就把自己的权力命运交给了别人的注意力。

{{< sentence >}}
让别人依赖你。维持独立不如维持不可替代性。
{{< /sentence >}}

喜欢你的人可以在下一秒不喜欢你。但依赖你的人不会轻易和你翻脸，因为拆除依赖关系的成本比忍受你更高。

## 信息与节奏

{{< sentence >}}
隐藏你的意图。让对手永远不知道你的下一步。
{{< /sentence >}}

过早暴露计划给了对手反制的时间。塔列朗在法国五次政权更迭中全身而退；他从不让任何一方确定他站在哪一边。不可预测性本身就是一种保护。

{{< sentence >}}
用选择性的诚实和慷慨来瓦解对手的防线。
{{< /sentence >}}

一个真诚的举动可以掩盖十个隐秘的意图。格林指出，完全不可信的人反而安全——因为所有人都在防备。最危险的是那些"看起来很真诚"的人。

## 战场选择

{{< sentence >}}
要学会适时投降。投降是一种工具，为你赢得时间和空间。
{{< /sentence >}}

多数人把投降等同于失败。格林把它重新定义为一种策略——在力量不够的时候保存实力，等待反击的时机。硬撑到底是勇气，但也可能是愚蠢。

{{< sentence >}}
轻视你无法得到的东西。忽视是最好的报复。
{{< /sentence >}}

你越关注得不到的东西，它在你生活里的分量就越大。格林的建议是：移开注意力就等于消解了对方的权力。不回应，本身就是一种权力行为。
