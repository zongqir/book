## 关于表现与看见

{{< sentence >}}
别人对你的第一印象，不是你装出来的样子，而是你在乎他们印象时泄露的焦虑。
{{< /sentence >}}

你越想让别人喜欢你，对方越能感受到你不是在和他互动，而是在和他的评价互动。放松下来吧，大多数人对你没那么在乎。

{{< sentence >}}
人际关系最有效的开场，不是展示你有多好，而是好奇他是怎样的。
{{< /sentence >}}

问一个真诚的问题，听对方回答。这比任何自我介绍都有效。

## 关于冲突与需求

{{< sentence >}}
每次对方生气，背后都是你忽略了他的一个需要。学会找到那个需要，冲突就有了出口。
{{< /sentence >}}

别问"你为什么生气"。问"你现在最需要什么"。

{{< sentence >}}
不接纳对方的负面情绪，就是在说：我只在乎你对我有利的部分，你的真实对我没价值。
{{< /sentence >}}

当伴侣或朋友难过时，你第一反应是想让他快点好起来，其实是在逃避他的情绪。完整的陪伴，包括陪他难过。

## 关于信任与一致性

{{< sentence >}}
信任不是因为别人有多好，是因为别人做的和说的一样。
{{< /sentence >}}

一个有缺点但可靠的人，比一个完美但不稳定的人更值得信任。承诺兑现才能赚到信任。

{{< sentence >}}
你对关系的态度，别人能从你的细节里看出来。记住他说的话，比任何大承诺都更有力。
{{< /sentence >}}

下次约见面，试试看：他上次说的那件事，你还记得吗？

## 关于边界与关心

{{< sentence >}}
拒绝别人，往往是保护这段关系的最好方式。
{{< /sentence >}}

说"不行"的人比说"可以"的人更值得相信。因为他的承诺都是经过思考的。

{{< sentence >}}
你为了讨好付出的代价，最终会被计入这段关系的成本。对方会感到有欠，而不是感到被爱。
{{< /sentence >}}

有原则地拒绝，才能有质量地给予。

## 关于倾听与共鸣

{{< sentence >}}
倾听不是等对方说完，而是真的想听见他在说什么。区别在注意力。
{{< /sentence >}}

下次和朋友聊天，如果你发现自己在组织回应，说明你没在听。停下来，清空头脑，再试一次。

{{< sentence >}}
共鸣不是"我也经历过"，而是"我能感受到你为什么会这样"。少说你自己，多问他怎样。
{{< /sentence >}}

把对话从"我也…"转向"你是怎样…"，关系深度会完全不同。
