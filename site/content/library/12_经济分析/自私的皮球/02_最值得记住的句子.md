## 关于无私的伪装

{{< sentence >}}
当一个人说"我是为了你好"，他可能同时是利己的。他想要被感谢、被记住、被看见。
{{< /sentence >}}

{{< sentence >}}
最危险的无私，是那些自己都相信了自己的无私的人。因为这时候，即使有人指出他的自私，他也会更加坚定地否认。
{{< /sentence >}}

## 关于无奈和被迫

{{< sentence >}}
"我被迫这样做"往往意味着"有很多选择，但这个对我最有利"。
{{< /sentence >}}

{{< sentence >}}
一个人如果真的无路可走，反而会坦白说出来。那些强调自己"被迫"的人，往往有选择，但不想承认选择意味着的负责。
{{< /sentence >}}

## 关于牺牲

{{< sentence >}}
每一个"我的牺牲"故事，都应该被理解为"我用这个代价换取这个好处"。一旦你这样理解，就能更清楚地看到这笔交易是否真的值得。
{{< /sentence >}}

{{< sentence >}}
最让人窒息的牺牲，来自默默牺牲的人。因为他们既获得了牺牲的好处（道德高地、权力、控制），又拒绝被看见为在交易。
{{< /sentence >}}

## 关于好人的标签

{{< sentence >}}
一个人对"好人"标签的执着，往往和他需要这个标签的程度成正比。
{{< /sentence >}}

{{< sentence >}}
真正的好行为，不需要时时被确认和赞美。需要不断被确认的"好"，其实是在喂养一个不安的自我。
{{< /sentence >}}
