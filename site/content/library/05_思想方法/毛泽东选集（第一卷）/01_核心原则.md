# 五条判断准则：从阶级分析到实践校正

方法细节会忘，年代语境会变，但有些判断方向不该丢。

这组文章一直在约束同一种误判：现实还没看清，结论已经先到了。下面五条准则提炼的就是这个方向。方法可以因题目而变，方向一丢，后面的分析和动作都容易越走越偏。

{{< principle no="1" type="core" statement="判断必须从具体结构起步，不能从抽象口号直接推出有效动作" >}}
最容易先出问题的习惯：先有结论，再去现实里找例子。

《中国社会各阶级的分析》给出的起手式是先拆局面里的参与者、利益关系和受压位置。口号可以鼓舞人，但口号替代不了结构判断。

常见误读是把「回到结构」理解成不需要站位。站位当然要有，但站位建立在看清结构之后，不能拿来替代分析。

边界：不是所有问题都要拆成宏大结构。但只要题目涉及群体位置、利益分布和力量关系，这一步就不能跳过。
{{< /principle >}}

{{< principle no="2" type="core" statement="力量必须先集中在主要矛盾上，平均用力是最常见的失焦" >}}
《矛盾论》反复压的一件事：局面越复杂，越要压缩焦点。先找决定全局走向的那条矛盾，集中力量打穿。

平均用力听起来全面，实际最容易失焦。什么都重要，等于什么都没打穿。

常见误读是把「抓主要矛盾」当成别的问题都不管。次要问题当然存在，只是顺序不同。先后错了，主次一起坏。

还有一层：只盯议题、不盯主力也会落空。问题看对了，力量判断错了，动作一样悬空。
{{< /principle >}}

{{< principle no="3" type="support" statement="理论只有在实践里来回碰撞，才不会僵成口令" >}}
《实践论》给出的结构不是「理论指导实践」这么简单的单向链。它要求理论进入现实，被现实修正，再回来改判断。

约束的是两种偷懒：书本崇拜和经验崇拜。只靠引文，判断脱离现实；只靠经验，判断陷在局部。

更硬的一层：实践不是做完就算。反馈必须回来拆你的原判断。现实已经给出反证还不肯改，理论就从工具变成了护身符。
{{< /principle >}}

{{< principle no="4" type="support" statement="表面强弱不能直接当终局，要追问条件和裂缝" >}}
《星星之火，可以燎原》处理的核心问题：弱势局面该怎么看。

它继续追问：对手内部有没有裂口，本地有没有可以站住的条件，真实需要有没有在积累。条件在变化，表面强弱只是某一刻的截面。

常见误读是把它当成盲目乐观。边界恰恰在这里：裂缝和条件必须真实存在，不能靠意志补出来。

纠正的是把表面强弱直接当终局的偷懒，不是否认约束本身。
{{< /principle >}}

{{< principle no="5" type="support" statement="组织内部最危险的错，常常是方向先歪了却只在执行力上加码" >}}
《关于纠正党内的错误思想》处理的就是这类问题：错误不在不努力，在方向本身有偏。

路线、认识和组织目标一旦偏了，执行越猛，偏得越快。纪律能放大方向，不能代替方向。

今天尤其容易被忽略。团队一遇到受挫就加会、加报表、加考核。如果问题定义先错了，组织越整齐，离真实问题只会越远。

切口不是放松纪律，是先停下来问：我们到底在解决什么问题，这个定义是不是已经错了。
{{< /principle >}}

五条放在一起，方向很清楚：先看现实，先看位置，先看条件，再决定动作和节奏。

以后回头看这些文章，最值得检验的不是「我还记得多少内容」，是这些准则有没有在你准备走偏时先一步冒出来。
