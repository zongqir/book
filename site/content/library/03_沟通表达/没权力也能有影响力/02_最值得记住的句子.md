# 没权力也能有影响力里最该留下来的，是这些会在关键时候拽住你的提醒句

{{< sentence >}}
没权力时，先别问别人为什么不配合，先问他为什么要配合。
{{< /sentence >}}

这句能把人从委屈里拉回现实。

{{< sentence >}}
影响力不是抢赢一场争论，是让别人也有理由动。
{{< /sentence >}}

只靠你自己想得通，事情推不起来。

{{< sentence >}}
职位不能替你建立信任，方案也不能自动替你换资源。
{{< /sentence >}}

很多人输在把“我说得对”误当成“别人就该支持”。

{{< sentence >}}
联盟不是点赞，是有人真的愿意为你承担一点成本。
{{< /sentence >}}

没有成本，就还不算联盟。

{{< sentence >}}
横向推动最怕把组织政治想得太脏，于是干脆假装它不存在。
{{< /sentence >}}

看不见它，就会反复撞墙。

- 一开口就想解释很多的时候：先翻这页。
- 你觉得自己明明没说错，结果事情还是做坏的时候：先翻这页。
- 你知道问题在哪，却迟迟不知道从哪一句开始的时候：先翻这页。
- 对方的反应、组织的惯性或关系的压力开始把你带偏的时候：先翻这页。
