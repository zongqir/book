# 写故事前，先把这几句钉在桌上

## 最该反复调用的提醒

{{< sentence no="1" quote="故事不是事件堆，它是人物在压力下被迫变化的线。" >}}
事情很多，不等于故事已经成立。

只要人物没有被逼到必须选择，情节再满也只是流水账。
{{< /sentence >}}

{{< sentence no="2" quote="人物想要什么，必须具体到这一场戏里能争、能失、能错过。" >}}
目标一虚，冲突就会跟着发虚。

“想幸福”太大，“今晚必须拿到这份证据”才足够推动场景。
{{< /sentence >}}

{{< sentence no="3" quote="冲突不是热闹，而是阻力真的咬住了欲望。" >}}
吵架、打斗、喊叫都不自动等于冲突。

只有当阻力会改写人物能不能拿到目标时，张力才会起来。
{{< /sentence >}}

{{< sentence no="4" quote="每一场戏都该让局面偏转，而不是只多给一点信息。" >}}
场景写完以后，如果关系、风险和判断都没变，这场戏多半太轻。

设定说明可以补知识，转折才能推动故事。
{{< /sentence >}}

{{< sentence no="5" quote="能靠动作暴露的，就不要先靠解释交代。" >}}
人物值不值得信，不该先写在旁白里。

让他在选择里露出来，比任何说明都更有力。
{{< /sentence >}}

{{< sentence no="6" quote="反转不是突然发生，而是前面因果累到这里才不得不发生。" >}}
硬拐弯会让人惊一下，却留不住。

真正站得住的转折，回头看时应该觉得它早就埋在前面了。
{{< /sentence >}}

{{< sentence no="7" quote="结尾不是把信息收完，而是把最核心的欲望线收紧。" >}}
故事结束时，观众最想确认的不是你还有多少设定没讲。

他们更在意人物最后成了什么，代价落在了哪里。
{{< /sentence >}}

{{< sentence no="8" quote="主题最好从后果里长出来，不要抢在前面替故事发言。" >}}
一句“我想表达什么”本身撑不起故事。

先让人物去做、去错、去承担，主题才会自己浮出来。
{{< /sentence >}}

## 这些句子最适合在哪些场景拿出来

- 写了三页梗概，还说不清主角这一刻到底要什么，先回到第 2 句。
- 场面很热闹，却总觉得读者不紧张，先回到第 3 句。
- 一场戏写完只多了说明，没有多出变化，先回到第 4 句。
- 人物介绍很多，形象还是立不住，先回到第 5 句。
- 反转像机关，不像结果，先回到第 6 句。
- 结尾在收设定，没收人物命运，先回到第 7 句和第 8 句。

## 真正的用法不是背句子，而是拿它们做删改

写故事最怕一口气往前冲。

更稳的做法，是每写完一场戏就回来查几句：人物这场到底要什么？阻力是不是够硬？局面有没有偏转？主题是不是还躲在动作后面？只要你开始这样回看，很多问题会在定稿前自己露出来。
