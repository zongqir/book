## 关于财富转移

{{< sentence >}}
房价上升不是中产变富，而是财富在重新分配。
{{< /sentence >}}

有资产的人通过资产升值获利；没有或少有资产的人则通过更高的生活成本被变相减薪。

{{< sentence >}}
只有卖出才能变现，不卖就只是账面数字。
{{< /sentence >}}

很多人的"虚拟财富"其实无法支撑生活中的真实需求。

## 关于货币

{{< sentence >}}
你的工资看起来在增加，但它相对于资产的升值在贬值。
{{< /sentence >}}

名义增长≠实际增长。关键要看实际购买力。

{{< sentence >}}
通货膨胀是对有存款的人的隐形征税。
{{< /sentence >}}

年利息2%的定期存款，但物价涨3%，你的购买力在下降。

## 关于结构

{{< sentence >}}
你个人的努力改变不了结构性趋势。
{{< /sentence >}}

不是在否定努力，而是说明有些困局不靠个人就能突破。

{{< sentence >}}
中产的危险在于被固定支出锁死，没有调整的灵活性。
{{< /sentence >}}

房贷、教育、医疗一旦承诺，就很难改变。

## 关于选择

{{< sentence >}}
没有绝对的"应该怎样"，只有基于认知的"能够怎样"。
{{< /sentence >}}

理解了困局后，你才能做出符合自己处境的决定。

{{< sentence >}}
有些选择不是"成功"或"失败"，而是"风险程度不同"。
{{< /sentence >}}

背房贷和不背房贷都有代价，关键是你是否理解了代价。
