下面五条原则不是康德哲学的概述，是从《纯粹理性批判》的论证中提取出来的判断约束。它们的用途是：当你遇到一个关于"人能不能知道X"的主张时，用这些约束检查它站不站得住。

{{< principle no="1" type="core" >}}
## 认识是主体构建的，不是被动接收的

在感官接触到任何对象之前，人已经带着一套认知装置在工作了。空间和时间是感知的先天形式——你无法感知到一个"不在空间中"或"不在时间中"的东西。因果、实体、数量这些范畴是知性的组织工具——没有它们，感官材料就是一堆无法理解的杂多。

这条原则的判断约束：任何关于"客观世界本来是什么样"的主张，都需要先回答一个问题——你怎么确定你描述的是对象本身，而不是你的认知装置加工之后的产物？
{{< /principle >}}

{{< principle no="2" type="core" >}}
## 离开经验的纯概念推理不产生知识

概念没有直觉——也就是感官提供的具体内容——就是空的。一个推理链条无论多严密，如果前提全部是纯概念、没有任何经验材料参与，它的结论就不构成知识。

康德用这条原则终结了一类持续两千年的论证：用纯概念推导出上帝存在、灵魂不朽、世界有开端。这些推理的逻辑都没有错，但前提没有经验内容，结论虽然不可反驳，也不构成知识。

判断约束：遇到一个看起来完美的论证，先问——前提有没有经验着陆点？如果全部是定义和概念之间的推演，这个论证的结论最多是"逻辑上可能"，不是"实际上为真"。
{{< /principle >}}

{{< principle no="3" type="support" >}}
## 时间和空间是感知的先天形式

时间和空间不是从经验中归纳出来的概念，也不是独立于人存在的容器。它们是人接收感官信息的方式——人只能在时间和空间中感知事物，就像人只能用眼睛接收光信号。

这条原则支撑了一个困扰哲学家很久的问题：数学为什么既不是纯经验也不是纯定义？几何学的命题——比如"三角形内角和等于180度"——不需要做实验就知道是真的，但它又不是空洞的同义反复。康德的解释：几何学描述的是空间直觉的结构，而空间直觉是先天的。

判断约束：当有人说"时间和空间只是人的错觉"或者"时间和空间是独立于人的绝对存在"时，康德的位置在两者之间——它们是真实的，但它们是人类认知的结构，不是物自体的属性。"物自体"指事物脱离人的认知条件之后的样子——康德认为它存在但不可知。
{{< /principle >}}

{{< principle no="4" type="support" >}}
## 范畴是知性整理经验的必备工具

感官提供材料，但材料本身不是知识。把一堆颜色、形状、温度变成"这是一张桌子""火导致热"，需要知性用范畴来组织。范畴——因果、实体、数量、可能性——是知性自带的组织工具，不是从经验中学来的。

先验演绎的核心论证就在这里：如果没有范畴，经验本身就不可能。你不可能拥有一个没有因果关系、没有实体概念、没有数量区分的经验。范畴不是可选的思维习惯，是经验成立的必要条件。

判断约束：遇到"因果关系只是人的主观联想"这类说法时，康德的追问是——如果取消因果范畴，你还能拥有连贯的经验吗？范畴的有效性不是靠归纳确认的，是靠"取消它则经验崩溃"来保证的。
{{< /principle >}}

{{< principle no="5" type="support" >}}
## 理性在经验之外会制造系统性幻觉

理性有一个内置倾向：把有条件的知识推向无条件的统一。在经验范围内，这个倾向很有用——它驱动科学不断追问更深的原因。但一旦越出经验范围，这个倾向就变成陷阱。

康德把这种陷阱叫"辩证幻相"。它不是偶尔犯的错，是理性的结构性倾向。表现形式就是二律背反：世界有没有开端，自由意志存不存在——正反两方都能给出同样严密的论证，因为双方的推理链条都脱离了经验、在纯概念层面空转。

判断约束：当你发现正反两个论证都无懈可击，而且你无法想象任何经验能判定哪边对的时候，最可能的状况不是"真理在其中一方"，而是"这个问题不在理性的有效射程内"。
{{< /principle >}}
