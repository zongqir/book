# 安全感破了，什么道理都传不过去

## 识别对话何时变得危险

{{< sentence no="1" quote="对话之所以失败，不是因为内容有问题，而是因为对方不再觉得安全。" >}}
你发现自己越讲越认真、对方越来越冷的时候，停一下。问题不在你说了什么——在于安全感已经碎了。
{{< /sentence >}}

{{< sentence no="2" quote="人在觉得不安全的时候，只有两种反应：沉默或暴力。" >}}
沉默不是同意，是退出。暴力不是反驳，是防御。看到这两个信号，先别急着推内容——回去修氛围。
{{< /sentence >}}

{{< sentence no="3" quote="你越是认为自己对，就越容易忽略氛围正在崩。" >}}
正确感是氛围判断力的天敌。越确信自己有道理的时候，越容易忽略对方的脸已经变了。
{{< /sentence >}}

## 修复安全的两个抓手

{{< sentence no="4" quote="在对方觉得你的目的和他的利益冲突时，任何技巧都不管用。" >}}
安全感的基座是共同目标。对方相信你们想要的东西有交集，才愿意继续说下去。没有这个基座，STATE 五步走得再标准也是白搭。
{{< /sentence >}}

{{< sentence no="5" quote="先说你不想造成的结果，再说你的意图。" >}}
"我不是要追责，我是想搞清楚接下来怎么合作。"这两句话的顺序很重要。先排除威胁，再亮出目的，对方的防御才可能松开。
{{< /sentence >}}

{{< sentence no="6" quote="当你在对话中觉得自己是受害者，停下来问一问——你漏掉了什么事实。" >}}
帕特森管这叫"审视你的故事"。你觉得委屈的那一刻，你看到的事实通常是经过筛选的。把被你忽略的那部分捡回来，情绪会自己降温。
{{< /sentence >}}

## 从事实开口，一步步走到真话

{{< sentence no="7" quote="从事实开始，不要从判断开始。" >}}
"上周五你说周一交付，现在周三了"是事实。"你不靠谱"是判断。同一件事，事实对方能接住；判断当场就炸。
{{< /sentence >}}

{{< sentence no="8" quote="说出你的想法，但把它当作一种可能，而不是既定事实。" >}}
"我担心时间线会继续滑"比"你肯定又要拖"多了一个转圜空间。试探语气不是软弱，是给对方留一个进入对话的门。
{{< /sentence >}}

{{< sentence no="9" quote="真正问对方怎么看——然后闭嘴听。" >}}
征询不是走过场。问完"你怎么看"之后不等对方说完就接话，跟没问一样。对方的沉默需要时间变成语言，忍住前十秒。
{{< /sentence >}}
