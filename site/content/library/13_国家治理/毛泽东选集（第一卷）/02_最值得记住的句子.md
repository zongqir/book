# 第一句就要先把你拽回现实

{{< sentence >}}
先问中国是什么样，再问革命该怎么走。
{{< /sentence >}}

很多判断一上来就错，不是因为后面推理不努力，而是起点不是现实，是先验想象。读第一卷，这句最该排前面。

{{< sentence >}}
谁是主力，谁会摇摆，谁能联合，都不能靠好恶来定。
{{< /sentence >}}

这句专门纠正“我喜欢谁就觉得谁可靠”的冲动。第一卷最硬的地方，就是逼你回到力量关系本身。

{{< sentence >}}
弱不等于没有路，强也不等于已经稳了。
{{< /sentence >}}

遇到最坏局面时，这句最有用。它逼你别急着判死，也别只看表面声势。

{{< sentence >}}
没有调查，很多自信只是借来的。
{{< /sentence >}}

这句比“没有调查没有发言权”更该落到今天的动作里。材料不是装饰，是判断资格本身。

{{< sentence >}}
路线先歪了，后面越整齐，偏得越快。
{{< /sentence >}}

这句用来挡住“先加码执行再说”的冲动。第一卷一直在提醒，方向错了，纪律和热情都救不了。

{{< sentence >}}
主要矛盾不是背出来的，是在具体处境里重新判出来的。
{{< /sentence >}}

很多人会说“抓主要矛盾”，真到现实里却只会挑一个顺手的大词。这句专门防这个。

{{< sentence >}}
理论不回到实践里改，就会从工具变成护身符。
{{< /sentence >}}

这句最适合放在争论卡住、概念越来越熟、现实越来越模糊的时候看。

{{< sentence >}}
现实不是替你证明早就对了，它是来拆你原判断的。
{{< /sentence >}}

这是第一卷后半最该留下的一句狠话。真会用的人，最先学会的往往不是坚持，而是改判。

## 这些句子什么时候最该翻出来

- 一开口就想表态时：先看第 1、2 句。
- 一碰见弱势局面就想判死时：先看第 3 句。
- 材料还很薄，结论已经很重时：先看第 4 句。
- 出了问题就想靠加压补救时：先看第 5 句。
- 什么都重要、什么都想抓时：先看第 6 句。
- 理论说得越来越顺，现实却越来越虚时：先看第 7、8 句。

第一卷最该留下来的句子，不是拿来显得懂，而是拿来挡住自己最容易犯的那些错。
