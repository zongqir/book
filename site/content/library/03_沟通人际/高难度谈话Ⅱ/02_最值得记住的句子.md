## 反馈的三张面孔

{{< sentence no="1" quote="反馈有三种：感谢、指导、评价。混在一起给，听的人只会接住评价，丢掉指导。" >}}
{{< /sentence >}}

大部分反馈对话出问题的根源：给的人在指导，听的人以为在被打分。分辨反馈类型，是接住反馈的前置动作。不分辨就接，等于闭着眼接球。
{{< sentence no="2" quote="你不需要接受所有反馈。但你需要先听懂它，才有资格拒绝。" >}}
{{< /sentence >}}

拒绝和屏蔽是两回事。屏蔽是情绪驱动——对方一开口你就关上了门。拒绝是判断驱动——听完了、看清了，然后决定不采纳。前者是逃跑，后者是选择。
## 触发器在替你做决定

{{< sentence no="3" quote="你对反馈的第一反应，暴露的不是反馈质量，而是你的触发器类型。" >}}
{{< /sentence >}}

生气、委屈、想反驳。这些反应看起来像在评估反馈，其实是触发器替你做了决定。识别触发器那一刻，才是主动权回到你手上的起点。
{{< sentence no="4" quote="当你在想&quot;凭什么是你来说这话&quot;，你已经从内容切换到了关系。" >}}
{{< /sentence >}}

关系触发器最隐蔽。你觉得自己在讨论对错，其实在纠结资格。内容和来源必须拆开处理，否则有价值的反馈会因为来源不顺眼而被丢掉。
## 身份震荡与第二分数

{{< sentence no="5" quote="一条反馈的杀伤力，往往来自它踩中了你对自己的某个核心定义。" >}}
{{< /sentence >}}

"我一直以为自己是个好父亲。"身份触发器的杀伤力最大——它挑战的不是你今天做了什么，而是你认为自己是谁。反馈的内容可能很小，但它戳中的东西可以很大。
{{< sentence no="6" quote="你对一条反馈的处理方式——你的&quot;第二分数&quot;——比反馈本身更能定义你的成长轨迹。" >}}
{{< /sentence >}}

老板给了差评，这是第一分数。你因此消沉两周、跟同事抱怨三轮、回家冲家人发火——这是第二分数。你能控制的、也真正决定成长方向的，是第二分数。
## 接收的边界

{{< sentence no="7" quote="接收反馈的能力，有一半是知道什么时候该拒绝。" >}}
{{< /sentence >}}

会接反馈的人也会拒绝反馈。关键区别在于：拒绝的决定是在理解之后做出的，而不是在情绪里完成的。没有边界的接收等于讨好。
