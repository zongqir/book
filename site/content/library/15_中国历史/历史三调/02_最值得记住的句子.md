{{< sentence >}}
历史作为事件、历史作为经历、历史作为神话——三者之间的距离，就是理解和误解之间的距离。
{{< /sentence >}}

柯文的核心框架。事件是"客观发生的"，经历是"主观感受到的"，神话是"后人重新塑造的"。混淆三者是绝大多数历史误读的来源。

{{< sentence >}}
经历者眼中的历史和历史学家眼中的历史几乎是两种东西。
{{< /sentence >}}

历史学家看到的是结构、因果、趋势。经历者看到的是恐惧、饥饿、不确定性。两种视角都是真实的，但它们很少能对上。

{{< sentence >}}
神话不是谎言——它是一种有选择的真实，选择本身服务于当下的需要。
{{< /sentence >}}

义和团在不同时期被塑造成不同的形象。每个形象都包含真实的元素，但被选择和组合的方式反映的是塑造者的需要而不是事件本身。

{{< sentence >}}
干旱、饥荒、失业——义和团运动的直接动因不是"反帝爱国"，而是生存危机。
{{< /sentence >}}

柯文提醒：宏大叙事往往遮蔽了最朴素的因果关系。人们加入义和团首先是因为活不下去，而不是因为有了政治觉悟。

{{< sentence >}}
同一个事件在不同人的经历中可以同时是解放和灾难。
{{< /sentence >}}

对拳民来说义和团运动是一次赋权，对教民来说是一场屠杀，对清廷来说是一次赌博。单一叙事无法容纳这种矛盾，而矛盾本身就是历史的真实面貌。
