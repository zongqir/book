## 颠倒顺序

{{< sentence >}}
你以为自己在想清楚了再写。但大脑不是这样工作的——写出来的过程就是想清楚的过程。
{{< /sentence >}}

脑子里的想法永远是模糊的、片段的、跳跃的。你觉得自己"已经想好了"，一开口就发现组织不起来。因为"想清楚"需要一个外部化的载体——笔和纸。把念头逼到纸上，你才能看见它们的形状。

{{< sentence >}}
一分钟之内写不出来的想法，给你五分钟也写不出来。多出来的四分钟全用在自我审查上了。
{{< /sentence >}}

一分钟限制的精妙之处在于：它逼你放弃修饰和审查。你没时间想"这样说好不好"，只能写出第一个冒出来的东西。那个"第一个冒出来的东西"通常就是你真正的想法——只是你平时被审查机制挡住了。

## 量变产生质变

{{< sentence >}}
每天十张备忘录，连续做三个月。不是写得好不好的问题，是量变引起的思考能力升级。
{{< /sentence >}}

十张听起来很多，但每张只需要一分钟。十分钟的事。赤羽强调的不是单张备忘录的质量，而是累积效应。三个月是九百张备忘录。九百次"把模糊念头逼成清晰文字"的练习。这个量级的训练会改变你的思考速度和清晰度。

{{< sentence >}}
标题写得越具体，一分钟里能写出的内容越多。"工作" 太大了；"明天早上和客户会议我最想确认的三件事"刚好。
{{< /sentence >}}

标题是思考的入口。入口越窄，一分钟内能到达的深度越大。宽泛的标题让你在一分钟里到处打转，什么都碰了一下但什么都没深入。具体的标题让你直接进入核心。

## 写下来的价值

{{< sentence >}}
模糊的焦虑写下来之后，体积会缩小一半。因为你看清了它到底是什么。
{{< /sentence >}}

你脑子里有一团说不清的焦虑。感觉很大、很重。写下来之后发现：它其实只有三件事——明天的会议、下周的截止日期、一个还没回的邮件。三件具体的事，每件都可以处理。焦虑的体积之所以大，是因为你没把它拆开看。

{{< sentence >}}
好的标题从你此刻最困扰的事情开始。不要想"写什么有价值"——写让你现在最难受的那件事。
{{< /sentence >}}

赤羽的建议是：别把 A4 备忘录当成规划工具。它是思考工具。最好的标题就是你此刻最烦的那件事。因为那件事最需要被外部化——在脑子里转来转去只会越来越重。
