{{< sentence >}}
在没有银行的社会里，人际关系就是信用体系。
{{< /sentence >}}

黄东林能借到钱不是因为他有抵押物，而是因为他在关系网络中的声誉。声誉替代了征信报告，宗族替代了担保机构。这不是落后——这是在特定条件下的合理解决方案。

{{< sentence >}}
一个家族的兴衰不取决于某个人的能力，而取决于这个家族在关系网络中的位置。
{{< /sentence >}}

黄东林和黄茂林的分化不是因为能力差异，而是因为一方在关键时刻维持住了关系网络的核心位置，另一方没有。位置决定资源流向。

{{< sentence >}}
婚姻不只是两个人的事，它是两个家族资源网络的合并。
{{< /sentence >}}

在金翼的世界里，嫁女儿和风险投资在功能上没有本质区别——都是用关系的建立来换取资源通道的扩展。

{{< sentence >}}
面子不是虚荣，它是这个社会里的信用评级。
{{< /sentence >}}

"丢面子"在金翼的分析框架里不是情感问题而是经济问题——你的面子下降等于你的信用评级下调，直接影响你能调动的资源。

{{< sentence >}}
现代化不只是技术的替换，它是整个社会基础设施的替换——从人际关系切换到制度和合同。
{{< /sentence >}}

林耀华隐含的判断：现代化过程中消失的不只是"落后的风俗"，还有一整套在特定环境下有效运转的社会基础设施。
