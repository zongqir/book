# 用数据讲故事里最该留下来的，是这些会在关键时候拽住你的提醒句

{{< sentence >}}
先定结论，再决定图怎么画。
{{< /sentence >}}

顺序一反，图很容易变成装饰。

{{< sentence >}}
图里每多一点噪音，结论就少一点力气。
{{< /sentence >}}

删不是美化，是让判断活下来。

{{< sentence >}}
不要指望读者自己看出重点。
{{< /sentence >}}

图表也需要带路。

{{< sentence >}}
好图不是信息最多的图，是判断最清楚的图。
{{< /sentence >}}

这句最适合压住“我怕漏掉”的焦虑。

{{< sentence >}}
如果图不能替你说话，你最后还得靠嘴把它救回来。
{{< /sentence >}}

那说明图本身没完成任务。

## 这些句子什么时候最该翻出来

- 一开口就想解释很多的时候：先翻这页。
- 你觉得自己明明没说错，结果事情还是做坏的时候：先翻这页。
- 你知道问题在哪，却迟迟不知道从哪一句开始的时候：先翻这页。
- 对方的反应、组织的惯性或关系的压力开始把你带偏的时候：先翻这页。
