# 先别忙着领导别人，先看自己是不是已经发空

{{< sentence >}}
离开中国现实越远，学得越多也可能越空。
{{< /sentence >}}

第三卷第一刀先砍学习入口。理论再熟，现实进不来，后面都白搭。

{{< sentence >}}
真正危险的不是有人公开反对，而是没人把真问题说出来。
{{< /sentence >}}

这句专门防“表面统一”。第三卷最怕的不是争论，是沉默。

{{< sentence >}}
学风坏了，判断会假；党风坏了，组织会裂；文风坏了，路线根本传不下去。
{{< /sentence >}}

这句把第三卷最关键的三件事重新拧回一股绳。

{{< sentence >}}
空话不是修辞毛病，是组织已经开始失去现实感。
{{< /sentence >}}

第三卷为什么对党八股下重手，这句最能说明。

{{< sentence >}}
领导不是替群众想明白，是把群众经验压成能回去验证的办法。
{{< /sentence >}}

这句最适合在“上面全懂、下面全怨”的局面里调出来。

{{< sentence >}}
路线如果只在会上正确，进不了干部和群众，等于没有路线。
{{< /sentence >}}

这句把表达重新压回政治功能，不让它只停在“会不会说话”。

{{< sentence >}}
越接近胜利，越不能把“以后怎么办”拖到以后。
{{< /sentence >}}

这句把《论联合政府》和前面的整风、群众路线接起来了。

{{< sentence >}}
党要领导中国，不能只靠反对谁，还得拿得出面向全国的秩序说法。
{{< /sentence >}}

第三卷最深的地方，就压在这句里。

## 这些句子什么时候最该翻出来

- 学得越多，判断越像概念时：先看第 1 句。
- 会议特别顺、谁都不提反对时：先看第 2、3 句。
- 讲话越来越正确，别人却越来越听不进去时：先看第 4、6 句。
- 上下脱节、上面只会布置、下面只会抱怨时：先看第 5 句。
- 局势越来越大，大家却只盯眼前最后一程时：先看第 7、8 句。

第三卷真正该留下来的，不是气势话，而是这些让一支党先回头看自己的话。
