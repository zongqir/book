## 方向比速度值钱

{{< sentence >}}
"没有战略的战术是通往失败最嘈杂的路。"
{{< /sentence >}}

很多人忙得停不下来，但从没问过自己在忙的方向对不对。战术上的勤奋解决不了战略上的懒惰。下次觉得自己"一直在做但没什么进展"的时候，先检查方向。

{{< sentence >}}
"在你看清全局之前，任何一步都可能是浪费。"
{{< /sentence >}}

急着动手是本能，但卡斯帕罗夫在棋盘上学到的教训是：先看清楚局面再下手，哪怕多花三十秒，省的可能是后面三十步的弯路。

## 直觉不是猜，但也不是真理

{{< sentence >}}
"直觉是压缩后的经验，不是未经检验的偏好。"
{{< /sentence >}}

直觉有价值，前提是它背后有足够多的经验。一个下了一万盘棋的人的直觉和一个下了十盘棋的人的直觉，精度完全不同。信直觉之前，先估一下自己在这个领域的经验存量。

{{< sentence >}}
"计算会告诉你哪步能赢；直觉会告诉你哪几步值得算。"
{{< /sentence >}}

两者不是对立的。直觉负责缩小范围，计算负责验证。问题是大多数人要么全靠感觉，要么试图把所有选项都算一遍——两种极端都不可靠。

## 准备比临场反应重要

{{< sentence >}}
"你在比赛前的准备质量，决定了你在比赛中能自由思考的空间有多大。"
{{< /sentence >}}

准备不是为了背答案。准备是为了让你在压力最大的时候，把注意力释放给需要临场判断的部分。如果所有事情都等到上场再想，你的注意力会被基础问题吃光。

{{< sentence >}}
"对手会告诉你该怎么准备——如果你愿意去研究他。"
{{< /sentence >}}

卡斯帕罗夫花在研究对手上的时间不比花在研究棋局上的少。了解对手的风格、弱点和偏好，不是为了模仿，是为了让自己的准备有靶心。

## 输了以后怎么办

{{< sentence >}}
"从失败中学到的东西，比从胜利中学到的多得多——前提是你愿意去看。"
{{< /sentence >}}

赢了很少有人复盘。输了才是学习窗口打开的时候。但大多数人输了以后忙着找借口或者自我安慰，错过了最有价值的复盘时机。

{{< sentence >}}
"危险的不是犯错，是犯了错还继续沿着错误的路线走。"
{{< /sentence >}}

棋盘上走错一步不一定输。连续沿着错误方向走才会输。发现走偏的第一时间就该停下来重新评估，而不是想着"再走两步看看"。
