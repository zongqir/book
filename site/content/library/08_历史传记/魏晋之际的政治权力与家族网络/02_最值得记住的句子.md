{{< sentence no="1" quote="婚姻是政治投资，不是家庭事务" >}}
{{< /sentence >}}

仇鹿鸣反复展示一个事实：魏晋高门的每一桩婚姻，都能在政治版图上找到对应坐标。把婚姻当作私人领域来理解，会漏掉权力分析中最关键的变量。

提醒场景：读到某个魏晋人物"娶了某家之女"时，默认先问"这桩婚姻连接了什么政治资源"。

{{< sentence no="2" quote="站队不需要密谋，姻亲自动完成对齐" >}}
{{< /sentence >}}

高平陵之变中，许多人的立场不需要"被说服"。姻亲关系已经预设了激励方向。大多数关键人物的站队可以用姻亲关系直接预测，不需要引入"忠诚""野心"或"阴谋"。

提醒场景：分析某次政治站队时，如果姻亲关系已足以解释方向，停下来。不必继续向"个人动机"深挖。

{{< sentence no="3" quote="排斥不需要明令，只需要在关键节点上偏向自己人" >}}
{{< /sentence >}}

司马氏的政治排斥手段极少表现为直接打击。更常见的模式：在关键人事节点上持续安排网络内部的人，让外部家族逐渐失去上升空间。

提醒场景：分析某个家族的衰落时，先检查他们是否被系统性排除在核心任命之外。

{{< sentence no="4" quote="网络边缘的人先变，核心的人最后才变" >}}
{{< /sentence >}}

两个政治网络的交叉地带，站位最不稳固。退出成本低、重新下注的空间大。仇鹿鸣的案例分析中，最早叛变的人几乎都处于网络边缘。

提醒场景：预判谁会先转向，看网络位置，不看"品性"。

{{< sentence no="5" quote="个人看清了局势也未必能动" >}}
{{< /sentence >}}

家族网络的约束不只作用于信息层面，还作用于行动层面。你的父兄、岳父、姻亲都在同一张网上。个人跳到对面，意味着把整个家族推向危险。

提醒场景：遇到"为什么他看清了形势还是选错了"，先检查家族绑定。

{{< sentence no="6" quote="禅代是持续数十年的网络工程" >}}
{{< /sentence >}}

从司马懿到司马炎完成禅代，经历了三代人。仇鹿鸣关注的是一个持续数十年的网络编织过程。禅代的"成功"回溯到每一桩联姻、每一次任命、每一个被拉入或排出网络的家族。

提醒场景：分析长周期的权力转移时，不要聚焦于"关键转折点"。往前查二十年的婚姻和人事记录。

{{< sentence no="7" quote="门阀格局的生成过程比定型状态更难拆" >}}
{{< /sentence >}}

田余庆写的东晋门阀是已经定型的格局。仇鹿鸣拆的是格局的生成过程——网络如何从零散的利益交换凝结为结构性的政治力量。理解生成过程，对证据精度的要求更高。

提醒场景：研究任何一种制度化的权力格局时，优先追溯生成期。
