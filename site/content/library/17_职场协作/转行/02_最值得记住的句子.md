## 还没动的时候

{{< sentence >}}
你不可能坐在现有身份里想清楚下一个身份是什么。
{{< /sentence >}}

用旧身份的认知框架去规划新身份，等于用陆地上的经验设计航海路线。你能列出所有已知变量，但决定结果的，是下水之后才遇到的东西。自我评估做得再细致，也只是在旧地图上画路线。

{{< sentence >}}
先行动再想清楚，不是冲动——是唯一有效的路径。
{{< /sentence >}}

直觉告诉你"想好了再动才稳妥"。但伊巴拉跟踪的39个转行者没有一个是先想清楚的。行动产生的体验才能更新认知。想清楚了再动的人，通常一直没动。

## 开始试的时候

{{< sentence >}}
你不是只有一个"真实自我"等着被发现。你有好几个可能的自我，等着被测试。
{{< /sentence >}}

"找到真实自我"是个浪漫但危险的说法。它暗示存在唯一正确答案，让你在没找到之前不敢行动。"可能的自我"替换了这个框架：你在任何时间点都有多个潜在身份值得探索。哪个最终成立，取决于你试了哪些。

{{< sentence >}}
不要辞职去转行。用小实验，让新旧身份并行一段时间。
{{< /sentence >}}

全押式转行——先裸辞再找方向——风险极大，切断所有退路。有效的做法是保留现有工作的同时，用兼职、志愿者、副业或探索性项目接触新身份。并行跑一段，让体验告诉你答案。

## 中间摇摆的时候

{{< sentence >}}
你现有的朋友和同事，会不自觉地把你拉回旧身份。
{{< /sentence >}}

不是他们不支持你，是他们对你的认知锚定在过去。你说"我想转行做教育"，老朋友说"你做得好好的为什么要换"。这不是恶意，是社交网络的认知惯性。想让新身份站住脚，必须去接触新圈子里的人。

{{< sentence >}}
转行不是一个决策，是一段过程。中间必然有一个两头不靠的时期。
{{< /sentence >}}

很多人以为转行应该像换工作——周五离开 A 公司，周一走进 B 公司。真实的身份转换有一段模糊的过渡期：你既不完全是旧身份，也没确立新身份。这段时期让人焦虑，但它不是转行出了问题——它就是转行本身。

{{< sentence >}}
你的职业故事需要被重写，不是被发现。
{{< /sentence >}}

过去的经历不会自动连成一条线。你需要主动重新编织叙事——把过去的经验、现在的探索和未来的方向编成一个说得通的故事。不是编造，是重新诠释。这个故事不仅说给别人听；更是说给自己听，帮你确认"我正在变成谁"。
