{{< sentence author="茅海建" book="近代的尺度" >}}
史学研究的根本问题不在于史料的多少，而在于史料的可靠程度。
{{< /sentence >}}

史学界常说要重视一手资料，但茅海建指出更关键的是判断资料的可信度。同样是当事人记录，亲历者的日记和多年后的回忆录可信度完全不同。

{{< sentence author="茅海建" book="近代的尺度" >}}
历史的真相往往不在宏大叙事中，而在那些看似无关紧要的细节里。
{{< /sentence >}}

茅海建通过具体案例展示，那些被史家忽略的小细节——比如一封信的寄送时间、一个官员的具体职务——往往能推翻流传很广的"史学定论"。

{{< sentence author="茅海建" book="近代的尺度" >}}
做史学研究，最怕的不是找不到资料，而是相信了不该相信的资料。
{{< /sentence >}}

很多史学错误的根源是史家对某份看起来权威的资料缺乏足够的怀疑。茅海建强调每一份史料都需要交叉验证，包括那些来自官方档案的记录。

{{< sentence author="茅海建" book="近代的尺度" >}}
史料考证不是为了炫耀博学，而是为了接近历史的本来面目。
{{< /sentence >}}

茅海建反对为了考证而考证的学院派作风，他的史料辨析最终指向一个目标：还原历史事件的真实过程，而不是维护某种既定的史学观点。

{{< sentence author="茅海建" book="近代的尺度" >}}
中国近代史研究中最大的问题，是用后来的观念去理解当时的行为。
{{< /sentence >}}

茅海建提醒史学研究者要回到历史现场，用当时人的知识背景和价值观念去理解当时人的选择，而不是站在后世的立场上做事后诸葛亮式的评判。

{{< sentence author="茅海建" book="近代的尺度" >}}
史学研究的价值不在于提供标准答案，而在于展示思考问题的严谨过程。
{{< /sentence >}}

茅海建认为史学研究的意义不是给历史问题下定论，而是展示如何用严谨的方法处理复杂的历史材料，这个思考过程比结论更有价值。
