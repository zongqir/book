{{< sentence >}}
付出不亚于任何人的努力——这不是励志口号而是稻盛和夫对"什么是够努力"的回答：你觉得自己够努力了？看看周围有没有比你更努力的人。有的话就还不够。
{{< /sentence >}}

努力的标准不是自我感觉而是和同行比较。这个标准残酷但清晰。

{{< sentence >}}
动机至善，私心了无——稻盛和夫创立KDDI时反复问自己：是不是为了自己的虚荣心？他花了六个月确认自己的动机是纯粹的才开始行动。
{{< /sentence >}}

大多数人在确认动机上花的时间是零。稻盛和夫花了六个月。动机的纯度直接影响决策的质量。

{{< sentence >}}
阿米巴经营的核心不是分权——是让每一个小单元都像经营者一样思考。不是"你执行我的计划"而是"你来决定怎么经营你的小团队"。
{{< /sentence >}}

把员工变成经营者——这比任何激励体系都有效，但也比任何体系都难建。

{{< sentence >}}
利润是为社会做贡献后社会给你的回报——不是你从社会手里抢来的。如果利润是抢来的，迟早要还回去。
{{< /sentence >}}

这不是道德说教——是稻盛和夫对商业可持续性的判断。短期利润可以靠抢但长期利润必须靠创造价值。

{{< sentence >}}
拯救日航时稻盛和夫做的第一件事不是看财务报表——是和一线员工谈话。他发现日航的问题不在财务而在人心：员工不相信公司还有未来。
{{< /sentence >}}

数字是结果不是原因。修数字之前先修人心。
