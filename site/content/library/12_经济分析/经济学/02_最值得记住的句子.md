# 经济学思维的几个关键洞察

## 经济学的核心是研究选择和激励

{{< sentence >}}
经济学不是研究钱的科学，而是研究选择的科学。
{{< /sentence >}}

所有涉及选择的现象都能从经济学角度来理解。

{{< sentence >}}
人的行为永远遵循激励，无论他自己有没有意识到。
{{< /sentence >}}

改变激励，就能改变行为，这简单而有力。

{{< sentence >}}
看起来不合理的现象，往往是人们在特定约束条件下的理性选择。
{{< /sentence >}}

别用你的偏好判断别人的选择，先看看他们面临什么约束。

## 关于成本和代价的常见误判

{{< sentence >}}
最重要的成本往往是看不见的机会成本。
{{< /sentence >}}

做这个选择意味着放弃做那个选择，这个放弃就是成本。

{{< sentence >}}
政策的意图好不好，不等于结果好不好。
{{< /sentence >}}

最有害的政策往往来自最好的意图，因为忽视了真实的激励效应。

{{< sentence >}}
看起来最合理的政策，往往带来最意想不到的后果。
{{< /sentence >}}

因为决策者没有看清真实激励链条的全部效应。

## 经济学思维的实际应用

{{< sentence >}}
任何规则都会被人在新的约束条件下创意性地突破。
{{< /sentence >}}

这不是人坏，而是人总在寻求最大化收益的方式。

{{< sentence >}}
短期和长期的激励往往是矛盾的。
{{< /sentence >}}

很多社会问题的根源就在这里，需要设计激励来对齐两者。

{{< sentence >}}
看一个社会怎样对待稀缺资源，就能看出它的真实价值观。
{{< /sentence >}}

不是看它说什么最重要，而是看它真正在哪里花钱。
