## 知识与言说的关系

{{< sentence >}}
能言说的东西，往往是最浅层的理解。最深的理解，往往无法言说。
{{< /sentence >}}

教科书能把原理说清楚。但怎样在现实中灵活应用这个原理，往往说不清。

{{< sentence >}}
知识是可以跨界传递的，但经验是本地的、无法复制的。
{{< /sentence >}}

同一个学生，在A老师的课堂上掌握了某个知识，转到B老师的课堂上还是同样的知识。但一个医生在北京积累的临床经验，拿到另一个医生那里是无法直接用的。

## 经验与判断力

{{< sentence >}}
判断力不是从学习知识来的，是从在错误和成功中反复积累来的。
{{< /sentence >}}

多少人读过决策学的理论，但一到真实的决策时刻还是很纠结。为什么？因为知识告诉你的是原则，但实际情况往往在灰色区间里。

{{< sentence >}}
直觉看起来凭空出现，其实是大量经验的压缩。
{{< /sentence >}}

一个有二十年经验的工程师，看一眼某个故障现象，立刻知道问题出在哪。这不是神奇，而是他的脑子已经把这个领域的所有情况都建了一个无意识的模型。新手需要查资料推理，老手凭直觉秒判。

## 学习的真实周期

{{< sentence >}}
一个人从理论理解到实际能力，往往需要比想象长得多的时间。
{{< /sentence >}}

教科书说掌握一门知识需要 100 小时。但从掌握知识到真的能用，可能还要 1000 小时的实践。

{{< sentence >}}
如果你觉得自己掌握得很快，往往说明你理解还不够深，或者项目还不够复杂。
{{< /sentence >}}

深度学习不是学得快，是学得慢。要停留足够长的时间，让无意识的学习发生。

## 什么时候知识最有价值

{{< sentence >}}
知识最有价值的时刻，是在你已经有基本经验之后。
{{< /sentence >}}

一个完全的新手学最复杂的理论往往没用，因为没有实践的土壤。但一个在某个领域干了两三年的人再学理论，往往会有豁然开朗的感觉。

{{< sentence >}}
理论不是学习的起点，而是在一定经验基础上的总结。
{{< /sentence >}}

传统的教育顺序是先学理论再实践。有时候反过来——先实践再学理论——效果会更好。
