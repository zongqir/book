# 十句提醒，把老板从情绪来源变回关键接口

{{< sentence no="1" quote="向上管理不是管理老板本人，而是管理你和老板之间的工作接口。" >}}
{{< /sentence >}}

{{< sentence no="2" quote="任务一到手先确认老板要的是什么，不要先默认自己理解得对。" >}}
{{< /sentence >}}

{{< sentence no="3" quote="很多返工不是做错了，而是一开始对优先级和交付口理解就不同。" >}}
{{< /sentence >}}

{{< sentence no="4" quote="只会上报问题，不会同步选项，会上级接口里的信息价值会很低。" >}}
{{< /sentence >}}

{{< sentence no="5" quote="越忙的老板，越需要你用他能快速判断的方式讲话。" >}}
{{< /sentence >}}

{{< sentence no="6" quote="风险越早同步，越像合作；越晚解释，越像甩锅。" >}}
{{< /sentence >}}

{{< sentence no="7" quote="老板说“你看着办”，并不等于不存在隐含标准。" >}}
{{< /sentence >}}

{{< sentence no="8" quote="向上管理做得好，往往不是关系更热，而是误判更少、往返更少。" >}}
{{< /sentence >}}

{{< sentence no="9" quote="真正成熟的汇报，不只是让老板知道你做了什么，而是让老板能更快决定下一步。" >}}
{{< /sentence >}}

{{< sentence no="10" quote="这本书最该留下来的，不是迎合技巧，而是更早对齐目标和节奏的能力。" >}}
{{< /sentence >}}
