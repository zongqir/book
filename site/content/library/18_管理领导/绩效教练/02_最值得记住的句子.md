{{< sentence >}}人的表现等于潜力减去干扰{{< /sentence >}}

这个公式直击教练的本质。大多数绩效问题不是能力不足，而是内在干扰过多——自我怀疑、焦虑、注意力分散。教练的工作是清除干扰，而不是增加技能。

{{< sentence >}}提问的目的是提高觉察和责任感{{< /sentence >}}

觉察让人看清现状，责任感让人主动行动。这两者缺一不可。纯粹的觉察容易变成分析瘫痪，纯粹的责任感容易变成盲目执行。

{{< sentence >}}指令式管理培养依赖性，教练式管理培养独立性{{< /sentence >}}

每次你直接给答案，都是在训练下属等待指示。每次你通过提问让他们自己找到答案，都是在建立他们独立思考的能力。

{{< sentence >}}好问题比好答案更有价值{{< /sentence >}}

一个精准的问题能够重新框定整个问题，触发全新的思考路径。而直接的答案只能解决当前这一个具体情况。

{{< sentence >}}目标必须是被教练者自己的，而不是教练的{{< /sentence >}}

强行推销你认为对他们有益的目标，即使达成也很难持续。只有当目标与他们的内在动机一致时，才能产生持久的驱动力。

{{< sentence >}}选择权在被教练者手中，支持权在教练手中{{< /sentence >}}

教练不是决策者，而是决策支持者。你的职责是帮助他们看清选择，而不是替他们做选择。

{{< sentence >}}听见话语背后的情感和需求{{< /sentence >}}

表面的问题往往不是真问题。一个关于时间管理的抱怨，可能背后是对工作价值的质疑。深层倾听才能发现真正需要处理的议题。
