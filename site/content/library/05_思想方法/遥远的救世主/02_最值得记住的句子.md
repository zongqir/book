丁元英说话不多，但他说出来的句子经常在对话结束很久之后才开始起作用。下面八句从小说对话里筛出来，不是按出场顺序，是按"你卡住时调用它的价值"排的。

## 在判断犹豫时先调这几句

{{< sentence >}}
神即道，道法自然，如来。
{{< /sentence >}}

丁元英对宗教和哲学的理解压成了这一句。它不是在谈神学，是在说一件事：规律就在那里，不因为你信不信它就改变。做判断时先找规律，别先找信仰。

这句在你试图用态度、意愿、决心代替分析的时候最管用。

{{< sentence >}}
强势文化就是遵循事物规律的文化，弱势文化就是依赖强者的道德期望破格获取的文化。
{{< /sentence >}}

全书最底层的判断框架。它区分了两种做判断的起点：一种从规律出发，一种从期望出发。

你下次做决策前可以拿它照一下镜子：你现在的判断起点，是在看规律，还是在期望谁来帮忙？

{{< sentence >}}
你不知道你，所以你是你。如果你知道了你，你就不是你了。
{{< /sentence >}}

丁元英对芮小丹说的。听起来绕口，拆开很简单：一个人之所以困在现在的状态里，恰恰是因为看不见自己的模式。一旦看见了，模式就开始松动。

卡在同一个地方反复出错的时候，先别急着换方法，先问：我是不是有一个自己看不见的判断惯性？

## 在路径依赖的时候翻出来看

{{< sentence >}}
想要，是在世俗社会能得到的；不想要，是在世俗社会得不到的。别把"想要"和"能要"搞混了。
{{< /sentence >}}

这句话的杀伤力在最后一刀。大部分目标设定的问题不是目标太高，是根本没分清"想要"和"能要"。判断力弱的人容易在这两个之间来回跳——想要的时候热血沸腾，发现够不着就怪环境。

{{< sentence >}}
有道无术，术尚可求；有术无道，止于术。
{{< /sentence >}}

方向对了，具体方法慢慢能找到。但如果只有方法没有方向，方法就是你的天花板。

适合在你觉得"技能够了但总差一口气"的时候拿出来用。差的那口气大概率不是技能，是看问题的位置。

{{< sentence >}}
这世上原来就没什么神话。所谓的神话，不过是常人的思维所不易理解的平常事。
{{< /sentence >}}

格律诗音响的成功在旁人眼里是"商业奇迹"。丁元英从头到尾没做过任何超常规操作——只是把成本结构看清楚了。

当你觉得某件事"太神了、我做不到"的时候，先拿这句话照一下：你看不懂，可能不是因为它复杂，是因为你没从对的位置去看。

## 在自我欺骗的时候用这两句

{{< sentence >}}
本是后山人，偶做前堂客。醉舞经阁半卷书，坐井说天阔。
{{< /sentence >}}

丁元英写给自己的。读了半本书就敢论天下，是最常见的知识虚荣。这句不是在谦虚——是在做精度校准。你的判断到底建立在多厚的事实上？

{{< sentence >}}
透视社会依次有三个层面：技术、制度和文化。小到一个人，大到一个国家一个民族，任何一种命运归根到底都是那种文化属性的产物。
{{< /sentence >}}

丁元英分析问题的坐标系。遇到复杂问题时，不要一头扎进技术细节。先问：这个问题在技术层面能解决，还是必须在制度甚至文化层面才能解决？

层级搞错了，越努力越偏。
