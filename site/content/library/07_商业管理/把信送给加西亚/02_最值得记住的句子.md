# 八句提醒，把执行起点从借口拉回承担

{{< sentence no="1" quote="任务一到手，先想怎么开始找路，比先证明路有多难更重要。" >}}
{{< /sentence >}}

{{< sentence no="2" quote="很多执行失败，不是输在做不到，而是输在还没做就先把理由说满。" >}}
{{< /sentence >}}

{{< sentence no="3" quote="真正稀缺的人，常常不是最会表态的人，而是任务下来后最先开始动的人。" >}}
{{< /sentence >}}

{{< sentence no="4" quote="主动承担不是鲁莽硬冲，而是在目标明确时不把自己第一反应训练成等待。" >}}
{{< /sentence >}}

{{< sentence no="5" quote="组织里最常见的惰性，往往不是偷懒，而是永远等更完整说明才肯开始。" >}}
{{< /sentence >}}

{{< sentence no="6" quote="你能不能自己去补路径，常常比你会不会讲困难更决定结果。" >}}
{{< /sentence >}}

{{< sentence no="7" quote="执行起点一旦总是外推，后面的责任感很难再突然长出来。" >}}
{{< /sentence >}}

{{< sentence no="8" quote="这本书真正想留下来的，不是服从姿态，而是少让借口比行动先到场。" >}}
{{< /sentence >}}
