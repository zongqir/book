{{< sentence >}}
星巴克不是在卖咖啡——是在卖一个你可以安静待着的地方。咖啡是入场券，空间才是产品。
{{< /sentence >}}

产品功能和产品价值是两件事。很多公司混淆了这两者——以为把功能做好就够了。

{{< sentence >}}
舒尔茨在米兰看到咖啡馆老板叫得出每个常客的名字。他意识到咖啡馆卖的不是浓缩咖啡而是归属感。
{{< /sentence >}}

归属感是人类最基本的需求之一——如果你的商业场景能提供它，你就拥有了比价格优势更持久的壁垒。

{{< sentence >}}
舒尔茨坚持给所有员工（包括兼职）提供医疗保险和股票期权。华尔街觉得他疯了。他认为这不是成本而是品牌的一部分——员工的态度就是品牌的脸。
{{< /sentence >}}

品牌不只在广告里——品牌在每一个员工和顾客的交互中。你在员工身上省的钱会从顾客体验中漏出去。

{{< sentence >}}
星巴克早期拒绝做加盟——因为加盟意味着失去对体验的控制。舒尔茨宁可慢一点也不愿意让体验不一致。
{{< /sentence >}}

速度和控制之间永远有张力。选择哪一个取决于你的核心竞争力是什么——如果是体验，就选控制。

{{< sentence >}}
舒尔茨的父亲是蓝领工人，受过工伤但没有保险。这个童年记忆直接影响了星巴克的员工福利政策。创始人的伤疤经常成为企业的基因。
{{< /sentence >}}

企业的价值观不是策划出来的——是从创始人的真实经历中长出来的。
