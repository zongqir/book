## 识别出发点的问题

{{< sentence >}}
你听到最多的反馈，往往是最不属于你的反馈。
{{< /sentence >}}

这不是说外部反馈有问题，而是说反馈的信号越强、越来自权威、越符合社会期待，你就越容易把它当成自己的目标。久而久之，你忙得不亦乐乎，却不知道自己在为谁工作。

{{< sentence >}}
一个改变能不能坚持，决定因素不是方法的难度，而是动机的来源。
{{< /sentence >}}

来自外部的动机（父母期待、社会评价、同龄人竞争）能提供短期的冲力，但无法支撑长期的坚持。来自内部的动机（想要的自我形象、真实的好奇心、对自己的承诺）才能在没人监督的时候还能继续。

{{< sentence >}}
自我认知不改变，习惯改变了也会被拉回来。
{{< /sentence >}}

如果你对自己的认知是"我就是个容易分心的人"，即使学了再多的专注技巧，大脑也会找到理由回到旧的模式。认知是前提，习惯是结果。改结果而不改前提，费力不讨好。

## 调整方向的要点

{{< sentence >}}
好的发展目标是内部一致的，不是外部漂亮的。
{{< /sentence >}}

外部漂亮的目标（高薪、大房子、名校学位）看起来吸引人，但如果它和你内部的价值观不对齐，执行起来就会很别扭。好的目标是那种你即使没人看见也会去做的事情。

{{< sentence >}}
找到你自己的发展杠杆，比复制别人的成功方法重要得多。
{{< /sentence >}}

对一个人有效的杠杆对另一个人可能完全失效。有人的杠杆是理解自己的恐惧，有人是确定自己真正在乎什么，有人是停止复制别人。关键是找到那个触发点，一按就能动的那个。

{{< sentence >}}
当两个目标冲突时，这不是时间管理问题，是信号问题。
{{< /sentence >}}

如果你既想要职业成功又想要陪家人的时间，而现在的努力在分裂你，那问题不在怎样同时做两件事，而在需要重新审视这两个目标的优先级。不解决这个优先级，时间分配技巧帮不了你。

{{< sentence >}}
个人发展的最大浪费，是在错误的方向上使力。
{{< /sentence >}}

比能力不足更糟的，是在一个不属于你的方向上花了五年、十年的精力，才发现走错了路。这就是为什么自我认知要先于行动力——确定对了再往前冲。
