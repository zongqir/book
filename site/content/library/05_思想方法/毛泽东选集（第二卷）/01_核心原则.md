# 五条战略准则：从判断阶段到保住独立性

第一卷提取的准则落在"判断方法"这一层：从结构起步、集中打主要矛盾、让实践校正理论。

第二卷的准则落在另一层——"战略判断"。方法已经有了，关键是在力量严重不对称的长线博弈里，怎样做出方向不偏、节奏不乱的选择。

以下五条准则从《论持久战》《新民主主义论》《统一战线中的独立自主问题》《反对自由主义》等篇目中提取。前两条是核心准则，后三条是支撑准则。

## 核心准则

{{< principle no="1" type="core" statement="力量对比不是静态快照——要判断的是结构变化的方向" >}}

速胜论盯着对手的弱点就觉得能赢。亡国论盯着自己的弱点就觉得必输。两种都把此刻的力量排名当成了终局。

《论持久战》的判断建立在一个不同的前提上：强弱是会变的。日本越打越孤立、越打资源越紧；中国越打民心越聚、越打国际支持越多。关键变量是结构变化的方向。

判断阶段先于判断胜负。防御阶段的目标是保存力量、消耗对手；相持阶段的目标是积累优势、等待转折；反攻阶段才追求决定性胜利。每个阶段该做的事完全不同。

这条准则防的是两种偏：急着摊牌，或者早早认输。

{{< /principle >}}

{{< principle no="2" type="core" statement="联盟中必须保住独立的判断标准、组织能力和调整节奏的权利" >}}

劣势方需要联盟。但联盟里最常见的陷阱是：为了维持合作，把判断标准、行动节奏甚至组织权交给更大的伙伴。

合作中交出独立性，表面上关系顺畅，实际上丧失了纠偏能力。等到发现方向不对，已经调不了头。

"又联合又斗争"的判断是：联合是为了共同应对更大威胁；斗争是为了在联合中保住位置和空间。这两件事必须同时做。

这条准则防的是：把合作理解成听大伙伴的。

{{< /principle >}}

## 支撑准则

{{< principle no="3" type="supporting" statement="只有'反对什么'撑不起长线合作——'建成什么'才是真正的锚" >}}

反对同一个敌人，足以让一群力量走到一起。但打赢之后呢？

《新民主主义论》在战争中期就回答了"建成什么"：政治上联合、经济上多种成分并存、文化上民族的科学的大众的。

这个愿景的战略价值在于：它让联盟在最困难的阶段也有粘合点；让内部分歧出现时有裁判标准。没有它，每次遇到分歧都得从头谈判。

{{< /principle >}}

{{< principle no="4" type="supporting" statement="战略一致靠的是每步具体行动的灵活调整——不是僵在一个计划上不动" >}}

持久战的三个阶段，每个阶段的打法完全不同。防御阶段打运动战，相持阶段打游击战，反攻阶段打阵地战。

战略方向不变，但战术必须随阶段调整。把防御阶段的打法搬到反攻阶段，或者在相持阶段急着决战，都会出大问题。

灵活调整不是没有方向。恰恰相反：正因为战略方向足够清楚，每一步才知道该怎么调。

{{< /principle >}}

{{< principle no="5" type="supporting" statement="无原则的内部和平是一种隐性瓦解——比正面分歧更危险" >}}

团队里最容易忽视的裂缝是：大家都不说真话，都不做原则讨论，决定了不执行但也不提反对。

《反对自由主义》的判断是：这些小裂缝叠起来，效果等于组织在形式上统一、行动上解体。比公开分歧更难察觉，也更难修复。

纪律的基础是认同。如果成员对共同判断标准没有真正认同，靠加压维持的纪律迟早会碎。

{{< /principle >}}
