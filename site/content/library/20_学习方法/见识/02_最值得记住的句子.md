# 七句判断，拉开选择和努力之间的差距

{{< sentence >}}
走得远的人做对的那些事，往往不是技巧性的，而是判断性的。
{{< /sentence >}}

整本书最底层的一句话。吴军观察了大量硅谷工程师、投资人和企业创始人，发现差距很少来自智力或勤奋。选了什么赛道、砍掉了什么事、在分叉点上做了什么决定——这些判断性的选择，决定了最终走到哪里。

每次面对重要选择时，先问自己：我现在是在挑技巧还是在做判断？

{{< sentence >}}
一个人一天能做好的事不超过三件。
{{< /sentence >}}

做减法最具体的锚点。不是"少做一点"的模糊建议，是一条硬约束。你今天日程上超过三件重要的事，大概率一件都做不好。

这句话适合在排日程时拿出来。砍到三件，才有空间把每件做透。

{{< sentence >}}
凡是看起来忙但不产生实质推进的，都是伪工作。
{{< /sentence >}}

识别伪工作的判断标准。回复不紧急的消息、参加不产出决策的会议、整理不会再看的笔记——这些活动消耗精力但不推动任何事。

下次觉得自己"很忙"时翻出来，对照一遍。

{{< sentence >}}
格局就是做选择时能考虑多远的未来和多大的范围。
{{< /sentence >}}

把"格局"从虚词变成可操作定义。只看三个月薪资的人和同时看十年行业走向的人，选择质量天然不同。

适合在做长期规划时提醒自己：你考虑的时间跨度够不够长。

{{< sentence >}}
人脉不是你认识多少人，而是多少人在需要帮助时会想到你。
{{< /sentence >}}

社交判断的翻转。多数人经营人脉的方式是多认识人、多换名片。吴军指出方向反了——先让自己变得有价值，别人自然会找过来。

在社交投入上犹豫时拿出来用。与其去参加第十个行业聚会，不如把一件事做到别人需要你帮忙。

{{< sentence >}}
投资自己永远是回报率最高的投资。
{{< /sentence >}}

资源分配的优先级。吴军在投资领域见过无数种回报曲线，最后得出的结论指向自己——提升自己的判断力和能力，比任何理财产品的回报都稳定。

适合在纠结"钱该往哪放"的时候提醒自己。

{{< sentence >}}
与其多做一件事，不如把一件事做到别人的十倍好。
{{< /sentence >}}

十倍标准。不是"做好"，是"十倍好"。吴军认为在竞争充分的领域，稍微好一点没有意义，只有好到一个数量级以上才能拉开差距。

适合在犹豫要不要开新项目时用来刹车：你手上这件事做到十倍好了吗？
