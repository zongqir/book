# 七句古老到让人没法反驳的财富纪律

## 存钱是所有后续动作的起点

{{< sentence >}}
你赚来的每一枚金币，至少有一成必须属于你自己。
{{< /sentence >}}

阿卡德说的"属于自己"不是指你花在自己身上的钱。而是你从收入中扣下来、不动的那部分。"先付自己"的意思是：在支付房租、伙食、一切账单之前，先把这一成拿走。如果顺序反过来——先花完再看剩多少——结果永远是零。

{{< sentence >}}
你的钱包是要养胖的——如果进去十枚、出来十枚，它永远是空的。
{{< /sentence >}}

克拉森用了一个简单到让人不好意思反驳的比喻。你的钱包不是中转站，应该是蓄水池。每次进出平衡等于没有积累。

## 支出是需要治理的，不是自然发生的

{{< sentence >}}
不要把欲望和需求混为一谈——你的欲望永远比你的收入增长得更快。
{{< /sentence >}}

收入翻倍，消费也翻倍——这是大部分人的真实轨迹。克拉森说问题不在收入，在于欲望的膨胀没有上限。唯一的解决方案是给消费设一个和收入无关的硬顶。

{{< sentence >}}
预算不是限制你的自由，而是确保你的钱去了你想让它去的地方。
{{< /sentence >}}

大部分人抵触预算，觉得它像枷锁。克拉森把预算重新定义为决策工具——没有预算的人不是更自由，而是更容易被冲动和惯性控制。

## 钱存下来之后，要让它去工作

{{< sentence >}}
每一枚闲置的金币都是一个偷懒的工人。
{{< /sentence >}}

存下的钱如果只是放在账户里不动，你只完成了一半。另一半是让它产生收益——利息、投资回报、资产增值。克拉森不追求高回报，追求的是"你的钱在替你赚钱"这个状态持续运行。

## 保护先于回报

{{< sentence >}}
投资的第一条原则是不亏损本金——永远不要把钱交给自己不懂的领域。
{{< /sentence >}}

阿卡德在故事里吃过一个大亏：把积蓄交给一个砖匠去做珠宝生意。砖匠是好人，但不懂珠宝——钱全赔了。克拉森的教训是：投资之前先问一个问题——管你钱的人在这个领域有没有被验证过的能力？如果没有，再高的回报承诺都不值得冒险。

{{< sentence >}}
好运气更偏爱有准备的人——而准备的意思是：别犹豫，别等"完美时机"。
{{< /sentence >}}

阿卡德说很多人明明看到了机会，但因为犹豫而错过。好运不是天上掉馅饼，而是机会出现时你有资源、有判断、有行动力去接住它。前面的储蓄和纪律，都是在为这个时刻做准备。
