{{< sentence >}}
美元的锚从黄金换成了石油，看上去断了根，实际上更结实了。
{{< /sentence >}}

布雷顿森林体系崩溃后，美元没有贬值反而巩固了——石油结算绑定比黄金储备更难绕开。

下次听到"美元要崩盘"的预测时，先问一句：全球石油交易还在用美元结算吗？只要这个锚还在，预测就缺了根基。

{{< sentence >}}
谁控制了石油的定价权，谁就控制了工业国家的命脉。
{{< /sentence >}}

石油不是普通商品，是工业体系运转的前提。控制定价不是控制一种资源，是控制所有依赖它的经济体的行为空间。

理解中东冲突时，先看石油定价权在谁手里，再看各方立场。因果关系经常是反过来的。

{{< sentence >}}
战争是政治的延续，而政治在很多时候是经济的延续。
{{< /sentence >}}

克劳塞维茨的判断再往前推一层。

分析一场军事行动的动机时，政治口号往往是表层；追到经济利益和资源争夺，才能理解为什么"不惜代价"。

{{< sentence >}}
布雷顿森林体系不是一个经济安排，是用制度锁定全球货币权力的起点。
{{< /sentence >}}

1944年的制度设计让美元成为全球结算货币，政治后果远大于经济后果。制度设计本身就是权力行为。

每次看到"国际制度是中性的"这类说法，值得先追问：制度由谁发起、规则由谁制定、违约由谁惩罚？

{{< sentence >}}
美国维持霸权靠的不只是航母，更是一张覆盖全球的金融结算网络。
{{< /sentence >}}

SWIFT系统、美元结算通道、制裁清单——金融基础设施的控制力在日常中看不见，危机中比军事力量反应更快、打击更精准。

一个国家被切断美元结算通道时，不需要一艘航母出动，外贸就能瘫痪大半。

{{< sentence >}}
冷战不是两种意识形态的对决，是两个超级大国争夺资源和市场的总竞赛。
{{< /sentence >}}

不是说意识形态不重要，而是它无法解释很多具体决策的优先级。美国在冷战中多次支持独裁政权，苏联也和资本主义国家做生意——只看信仰，这些行为就成了"例外"；看资源和市场，它们是常规操作。
