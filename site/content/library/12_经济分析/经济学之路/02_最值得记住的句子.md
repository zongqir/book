{{< sentence >}}
稀缺性是经济学所有问题的起点。
{{< /sentence >}}

没有稀缺，就没有经济现象。一旦有限，就开始有价格、有竞争、有配置问题。

{{< sentence >}}
激励改变人的行为方式。
{{< /sentence >}}

所有的经济政策、商业决策、社会现象，都可以追溯到"什么激励导致了什么行为"。

{{< sentence >}}
没有免费午餐，每个选择都有代价。
{{< /sentence >}}

看似完美的政策，往往有隐形的成本。理解权衡，就能预测意想不到的后果。

{{< sentence >}}
用经济学思维看现象，就要问"这个决定背后的激励是什么"。
{{< /sentence >}}

不问"这个政策好不好"，而问"这个政策激励人们做什么"。答案就在行为改变里。

{{< sentence >}}
个人理性和集体结果往往不同。
{{< /sentence >}}

每个人都在做理性决策，但总体的结果可能对所有人都不利。这是经济学最深的悖论。

{{< sentence >}}
经济学不是预测未来，是理解"如果……会怎样"。
{{< /sentence >}}

经济学的价值不在算出精确答案，而在理解因果关系和权衡。
