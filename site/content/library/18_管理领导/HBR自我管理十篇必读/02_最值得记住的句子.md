{{< sentence >}}
大多数人以为他们知道自己擅长什么。通常他们错了。
{{< /sentence >}}

德鲁克在《管理自己》中的开场判断。你以为自己擅长战略思考，实际上你擅长的是执行。你以为自己不擅长社交，实际上你在一对一沟通中比大多数人强。不要猜——用反馈分析法验证。

{{< sentence >}}
管理精力，而非时间，是高效能的关键。
{{< /sentence >}}

洛尔和施瓦茨指出：时间是固定的，精力是可管理的。你有八小时工作时间但只有四小时高精力状态。把最重要的事放在高精力时段做，把机械性任务放在低精力时段——产出质量会有显著提升。

{{< sentence >}}
不要用你衡量事业成功的方式来衡量人生。
{{< /sentence >}}

克里斯坦森的警告：很多高管在事业上极其成功但家庭破裂、健康崩溃、没有真正的朋友。因为他们把工作中的优化思维（追求最大化可衡量的指标）套用到了人生的其他领域——而人生中最重要的事情往往不可衡量。

{{< sentence >}}
你的价值观不是你说出来的，是你做出来的。
{{< /sentence >}}

德鲁克要求每个人搞清楚自己的价值观——不是写在纸上的那种，而是在面对选择时实际引导你行为的那种。你说你重视家庭，但每天工作到晚上十点——你的行为价值观和言语价值观不一致。

{{< sentence >}}
你不需要修补所有弱点，你需要找到你能做出独特贡献的地方。
{{< /sentence >}}

很多人把大量精力花在补短板上。德鲁克说这是浪费——你在弱项上投入再多也只能达到平庸。把精力放在优势上，让自己在强项上从优秀变成卓越。
