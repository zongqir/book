# 八句行动指令，把性格从"天生如此"拉回"做了才知道"

{{< sentence >}}
性格是行动的结果，不是行动的原因。
{{< /sentence >}}

整本书的地基就在这一句。你以为自己内向所以不社交，竹内均说反了——你不社交所以变得内向。因果方向一旦翻转，改变就从"不可能"变成了"换个做法"。

{{< sentence >}}
不要等到有干劲了才开始做，做起来干劲才会来。
{{< /sentence >}}

等动力是最常见的拖延借口。动力不是燃料，是副产品。先把手放到键盘上、先把鞋穿上、先打开那个文件——动起来之后，阻力会迅速下降。

{{< sentence >}}
改变性格的最短路径，是改变每天的行为习惯。
{{< /sentence >}}

不需要顿悟，不需要心理分析，不需要找到"根源"。每天早起十分钟、每天主动跟一个人打招呼、每天写三行笔记——这些看起来微不足道的行为，持续做下去就是性格的重写。

{{< sentence >}}
先让身体动起来，头脑会跟上。
{{< /sentence >}}

身体和心理不是分开的。运动改善情绪，姿态影响自信，表情能反向调节心情。这不是鸡汤，是有实验支撑的身心回路。

{{< sentence >}}
行动可以制造自信，思考不行。
{{< /sentence >}}

自信不是想出来的。做成一件小事，自信增加一点。再做成一件，再增加一点。靠反思获得的"自信"撑不过下一次打击；靠行动累积的自信有地基。

{{< sentence >}}
不要试图一次改变全部，先在一个点上做到。
{{< /sentence >}}

同时改五个习惯，结果是一个也改不成。挑一个最小的行为改变，做到稳定为止，再换下一个。竹内均自己就是这么一个一个改过来的。

{{< sentence >}}
人是习惯的动物——这句话的另一面是，你可以选择新的习惯。
{{< /sentence >}}

"习惯的动物"通常用来解释人为什么难改。但竹内均把重点放在"可以选择"上。既然旧习惯是被无意识安装的，新习惯也可以被有意识安装。

{{< sentence >}}
所谓天性，不过是长期行为留下的痕迹。
{{< /sentence >}}

把"天性"还原为"行为痕迹"，威力很大。痕迹可以被新痕迹覆盖。你觉得自己天生急躁，可能只是从来没练过慢下来。试三个月每天做一件需要耐心的事，再看看"天性"还在不在。
