{{< sentence >}}
奖励一项活动的代价之一，就是这项活动本身的乐趣会减少。
{{< /sentence >}}

德西的经典实验：给本来就喜欢画画的孩子发奖金，一段时间后取消奖金——孩子就不画了。外在奖励挤出了内在动机。平克把这叫做"汤姆索亚效应"的反面：你一旦开始为一件事付费，人们就不再自愿做它。

{{< sentence >}}
控制导致服从，自主导致投入。
{{< /sentence >}}

你能用规则和监控让人按时完成任务。但你不能用规则和监控让人主动创新、深入思考、超出预期。前者是服从，后者是投入。创造性工作需要的是后者。

{{< sentence >}}
精熟是一种痛苦。它需要努力、坚持和刻意练习。但追求精熟本身就是回报。
{{< /sentence >}}

精熟不等于"快乐工作"。在精熟的过程中你会沮丧、挣扎、想放弃——但这种挣扎本身会带来深层满足。打游戏的人不是因为游戏简单才上瘾，恰恰相反。

{{< sentence >}}
利润就像呼吸：你需要它活着，但你不会说活着的目的是呼吸。
{{< /sentence >}}

平克区分利润最大化和目的驱动不是要否定商业——企业当然需要赚钱。但把赚钱当成唯一目标的公司，会用短视的方式伤害员工和客户。目的不是利润的反面，是利润的前提。

{{< sentence >}}
如果外在奖励是做某件事的唯一理由，那它就是一项苦差事。
{{< /sentence >}}

测试一个工作是否有内在动机的简单方法：如果薪水减少10%，你还愿意做这份工作吗？如果答案是"不"，你可能正在做一份没有内在驱动力的工作。
