{{< sentence >}}
奥斯曼帝国对阿拉伯世界的统治不是单纯的压迫——它提供了一种秩序，当这种秩序消失时，替代它的不是自由而是更深的混乱。
{{< /sentence >}}

帝国的崩溃不一定带来解放。当旧秩序消失时，新秩序的建立需要时间——空白期通常被外部势力和内部冲突填满。

{{< sentence >}}
一战后的中东边界不是阿拉伯人画的。赛克斯-皮科协定用一把尺子在地图上划了几条直线——这些直线切开了部落、族群和贸易路线，制造了持续至今的冲突。
{{< /sentence >}}

人为划定的边界不会自动获得合法性。当边界不反映地面现实时，它不是在解决问题而是在制造问题。

{{< sentence >}}
阿拉伯民族主义运动不是一场运动而是很多场——每一个国家有自己的版本，彼此之间的矛盾有时比它们和外部敌人的矛盾更大。
{{< /sentence >}}

"阿拉伯世界"是一个地理概念不是一个政治实体。内部的多样性远大于外部观察者想象的。

{{< sentence >}}
纳赛尔的泛阿拉伯主义在1967年战争中破碎了。一场六天的战争摧毁了一代人的政治梦想——接下来填补空白的是政治伊斯兰。
{{< /sentence >}}

世俗民族主义的失败为宗教政治打开了空间。政治真空不会保持空白——总会有新的力量来填充。

{{< sentence >}}
阿拉伯之春揭示了一个悖论：推翻独裁者比建立替代秩序容易得多。革命的最难部分不是开始而是之后。
{{< /sentence >}}

破坏旧秩序需要的是愤怒和勇气；建设新秩序需要的是制度设计和妥协能力——后者远比前者稀缺。
