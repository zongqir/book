# 拿到一本书，先别急着从第一页读起

## 三步最小版

如果只做三件事：

1. **读前十分钟**：看目录、序言、结尾。写下一句话——在解决什么问题，我为什么读，值不值得精读。
2. **读中抓三件**：作者的核心主张是什么；靠什么论据支撑；结构怎么展开。
3. **读后落一版**：合上书，用自己的话写出三条主张、一段论证摘要、一条个人判断。

做到这三步，一轮阅读就从"翻完"升级成了"读懂并记住"。

后面的展开版是在这个基础上加细节，不是另一套流程。

## 读前：建立判断再投入

拿到一本书，先别翻正文。花十分钟看四个地方：目录、序言或前言、最后一章的小标题、封底介绍。

然后写下三行：

- 在解决什么问题
- 我当前为什么需要读它
- 我准备读到什么程度（扫一遍 / 只读核心章 / 全书精读）

这三行写不出来，先不要开始精读。

## 检视阅读：十分钟建地图

按目录过一遍全书。每章用一句话标注用途：核心论证？背景铺垫？案例补充？可以跳过？

快速翻读每章的开头和结尾。遇到不懂的地方标记，不停下来钻。

抓反复出现的关键词。一本书通常有三到五个核心概念——它们决定后面的理解门槛。

这一轮结束，至少能回答四个问题：

- 核心问题是什么
- 作者大致怎么展开
- 哪几章最重要
- 第二遍重点看什么

## 分析阅读：拆开作者的论证

现在才开始认真读。盯住四件事：

1. **找主张**——作者最想让你接受的结论是什么。
2. **找概念**——作者定义了哪些关键词，这些词在书里的意思和日常用法有没有区别。
3. **找推理**——结论靠什么前提和论据推出来的。哪些环节强，哪些环节弱。
4. **找边界**——作者回答了什么问题，没回答什么问题。

读完重点章节后，做一个测试：合上书，用自己的话把核心论证讲一遍。讲不出来，就回去重读。

## 评价：理解站稳之后再开口

评价有一个前置条件：你能准确复述作者的主张和推理。

满足这个条件之后，写下三组判断：

- 我同意什么，为什么
- 我保留什么，卡在哪里
- 我反对什么，我的依据是什么

区分清楚：你的依据是事实、逻辑，还是个人偏好。三者混在一起的评价没有复用价值。

## 读后落盘：五项必留

读完一本值得保留的书，至少留下这些：

1. 一句话：在解决什么问题
2. 三条核心主张
3. 一段主干论证摘要
4. 一段个人判断（同意 / 保留 / 反对 + 理由）
5. 一条下一步动作（进入专题？再读一本相关书？用在某个项目上？）

前四项是这轮阅读的沉淀。第五项决定它会不会变成死档案。

## 按书的类型调整力度

不是每本书都要跑全流程。

- **入门书、概览书**：检视阅读 + 三条主张就够了。
- **高密度论述书**：完整走分析阅读，尤其抓论证链。
- **工具书、手册**：按需翻查，不需要整本拆解。
- **小说、散文**：不套分析阅读框架。关注体验和表达，摘录就是产物。

判断标准不复杂：需要理解作者的论证才算读懂吗？需要就拆，不需要就别硬套。
