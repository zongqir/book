## 识别自动反应的特征

{{< sentence >}}
你在压力下的第一反应，往往最能代表你的真实样子，而不是你想要的样子。
{{< /sentence >}}

在放松的时候，你能控制自己的表现。但当压力突然增加、情绪被触发时，防御机制会接管你的行为。此时你做出的反应，往往比你平时想象的样子更接近真实。

{{< sentence >}}
被批评时立刻反驳，被拒绝时立刻否认，被忽视时立刻自怜——这些都不是你的缺点，而是心理防御机制在工作。
{{< /sentence >}}

这些反应有个共同特点：它们都很快，往往在你还没意识到之前就已经发生了。快速到什么程度？快到你甚至感觉不到是"反应"，而是觉得"这就是事实"。

{{< sentence >}}
你认识的"自己"，往往是你想象中的自己，而不是压力下的真实自己。
{{< /sentence >}}

在镜子前面或和朋友聊天时，你可能说"我很理性""我不会被批评打击"。但真的遭遇批评时，你可能会发现自己的反应和这个自我评价完全不符。这不是你在说谎，而是"你想象的自己"和"压力下的自己"是两个不同的系统。

## 理解防御机制的起源

{{< sentence >}}
你的很多防御方式，都是在你最没有能力应对的时候学会的。
{{< /sentence >}}

不管是童年的环境还是过去的创伤，都会塑造出一套"在这种情况下怎样才能活下去"的反应方式。当时这些方式可能是有用的、甚至是救命的。但现在，同样的情况已经改变了，这些防御方式往往变成了你的限制。

{{< sentence >}}
防御机制的目的是保护你，但它往往过度保护，反而限制了你的生活。
{{< /sentence >}}

如果你曾经因为表达自己而被批评，你的心理系统可能学会了"沉默会更安全"。这个防御在那时可能是对的，但现在它可能让你错失了表达自己、被倾听、建立深刻关系的机会。
