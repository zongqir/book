{{< sentence >}}
奋斗不是看你有多累，而是看你获得了什么。
{{< /sentence >}}

有些人每天加班到晚上十点，却没有任何成就。有些人工作高效，虽然时间不长但获得了明显的成果。衡量奋斗的标准应该是结果，而不是时间。

{{< sentence >}}
最聪慧的奋斗往往看起来很轻松。
{{< /sentence >}}

那些通过过度加班才完成任务的人，说明他的工作方法有问题。真正的奋斗高手往往能够用更少的时间完成更多的工作——因为他们的方法更聪慧。

{{< sentence >}}
奋斗一时容易，坚持奋斗很难。关键是保持热情而不是被消耗。
{{< /sentence >}}

长期的职业发展不是百米冲刺，而是马拉松。需要的是能够持续多年的热情，而不是短期的冲动。

{{< sentence >}}
有时候停止也是奋斗的一部分。
{{< /sentence >}}

有些方向已经证明不可行，继续奋斗反而是浪费。聪慧的人知道什么时候应该改变方向或停止某个项目。

{{< sentence >}}
奋斗的最大敌人是舒适。
{{< /sentence >}}

当生活变得舒适时，人往往就停止了奋斗。但恰恰是在舒适中，人最容易被时代淘汰。

{{< sentence >}}
奋斗中的失败是学习，不是失败。
{{< /sentence >}}

每一次失败都提供了宝贵的学习机会。把失败当作学习，而不是失败，能帮助你从挫折中快速恢复。
