## 自尊的结构

{{< sentence >}}
自尊建立在三根柱子上：自爱、自我形象和自信。三根柱子可以分别出问题。
{{< /sentence >}}

自爱是无条件的底色——接受自己，包括接受缺点。自我形象是对自己的认知是否准确。自信是面对具体事情时相信自己能应对。哪根柱子松了，表现完全不同，修法也不同。

{{< sentence >}}
健康的自尊不是高自尊，而是稳定的自尊。
{{< /sentence >}}

稳定的自尊遇到打击会弯，但不会断。脆弱的高自尊看起来比低自尊更好，实际上因为需要不断被确认，反而更容易崩塌。判断标准不是"你觉得自己好不好"，而是"遇到挫折之后你需要多久恢复"。

{{< sentence >}}
过高的自尊和过低的自尊一样危险，只是伤害的方式不同。
{{< /sentence >}}

低自尊让人退缩、回避、放弃尝试。过高的自尊让人听不进反馈、无法从错误中学习、在关系里不断制造冲突。两个极端都是自尊不健康的信号。

## 外部确认的陷阱

{{< sentence >}}
需要别人不断确认才能维持的自尊，是借来的自尊。
{{< /sentence >}}

如果三天没有收到正面反馈你就开始怀疑自己，自尊的供能系统就是外挂的。修复方向不是"找更多人夸你"，而是建一个不依赖外部输入也能运转的内部评价系统。

{{< sentence >}}
自信可以通过行为训练获得，自爱不能——自爱需要的是接纳，不是成就。
{{< /sentence >}}

考了证、升了职、跑了马拉松——自信可以随着这些成就增长。但自爱是另一回事。一个成就等身的人如果不能接受自己的脆弱和局限，自爱那根柱子仍然是空的。

## 修复的方向

{{< sentence >}}
鼓励对自尊的作用，像止痛药对骨折的作用——暂时不痛了，但什么都没接上。
{{< /sentence >}}

"你很棒""你值得被爱"这类话在说出来的那一刻确实让人舒服。但它不修复任何结构性的问题。真正的修复从诊断开始：到底是自爱缺失、自我形象歪曲，还是特定领域的自信不足。

{{< sentence >}}
自尊的改变不是顿悟，是无数次小修正的累积。
{{< /sentence >}}

不存在一个"想通了就好了"的时刻。每次在批评面前停住自动贬低反应，每次在赞美面前不急着否认，都是一次微小的结构修复。累积到一定量，稳定性自然会变化。
