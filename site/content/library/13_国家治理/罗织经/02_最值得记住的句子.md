下面这些句子都是来俊臣站在施害者视角写的。它们读起来冷，但每一句背后都对应一个可以被识别的操作节点。

## 侦察与揣摩阶段

{{< sentence >}}
"人者多欲，其性尚私。"
{{< /sentence >}}

所有构陷的起点都是摸清目标的欲望和隐私。来俊臣的意思很直接：人都有弱点，找到弱点就找到了杠杆。防御动作：当有人不寻常地关心你的私事、利益纠葛或人际关系时，先问自己——对方要用这些信息做什么。

{{< sentence >}}
"察其所好，乃投之以计。"
{{< /sentence >}}

知道目标喜欢什么之后，下一步是用目标的偏好设计陷阱。不是正面攻击，是顺着目标的惯性走，让目标自己走进预设的位置。在职场中常见的变体：对方主动帮你推荐一个看似有利的机会——先想想这个"机会"服务谁的利益。

## 孤立与信息操控阶段

{{< sentence >}}
"恩不可过，过则招嫉。"
{{< /sentence >}}

这句话拿来构陷的用法是：对目标的上级强调"他受的恩太多了"，激发嫉妒和猜忌。让上级觉得目标"功高震主"或"忘恩负义"，比编造罪名有效得多。

防御动作：如果有人开始在你的上级面前"替你说好话"说得太过，当心——过度吹捧和过度打压在构陷术里是同一个工具的两面。

{{< sentence >}}
"敌者，一人之敌也；友者，亦一人之友也。"
{{< /sentence >}}

来俊臣提醒自己：敌人的敌人可以暂时变成朋友，朋友的利益也可以随时让他变成敌人。关系网络是可以被重新编织的。他做的就是重新编织目标的关系网，让原来支持目标的人变成中立者甚至帮手。

遇到平时的盟友突然转变立场时，不要先归因为"他变了"。先判断有没有人在中间做了什么。

## 审讯与定罪阶段

{{< sentence >}}
"人不自疑，无以伐善。"
{{< /sentence >}}

整部《罗织经》里最冷的一句。让目标自己开始怀疑自己，是构陷术的最高效率点。来俊臣的审讯不靠酷刑——靠反复、疲劳、信息隔离和话术引导，让目标在精神消耗后主动"承认"或"解释"，而每一次解释都变成新的指控材料。

在任何对抗性对话里，对方越想让你"好好说清楚"，你越要节制表达。

{{< sentence >}}
"事不至大，无以惊人。"
{{< /sentence >}}

来俊臣从不满足于小案子。他把每个案子扩大、牵连更多人，直到参与定罪的人太多、没有人敢翻案。规模本身就是武器。

在舆论场上，一个定性一旦被大量转发，纠错的成本就不是"提供真相"能解决的了。窗口期在定性刚出现的那一两天。

{{< sentence >}}
"无信则人不附，附而不信则必叛。"
{{< /sentence >}}

来俊臣用这句话管理自己的线人和打手网络：要让手下觉得跟着你有好处，但不能让他们真的信任你——信任太深会滋生独立判断。他对工具人的管理方式是"有利可图但永远不安全"。

反过来看，当你被一个组织用"有利可图但永远没有安全感"的方式管理时，你很可能正在被当作工具使用。

## 场景速查

| 你遇到的情况 | 对应的操作阶段 | 回看哪句 |
|---|---|---|
| 有人异常关心你的私事和利益关系 | 侦察与揣摩 | 人者多欲，其性尚私 |
| 一个"好机会"突然出现，来路不明 | 投放诱饵 | 察其所好，乃投之以计 |
| 你的盟友集体沉默或回避 | 孤立 | 敌者/友者 |
| 有人在上级面前过度替你"说好话" | 激发猜忌 | 恩不可过，过则招嫉 |
| 对方要求你"把事情详细讲清楚" | 诱导自证 | 人不自疑，无以伐善 |
| 事件规模迅速扩大，牵连越来越多 | 案件膨胀 | 事不至大，无以惊人 |
