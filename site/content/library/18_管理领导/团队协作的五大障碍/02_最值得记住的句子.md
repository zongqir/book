{{< sentence >}}信任是一切团队合作的基础，没有信任就没有真正的冲突，没有冲突就没有真正的投入{{< /sentence >}}

信任不是相信别人不会犯错，而是相信即使暴露弱点也不会被利用。团队成员只有在这种安全感下，才敢于表达真实想法，进入建设性冲突。

{{< sentence >}}恐惧冲突的团队永远不会有真正的共识，因为他们从未经历过真正的辩论{{< /sentence >}}

表面和谐掩盖了根本分歧。没有经过充分讨论和质疑的决定，团队成员内心并不真正认同，执行时自然会出现各种问题。

{{< sentence >}}投入不是参与，而是对决定的完全支持，即使这个决定不是你的首选{{< /sentence >}}

真正的投入意味着即使你在讨论中是少数派，一旦团队做出决定，你就会全力支持。这需要充分的讨论过程作为前提。

{{< sentence >}}逃避责任的本质是不愿意让队友感到不适，但这种"善意"实际上伤害了整个团队{{< /sentence >}}

很多管理者误以为维护关系就是避免直接反馈。但真正的责任文化要求团队成员互相提醒和纠正，这是对共同目标的承诺。

{{< sentence >}}当团队成员把个人需求或部门利益放在团队结果之前时，整个组织就会陷入政治游戏{{< /sentence >}}

结果导向不仅仅是关注业绩指标，更重要的是让所有人明确什么是真正的成功，并愿意为了团队成功而暂时牺牲个人或部门的短期利益。

{{< sentence >}}金字塔的每一层障碍都必须先解决下层问题，跨层级的解决方案注定失败{{< /sentence >}}

这是整个模型的核心逻辑。很多团队干预失败的原因就在于跳过了基础层面，试图在缺乏信任的基础上建立责任制，或者在回避冲突的文化中推行结果导向。

{{< sentence >}}领导者的首要任务不是避免冲突，而是示范脆弱性，为团队创造信任的起点{{< /sentence >}}

领导者必须首先展示自己的不完美，承认错误，请求帮助。这种脆弱性示范会鼓励团队成员也这样做，从而建立真正的信任基础。这是整个金字塔的地基。
