## 价格与管制

{{< sentence >}}
价格不是商家的意愿，是供需关系的信号。管制价格不是消灭问题，是消灭信号。
{{< /sentence >}}

当政府限制某种商品的价格，短缺不会因为价格被压低而消失——它只是从"贵"变成了"买不到"。价格管制消灭的是让供需匹配的那个信号。

{{< sentence >}}
租金管制保护的不是穷人，是已经住在里面的人。
{{< /sentence >}}

租金管制降低的是现有租户的成本，但同时减少了新建住房的动力。新来者面对的是更少的房源和更长的等待。受保护的群体和受损的群体不是同一群人。

## 贸易与就业

{{< sentence >}}
贸易保护拯救的工作岗位是看得见的，毁掉的工作岗位是看不见的。
{{< /sentence >}}

对进口钢铁加征关税确实保住了钢铁厂的工人。但用钢铁的汽车厂、建筑公司因为成本上升而裁掉的人，没有出现在新闻标题里。看得见的受益者和看不见的受害者，这是经济政策评估中最常见的盲区。

{{< sentence >}}
一个国家不需要在任何领域都有优势，才能从贸易中获益。
{{< /sentence >}}

比较优势原理的核心不是"你要擅长什么"，而是"你把有限资源投在哪里损失最小"。即使一个国家什么都比另一个国家差，两个国家通过贸易仍然可以同时获益。

## 歧视与差异

{{< sentence >}}
差异不等于歧视，但歧视的结论往往比差异的原因更容易传播。
{{< /sentence >}}

不同群体之间的收入差异、就业差异确实存在。但差异的存在不能自动证明歧视是原因。年龄结构、教育分布、行业偏好、地理分布——这些变量在被控制之前，差异的数字本身什么也说明不了。

{{< sentence >}}
统计数据给你的是相关性，因果推断需要你自己负责。
{{< /sentence >}}

"A 群体收入低于 B 群体"是统计事实。"A 群体收入低是因为受到歧视"是因果推断。前者可以直接从数据里读出来，后者需要排除所有替代解释才能成立。混淆这两层是经济争论中最高频的错误。
