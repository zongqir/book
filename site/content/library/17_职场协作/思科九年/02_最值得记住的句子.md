{{< sentence >}}
管理不是控制，是服务。
{{< /sentence >}}

林正刚发现最有效的管理方式不是告诉团队做什么——是帮他们扫清障碍。当你的工作从"指挥"变成"服务"，团队的主动性会大幅提升。

{{< sentence >}}
在大公司里，横向关系比纵向关系更重要。
{{< /sentence >}}

你的直属上下级关系当然重要。但在跨部门协作密集的大公司里，和其他部门的关系往往决定了你能不能做成事。花时间维护跨部门关系不是浪费——是投资。

{{< sentence >}}
每一次横向调动都是一次学习机会。
{{< /sentence >}}

很多人把横向调动当降级。林正刚的经验相反——每一次换部门、换职能，都让他理解了公司的另一个维度。最终这些跨领域经验成了他升到高管的关键优势。

{{< sentence >}}
和总部打交道的关键是让他们看到你的贡献，而不是你的需求。
{{< /sentence >}}

向总部要资源的最好方式不是说"我需要"——而是说"我已经做到了什么，如果再给我这些资源我可以做到更多"。先展示价值，再要支持。

{{< sentence >}}
做对的事比做对事情更重要。
{{< /sentence >}}

方向错了，执行力越强损失越大。林正刚在九年里做过几次重大方向选择——有些选对了，有些选错了。选对方向的收益远超执行细节上的优化。
