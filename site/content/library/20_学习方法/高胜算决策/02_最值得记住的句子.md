# 高胜算决策：最值得记住的句子

## 拆掉结果论的底层提醒

{{< sentence >}}生活不是象棋，而是扑克。{{< /sentence >}}

象棋信息完全透明，输赢取决于计算能力。扑克有隐藏信息、随机发牌、对手的伪装。绝大多数现实决策更接近扑克——你永远在信息不完整的条件下行动。承认这一点，是停止用结果倒推决策质量的起点。

{{< sentence >}}结果不好不等于决策不好；结果好也不等于决策好。{{< /sentence >}}

Duke 把用结果评价决策的习惯叫 resulting。一次正确的下注可能输钱；一次错误的全押可能碰巧赢了。如果只看结果学经验，学到的很可能是错误的经验。

{{< sentence >}}想要做出更好的决策，先要学会对"我不知道"感到舒适。{{< /sentence >}}

确定感是大脑的默认奖励。承认不确定让人不舒服，但只有在不确定的缝隙里，校准才有可能发生。扑克手的基本功不是胆大，而是精确地知道自己不知道什么。

## 让概率直觉落地的触发器

{{< sentence >}}每次你说"我确定"的时候，试着改成"我愿意赌多少"。{{< /sentence >}}

全书最核心的行为触发器。一旦把断言转化为下注，大脑会从"捍卫立场"切换到"评估概率"。你不再关心对错，开始关心把握有多大。

{{< sentence >}}信念像赌注，不像事实——它们应该随新证据更新。{{< /sentence >}}

大多数人把信念当成身份的一部分来保护。Duke 要求你把信念当成概率估计：70% 相信一件事，意味着还有 30% 的空间留给反面证据。新信息进来时，调整概率，不调整自尊。

{{< sentence >}}赢的时候少归功于自己，输的时候少归咎于运气。{{< /sentence >}}

自我归因偏差的一句话纠偏。人的默认模式是成功归能力、失败归运气。反过来练习：赢了多想运气成分，输了多想决策缺陷。长期下来，归因更准。

## 在团体和时间维度上的校准

{{< sentence >}}如果你周围的人只会说"你说得对"，你的判断在持续恶化。{{< /sentence >}}

求真团体的核心不是"大家一起变聪明"，而是有人认真质疑你的推理过程。没有这种外部压力，确认偏差会把判断越带越偏——而你完全感觉不到。

{{< sentence >}}做决定之前，问自己：十分钟后、十个月后、十年后，我会怎么看这个选择？{{< /sentence >}}

10-10-10 规则是对抗短期情绪绑架的工具。当下的愤怒、兴奋、恐惧会扭曲判断；拉开三个时间窗口，短期情绪的权重自动下降。
