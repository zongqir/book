{{< sentence >}}
说话前先想清楚，你希望对方听懂什么。
{{< /sentence >}}

很多表达失败，不是因为词没选好，是因为目标本来就没想明白。你只是急着说，却没先定义结果。

{{< sentence >}}
把感受直接说出来，通常比绕着说更不伤人。
{{< /sentence >}}

绕着说看起来委婉，实际常常让对方只感到压力，不知道你真正卡在哪里。直说不是硬，说清楚才是。

{{< sentence >}}
情绪最满的时候，最不适合补充解释。
{{< /sentence >}}

你以为自己在说明白，对方听到的往往只有情绪强度。先降温，再补信息，解释才有入口。

{{< sentence >}}
请求要说到动作，不要停在态度。
{{< /sentence >}}

“你重视一点”太空，“周五前把这件事回我”才有执行面。说到动作，对方才知道自己该怎么做。

{{< sentence >}}
对方反应很差时，先检查你说的是内容，还是夹带了评判。
{{< /sentence >}}

很多话表面在谈事，里面其实藏着“你怎么总这样”。对方一旦先听到评判，内容就进不去了。

{{< sentence >}}
好好说话的第一步，不是更会说，而是先不乱说。
{{< /sentence >}}

少抢答，少补刀，少在没想清楚时把话扔出去。很多沟通质量，先靠收住，再靠展开。
