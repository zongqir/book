## 攻击面前的第一反应

{{< sentence >}}
当你开始解释、辩护或反击时，冲突就进入了升级通道。
{{< /sentence >}}

这是整本书最核心的一个判断。冲突里的防御——无论是解释、辩护还是反击——在对方眼里全都是攻击。你以为自己在澄清，对方看到的是你在反驳。

{{< sentence >}}
冲突中最有用的第一个动作不是说什么，是停下来。
{{< /sentence >}}

停下来不是认输。停下来是切断"攻击→防御→升级"这个自动循环。三十秒的沉默比三十秒的辩解有用一百倍。

## 倾听不是沉默，是一个动作序列

{{< sentence >}}
你以为你在听，其实你在准备反驳。
{{< /sentence >}}

冲突中的"倾听"几乎不存在。对方说话时，你的注意力全部用来组织自己的下一句。卡斯帕森的倾听循环要求你把注意力从自己的反驳转移到对方的话上——听，复述，确认，再听。

{{< sentence >}}
承认对方的感受，不等于承认对方是对的。
{{< /sentence >}}

这是多数人在冲突中最大的混淆。"我能看出来你很生气"不是"你生气是对的"。把房间里已经存在的情绪说出来，本身就能降温。

## 区分人和问题

{{< sentence >}}
攻击对方的人格，你们就不再是在解决同一个问题了。
{{< /sentence >}}

从"这个方案有问题"滑到"你怎么总是这样"只需要一句话。但滑过去之后，对话就不再是关于方案了，而是关于"你是什么人"。回不来的。

## 需要和策略的分界

{{< sentence >}}
你以为你们在争一件事，其实你们在争满足需要的方式。
{{< /sentence >}}

两个人抢同一间会议室。表面上冲突不可调和。但一个人需要的是安静的空间，另一个需要的是投影设备。需要不同，可以各退一步；策略相同，就只能硬抢。

{{< sentence >}}
找到对方的需要，冲突就从对抗变成了联合解题。
{{< /sentence >}}

卡斯帕森和费希尔在这里说的是同一件事：从立场退到利益。但她的语境是冲突现场，不是谈判桌。在冲突现场做到这一步比谈判桌上难十倍，因为情绪在烧。
