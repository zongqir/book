鸦片战争的故事被讲过无数遍。"落后就要挨打"是最常见的总结。但茅海建翻遍档案之后发现，问题不在"落后"本身——清军在局部战斗中并非毫无还手之力——问题在于，整个决策系统始终在处理失真信息，始终用一套封闭的认知框架理解陌生对手。

以下五条判断准则，从鸦片战争的具体决策环节中提取，指向信息机制和认知框架层面的系统性失灵。

{{< principle no="1" type="core" statement="一、信息失真是激励结构的产物" >}}
道光年间的官僚系统不缺聪明人。林则徐能读外文报纸，琦善能判断军事形势，前线将领中也有人清楚英军实力远超清军。

失真的根源不在"谁撒了谎"，而在"什么样的信息能安全地向上传递"。奕山在广州打了败仗，奏折里写成"大捷"；道光信以为真，下旨嘉奖。如实汇报广州军事劣势的琦善，被锁拿进京。

信号非常明确：系统奖励报喜，惩罚报忧。在这种激励结构下，失真不需要有人蓄意策划。每个环节的理性选择汇聚起来，就是系统性的谎言。

**判断准则：** 如果一个组织里坏消息的传递者比坏消息本身受到更严厉的对待，信息链已经在失真——无论表面流程多么规范。
{{< /principle >}}

{{< principle no="2" type="core" statement="二、认知框架封闭比信息失真更难修复" >}}
信息失真至少还有一个理论上的修复路径：改变激励结构，让说真话的人不受罚。认知框架的封闭没有这种修复接口。

"天朝"不只是自大的称呼。它是一整套关于世界秩序的解释系统：中国是中心，四方是藩属，英国是蛮夷。在这套系统里，"蛮夷"打不过天朝是公理，不是需要检验的假设。

道光不理解英国为什么要打仗。他的解释框架里，蛮夷无非想通商和赏赐；给了好处自然会退。林则徐最初也在这个框架内：烟禁了，贸易断了，蛮夷自然服软。等到发现英军战力远超预期，林则徐开始局部修正——但框架本身没有被动摇。

更危险的是，封闭是集体性的。从皇帝到军机大臣到地方督抚，共享同一套天朝世界观。没有人拥有一个替代性的理解框架来重新解释英国的行为动机和军事能力。

**判断准则：** 如果一个组织对外部对手的理解建立在不可修正的前提假设上——比如"他们不可能比我们强"或"他们的目的一定是我们能理解的"——再多的信息采集也无法转化为有效决策。
{{< /principle >}}

{{< principle no="3" type="core" statement="三、惩罚说真话的系统丧失自我纠错能力" >}}
琦善的遭遇是鸦片战争中值得逐日细看的一段。

他到广州后，亲眼看到英军火力和清军之间的巨大差距。他试图通过谈判争取时间，也试图向道光传递战场真实状况。但道光刚刚收到奕山"大捷"的奏报，两份信息放在一起，琦善的版本显然"不忠诚"。

最终琦善被撤职锁拿，罪名之一是"擅自割地"。此后接任的将领，没有人再敢如实汇报。

纠错依赖真实信息的上传；上传依赖传递者不被惩罚；惩罚琦善这件事，等于向全系统广播了"说实话的下场"。系统的自我纠错能力就此关闭。

**判断准则：** 观察一个组织的纠错能力，不要看规章制度，要看最近一个说真话的人后来怎么样了。
{{< /principle >}}

{{< principle no="4" type="support" statement="四、军事差距可弥合，信息链断裂不可逆" >}}
清军与英军的武器差距确实存在。英舰的火炮射程、精度和威力都超过清军岸防。但茅海建用大量战例证明，差距没有大到"完全不能打"的程度。

关天培在虎门坚守到战死。陈化成在吴淞口拼死抵抗。三元里民众在近战中给英军造成了麻烦。如果能在战略层面做出正确判断——集中兵力、选择有利地形、争取外交缓冲——结果未必是单方面溃败。

不可逆的是信息链的断裂。当决策中枢收不到真实战况，当前线指挥官不敢请求增援——因为请求增援等于承认打不赢——整个系统就丧失了协调行动的能力。

武器可以买，技术可以学。一条已经断裂的信息链，在战争进行中几乎无法修复。

**判断准则：** 评估一个组织的危机应对能力，硬件差距不是第一优先；先看信息链是否还在传递真实信号。
{{< /principle >}}

{{< principle no="5" type="support" statement="五、封闭系统对全新对手没有识别接口" >}}
清朝不是没有处理过外部威胁。准噶尔、大小金川、缅甸、安南——此前的军事冲突对手都在"天朝"认知框架之内。争夺领土、索取利益、反抗管辖——都是可理解的。

英国不在这个框架里。它的战争目的——自由贸易、外交平等、领事裁判权——对清朝来说不可理解。不是"不愿意理解"，是认知系统里没有对应的概念槽位。

道光反复用"蛮夷滋事"来解释战争，反复试图用"赏赐通商"来结束战争。这不是个人的愚蠢，而是整个认知框架对全新类型的对手缺乏识别机制。

**判断准则：** 一个系统过去成功应对过的对手类型，恰恰可能成为它识别新类型对手的障碍。"我们以前都能搞定"这种经验，在面对全新对手时是最危险的麻醉剂。

## 五条准则的共同指向

这五条判断准则可以拆开用，但它们的力量在合并时最大。

信息失真提供了错误的决策输入。认知框架封闭阻止了对输入的修正。惩罚说真话的机制关闭了纠错通道。信息链断裂使协调行动变得不可能。缺乏对新对手的识别接口使所有应对都建立在错误前提上。

五条同时发作，任何单点修复都来不及。鸦片战争的决策溃败不是某个环节出了问题，而是每个环节都在各自失灵的同时互相加强。
