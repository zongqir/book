{{< sentence >}}
每一次高难度谈话都包含三层对话：发生了什么、情绪、身份认同。
{{< /sentence >}}

表面上你们在争论事实（"到底是谁搞错了"），底下在管理情绪（"我很受伤但不敢说"），最深处在保护身份（"我是不是一个不称职的人"）。只在事实层纠缠永远解决不了问题。

{{< sentence >}}
不要假设你知道对方的意图。你看到的是对方的行为，不是对方的意图。
{{< /sentence >}}

"他故意让我难堪"——这是你对行为的解读，不是事实。同样的行为背后可能有完全不同的意图。在弄清之前就把负面意图归给对方，对话还没开始就已经失败了。

{{< sentence >}}
高难度谈话的目标不是赢，而是学习。
{{< /sentence >}}

如果你进入对话的目的是说服对方你是对的，你会错过所有新信息。把目标从"让对方认错"改成"理解对方为什么这么看"，对话的质量会发生根本变化。

{{< sentence >}}
每个人都有一个"身份对话"在后台运行。
{{< /sentence >}}

给下属提负面反馈时，他的后台程序可能在跑"我是不是要被开除了"或"我是不是真的很差"。这个身份威胁如果不被处理，你说什么他都听不进去。

{{< sentence >}}
情绪不是谈话的障碍，情绪是谈话的一部分。
{{< /sentence >}}

压抑情绪不会让情绪消失，只会让它以更隐蔽的方式影响对话——讽刺、冷漠、被动攻击。承认情绪的存在（"我注意到我自己有些沮丧"），反而能让对话回到正轨。

{{< sentence >}}
从第三个故事开始谈话。
{{< /sentence >}}

不要从"你错了"开始，也不要从"我觉得"开始。从"我们对这件事有不同的看法"开始——这是第三个故事，一个双方都能接受的起点。
