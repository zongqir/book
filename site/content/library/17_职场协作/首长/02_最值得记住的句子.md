{{< sentence >}}
最困难的决策往往不是大方向错了，而是小环节执行变形了。
{{< /sentence >}}

赵德良推进一项重大改革时，大的方向得到了大多数人同意。问题出在执行环节。下级部门为了规避风险，把改革条款改得面目全非。他最后花费的精力，不是在论证改革必要性，而是在一个一个环节上纠正执行偏差。这个时间往往比初期的方向决策长得多。

{{< sentence >}}
有人不执行是态度问题，有人执行得慢是能力问题，有人执行得变形是对你决策的微弱反对。
{{< /sentence >}}

赵德良通过执行细节来判断下属的真实态度。直接抗命的人很少。更常见的是那些用"有难度"、"需要时间"、"要调研一下"来变相反对的人。他们没有说"不"，但他们用执行速度和执行方式来表达不同意。你要能读出这个信号，并判断是该强推还是该听他们的实际意见。

{{< sentence >}}
权力越大，信息越容易被扭曲。所以领导者要养成直接听负面信息的习惯。
{{< /sentence >}}

省委书记是全省最有权力的人，也因此最容易被伪装的汇报所蒙蔽。很多人在赵德良面前都会说好听的话。他要做的是主动询问坏消息、问题和反对意见。有些部下到最后都不敢在他面前表达真实看法，不是因为他脾气坏，而是因为他的权力本身形成了一道隐形的压力。

{{< sentence >}}
在你改变某个制度前，先问这个制度为什么会存在。它的存在往往解决了什么问题，你的改变也会破坏什么。
{{< /sentence >}}

赵德良想改掉的一项工作流程看起来很低效。但当他深入了解后发现，这个流程看起来低效的部分，恰好是用来防止另一种风险的。改革时他不是完全推翻，而是在保留防风险机制的前提下提高效率。这需要他理解制度的历史——每一个环节通常都是在过去的某个事件后加进来的。

{{< sentence >}}
不是所有的不同意都值得听。但你要能分辨哪些不同意来自真正的专业见解，哪些来自既得利益的保护。
{{< /sentence >}}

赵德良面对多种反对声音。有人反对是因为改革触动了他们的部门权力，有人反对是因为他们确实看到了方案里的漏洞。前者再多的讨论也解决不了问题，后者需要他认真对待。他要学会区分这两种反对，这需要同时了解反对者的利益位置和他们的专业水平。

{{< sentence >}}
战术上的坚定和战略上的灵活不矛盾。你在大原则上可以一步不退，在实现的细节上可以根据阻力调整。
{{< /sentence >}}

赵德良推行的一项政策遇到了很强的既得利益集团反对。他没有在"这个政策必须推"上妥协，但在"怎么推、什么时间表、谁先受影响"上做了很大的灵活调整。坚定的是目标，灵活的是路径。混淆这两者的领导者，要么变成了固执己见，要么变成了妥协的人。
