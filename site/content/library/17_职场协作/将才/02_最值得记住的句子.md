# 勤奋是最容易让人放弃思考的东西

## 瓶颈在思维层次，不在努力程度

{{< sentence >}}
将才和兵才的差别不在能力高低，在思考的层次。
{{< /sentence >}}

能力够了还升不上去的人，问题几乎都在这里。执行层思考和全局思考之间的鸿沟，不靠加班能填。

{{< sentence >}}
用勤奋回避思考，是职场里最隐蔽的偷懒。
{{< /sentence >}}

很多人忙到没时间想，以为自己特别拼。杜书伍的判断正好反过来：忙到没时间想，说明你在用体力逃避脑力。

{{< sentence >}}
观念的天花板比能力的天花板更难打破——因为你看不见它。
{{< /sentence >}}

能力不够可以学，但你觉得"我的做法没问题"的时候，连学的方向都找不到。观念瓶颈最危险的地方是：卡住你的东西，你完全意识不到。

## 积累优先于冲刺

{{< sentence >}}
职涯是爬楼梯，不是坐电梯——跳过的每一级，后面都要补回来。
{{< /sentence >}}

急着升迁的人最该记住这句。每次跳过基础去够更高的位子，欠下的债迟早要还，而且利息很高。

{{< sentence >}}
先把事情做对，再把事情做快。
{{< /sentence >}}

追求效率之前先确认方向。做得快但方向错了，只是更快地浪费时间。

{{< sentence >}}
做到别人替代不了，比做到最快更有价值。
{{< /sentence >}}

速度谁都能追，不可替代性才是护城河。想清楚你的价值是"做得快"还是"只有你能做"。

## 格局和自我修炼

{{< sentence >}}
格局不是看得远，是看得到自己和全局的关系。
{{< /sentence >}}

很多人以为格局就是"站得高看得远"。杜书伍的定义更实在：你能不能看清自己手上这件事，在组织里处于什么位置、会牵动什么。

{{< sentence >}}
不要等"准备好了"再往上走——你在当前层次永远觉得自己没准备好。
{{< /sentence >}}

等到万事俱备再行动，是最合理的拖延借口。切换思维层次只能在行动中完成，不能在等待中完成。

{{< sentence >}}
一个人最容易被自己的强项困住。
{{< /sentence >}}

你最擅长的事，往往就是锁死你的事。因为你会反复做它来获得安全感，而放弃了不擅长但更重要的事。

## 什么场景调用哪句

接到任务就想马上动手时——调出"用勤奋回避思考"。

觉得升不上去是运气问题时——调出"观念的天花板比能力的天花板更难打破"。

刚到新岗位想赶紧出成绩时——调出"先把事情做对，再把事情做快"。

觉得自己在某个领域已经很强了——调出"一个人最容易被自己的强项困住"。

觉得现在的做法没什么问题——调出"观念的天花板比能力的天花板更难打破"。

犹豫要不要接受新挑战时——调出"不要等准备好了再往上走"。

工作三五年感觉原地踏步时——调出"将才和兵才的差别不在能力高低，在思考的层次"。

面对职场焦虑但说不清焦虑什么时——调出"职涯是爬楼梯，不是坐电梯"。先确认自己在哪一级，再决定下一步做什么。
