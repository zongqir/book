{{< sentence >}}
苦不是因为世界总不顺，而是因为你一直想把它抓成你要的样子。
{{< /sentence >}}

外部条件当然会带来痛感，但让痛感扩成长期缠绕的，常常是抓住不放。越想固定，越容易受伤。

{{< sentence >}}
感受会升起，但你不必立刻把它认成“我”。
{{< /sentence >}}

烦、怕、怒、失落都是真实经验，但它们来了，不等于你就必须顺着它们走。先分开“有这个感受”和“我就是这样的人”。

{{< sentence >}}
第一支箭常常躲不开，第二支箭通常是自己补上的。
{{< /sentence >}}

事情已经发生是一层痛，反复咀嚼、放大、自责又是另一层。很多人真正被压垮的，是后面那一层。

{{< sentence >}}
看见无常，不是叫你冷淡，而是别再拿变化当背叛。
{{< /sentence >}}

状态会变，关系会变，身体会变，心境也会变。接受变化不是消极，是停止和现实硬碰硬。

{{< sentence >}}
真正的修行，不在你想通了什么，而在反应要冲出去时你能不能停一下。
{{< /sentence >}}

书里讲再多道理，最后都落在一个瞬间：那股熟悉的反应又来了，你有没有一点点不一样。

{{< sentence >}}
解脱不是把生活过没了，是把缠住你的那层东西慢慢卸掉。
{{< /sentence >}}

该做的事还是做，该承担的关系还是承担。变化的是你和这些事情之间的黏连程度。
