收银台前，每个价格都在做两件事：收钱，和把你归类。

## 价格在替商家挑人

{{< sentence no="1" quote="价格差异的目的不是区分产品，是区分顾客。" >}}

大杯比小杯贵的幅度，远超那口咖啡的成本。商家用价格梯度让你自己暴露支付意愿。

这就是价格歧视。听起来像骂人，每家店天天在用。

{{< /sentence >}}

{{< sentence no="2" quote="超市自有品牌故意做得丑，是为了让你心甘情愿为别的牌子多掏钱。" >}}

朴素包装让你下意识觉得"便宜货"。你转头拿了包装精美的那款，多花十几块。

超市主动压低自有品牌的吸引力，把顾客按价格敏感度分拣开。分开了，才能分别定价。

航空公司经济舱座椅做得窄，同一个道理。

{{< /sentence >}}

{{< sentence no="3" quote="你的时间值多少，商家比你清楚。" >}}

愿意翻传单、剪优惠券的人对价格敏感。懒得剪的人默认交了一笔"怕麻烦税"。

你以为在捡便宜，商家以为在精确归类。双方都对。

{{< /sentence >}}

## 位置吃掉了最大一块钱

{{< sentence no="4" quote="谁控制了稀缺位置，谁就拿走了定价权。" >}}

火车站出口的咖啡比两条街外贵一倍。豆子一样，手艺一样。贵的是那个你赶时间、没得选的位置。

学区房贵的是学位名额，砖头本身不值那个价。哈福德管这个叫"稀缺权力"。抓住这个词，很多离奇价格就不离奇了。

{{< /sentence >}}

{{< sentence no="5" quote="房租涨了，多出来的钱几乎全进了房东口袋——他什么都没多做。" >}}

城市越发展，占住好位置的人越先受益。位置没法复制，地段的稀缺性让土地所有者坐享其成。

这条规律从伦敦到深圳都成立。

{{< /sentence >}}

## 谁在交易之外买单

{{< sentence no="6" quote="有人在交易之外替你买单时，市场就算不清完整的成本。" >}}

工厂排废气，代价由周边居民承担，工厂账上看不见。经济学叫外部性。

只要这笔账没被塞进价格，市场价格就是失真的。下次听到"市场会自己调节"，先追一句：外部性处理了吗？

{{< /sentence >}}

{{< sentence no="7" quote="买家越压价，好卖家越不出场——最后留下的反而多是烂货。" >}}

二手车市场是经典案例。卖家比买家更清楚车况；买家怕坑拼命压价；好车主嫌价低干脆不卖了。

市场自己把自己搞坏了。任何信息严重不对等的交易，都可能出现这种"劣币驱逐良币"。

{{< /sentence >}}

## 规则撑不住，价格就空转

{{< sentence no="8" quote="制度崩坏的地方，再聪明的人也不敢做长期投入。" >}}

喀麦隆穷，缺的不是资源和人才。贿赂是默认流程，产权随时可能被没收。能力再强也白搭，规则兜不住长期行为。

微观层面价格机制再精巧，没有制度兜底就跑不起来。

{{< /sentence >}}

{{< sentence no="9" quote="市场需要规则撑着——这个前提本身不是市场能提供的。" >}}

哈福德写了一整本书讲价格多聪明。但他落脚在一个限定条件上。产权、规则、司法，三样都缺的地方，价格信号就是空转。

一个国家的贫富，往往卡在这一层。

{{< /sentence >}}

## 哪个场景调哪句

逛超市、买咖啡、比价格时——调第 1、2、3 句。价格梯度在替商家分拣你。

看房价、选地段、谈租金时——调第 4、5 句。先看谁控制了稀缺位置。

讨论环保、食品安全、公共政策时——调第 6、7 句。先问外部性和信息差有没有处理。

判断一个地方能不能长期投入时——调第 8、9 句。制度比资源重要。
