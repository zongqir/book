红军的力量不在武器，在于每个士兵都知道自己为什么打仗。
{{< /sentence >}}

斯诺反复提到这一点。国民党军队装备远优于红军，但红军士兵能说清楚自己在为谁打仗。这个差别在战场上直接转化为意志力差距。

{{< sentence >}}
我在中国见过的最自由、最快乐的人，居然是在这片被封锁的穷山沟里。
{{< /sentence >}}

苏区物资极度匮乏，但斯诺观察到的精神面貌和外面截然不同。一个外来者的直觉反差，比任何分析框架都更有冲击力。

{{< sentence >}}
毛泽东是一个精通中国旧学的人，他引用的典故比他的任何同事都多。
{{< /sentence >}}

斯诺注意到的不只是毛的政治判断力，还有他的知识结构。一个农民运动的领导者熟读古书——这个细节打破了当时西方对共产党人的刻板想象。

{{< sentence >}}
在西北的荒原上存在着一个运转的政府，有邮局、学校和货币，但外界对此一无所知。
{{< /sentence >}}

封锁的代价远超军事层面。信息封锁让一个完整的政权在世界面前隐形了好几年。斯诺的报道首先打破的是这层信息壁垒。

{{< sentence >}}
他们走过的路是人类历史上最漫长的军事行军，但他们自己谈起来就像在说一次远足。
{{< /sentence >}}

长征亲历者的讲述方式让斯诺吃惊。两万五千里，饥饿、死亡、雪山、沼泽——当事人用轻描淡写的语气说出来。经历过极端处境的人不会像旁观者那样渲染苦难。

{{< sentence >}}
我不得不承认，他们所做的事，无论最终结果如何，都已经是一项卓越的成就。
{{< /sentence >}}

斯诺给出的是一个现场判断，带着明确的保留。"无论最终结果如何"——他没有预测结局，只确认了一件事：他亲眼看到的变化是实在的。
