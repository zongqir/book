{{< sentence no="1" quote="家族组织的类型差异，根源不在文化传统，而在经济结构。" >}}
{{< /sentence >}}

莆田平原和闽西山区的对比是全书基本坐标。经济条件不同，家族走出了完全不同的组织路径。
{{< sentence no="2" quote="继承制度决定了家族是走向集中还是走向分散。" >}}
{{< /sentence >}}

诸子均分让家产代代切碎，宗族权威难以累积。部分集中继承则为大宗族提供了经济基础。分家规则是预测家族形态的第一个变量。
{{< sentence no="3" quote="宗族是明中叶以后特定条件下的组织重建，不是古已有之的延续。" >}}
{{< /sentence >}}

大规模建祠修谱集中在嘉靖至万历年间。驱动力来自制度条件的变化，有明确的时间起点。
{{< sentence no="4" quote="大礼议放开了民间祭祖的礼制限制，普通百姓第一次可以合法建祠堂。" >}}
{{< /sentence >}}

一项高层政治争论的意外后果，成了福建宗族化的直接推手。宫廷博弈和乡村组织之间的因果链，比想象中短。
{{< sentence no="5" quote="里甲制的编户逻辑和家族的组织逻辑并不天然一致。" >}}
{{< /sentence >}}

国家编户需要固定的赋税单位。家族的分合节奏由继承和人口决定。两套逻辑的冲突催生了多种适应策略。
{{< sentence no="6" quote="里甲制瓦解之后，家族组织走向了多种新的组织形式。" >}}
{{< /sentence >}}

基层社会自发找到新的组织均衡。有些地方转向祭祀整合，有些转向契约合作。制度崩溃不等于社会停滞。
{{< sentence no="7" quote="同一个&#x27;祠堂&#x27;，在莆田是权力中心，在闽西是合股议事场所。" >}}
{{< /sentence >}}

名称相同，运作机制完全不同。仅凭名称做比较研究，容易系统性地得出误导结论。
{{< sentence no="8" quote="族谱的首要功能是确认成员资格、分配权利义务。" >}}
{{< /sentence >}}

谁被写进去、怎么排列世系，都是当时的权力决定。修谱是组织行为，附带产生了历史叙事。
{{< sentence no="9" quote="族田的设立和管理方式直接反映家族的权力结构。" >}}
{{< /sentence >}}

谁管田、谁受益，就是谁掌权。公共财产的分配规则是家族内部权力格局的镜子。
{{< sentence no="10" quote="一条鞭法改变了赋税计算方式，连带改变了家族成员之间的利益分配。" >}}
{{< /sentence >}}

税制改革看似外部冲击，实际直接改写了家族内部的合作公式。制度变动不止影响国家和个人的关系，也改造家族内部的利益结构。
{{< sentence no="11" quote="福建不同地区的家族形态差异，大于福建与其他省份之间的差异。" >}}
{{< /sentence >}}

省级行政区划掩盖了内部的巨大分化。用"福建宗族"做整体概括，本身就遮蔽了最重要的研究发现。
{{< sentence no="12" quote="到每个地方，先看这里的人实际怎么组织、怎么分配、怎么决策。" >}}
{{< /sentence >}}

概念先行的研究反复在地方差异面前碰壁。从材料出发是被经验教训逼出来的选择，也是郑振满全书的方法论底色。
