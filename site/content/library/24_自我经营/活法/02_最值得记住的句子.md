# 思维方式带正负号，其余全是乘数

## 方程式与方向

{{< sentence >}}
人生·工作的结果 = 思维方式 × 热情 × 能力。
{{< /sentence >}}

三个变量的乘法，不是加法。思维方式取值范围从负一百到正一百。能力九十分、热情九十分，但思维方式是负数，结果就是大面积负数。判断自己卡在哪里时，先查正负号。

{{< sentence >}}
思维方式哪怕只有一点点偏向消极，人生的结果就会全部变成负值。
{{< /sentence >}}

不需要大面积崩坏。一个习惯性的推责反应、一种"差不多就行"的默认心态，就够把热情和能力的乘数全部翻转。

{{< sentence >}}
心不唤物，物不至。
{{< /sentence >}}

稻盛不是在讲吸引力法则。他说的是：你不主动设定目标并持续想着它，你的注意力不会聚焦，资源不会向你集中。想清楚要什么，才能看见机会。

## 工作与修行

{{< sentence >}}
劳动的目的不应仅仅是获取粮食，劳动本身就是人生最尊贵的行为。
{{< /sentence >}}

把工作当成纯交易的人，永远在计算投入产出比。一旦觉得亏了，就开始应付。应付的人不可能做出好东西。工作是唯一一个你每天都能练习全力以赴的场景。

{{< sentence >}}
付出不亚于任何人的努力。
{{< /sentence >}}

六项精进的第一条。不是比谁加班多，是问自己：今天有没有在能力范围内做到最好。回答是"差不多"的那一天，就是热情变量在打折扣的一天。

## 利他与决策

{{< sentence >}}
先利他，后利己——这是宇宙的法则。
{{< /sentence >}}

利他在稻盛的语境里不是道德训诫，是优先级排序：先搞清楚对方需要什么，再决定自己做什么。接手日航时，他不看报表先看乘客需求——利他是经营决策的起手式。

{{< sentence >}}
以善意回报善意，以正面心态面对困境。
{{< /sentence >}}

遇到麻烦时的第一反应决定后续走向。先追责、先抱怨、先计较，这些反应本身就在把思维方式往负数拧。先想"我能做什么"，再处理其余的。

## 反省与校准

{{< sentence >}}
每天反省，每天进步一点点。
{{< /sentence >}}

反省不是写日记，是每天结束前问自己一个问题：今天有没有因为偷懒、傲慢或短视做了减分的事。能说出具体的一件，就是有效反省。

{{< sentence >}}
不要有感性的烦恼。
{{< /sentence >}}

已经发生的事，反复焦虑改变不了结果，只消耗精力。烦恼可以有，但在做出判断和行动后就该放下。用在烦恼上的时间，全是从热情变量里扣的。

## 什么场景拿出来翻

- 觉得自己够努力但结果不对：翻方程式，查思维方式的正负号。
- 工作中开始应付了事：翻"付出不亚于任何人的努力"，问自己今天打了几折。
- 做决定时只想着自己的利益：翻利他句，先想对方需要什么。
- 犯了错开始焦虑或推卸：翻"不要有感性的烦恼"和反省句，先行动再放下。
- 不确定自己的方向对不对：翻"心不唤物"，问自己到底想要什么。
