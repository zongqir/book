## 认知差异

{{< sentence >}}
你觉得对方不可理喻，是因为你在用自己的操作系统评估对方的输出。
{{< /sentence >}}

一个直觉型的人觉得细节控"太死板"，一个思维型的人觉得情感丰富的人"太感性"——这不是评价在起作用，是参照系在起作用。换一个参照系，同样的行为会得出完全不同的评价。

{{< sentence >}}
四种气质没有高低之分，但有适配场景之分。
{{< /sentence >}}

理性型在需要逻辑决策的场景里效率最高。理想型在需要激发他人动力的场景里不可替代。把一种气质放在不适配的场景里，看起来就像"能力不行"——但真正的问题是场景和气质不匹配。

## 沟通错位

{{< sentence >}}
大多数沟通失败不是因为说得不够清楚，而是因为听的人在用不同的频道接收。
{{< /sentence >}}

一个感觉型的人说"我觉得这个项目有问题"，意思是"直觉告诉我某些地方不对"。一个思维型的人听到的是"她没有数据就下结论"。双方说的和听到的是两件事。

{{< sentence >}}
改变一个人的气质类型几乎不可能，改变你与那个类型沟通的方式完全可能。
{{< /sentence >}}

试图让一个判断型的人"放松一点别那么较真"，几乎一定会失败。但如果你在沟通时先给结论再给过程（这是判断型偏好的信息接收顺序），对方会觉得你说话"特别清楚"。你没有改变对方，你改变的是信号的编码方式。

## 关系维护

{{< sentence >}}
长期关系中最大的伤害不是冲突，而是"你为什么就不能像我一样"这种持续性的失望。
{{< /sentence >}}

这种失望的根源是：期待对方用你的方式运行。一旦理解了对方的气质类型，失望不会完全消失，但会从"他不愿意"变成"他的系统就是这样运转的"。后者至少不伤感情。

{{< sentence >}}
理解一个人的气质类型不是为了预测他的每一步行动，而是为了在意外发生时少一些惊讶。
{{< /sentence >}}

预测行为是不可能的。但知道对方在压力下倾向于退缩还是爆发、在做决策时依赖数据还是感受，能帮你在关键时刻不做出让关系恶化的反应。
