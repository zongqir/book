# 先闭嘴，才轮得到你说的话被听见

## 赢了嘴上的，输了关系里的

{{< sentence >}}
天底下只有一种方法能赢得争论——避免争论。
{{< /sentence >}}

不是说你不能有不同意见。是把"我要证明你错了"换成"我想让你自己发现另一种可能"。区别不大，效果天差地别。

{{< sentence >}}
如果你错了，迅速而真诚地承认。
{{< /sentence >}}

认错的速度决定了对方的反应。你先认，对方反而会替你找理由；你硬撑，对方只会加力证明你确实错了。

{{< sentence >}}
用提问代替直接命令。
{{< /sentence >}}

"你觉得这样调整会不会更好？"和"你应该这样改"——信息量一样，一个让对方参与决策，一个让对方服从指令。

## 名字是最小的重视信号

{{< sentence >}}
一个人的名字对他来说是任何语言中最甜蜜、最重要的声音。
{{< /sentence >}}

执行版本很简单：见面时叫名字，告别时叫名字。多数人做不到，因为他们根本没记住。

{{< sentence >}}
每个你遇到的人都觉得自己在某方面比你强。一条通向他内心的路，就是不经意地让他感到自己很重要。
{{< /sentence >}}

"不经意"是关键。刻意让对方觉得重要，对方能闻出来。不经意让对方觉得重要，对方只觉得你这个人不错。

## 嘴巴闭上，耳朵才打开

{{< sentence >}}
要成为一个好的交谈者，首先要成为一个好的倾听者。
{{< /sentence >}}

"好的倾听者"不是沉默。是在对方说话的时候，你的注意力确实在对方身上，不是在准备自己下一句要说什么。

{{< sentence >}}
谈论对方感兴趣的话题。
{{< /sentence >}}

罗斯福在接见客人前一晚，会专门研究对方的兴趣爱好。不是因为他喜欢研究别人——他知道：用对方的兴趣开场，五分钟就能建立信任。

{{< sentence >}}
如果你想让别人喜欢你，真诚地对他人感兴趣。
{{< /sentence >}}

注意不是"表现得对他人感兴趣"。你得真的好奇这个人是谁、经历了什么、在乎什么。装出来的好奇不超过三次就会被识破。

## 批评的门一开，对方的门就关了

{{< sentence >}}
批评是没用的，因为它使人采取防御姿态，并通常使人努力为自己辩护。
{{< /sentence >}}

你批评一个人，他的第一反应永远不是"你说得对"，而是"你凭什么"。不管你的批评多准确，对方的大门在你开口那一秒就关上了。

{{< sentence >}}
间接地提醒他人注意自己的错误。
{{< /sentence >}}

"你这份报告数据有误"和"这份报告的框架很清楚，如果数据部分再核实一下就完美了"——信息量一样，接受度完全不同。

{{< sentence >}}
先谈你自己的错误，再指出对方的问题。
{{< /sentence >}}

"我之前也犯过类似的错，后来发现关键是……"——当你先把自己放到和对方同一侧，对方才不会觉得你站在审判席上。

## 什么场景调出哪句

和同事观点不同，想开口反驳——先调出"赢了争论就输了好感"。

跟人初次见面——先调出"谈对方感兴趣的话题"。

需要指出别人的问题——先调出"间接提醒"和"先谈自己的错误"。

对方正在说话而你急着插嘴——先调出"好的倾听者"。

发现自己犯了错——先调出"迅速而真诚地承认"。

想让对方帮你做一件事——先调出"用提问代替直接命令"。
