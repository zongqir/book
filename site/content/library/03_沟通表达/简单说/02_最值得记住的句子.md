# 《简单说》里最该留下来的，是这些会把你从前摇里拽出来的提醒句

{{< sentence >}}
先说对方最需要知道的，不要先说你最想交代的。
{{< /sentence >}}

这句最该排第一。你一绕，通常就是因为你先站到了自己这边。

{{< sentence >}}
重点出来太晚，等于重点没出来。
{{< /sentence >}}

很多话不是内容不对，是来得太晚，对方已经没有耐心接了。

{{< sentence >}}
短不是少说，是先把最该活的那一句救出来。
{{< /sentence >}}

这句专门防“我一精简就会说不完整”的焦虑。

{{< sentence >}}
别人不是没耐心，是你把整理工作留给了别人。
{{< /sentence >}}

这句很扎人，但很准。很多前摇，本质就是把信息整理外包给听的人。

{{< sentence >}}
背景不是开头的义务，是重点出来以后才需要的补充。
{{< /sentence >}}

这句适合压住“让我先解释一下”的本能。

{{< sentence >}}
你讲得越像在替自己免责，别人越抓不到你到底要说什么。
{{< /sentence >}}

很多前摇长，不是信息多，是自我保护太早上场。

{{< sentence >}}
一句话如果不能先站住，后面再多都只是在陪它摇晃。
{{< /sentence >}}

这句能帮你在开口前逼自己先把主句立住。

## 这些句子什么时候最该翻出来

- 你想说“我先补个背景”的时候：先看第 1、5 句。
- 你知道重点，但总想放到最后再说的时候：先看第 2、4 句。
- 你怕自己说得太短不够严谨的时候：先看第 3、7 句。
- 你越解释越觉得自己说不清的时候：先看第 6 句。

这本书最该留下的，不是“更会表达”的感觉，而是这些能把你从前摇里硬拽出来的提醒句。
