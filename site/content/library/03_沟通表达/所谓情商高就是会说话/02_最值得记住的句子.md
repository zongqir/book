## 把请求变成对方的好处

{{< sentence >}}
不要直说你想要什么，先想对方能得到什么。
{{< /sentence >}}

"请把文件整理一下"和"你整理的文件大家都说好看"——同一个请求，前者是指令，后者是认可。投其所好的核心不是讨好，是把你的需求翻译成对方的收益。

## 人最难拒绝的是"只有你能做到"

{{< sentence >}}
比起被需要，被"独特地需要"更让人无法说不。
{{< /sentence >}}

"帮我搬个桌子"可以找任何人。"你力气最大，这个只有你能搬"把请求变成了一种认可。被认可欲和非你不可是佐佐木用得最多的两条路径——它们激活的不是义务感，是自我价值感。

## 给选项比给请求更容易得到"是"

{{< sentence >}}
不要问"要不要"，问"A 还是 B"——两个选项都是你想要的答案。
{{< /sentence >}}

"周末能来吗？"触发的是拒绝本能。"周六上午来还是下午来？"触发的是选择本能。选择的自由感让人觉得自己有控制权，而你已经锁定了范围。

## 先说谢谢再开口

{{< sentence >}}
人在被感谢之后，很难立刻拒绝你。
{{< /sentence >}}

这条路径短到只有两步：先说"谢谢你上次帮忙"，再提你的请求。感谢建立了一个微小的心理债务，不够大到让人不舒服，但够大到让人不好意思说不。

## 让普通话带上钩子

{{< sentence >}}
信息被记住不是因为正确，而是因为有感官冲击。
{{< /sentence >}}

"好吃"和"好吃到舌头要融化"——区别不在意思，在画面。赤裸裸法的核心是把身体感受塞进语言里。惊奇法的核心是在预期之外制造一个小错位。佐佐木的 8 个技巧，底层都是同一件事：给句子装上一个让人回头看一眼的钩子。

{{< sentence >}}
"不擅长说话"不是性格，是没练过配方。
{{< /sentence >}}

佐佐木反复强调的观点：说话技巧不是天赋，是食谱。你不需要理解心理学，你只需要按配方把原来的话改写一遍。这个判断是否过于简化，取决于你面对的是日常请求还是深层冲突——配方在前者里够用，在后者里不够。
