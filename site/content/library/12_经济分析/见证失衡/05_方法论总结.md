# 用会计恒等式拆解宏观经济失衡的方法

## 核心工具：国际收支恒等式

佩蒂斯的整套分析建立在一个等式上：经常账户余额 = 储蓄 − 投资。一个国家的贸易顺差，等于它的国内储蓄超出国内投资的部分。

这个等式不是假设，是会计恒等式——像"资产 = 负债 + 所有者权益"一样，它在定义上就成立。不需要经济理论支撑，不需要文化解释，不需要政治立场。

分析经济失衡问题时，第一步不是问"为什么会这样"，而是把现象塞进等式里看各项的数值变化。贸易顺差扩大了？那要么是储蓄增加了，要么是投资减少了。然后再追问：储蓄增加是因为什么制度安排？

## 从叙事回到结构：方法论的核心转向

经济分析领域有一种常见的做法：先讲一个故事（"中国人节俭所以储蓄率高""德国人严谨所以制造业强"），再用故事解释数据。佩蒂斯的方法论是反过来的——先看等式两边的数值变动，再追问哪些制度安排导致了这些数值。

这个转向的价值在于：叙事可以自圆其说，但等式不骗人。如果等式两边的数值不平，不是等式错了，是你的解释漏了什么。

具体操作是三步走：定位失衡的等式位置 → 追溯制度安排对等式各项的影响 → 判断调整路径。

## 制度安排如何影响等式各项

佩蒂斯不直接解释"为什么某国贸易顺差大"。他解释的是：哪些制度安排压低了居民收入在GDP中的占比？居民收入占比低，消费就低；消费低，储蓄率就高（因为储蓄 = GDP − 消费）；储蓄率高，在投资不变的条件下，经常账户就是顺差。

制度安排包括：汇率管制、金融抑制（利率管制）、劳动力市场结构、社会保障体系缺位、土地财政。每一个都不是独立变量，而是影响等式某一项的具体机制。

方法论的关键贡献在于：把"文化决定论"和"政策阴谋论"都排除在外。储蓄率高不是因为国民性格节俭，是因为制度安排压低了居民可支配收入。

## 两条调整路径的判断框架

经济失衡最终会调整——等式必须平衡。佩蒂斯把调整路径分成两类：增长型调整和衰退型调整。

增长型调整：提高居民收入占比 → 消费上升 → 储蓄率下降 → 贸易顺差缩小。经济总量可以不缩小甚至增长。

衰退型调整：投资回报下降 → 坏账增加 → 被迫缩减投资 → 储蓄和投资同时下降但储蓄降得更慢 → 等式勉强平衡，但经济总量萎缩。

判断某个经济体正在走哪条路径，看两个指标：居民收入增速是否快于GDP增速（增长型调整的信号）；坏账率和债务率是否在加速上升（衰退型调整的信号）。

## 方法论的组织逻辑：等式优先于因果

佩蒂斯的方法论不是一套经济理论，而是一个分析纪律：分析任何宏观经济问题时，先写出相关的会计恒等式，确认各项数值，然后在等式约束内寻找解释。

和传统经济分析的区别：传统方法先提出因果假说（"储蓄率高导致顺差"），再找数据验证。佩蒂斯先确认等式各项的数值关系，再追问哪些制度安排在驱动这些数值。

等式不给你答案，但等式帮你排除错误答案。任何违反会计恒等式的解释，不管叙事多精彩，都是错的。
