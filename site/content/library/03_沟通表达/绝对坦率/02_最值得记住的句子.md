# 绝对坦率里最该留下来的，是这些会在关键时候拽住你的提醒句

{{< sentence >}}
拖着不说，最后很少会更温和，只会更重。
{{< /sentence >}}

这句适合在你准备把问题再压一周的时候翻出来。

{{< sentence >}}
关系不是反馈的替代品，关系是反馈能被听进去的前提。
{{< /sentence >}}

没有这层前提，真话很容易被听成羞辱。

{{< sentence >}}
批评具体行为，比判断整个人更有力量。
{{< /sentence >}}

一旦上升到“你这个人就是……”这种句子，对话基本就废了。

{{< sentence >}}
表扬说不清，团队不知道什么值得继续。
{{< /sentence >}}

很多管理者只会纠错，不会说明“什么是对的”。

{{< sentence >}}
最糟的反馈不是刺耳，是模糊。
{{< /sentence >}}

刺耳还知道哪里痛，模糊只会让人回去猜。

## 这些句子什么时候最该翻出来

- 一开口就想解释很多的时候：先翻这页。
- 你觉得自己明明没说错，结果事情还是做坏的时候：先翻这页。
- 你知道问题在哪，却迟迟不知道从哪一句开始的时候：先翻这页。
- 对方的反应、组织的惯性或关系的压力开始把你带偏的时候：先翻这页。
