# 八句转折提醒：别被和谈拖住，也别把胜利做成失序

第四卷最适合反复调出来的句子，都发生在局势开始倒向你之后。

这时候最容易犯的，不是看不见机会，而是对机会太兴奋。和谈的表象、局部的胜利、接管前的忙乱，都可能把人带偏。下面这几句，适合在那种快要翻盘、也最容易做散的阶段拿出来压一压。

{{< sentence >}}
桌上还能谈，手上也不能停。
{{< /sentence >}}

第四卷开头最硬的提醒，就落在这句里。

谈判可以争取时间，也可以争取主动，但不能让准备离手。表面缓和，常常只是更大摊牌前的空档。

{{< sentence >}}
优势不集中，等于没有优势。
{{< /sentence >}}

总体未必处处强，但局部必须压得够厚。

这句适合在手里刚有一点资源、忍不住想四处出手的时候调出来。什么都顾，最后通常是什么都没打透。

{{< sentence >}}
一仗打赢不算数，能不能顺着它接第二步才算数。
{{< /sentence >}}

第四卷不迷信单点胜利。

改局靠的是一串连上的动作。拿下一处以后，兵力、舆论、秩序、干部安排，能不能立刻接上，决定了战果会不会漏掉。

{{< sentence >}}
翻利益账，要先把人分层，别把能稳局的人一起推走。
{{< /sentence >}}

土地和新区工作里，最怕分类不清。

该打的、该争取的、该稳住的，如果一起处理，局面会比原来更乱。这句纠正的是“越激烈越彻底”的冲动。

{{< sentence >}}
进场先立规则，别先忙着开新口子。
{{< /sentence >}}

局面打开以后，什么都想改最危险。

先定班子、议事法和纪律，才能接得住后面的事。没有这一步，接管会先变成内部混乱。

{{< sentence >}}
胜势越大，越不能四面出击。
{{< /sentence >}}

这一句看起来和前面很像，重点却不同。

前面讲的是资源集中，这里讲的是心态收束。越顺的时候，越容易把所有旧账一起翻出来。第四卷要人先忍住。

{{< sentence >}}
旧秩序倒得越快，越要提前讲清新秩序靠什么站。
{{< /sentence >}}

只会说旧东西该结束，还不够。

谁来组织，谁来服从，谁被联合，谁被压制，新的秩序怎样运行，这些都得在胜利前先有雏形。

{{< sentence >}}
接手之后还照老样子拍脑袋，前面的胜利很快会漏掉。
{{< /sentence >}}

第四卷越往后，越强调党委会和集体决断。

意思很直接。战场上的胆子，不能直接当成治理里的方法。局面一旦变大，个人冲锋会越来越不够用。

## 按场景调用

- **表面缓和、心里想松劲时**：第 1 句
- **资源刚多一点、忍不住想全面铺开时**：第 2、6 句
- **拿下一步以后，不知道下一步先接什么时**：第 3 句
- **准备动利益格局，却担心局面失控时**：第 4 句
- **刚准备接盘、手上事情一下变多时**：第 5、8 句
- **胜利在望，却没人说得清后面怎样运行时**：第 7 句
