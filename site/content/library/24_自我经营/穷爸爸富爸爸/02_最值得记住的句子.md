# 资产把钱放进口袋，负债把钱拿走——其余判断从这里开始

## 关于钱的方向

{{< sentence >}}
资产就是把钱放进你口袋里的东西，负债就是把钱从你口袋里拿走的东西。
{{< /sentence >}}

全书最底层的判断工具。不需要会计知识，不需要财务报表。每次花钱前问自己：这笔支出会让未来有更多钱流进来，还是让每个月的固定支出又多了一项。

{{< sentence >}}
富人买入资产，穷人只有支出，中产阶级买入他们以为是资产的负债。
{{< /sentence >}}

最危险的不是穷——穷的人知道自己没有资产。最危险的是中产阶级的错觉：贷款买了房、供了车、上了高端教育，以为自己在积累资产，实际每一项都在增加月度支出。

## 关于收入结构

{{< sentence >}}
不要为钱工作，让钱为你工作。
{{< /sentence >}}

听起来像口号，实际是结构性判断。为钱工作意味着你的收入和你的时间绑定——你停下来，收入就停。让钱为你工作意味着构建不依赖你个人时间的收入来源：租金、股息、版税、生意系统。

{{< sentence >}}
穷人和中产阶级为钱工作，富人让钱为自己工作。
{{< /sentence >}}

不是在说谁更努力。是在说收入的结构不同：一种是用时间换钱，另一种是用资产生钱。前者有天花板，后者没有。

## 关于恐惧和习惯

{{< sentence >}}
大多数人让恐惧和贪婪来控制自己的财务决策。
{{< /sentence >}}

怕失去工作，所以不敢尝试任何有风险的事。看到别人赚了钱，又冲动跟进。恐惧让你困在原地，贪婪让你在错误的时机出手。两种情绪交替控制，财务结构就不可能改善。

{{< sentence >}}
老鼠赛跑：起床，上班，付账单。起床，上班，付账单。
{{< /sentence >}}

赚得越多、花得越多、需要赚得更多。循环加速但不前进。识别信号：如果你涨了薪但依然觉得钱不够用，大概率在跑鼠赛。

## 关于教育和认知

{{< sentence >}}
学校教你如何为钱工作，但从不教你如何让钱为你工作。
{{< /sentence >}}

财商不在任何一门课程里。大多数人对钱的判断方式，来自父母的习惯和社会的默认路径，而不是系统学习的结果。

{{< sentence >}}
你挣多少钱不重要，重要的是你留下了多少钱。
{{< /sentence >}}

收入是入口，留存率才是结果。月入五万但月支出四万九的人，比月入两万但月支出八千的人更脆弱。这不是节俭问题，是结构问题。

## 什么场景拿出来翻

- 要做一笔大额支出时：翻资产负债定义，判断这笔钱是在增加资产还是增加负债。
- 涨了薪却没觉得宽裕：翻老鼠赛跑，检查支出结构是不是跟着收入一起膨胀了。
- 在犹豫要不要冒险时：翻恐惧和贪婪那句，分辨自己是在做理性判断还是在被情绪控制。
- 觉得自己理财知识不够：翻学校不教那句，提醒自己财商需要主动学，不会自动出现。
