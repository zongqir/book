# 用一句话击穿一个偏见——米塞斯最锋利的判断

米塞斯的写作风格不追求文采，追求不可反驳。下面这些句子，有的接近原文表述，有的是从他的论证中提炼的核心判断。标注为"提炼"的，是综合多处论述压缩而成，保留了判断方向和论证力度。

## 行动不是反射，是选择

{{< sentence no="1" quote="人的行为是有目的的行为。" >}}
六个字定义了整个学科的起点。把"有目的的"三个字去掉，经济学就退化成行为主义心理学——观察刺激和反应，不过问选择和判断。米塞斯拒绝这种退化。【接近原文】
{{< /sentence >}}

{{< sentence no="2" quote="试图否认人会有目的地行动的人，正在有目的地做这件事。" >}}
行为公理不可否认的原因。任何反驳尝试本身就是一个有目的的行为，因此反驳在逻辑上自我取消。碰到有人质疑"人不一定是有目的地行动的"，拿这句话回应就够了。【提炼】
{{< /sentence >}}

## 价值不在物品里，在判断中

{{< sentence no="3" quote="经济学研究的不是物品和服务，而是人的选择和行为。" >}}
一句话把经济学的对象从"物"拉回"人"。GDP、CPI、贸易额都是统计指标，不是经济学的基本对象。基本对象是做选择的人。【接近原文】
{{< /sentence >}}

{{< sentence no="4" quote="价值不是物品的内在属性，而是评价者对物品满足其需求能力的判断。" >}}
主观价值论的一句话版本。当有人说"石油本身就值这个价""房子的真实价值是……"时，用这句话纠偏：没有脱离评价者的"真实价值"。【接近原文】
{{< /sentence >}}

## 经济规律不靠数据验证

{{< sentence no="5" quote="经济学定理不是从经验中导出的——经验只能提供历史数据，不能提供普遍规律。" >}}
整个行为学方法论的一句话摘要。听到"数据表明……所以经济规律是……"时，可以用这句话重新审视：数据表明的是历史，不是规律。【提炼】
{{< /sentence >}}

{{< sentence no="6" quote="历史不能教给我们任何普遍规律，因为历史事件不可重复。" >}}
和自然科学的实验方法彻底划清界限。物理实验可以重复，历史事件不能。用历史数据归纳经济规律，在方法论上不成立。【接近原文】
{{< /sentence >}}

## 社会主义的致命缺陷不是效率低

{{< sentence no="7" quote="社会主义的问题不在于它缺少激励，而在于它取消了经济计算。" >}}
把社会主义批判从"人懒不懒"拉到"能不能算账"。即使每个人都勤奋无私，没有市场价格，资源配置仍然是盲目的。这是一个逻辑问题，不是道德问题。【提炼】
{{< /sentence >}}

{{< sentence no="8" quote="没有生产资料的市场价格，就不可能有合理的经济计算。" >}}
经济计算论证的核心判断。消费品有价格不够，生产资料——机器、原材料、土地——也必须有市场价格，否则生产决策无法做成本比较。【接近原文】
{{< /sentence >}}

## 利息不是制度产物

{{< sentence no="9" quote="利息是时间偏好的市场表达，不是银行或资本家的发明。" >}}
当有人主张"零利率合理"或"利息是剥削"时，米塞斯的回应：只要人偏好现在胜过未来，利息就必然存在。取消利息不是改革，是和人的行为逻辑作对。【提炼】
{{< /sentence >}}

## 干预不会停在第一步

{{< sentence no="10" quote="政府对市场的每一次干预，都会制造新的问题，然后为下一次干预提供理由。" >}}
干预主义的逻辑链条：价格管制导致短缺，短缺导致配给，配给导致黑市，黑市导致更多管制。每一步都是上一步的"合理"回应，但整条链条的方向是确定的。【接近原文】
{{< /sentence >}}

## 集体是隐喻，不是行动者

{{< sentence no="11" quote="只有个人才能思考、感受、判断和行动。集体是个隐喻，不是行动者。" >}}
当政策讨论中出现"国家应该""社会需要""人民要求"时，追问一句：到底是哪个具体的人在做这个决定？集体名词不会行动，行动的永远是个人。【提炼】
{{< /sentence >}}

## 预测不是经济学的工作

{{< sentence no="12" quote="经济学能告诉你'如果限价，必然短缺'，但不能告诉你'短缺有多严重'。" >}}
划清行为学的边界。定性因果是先验推理的领地，定量预测属于经济史。期待经济学家像气象学家一样给出精确预报，是把经济学放到了错误的位置上。【提炼】
{{< /sentence >}}
