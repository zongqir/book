# 争论一热起来，这七句最该先被想起来

## 先分清你在说结论，还是在说理由

{{< sentence >}}
立场不是理由，表态也不是论证。
{{< /sentence >}}

很多争论一上来先抢的是位置。“我就是反对”“我当然支持”能让别人知道你站哪边，却不能告诉别人你凭什么站在这里。

{{< sentence >}}
一句话如果只是在扩音情绪，它就还没进入讲道理的阶段。
{{< /sentence >}}

说得激烈、说得笃定、说得委屈，都可能是真的。但这些都替代不了理由本身。

## 理由碰不到结论，说再多也只是绕题

{{< sentence >}}
信息量很大，不等于它和结论真的有关。
{{< /sentence >}}

最常见的错法不是没话说，而是说了一堆不该算数的话。对方的人品、过往、动机、语气，常常被拿来假装回应当前论点。

{{< sentence >}}
抓住一个人的漏洞，不等于你已经推翻了他的主张。
{{< /sentence >}}

一个人不讨喜，方案也可能成立。一个人有私心，数据也仍然可能是真的。先回到论点，再决定这些背景该占多大权重。

## 前提不被接受，后面推得再顺也白搭

{{< sentence >}}
没有说出口的前提，往往才是真正在替结论承重的那部分。
{{< /sentence >}}

很多论证表面像在讲事实，底下其实压着价值排序。你不把这层翻出来，争半天也只是拿不同前提互撞。

## 证据和反驳决定你能把话说多满

{{< sentence >}}
证据只能证明它够得着的那一截，别替它多宣布半步。
{{< /sentence >}}

一个例子最多说明可能，一项研究最多说明它所覆盖的人群和条件。证据一旦超额使用，结论就会假装比它实际更稳。

{{< sentence >}}
不处理最强反驳的论证，通常只是暂时没被戳穿。
{{< /sentence >}}

反驳不是抬杠，它在告诉你结论到底扛不扛得住压力。能过这一步，话才配说重。

## 逻辑规则最先该拦住的人，是你自己

{{< sentence >}}
会识别谬误不稀奇，舍得把同样的尺子量回自己才稀奇。
{{< /sentence >}}

只会挑别人的毛病，很快就会把逻辑工具用成更精致的偏见保护壳。先查自己的理由，再拆别人的漏洞，论证才有公信力。
