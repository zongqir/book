# 先判阶段，再判打法；先联合多数，再保住自己的主动

最常要纠正的，不是勇气不足，而是判断顺序错了。

外部压力一大，人很容易在两个方向上一起走偏：要么急着速胜，要么过早认输；要么为了合作把自己交出去，要么把自己锁进小圈子里。

这一卷留下来的原则，就是为了挡住这些错法。

它们不会直接替你做决定，但会不断提醒你：哪些地方不能抢跑，哪些地方一偷就会把后面整条线一起带歪。

{{< principle no="1" type="core" statement="先判阶段，再判打法，不能拿一时强弱直接当终局" >}}
首先要约束的，是把眼前局面直接上升成最终命运的冲动。表面更强，不等于后面一直更强；表面更弱，也不等于再没有机会。真正要先看的，是阶段有没有变化，力量对比会不会随着时间、空间、组织和群众基础发生改写。

这条原则约束的是两种对称误判。一种是速胜幻想，觉得只要狠狠干一轮就能一把定输赢。另一种是消极退却，觉得对手更强就没必要再看后面。

常见误读是把“看阶段”理解成只会拖、只会等。不是。阶段判断不是拖延理由，而是决定你到底该快、该稳、该守还是该打的前提。

边界也很清楚。不是所有问题都值得拉到长线阶段来看，但只要题目已经进入长期对抗、力量变化和条件累积，这一步就不能省。
{{< /principle >}}

{{< principle no="2" type="core" statement="联合多数是必要的，但联合不能换成失去独立判断和组织能力" >}}
广泛合作里有一个绕不开的硬问题：你当然不能只剩自己，但你也不能为了合作把自己并没了。队伍越大，关系越复杂，越要保住判断、组织和行动的主动。

这条原则约束的是“先依附上去，以后再说”的短视。短期看，这样做像是在扩大力量；长期看，你会越来越不会独立判断，也越来越没有能力在合作里稳住自己的节奏。

常见误读是把“保住主动”理解成拒绝合作，或者一合作就警惕到什么都谈不成。不是。它约束的不是合作本身，而是在合作里失掉自己。

这条原则今天仍然很硬。做联盟、做跨部门协作、做平台合作，只要别人比你更大、更强、更有资源，就更需要把这条原则顶在前面。
{{< /principle >}}

{{< principle no="3" type="support" statement="战略上看长期，战术上抓局部可打点，不能把长期误读成空熬" >}}
这里说的长期，不是靠情绪熬过去，而是不断把局部空间、局部胜点和局部组织做出来。时间之所以有价值，是因为你在时间里真实积累了条件。

这条原则约束的是“既然是长期，那就先别急着做具体动作”的偷懒。真正的长期，从来不是无所作为。

常见误读是只会讲大方向，不会找局部切口。结果口头上说在打长线，实际上什么力量都没长出来。
{{< /principle >}}

{{< principle no="4" type="support" statement="外部压力越大，越要把群众动员和组织能力变成真实力量" >}}
胜负不能单纯看成两支力量正面碰撞。更要紧的是，你能不能把原本分散的支持、情绪和需要组织起来，变成一条会持续增长的现实力量。

这条原则约束的是只算账面实力、不看组织能力的短视。账面强弱当然重要，但如果只算这一层，你会漏掉更深的变化。

常见误读是把“群众”读成一句口号。不是。它真正要求的是组织、联结、动员和把支持变成持续能力。
{{< /principle >}}

{{< principle no="5" type="support" statement="合作面再大，也要区分对象层次：谁该联合，谁该争取，谁必须准备斗争" >}}
合作对象不能看成一团。有人可以稳定联合，有人更适合争取，有人阶段性同路但迟早会顶上来。对象一旦分层，策略和节奏才会跟着准。

这条原则约束的是“为了团结先别区分”的含糊。你越不区分，越容易在真正的关键点上吃亏。

常见误读是把分层理解成到处树敌。其实恰恰相反。分层的目的，是让你更知道谁值得下功夫争取，谁不能再抱不必要的幻想。
{{< /principle >}}

把这五条放在一起，方向就很清楚了：别让眼前强弱偷走你的阶段判断，别让广泛合作吃掉你的主动，别让长期变成空耗，别把支持停在口号，别把不同对象混成一堆。

所以它不该只被读成“更难的时候也要坚持”。

真正长期留在手里的，是这几条原则不断出来纠偏的频率。你准备抢跑时，它提醒你先判阶段；你准备附着时，它提醒你保住主动；你准备只靠硬拼时，它提醒你把局部和组织一起看进去。

只要这些原则还能先一步冒出来，这一卷就没有被读偏。
