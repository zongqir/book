## 选底版

{{< sentence >}}
你觉得从零开始创造的原创角色，背后几乎一定站着某个原型——只是你没有意识到。
{{< /sentence >}}

写作者以为自己在原创。但几千年的叙事传统已经穷尽了角色功能位置的基本类型。你的"原创角色"大概率是某个原型的无意识变体——区别在于：你知道底版是什么的时候，可以有意识地利用它和偏离它；不知道的时候，只能随机地落进某个原型的惯性里。

{{< sentence >}}
原型不是刻板印象。原型描述的是角色在故事中的功能位置和内在动力，不是某类人的标签。
{{< /sentence >}}

"保护者"原型不是说现实中的保护型人格。它描述的是故事中承担保护功能的角色通常会有什么样的驱动力——为所爱之人不惜一切——以及这种驱动力可能滑向什么方向——以保护为名实施控制。

## 找两极

{{< sentence >}}
每种原型的光明面和阴暗面不是两个角色，是同一个角色的两种可能性。弧线就藏在两极之间。
{{< /sentence >}}

扁平角色只有一面。给角色赋予另一极——哪怕在故事里不完全展开——角色就会获得深度。读者能感觉到这个人物还有另一种可能性。这种"可能性"本身就是吸引力。

{{< sentence >}}
角色的弧线不一定是从阴暗面走向光明面。反过来走——从光明滑向阴暗——是悲剧的基本结构。
{{< /sentence >}}

Schmidt 的系统不假设角色一定要成长。一个保护者变成了控制者，一个引领者变成了独裁者——这些下行弧线同样是有力的故事。关键是弧线有方向，不是角色从头到尾站在同一个位置。

## 用底版

{{< sentence >}}
底版保证角色在叙事功能上站得住。细节让角色在读者心里活起来。两层都要有。
{{< /sentence >}}

只有底版没有细节 = 功能正确但毫无个性（模板人物）。只有细节没有底版 = 很有趣但在故事里不知道为什么存在（装饰性人物）。好角色需要两层同时工作。

{{< sentence >}}
用原型做起点，然后偏离它——偏离的方向和幅度就是你的创造力。
{{< /sentence >}}

最有记忆点的角色往往是在某个原型的基础上做了意外的偏移。一个看起来像"智者"原型的角色有一个完全不理性的执念。一个看起来像"复仇者"的角色在关键时刻选择了宽恕。偏离必须建立在"读者已经隐约感觉到原型"的基础上——否则偏离没有参照，就不会产生惊喜。
