## 故事的生物学功能

{{< sentence >}}
故事是大脑的飞行模拟器——让你以零成本体验可能致命的处境。
{{< /sentence >}}

这不是比喻。fMRI 扫描显示，读故事时大脑激活的区域和真实体验那些事件时的区域高度重叠。大脑不是在"欣赏"你的故事，是在"经历"你的故事。如果你的故事没有提供值得模拟的经验，大脑就会关掉模拟器——读者就走神了。

{{< sentence >}}
读者的注意力不跟着"发生了什么"走，跟着"这对主角意味着什么"走。
{{< /sentence >}}

一个爆炸发生了。读者关心的不是爆炸的火焰多高，而是：主角现在怎么办？他的计划被打碎了吗？他最害怕的事情发生了吗？如果这些问题不在你的故事里，爆炸就只是特效。

## 内在冲突是引擎

{{< sentence >}}
所有的外在情节都是为了迫使主角面对他的内在冲突。如果没有内在冲突，外在情节只是噪音。
{{< /sentence >}}

内在冲突 = 主角需要的东西和他相信的东西之间的矛盾。他需要信任别人才能获得帮助，但他的信念是"每个人最终都会背叛我"。每一个外在事件都应该加深这个矛盾——要么验证他的旧信念，要么提供打破旧信念的证据。

{{< sentence >}}
主角在故事开始时就必须有一个错误信念。故事的全部作用就是迫使他面对和修正这个信念。
{{< /sentence >}}

没有错误信念的主角不会产生弧光。没有弧光的故事只是一串事件的清单。Cron 的这个判断切得很深：如果你还没有想清楚主角的错误信念是什么，你还没准备好开始写。

## 写作层面的操作

{{< sentence >}}
每一个场景都必须回答一个问题：这件事改变了主角对他的核心冲突的理解吗？如果没有，删掉。
{{< /sentence >}}

这是 Cron 给出的最残酷也最实用的剪辑标准。不是"这个场景写得好不好"，而是"这个场景对主角的内在旅程有没有推进作用"。写得再精彩但和内在冲突无关的场景，在好故事里没有位置。

{{< sentence >}}
具体的细节比抽象的描述有力一万倍——因为大脑只能模拟具体的东西。
{{< /sentence >}}

"她很难过"——大脑没反应。"她把咖啡杯握得太紧，指甲陷进了纸杯的边缘"——大脑立刻开始模拟那个手指的感觉。细节不是装饰，是模拟器的输入数据。没有具体细节，模拟器无法启动。
