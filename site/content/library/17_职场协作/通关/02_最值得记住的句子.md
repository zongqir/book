{{< sentence >}}
职业发展不是匀速爬坡，是走走停停中间夹着几个陡坎。
{{< /sentence >}}

大部分时间你在平路上，做着差不多的事。但突然有一天——一个项目需要新负责人、一个晋升名额空出来、一个跳槽窗口打开——你要在很短的时间内证明自己准备好了。平路上积累的东西，在这一刻才能变现。

{{< sentence >}}
关卡不等人。错过这一轮，下一轮什么时候来你控制不了。
{{< /sentence >}}

很多人的遗憾不是"我不行"，而是"那次机会来的时候我没准备好"。领导问你愿不愿意带一个项目——你犹豫了三天，名额给了别人。不是你能力差，是你的筹码没有提前到位。

{{< sentence >}}
你用来通过第一道关卡的方法，到了第二道关卡可能完全不管用。
{{< /sentence >}}

靠执行力赢得了第一次晋升。但第二次晋升需要的不是执行力——可能是跨部门的协调能力，可能是对业务全局的理解。每道关卡需要的筹码不一样。你不能拿同一套东西反复用。

{{< sentence >}}
在关卡面前犹豫三天和放弃是一样的。
{{< /sentence >}}

关卡的时间窗口很短。你在"要不要接""我行不行""风险大不大"之间纠结的时候，敢先举手的人已经走过去了。陈璐的建议是：在平路上就想好"如果机会来了我要不要接"，到了关卡面前不要再重新想一遍。

{{< sentence >}}
平时什么时候该抬头看路？当你发现周围的人在悄悄准备什么的时候。
{{< /sentence >}}

同事突然开始主动接跨部门的活、领导开始约谈你的平级——这些信号说明关卡可能快到了。大多数人注意到这些信号的时候已经晚了。保持对组织动态的基本敏感度，比埋头干活更重要。

{{< sentence >}}
勤奋解决平路上的问题，节奏感解决关卡上的问题。
{{< /sentence >}}

职场里不缺每天加班到十点的人。缺的是知道什么时候该加班、什么时候该社交、什么时候该学新东西、什么时候该主动请缨的人。这种节奏感无法从勤奋中自动获得——它需要你对组织、对时机、对自己的位置有清醒的判断。
