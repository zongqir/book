## 合作的结构条件

{{< sentence >}}
合作的基础不是信任，而是关系的持久性。
{{< /sentence >}}

整句话的重量在"持久性"三个字上。信任是合作运行一段时间之后的副产品，不是前提。你和供应商反复打交道不是因为信任他，是因为你们还会再见面——这个结构事实改变了双方的收益计算。

{{< sentence >}}
未来能把影子投射到当下，影响当下的选择。
{{< /sentence >}}

"未来的影子"是全书最核心的比喻。当再次相遇的概率足够高，眼前背叛的短期好处就被长期代价压下去。这个参数不需要双方讨论，只需要双方都感知到。

{{< sentence >}}
合作可以在没有友谊、没有远见、没有中央权威的条件下涌现、生长并自我保护。
{{< /sentence >}}

一句话概括全书的核心发现。日常直觉觉得合作要么靠信任，要么靠强制。阿克塞尔罗德证明：只要关系重复、行为可观察、回应对称，合作就能从纯粹追求自身利益的行为者中自发涌现。这句话同时否定了三种常见解释，值得反复咀嚼。

## 策略的生存性质

{{< sentence >}}
不要首先背叛。
{{< /sentence >}}

善良策略——从不主动背叛的策略——在两轮锦标赛中全部排在前列。开局释放善意不是天真，是给对方一个进入合作循环的机会。

{{< sentence >}}
不要嫉妒。
{{< /sentence >}}

三个字，最容易忽略。一报还一报从不追求"比对方多赢"。3:3 的结局优于试图 5:0 却引发报复后打成 1:1。嫉妒会把合作博弈当成零和博弈来打。

{{< sentence >}}
对合作和背叛都要给予回报。
{{< /sentence >}}

只回应合作不回应背叛，会被利用；只回应背叛不回应合作，会陷入僵局。双向回应让对方能够"读懂"你——可读性是合作的基础设施。

{{< sentence >}}
不要太聪明。
{{< /sentence >}}

过度精巧的策略在锦标赛中全军覆没。复杂让对方无法理解你的行为模式，而合作的前提恰恰是可预测性。四行代码的一报还一报赢了，不是因为简单"也行"，是因为简单本身就是竞争优势。

{{< sentence >}}
一报还一报的成功在于它同时具备四个性质：善良、可激怒、宽容、清晰。
{{< /sentence >}}

四个性质缺一个，整体效果就塌。善良但不可激怒会被利用；可激怒但不宽容会陷入永久报复；宽容但不清晰让对方读不懂意图。特别是可激怒和宽容之间的平衡——报复要快但不能记仇——这才是一报还一报最难把握的地方。
