## 关于捷径与自欺

{{< sentence >}}
我们倾向于高估自己的知识，低估自己的无知。
{{< /sentence >}}

过度自信偏差的一句话版本。多贝里在讲这个偏差时列举了大量专业人士的预测准确率——经济学家、股票分析师、政治评论员——结果都惨不忍睹。问题不是无知本身，是对无知毫无知觉。每次觉得"我很确定"的时候，拿这句话对照一下。

{{< sentence >}}
如果你因为已经付出太多而继续，你不是在做决策，你是在给过去的错误追加投资。
{{< /sentence >}}

沉没成本谬误的最干脆表达。装修追加预算、烂尾项目继续投入、不合适的关系拖着不分——场景换了，结构一样。唯一该问的问题是：如果从零开始，你还会做同样的选择吗？

{{< sentence >}}
你不是在寻找真相，你是在寻找支持你已有观点的证据。
{{< /sentence >}}

确认偏误最刺痛的说法。搜索引擎让这件事变得更致命——你打的关键词本身就带着结论方向。多贝里的建议很直接：做完一个判断后，花同样的时间找反面论据。

## 关于概率错觉

{{< sentence >}}
我们用'想起来有多容易'替代了'发生概率有多高'。
{{< /sentence >}}

可得性启发的核心机制。你怕鲨鱼但不怕蚊子，怕飞机但不怕汽车，怕恐怖袭击但不怕心脏病。每一组对比里，前者上新闻，后者不上。你的恐惧清单是媒体报道的投影，不是风险的排序。

{{< sentence >}}
在一个充满不确定性的世界里，一个好故事比一组好数据更有说服力——这恰恰是问题所在。
{{< /sentence >}}

叙事谬误的警告。人类天生需要因果故事来理解世界。一组随机数据配上一个合理叙事，你就觉得看到了规律。投资领域尤其危险：每一轮牛熊事后都能讲出完美的解释，但事前没人预测到。

## 关于社会性偏差

{{< sentence >}}
如果五千万人说了一句蠢话，这句话仍然是蠢话。
{{< /sentence >}}

从众效应的解毒剂。投票人数不改变论证质量。多贝里把这条用在了商业决策、投资跟风、社会舆论等场景上。流行不是正确的证据，共识不是真相的担保。

{{< sentence >}}
我们把一个人的某个突出特征扩展成对整个人的评价——然后深信不疑。
{{< /sentence >}}

光环效应的精确描述。长得好看的人被认为更聪明、更可靠、更有能力。一个品牌的某款产品好用，你就假设它所有产品都好用。评估系统不是逐项打分的；第一印象会污染全部后续判断。

## 关于行动与后果

{{< sentence >}}
我们系统性地高估了行动的好处，低估了不行动的好处。
{{< /sentence >}}

行动偏误的核心。足球守门员在罚点球时扑向左边或右边，其实站在原地不动的成功率更高。但站着不动"看起来不像在做事"。在投资、管理、政策制定中，"做点什么"的冲动让大量无效行动看起来理所当然。

有时候最好的决策就是不做决策。这句话听着像懒人借口，但多贝里拿出的统计数据说明它在特定条件下是严肃策略。

## 关于知与行的距离

每个偏差都有一个残酷的共同点：知道它的存在，不等于你能躲过去。多贝里自己也承认这一点。锚定效应的实验反复证明，即使受试者被明确告知存在锚定，影响只是轻微减弱。

偏差清单提供的不是免疫力。它提供的是命名——命名创造了一个识别窗口，识别窗口创造了一秒钟的犹豫，这一秒钟是所有纠偏动作的前提。
