# 八句话把带队从凭感觉切成照着做

{{< sentence no="1" quote="脑子里的想法不写出来，永远是一团糊。" >}}
管理者做决定的质量取决于一件事——想清楚了没有。

大部分管理者的日常状态是"好像想过了"。A4纸笔记法干的就是一件事：强制把"好像想过"变成"写出来了"。写出来就能检查，检查就能纠偏。
{{< /sentence >}}

{{< sentence no="2" quote="一分钟够了。时间限制是故意的。" >}}
赤羽雄二设定一分钟不是为了效率，是为了拦住完美主义。

给自己十分钟想问题，前八分钟都在犹豫措辞。给一分钟，只能写最核心的判断。管理者需要的不是漂亮的思考记录，是快速把脑子里最要紧的东西拽出来。
{{< /sentence >}}

{{< sentence no="3" quote="下属开口之后的三秒决定他下次还敢不敢来找你。" >}}
下属说完一个问题，你低头看手机、皱眉、或者沉默五秒——他读到的信号是"我不该来打扰"。

零秒回应不是让你不加思考就回答。是让你在下属开口之后立刻给出一个反应：点头、复述他的问题、或者说"等我三分钟"。反应本身就是信号——"我在听，你说的事我接住了。"
{{< /sentence >}}

{{< sentence no="4" quote="一对一不是加分项，是基础设施。" >}}
很多管理者把一对一当成"有时间就做、没时间就砍"的可选动作。

赤羽雄二的判断是：一对一是管理的基础设施，跟电力一样。砍掉一对一省下的那半小时，会在后面变成两小时的信息断裂和信任损耗。你不做，下属碰到问题就不说；不说，你就靠猜；靠猜，迟早猜错。
{{< /sentence >}}

{{< sentence no="5" quote="反馈隔一周就过期了。" >}}
周五发生的事，等到下周三再谈，下属已经记不清当时的细节。你的反馈从"帮他纠偏"变成了"翻旧账"。

即时反馈不需要正式。走过去说一句"刚才那个客户回复处理得很快，节奏不错"——三十秒足够。关键是在事件发生后24小时内说出来。
{{< /sentence >}}

{{< sentence no="6" quote="管理不需要领袖魅力，需要的是每天把该做的动作做完。" >}}
带好团队不要求你是那种走进房间所有人都被感染的人。要求的是：一对一你做了没有、A4纸你写了没有、反馈你给了没有、进度你追了没有。

每一项都是可检查的动作，不是需要修炼的品质。
{{< /sentence >}}

{{< sentence no="7" quote="交代任务的时候多说两分钟，省的是后面三天的返工。" >}}
分任务时说清楚四件事：做成什么样算合格、哪些地方可以自己决定、遇到什么情况要来找你、什么时候要完成。

四件事听起来啰嗦。但跳过任何一件，下属要么不敢动，要么做完发现方向错了。两分钟的交代成本远低于三天的返工成本。
{{< /sentence >}}

{{< sentence no="8" quote="信任不是等来的，是用具体动作做出来的。" >}}
新leader最常犯的错是等信任自然建立——"先磨合几个月再说"。

赤羽雄二的做法反过来：用具体动作制造信任。每周一对一不缺席，答应的事当天跟进，反馈说具体事实不说空话。下属看到你的动作是稳定的、可预期的，信任就开始长出来。不做这些动作，等三年也等不来。
{{< /sentence >}}
