以下句子按"你最可能在什么场景下需要它"排列。每条附带它纠正的具体判断偏差。

{{< sentence >}}
领导身边的人，本身就是一种权力。
{{< /sentence >}}

秘书常被理解为"高级助理"——帮领导订机票、整理文件。唐小舟进入角色后发现，省委大院里几乎所有人都在通过他来揣测书记的意图。他不需要做任何事情，仅仅因为"在那个位置上"，信息就会主动流向他。位置本身就是权力来源，和你有没有实权无关。

{{< sentence >}}
看一个人的真实地位，不看他的职级，看谁在等他。
{{< /sentence >}}

唐小舟观察省委大院的权力格局，最可靠的指标不是组织架构图。他看的是：谁在走廊里等着见谁？谁的电话打过来对方会立刻接？谁的车到了门口保安会主动开门？这些微信号比任何职级表都准确。在你的组织里，你能用同样的方式画出一张真实的权力地图。

{{< sentence >}}
在领导没有表态之前，你的任何判断都是冒险。
{{< /sentence >}}

唐小舟犯过几次这样的错——书记对某件事还没有明确态度，他提前按自己的理解做了安排。即使安排的方向和书记后来的决定一致，赵德良也不满意。原因不是结果错了，是过程越权了。在领导没有表态之前替领导做判断，哪怕猜对了，也是在削弱领导的决策权。

{{< sentence >}}
能让你知道的事情，领导自然会让你知道。他没说的，你不该打听。
{{< /sentence >}}

赵德良有些事情只跟特定的人商量，不告诉唐小舟。初期唐小舟会好奇，试图从其他渠道了解。后来他明白了：领导的信息分发本身就是信任的刻度尺。你被排除在某条信息之外，不是因为你不重要，是因为那条信息和你的职责不相关——或者领导还没决定是否信任你到那个程度。主动打听是在试探信任边界，通常的结果是把边界推远。

{{< sentence >}}
帮领导挡人是技术活——挡对了是保护，挡错了是截断。
{{< /sentence >}}

唐小舟的日常工作中有一大块是决定谁能见赵德良、谁不能。判断标准不是"书记忙不忙"，而是"这个人见书记对书记有没有好处"。挡掉一个不该见的人，赵德良事后会默认认可。放进一个不该见的人，赵德良可能不会说什么，但心里已经记了一笔。挡掉一个其实应该见的人，后果更严重——你切断了书记获取关键信息的通道。

{{< sentence >}}
饭局上的座次比会上的发言更说明问题。
{{< /sentence >}}

黄晓阳用大量篇幅写饭局。座次安排不是礼仪，是权力声明。谁坐在主位旁边、谁被安排在远端、谁被"不小心"漏掉了邀请——每个细节都在传递信号。唐小舟学会读饭局的速度比学会读文件快，因为饭局上的信号更直接、更难伪装。

{{< sentence >}}
越是重要的决定，越不会在正式场合宣布。
{{< /sentence >}}

省委大院里的人事调整、政策转向，往往在正式会议之前就已经在小范围内定了调。会议只是走程序。唐小舟发现，赵德良在做重大决策之前，会先在散步、吃饭或坐车时试探性地提几句。如果你只关注正式渠道的信息，你永远比知情者慢半拍。
