大脑的默认模式是自动驾驶。下面六句话不是道理，是刹车线。在你又被本能脑带着跑的时候，拿出其中一句，踩住。

{{< sentence >}}
成长发生在舒适区边缘。太简单的不算练习，太难的只会让你放弃。
{{< /sentence >}}

选学习内容或定运动目标时调用。做起来毫无压力，说明位置太低。刚开始就想放弃，说明跳进了困难区。往中间调。

{{< sentence >}}
元认知就是你能在做事的时候，退后一步，看见自己正在想什么。
{{< /sentence >}}

情绪上头、注意力被劫持、拖延发作时——先停一秒。问自己"我现在在干嘛"。不需要答案，注意到本身就是干预。

{{< sentence >}}
焦虑的本质是想同时做很多事，又想立刻看到效果。
{{< /sentence >}}

焦虑来袭时不急着压住它。先检查：是不是同时想要的太多？是不是等不了结果？把"十件事"砍到"一件事"，焦虑通常就降了一半。

{{< sentence >}}
大脑不喜欢模糊。你越是笼统地想"我要改变"，它越不动。
{{< /sentence >}}

计划列了一堆但迟迟没启动时调用。"我要改变"对大脑来说是噪音。"我明天早起五分钟"才是指令。模糊的决心感动自己，清晰的动作才启动大脑。

{{< sentence >}}
学习不是往脑子里灌信息。新东西必须和已有经验产生连接，才算学到。
{{< /sentence >}}

读完一章合上书，什么都说不出来——多半是因为读的时候没和自己的经验挂钩。下次试试：每读到一个新概念，停下来想三十秒，"这跟我经历过的什么事有关"。

{{< sentence >}}
耐心不是被动等待。是在看不到结果的时候，依然知道自己在对的方向上。
{{< /sentence >}}

坚持了两个月，体重没变、成绩没涨、文章没人看。这时候最容易放弃。拿这句话出来校准：方向对不对和结果快不快是两件事。方向对了，时间会给答案。
