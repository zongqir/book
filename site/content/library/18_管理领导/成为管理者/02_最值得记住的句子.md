{{< sentence >}}
管理者做正确的事。领导者做对的事。
{{< /sentence >}}

效率和效果的区别。管理者追求效率——把现有流程做到极致。领导者追求效果——判断方向是否正确，必要时推翻现有流程。一个人可以同时做管理和领导，但大多数人困在管理里，因为效率更容易衡量。

{{< sentence >}}
领导力不是天生的，是学出来的——通过自我认知。
{{< /sentence >}}

本尼斯采访的所有杰出领导者都有一个共同点：他们比一般人更了解自己的优势、弱点、价值观和驱动力。自我认知不是性格测试能给的，而是从经历中反复提炼出来的。

{{< sentence >}}
领导力最大的敌人不是失败，是失去好奇心。
{{< /sentence >}}

失败可以从中学习，好奇心一旦消失，学习就停止了。本尼斯发现优秀领导者到老都保持着对世界的强烈好奇——不是装出来的，是内在驱动力的外在表现。

{{< sentence >}}
你不可能在不了解自己的情况下领导别人。
{{< /sentence >}}

管理可以靠流程和工具。领导需要真实——而真实的前提是知道自己是谁。一个不了解自己价值观的人，在压力下做出的决定一定是混乱的。

{{< sentence >}}
管理者模仿，领导者原创。
{{< /sentence >}}

管理者学习最佳实践然后复制。领导者从自己的经验和价值观出发创造新方式。不是说最佳实践没用——而是说只靠模仿永远成不了领导者。

{{< sentence >}}
领导力的关键考验是在混乱中找到方向。
{{< /sentence >}}

确定性环境里管理者就够了——照章办事。不确定性环境才需要领导者——当地图失效时，你靠什么做判断？答案是价值观、经验和对自我的清醒认识。
