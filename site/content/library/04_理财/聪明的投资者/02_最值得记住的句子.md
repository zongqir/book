# 账户一波动，就把这几句拿出来

## 最该反复调用的提醒

{{< sentence no="1" quote="先把亏大钱的可能性压下去，再谈赚多少钱。" >}}
最适合在账户波动、情绪变重的时候拿出来，提醒自己先守住下限。
{{< /sentence >}}

{{< sentence no="2" quote="市场先生是来报价的，不是来指挥你的。" >}}
看到连续涨跌时，先把报价和价值分开。
{{< /sentence >}}

{{< sentence no="3" quote="波动不自动等于风险，永久损失才更该怕。" >}}
这句最能纠正把一切下跌都当成危险的本能反应。
{{< /sentence >}}

{{< sentence no="4" quote="防御型和进取型不是高下之分，是能力边界。" >}}
真正要先认清的，不是勇不勇敢，而是自己能做到什么程度。
{{< /sentence >}}

{{< sentence no="5" quote="价格和价值差得够远，安全边际才开始出现。" >}}
买入前回到这一句，能防止自己只因为跌多了就出手。
{{< /sentence >}}

{{< sentence no="6" quote="便宜不是理由，便宜又能活下去才是理由。" >}}
低估值对象一多，这句最能替你筛掉便宜陷阱。
{{< /sentence >}}

{{< sentence no="7" quote="组合是在替你承认自己会犯错。" >}}
一旦你开始觉得某个机会绝对确定，就该把这句拿出来。
{{< /sentence >}}

{{< sentence no="8" quote="真正昂贵的动作，常常是情绪最重的时候做的那个动作。" >}}
它提醒你：很多损失不是看错，而是被情绪催着提前做了事。
{{< /sentence >}}

## 这些句子最适合在哪些场景拿出来

- 账户一跌，你是先想价值还是先想止痛
- 看到低估值，你会不会先问它到底便宜在哪里
- 新机会出现时，你有没有先分清自己是防御型还是进取型
- 调仓时，你会不会主动回到组合纪律
- 市场很兴奋时，你还能不能等待价格

真正的用法不是背下来，而是在冲动、乐观、害怕和犹豫最重的时候，把顺序拽回来。
