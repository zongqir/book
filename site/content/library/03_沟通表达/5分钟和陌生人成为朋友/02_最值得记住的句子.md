## 破冰那一刻

{{< sentence >}}
开口的标准不是精彩，而是自然。
{{< /sentence >}}

你在等一个"完美的开场白"，但完美的开场白不存在。"你好，我是小王"就够了。对方不会因为你的开场白普通而拒绝你。他们会拒绝的是——你那种"我在执行一个社交任务"的紧绷感。放松，像和老同事打招呼一样开口。

{{< sentence >}}
对方之所以没和你说话，多半也是在等你先开口。
{{< /sentence >}}

社交场合里的沉默往往是双向的。你觉得对方不想被打扰，其实对方可能和你一样在犹豫。第一个开口的人掌握主动权——不是因为更勇敢，而是因为打破了双方都在等待的僵局。

## 推进对话

{{< sentence >}}
少说自己，多问对方。对方开口了，你就赢了一半。
{{< /sentence >}}

新手的本能是拼命展示自己——说自己的工作、经历、观点。但在初次交谈中，让对方说话比自己说话更有效。人在被倾听的时候会产生好感。一个好的问题比三分钟的自我介绍更让对方记住你。

{{< sentence >}}
跟着对方的话题走，别急着转到你想聊的方向。
{{< /sentence >}}

对方说"我最近在学画画"，你的反应应该是"什么画？怎么开始的？"——而不是"哦我也有个爱好，我喜欢打球"。后者把话题拉回了你自己，对方的分享被截断了。跟着对方的方向走，至少走两到三个回合，再自然地过渡到其他话题。

{{< sentence >}}
真实的好奇比假装的热情管用十倍。
{{< /sentence >}}

你不需要对所有人都感兴趣。但找到一个你真的想知道的点——对方的工作、经历、某个你没接触过的领域——然后认真地问。真实的好奇心会自然地产生追问，追问让对话深入，对话深入了，好感就来了。

## 收尾和后续

{{< sentence >}}
五分钟就够了。好的社交不需要长。
{{< /sentence >}}

不要觉得聊五分钟就走是不礼貌。短而舒适的对话比长而尴尬的对话好一百倍。五分钟聊完，对方的感受是"这个人挺好聊的"。硬撑二十分钟两个人都没什么新话题了，对方的感受变成了"终于聊完了"。

见好就收。"很高兴认识你，加个微信？"——简洁、自然、留下联系方式。

{{< sentence >}}
真正的社交发生在第二次联系的时候。
{{< /sentence >}}

第一次五分钟对话只是开了个口。如果你事后从不联系，这五分钟就白费了。隔一两天发一条消息——"上次你说的那个画展我搜了一下，确实有意思"——这条消息的价值远超当场聊了一小时。因为它说明你真的在意对方说了什么。

## 什么时候翻这些句子

- 社交场合犹豫要不要开口时——翻"对方也在等你先开口"
- 开口后不知道聊什么时——翻"少说自己多问对方"
- 觉得对话应该结束但不知道怎么走时——翻"五分钟就够了"
- 活动结束后想要维系联系时——翻"第二次联系"
