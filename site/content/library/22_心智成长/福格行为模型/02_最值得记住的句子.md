# 七句行为设计的底层判断，拦住你下次再靠意志力蛮干

又准备靠意志力硬撑了？

先看看这七句。它们更像七个刹车片。

你又要定大目标时，看一眼。
你又在骂自己不自律时，看一眼。
你做完了什么却没感觉时，也看一眼。

{{< sentence >}}
人不改变，不是因为不想改，是因为改变的方式没设计好。
{{< /sentence >}}

整个框架的起点。
又在怀疑自己意志力不够时，先停下来问：到底是意志力的问题，还是方法的问题。

{{< sentence >}}
行为 = 动机 × 能力 × 提示。三个同时到位，行为才会发生。
{{< /sentence >}}

B=MAP 公式。下次一件事做不到，别急着骂自己。
拿这三个变量逐个检查。

{{< sentence >}}
让行为小到你不可能说'我做不到'。
{{< /sentence >}}

两颗牙、两个俯卧撑、一页书。
不是偷懒，是把起步阻力降到零。

{{< sentence >}}
把新行为绑在你已经在做的事后面，它就有了根。
{{< /sentence >}}

锚点比闹钟靠谱。闹钟可以忽略，已有的日常动作不会消失。

{{< sentence >}}
庆祝不是矫情，是告诉大脑：这件事值得再做一次。
{{< /sentence >}}

做完微习惯后那一秒的正面情绪，是整个系统里最关键的一环。
跳过庆祝，行为很难被大脑标记为"值得保留"。

{{< sentence >}}
动机像潮水，会涨会退。别把行为建在潮水线上。
{{< /sentence >}}

所以 Fogg 不劝你"提高动力"。
他劝你缩小行为，让最没状态的时候也能完成。

{{< sentence >}}
设计行为的能力，比自控力可靠得多。
{{< /sentence >}}

最后一句收住整本书的方向。
别练意志力了。学会设计。

## 什么时候翻哪一句

- `又定了一个大目标`：先看第 3、6 句。大目标未必错，但起步动作必须小到不会失败。
- `又在骂自己不自律`：先看第 1、7 句。问题大概率不在你身上，在方法设计上。
- `做完了一件事却没什么感觉`：看第 5 句。庆祝缺失，大脑不会帮你记住这件事。
- `习惯总坚持三天就断`：看第 2、4 句。拿 B=MAP 逐个检查，大概率是提示没绑好或行为太大。
- `觉得微习惯太小没意义`：看第 3、6 句。小是手段，长出来才是目的。

把它们记成一个顺序会更好用：

先别靠意志力。
缩小行为。绑定锚点。
做完就庆祝。
