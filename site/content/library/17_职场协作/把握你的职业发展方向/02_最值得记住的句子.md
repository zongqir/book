{{< sentence >}}
方向不是想出来的，是走出来的。
{{< /sentence >}}

坐在家里做SWOT分析做一百遍，也不如出去试一次。蒋建敏反复强调：你在行动之前对自己的了解，大部分是想象。只有做了、拿到了反馈，你才知道自己真正擅长什么、真正受不了什么。

{{< sentence >}}
选项越多的人，做决定越慢。
{{< /sentence >}}

听起来反直觉——选项多不是好事吗？但蒋建敏观察到，机会多的人反而容易瘫痪。因为每放弃一个选项，你都在计算"如果那条路更好怎么办"。约束反而是解药：把不可能的排掉，剩下的反而更容易选。

{{< sentence >}}
你嘴上说感兴趣的事，和你实际会花周末去做的事，通常不是同一件。
{{< /sentence >}}

"我对什么感兴趣"是自我报告里最不可靠的数据。可靠的数据是行为记录：你在没有任何外部要求的情况下，自愿花时间做了什么？这些行为信号比任何性格测试都准。

{{< sentence >}}
等待完美时机的人，最后什么时机都等不到。
{{< /sentence >}}

"现在不合适""条件还不够""再等等看"——这些话翻译过来就是"我不敢"。蒋建敏说得很直接：你等的那个"合适的时机"，通常只有在你已经走出第一步之后才会出现。

{{< sentence >}}
方向可以错，但不能没有。
{{< /sentence >}}

选错方向的代价，远小于一直不选的代价。选错了，你至少知道了这条路不对——这本身就是信息。一直不选的人什么信息也没有，只有越来越深的焦虑。

{{< sentence >}}
你的职业方向，藏在你做过的事情里，不在你没做过的事情里。
{{< /sentence >}}

很多人找方向的方式是"看外面有什么机会"。蒋建敏建议反过来：先回头看你过去三年做过的所有事情里，哪些事做的时候你不觉得累、做完之后你会主动复盘、别人找你做这件事你会高兴。把这些事圈出来，方向的线索就在里面。
