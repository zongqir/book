# 感谢反馈里最该留下来的，是这些会在关键时候拽住你的提醒句

{{< sentence >}}
听反馈时，最先受伤的常常不是能力，是自我形象。
{{< /sentence >}}

这句能帮你解释为什么明明一句普通建议，也会让人别扭一天。

{{< sentence >}}
不是所有反馈都在回答同一个问题。
{{< /sentence >}}

先分清是在评价你、指导你，还是在表达感受。

{{< sentence >}}
语气很差，不等于内容全错。
{{< /sentence >}}

这句适合挡住那种“一难听我就全关掉”的反应。

{{< sentence >}}
先把能用的部分拣出来，再决定剩下的怎么处理。
{{< /sentence >}}

接反馈不是全收，也不是全扔。

{{< sentence >}}
成长不等于把每个评价都当判决书。
{{< /sentence >}}

这句能把你从反复内耗里拉回来。

## 这些句子什么时候最该翻出来

- 一开口就想解释很多的时候：先翻这页。
- 你觉得自己明明没说错，结果事情还是做坏的时候：先翻这页。
- 你知道问题在哪，却迟迟不知道从哪一句开始的时候：先翻这页。
- 对方的反应、组织的惯性或关系的压力开始把你带偏的时候：先翻这页。
