# 五条根本判断：第一卷怎样把中国革命的地基重新立起来

第一卷最怕被读偏成两种东西。

一种是几篇名文的集合。另一种是抽象方法论的合集。

两种都不对。第一卷的原则，都是围着一个更硬的问题长出来的：中国革命为什么能在中国成立，又为什么不能照抄别处的路。下面五条，守住了，才不至于把第一卷读成漂亮的概念库。

{{< principle no="1" type="core" statement="中国革命必须先从中国社会结构出发，不能从抽象革命公式出发" >}}
第一卷起手最重的一刀，就是把中国社会重新拆开。

谁被压迫最深，谁和旧秩序捆得最紧，谁会摇摆，谁可能成为主力，这些都要先从现实关系里看，而不能从一个先验的革命模板里往下套。

这条原则约束的，是最常见也最致命的错法：先信某种抽象正确，再逼现实去配它。第一卷反复说明，革命如果不先看中国社会到底是什么样，后面的敌友、主次、道路都会错。

它的边界也很清楚。不是说任何问题都要先做宏大结构分析，而是说凡是牵涉国家、阶级、土地、政权和革命道路的问题，都不能跳过这一层。
{{< /principle >}}

{{< principle no="2" type="core" statement="谁是主力、谁能联合、谁必须打击，只能从现实力量关系里判断，不能靠愿望指定" >}}
第一卷不是只把社会分层，它还要进一步回答革命依靠谁。

这一步非常难。因为它不是按道德好恶来分，也不是按口号响不响来分，而是按现实里能不能动员、会不会摇摆、在什么阶段能不能争取来分。

所以这条原则真正保护的，是革命的现实感。主力不是因为听上去最光荣就成为主力，同盟军也不是因为一时站在一起就永远可靠。

常见误读是把阶级分析读成死板标签学。第一卷真正要求的，恰恰是动态判断。谁在某个阶段能联合，谁在另一个阶段会转向，都得回到现实里重新看。
{{< /principle >}}

{{< principle no="3" type="support" statement="革命道路必须服从中国条件，弱小红色政权能不能存在也只能由条件决定" >}}
第一卷最震人的地方，是敢在最弱的时候讨论“为什么还能活”。

这背后的原则不是乐观，也不是豪气，而是条件判断。中国地域大、统治裂缝多、农民问题深、武装斗争能和土地问题结合，这些具体条件决定了革命道路不必照搬别国，也决定了红色政权并非绝无可能。

这条原则约束的是两种相反的误判。一种是机械照搬，把别国革命道路拿来直接套用。另一种是见到表面弱势就立刻判死，觉得一切都没可能。

它的边界同样很严。条件不成立，就不能硬说存在可能。第一卷从来不是鼓励空喊希望，而是要求你老老实实看条件。
{{< /principle >}}

{{< principle no="4" type="support" statement="理论只有在调查和实践里来回校正，才配叫认识" >}}
第一卷里的《反对本本主义》《实践论》，都在守这一条。

认识不是从书上直接搬来，也不是靠几次经验就自动长成。先调查，后判断；先进入实践，再收反馈；现实顶回来，就得改原来的理解。少任何一段，理论都会变空。

这条原则防的是两边都偏。只信书本，会脱离中国实际；只信局部经验，又会看不见更大的结构。

今天最容易把它读浅成一句“重视实践”。其实它要求得更狠：现实不是给理论背书，而是用来拆理论、改理论、逼你重想。
{{< /principle >}}

{{< principle no="5" type="support" statement="主要矛盾和关键动作只能在具体处境里判断，不能把一句正确话拿到哪里都照用" >}}
《矛盾论》后来太有名，常被压成一句“抓主要矛盾”。

第一卷原来的要求比这重得多。问题里有很多矛盾，主次会变，强弱会变，局面一变，先后顺序也可能一起变。也就是说，主要矛盾不是一个永远固定的答案，而是一种具体分析后的判断。

这条原则约束的，是口号化使用辩证法。把“抓主要矛盾”变成万用钥匙，反而会把复杂问题重新写扁。

它真正要求的是：每到一个具体处境，都得重新判断哪一头决定全局，哪一步现在最该先动。
{{< /principle >}}

把这五条连起来看，第一卷的根本方向就很清楚了。

先看中国，不先看口号。先看结构，不先看愿望。先看条件，不先看气势。先看调查和实践，不先看现成答案。先做具体分析，不把一句正确话拿去包打天下。

这才是第一卷真正留下来的原则层。
