{{< sentence >}}
计划就是在猜。
{{< /sentence >}}

商业计划书不是在做计划——是在猜测未来。用三年前的假设指导今天的决策，比不做计划还危险。不如做短期的、可调整的决定。

{{< sentence >}}
工作狂不是英雄。
{{< /sentence >}}

加班不代表投入，代表你管不好时间或者设计了错误的工作流程。Basecamp的人每周只工作40小时——产出不比硅谷996的团队少。

{{< sentence >}}
做少一点，做好一点。
{{< /sentence >}}

大多数产品的问题不是功能太少——是功能太多。砍掉80%的功能，把剩下的20%做到极致。用户不会因为你的产品什么都能做而选择你——会因为它某一件事做得特别好而选择你。

{{< sentence >}}
会议是毒药。
{{< /sentence >}}

一个小时的会议、八个人参加——不是花了一小时而是花了八小时。大多数会议完全可以用一封邮件或一条消息替代。如果必须开会：人越少越好、时间越短越好、必须有结论。

{{< sentence >}}
先赚钱，再花钱。
{{< /sentence >}}

不融资的好处不只是不稀释股份——是让你从第一天就必须做出有人愿意付钱的东西。融资让你可以推迟"有没有人买单"这个问题，但这个问题推迟得越久，发现答案是"没有"的代价越大。

{{< sentence >}}
拥抱约束。
{{< /sentence >}}

钱少、人少、时间少不是劣势——是强制你做减法的力量。最好的创意不是在资源无限的时候产生的，而是在不得不取舍的时候。
