{{< sentence quote="客户记不住你说了什么，但会记住你如何对待他。" >}}
细节不是装饰，是客户对你公司的印象来源
{{< /sentence >}}

{{< sentence quote="一个细节的改变可能改变成交概率，但一次失误往往毁掉整个销售过程。" >}}
细节管理的杠杆在于失败成本大于成功收益的倍数
{{< /sentence >}}

{{< sentence quote="销售的80%时间用在20%的有价值客户身上，剩下80%的时间被浪费在不值得的跟进上。" >}}
细节意识从客户分类和优先级开始
{{< /sentence >}}

{{< sentence quote="你问的问题决定了你得到的答案，而客户的答案决定了交易的可能性。" >}}
提问的措辞和顺序直接影响信息质量
{{< /sentence >}}

{{< sentence quote="异议不是拒绝，而是对方表达真实需求的方式。" >}}
细节营销在于理解异议背后的真实障碍
{{< /sentence >}}

{{< sentence quote="跟进的频度取决于交易的重要性，而不是销售人员的热情。" >}}
过度跟进和不足跟进都是成本
{{< /sentence >}}

{{< sentence quote="成交后的第一印象决定了客户的推荐意愿。" >}}
交付和服务的细节往往比销售细节更决定复购
{{< /sentence >}}

{{< sentence quote="记住客户上次提到的一个小细节，往往比一次精心准备的提案更有说服力。" >}}
人格化接待和记忆是信任的基础
{{< /sentence >}}
