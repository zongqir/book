## 利益结构决定政策方向

{{< sentence >}}
高房价不是市场失灵，是利益格局的精确表达。
{{< /sentence >}}

市场失灵暗示偶然和可修复。
但高房价背后有清晰的受益方和明确的维护机制。
地方财政、金融系统、开发商、投机资本——
利益一致时，价格方向就被锁定。

{{< sentence >}}
调控房价的人，同时也是高房价的受益者。
{{< /sentence >}}

裁判员同时是运动员，规则执行力度天然打折。
不改变这个身份冲突，每轮调控都只是缓兵之计。

## 购房者承受的远超价格

{{< sentence >}}
高房价压缩了整个社会的消费能力和创新空间。
{{< /sentence >}}

收入大部分用于还贷，消费被压缩。
年轻人不敢冒险创业，因为房贷不能断。
高房价的代价不止于住房；它抑制消费、创业和生育。

{{< sentence >}}
一套房绑架三代人——这不是修辞，是财务事实。
{{< /sentence >}}

父母出首付，夫妻背月供，孩子教育预算被挤压。
三代人的财务自由度被一套房集中消耗。
这种代际财务绑定，多数发达经济体没有对应物。

## 改革的真正障碍

{{< sentence >}}
土地财政不是过渡安排，已经变成地方政府的生存模式。
{{< /sentence >}}

"过渡"意味着有退出时间表。
但二十年过去，依赖程度不降反升。
生存模式一旦形成，改变它就不是技术问题，是利益博弈。

{{< sentence >}}
保障房建设速度，暴露了改革的真实优先级。
{{< /sentence >}}

口号上保障房是重点；执行上进度持续滞后。
建保障房意味着减少商品房用地、降低出让金收入。
看预算分配比听政策宣言更诚实。
