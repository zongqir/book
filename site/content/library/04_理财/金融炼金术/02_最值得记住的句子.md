# 一看到趋势和叙事，就先把这几句拿出来

## 最该反复调用的提醒

{{< sentence no="1" quote="市场不只是反映现实，它也会塑造现实。" >}}
这句能把你从静态均衡拉回到反馈链。
{{< /sentence >}}

{{< sentence no="2" quote="先找主叙事，再看主叙事怎样改写行为和融资条件。" >}}
叙事如果不能回到现实，它就还不是有用的研究。
{{< /sentence >}}

{{< sentence no="3" quote="偏差扩大时，不能用静态均衡去安慰自己。" >}}
趋势最强的时候，最容易犯这类错。
{{< /sentence >}}

{{< sentence no="4" quote="趋势里最贵的错，不一定是跟错，有时是太早站到对面。" >}}
它提醒你：正确方向和正确时点，不是一回事。
{{< /sentence >}}

{{< sentence no="5" quote="反身性不是口号，是要盯反馈链有没有继续加速。" >}}
真正值钱的，不是会说这个词，而是会追踪这条链。
{{< /sentence >}}

{{< sentence no="6" quote="一旦反馈链断了，旧逻辑往往会同时失灵。" >}}
转折发生时，别再拿原来那套理由硬撑。
{{< /sentence >}}

{{< sentence no="7" quote="不确定性不是噪音，它本身就是市场的一部分。" >}}
别因为无法精确，就假装自己已经看清。
{{< /sentence >}}

{{< sentence no="8" quote="能改口比硬撑更值钱。" >}}
在转折点前后，这句往往能少掉最重的损失。
{{< /sentence >}}

## 这些句子最适合在哪些场景拿出来

- 看到一轮趋势时，你会不会先找主叙事
- 叙事升温时，你还会不会跟踪它如何改写融资和行为
- 市场继续走强时，你能不能承认偏差会扩大而不是急着逆势
- 环境反转时，你会不会及时收缩旧逻辑
- 做判断时，你会不会区分自己看到的是趋势还是只是一句好听解释

真正的用法不是背下来，而是在冲动、乐观、害怕和犹豫最重的时候，把顺序拽回来。
