# 八个拿来纠偏效率直觉的提醒

## 动机和控制感

{{< sentence >}}动机的核心燃料不是激励，是控制感。人需要觉得自己在做选择，哪怕选择的范围很小。{{< /sentence >}}

当你发现自己对一件事提不起劲，先别问"我是不是不够自律"。先问"我在这件事上有选择权吗"。如果答案是"完全没有"——问题不在你的动力，在你的处境。

{{< sentence >}}最能摧毁动机的不是困难，是被剥夺选择权。{{< /sentence >}}

海军新兵训练里最消磨人的不是体能负荷，而是"一切都由别人决定"的无力感。恢复动力的方式不是减负，是把最小的选择权还给他们。

## 团队和心理安全

{{< sentence >}}高效团队的第一特征不是成员能力强，是成员敢开口。{{< /sentence >}}

Google 的亚里士多德计划发现，心理安全对团队效率的预测力超过了智力、经验和性格匹配。一个人不敢说话的团队，和一支有优秀球员但不传球的队伍一样——个体再强也赢不了。

## 专注和心智模型

{{< sentence >}}自动驾驶状态下完成的工作，看起来做了，其实脑子没到场。{{< /sentence >}}

飞行员最危险的时刻不是出了故障，是一切正常时大脑进入待机。上班族的日常版本：打开电脑，刷邮件，开会，写文档，一天过完了，但回想不起任何一个主动做出的判断。

{{< sentence >}}打破自动驾驶的办法是在动手之前先做心理预演：这次可能出什么意外，我准备怎么应对。{{< /sentence >}}

预演不需要精确预测未来。它的价值是强迫大脑从"执行模式"切换到"判断模式"。

## 目标的双层结构

{{< sentence >}}只有大目标的人容易迷失在日常里，只有小目标的人容易忘记自己在走向哪里。{{< /sentence >}}

都希格的建议不是"定一个好目标"，而是始终保持两层——一个有情感驱动力的远景，搭配一组可以今天就开始做的具体动作。

## 决策和概率思维

{{< sentence >}}好的决策者不是预测得更准，是同时持有更多种可能性。{{< /sentence >}}

扑克高手和商业决策者的共同点：他们不追求"猜对答案"，而是习惯性地给多种结果分配概率。这个思维习惯比任何决策框架都管用。

## 场景映射

| 你正在经历 | 先调出这句 |
|---|---|
| 对手上的任务提不起劲 | 动机燃料是控制感，不是激励 |
| 团队开会没人说话 | 高效团队第一特征是敢开口 |
| 一天过完了但想不起做了什么 | 自动驾驶状态下脑子没到场 |
| 完成了待办清单但感觉方向不对 | 缺的是大目标，不是小目标 |
| 要做一个不确定的决定 | 好决策者同时持有多种可能 |
