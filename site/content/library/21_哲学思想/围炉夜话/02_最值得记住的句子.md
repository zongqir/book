{{< sentence >}}
十分不耐烦，乃为人大病。
{{< /sentence >}}

急躁不是性格，是能力缺陷。你在微信群里打出去又撤回的那条消息、在会议上没忍住的那句反驳、在家里因为小事发的那通火——回头看，几乎都是"不耐烦"惹的祸。调用时机：每次感到烦躁想立刻行动时。

{{< sentence >}}
人品之不高，总为一利字看不破。
{{< /sentence >}}

不是说不该追求利益，是说利益一旦变成判断的唯一标尺，其他所有标尺都会失灵。你开始用"值不值"评估友情，用"有没有用"评估帮忙，最后发现身边全是交易、没有关系。调用时机：做选择时发现自己只在算利弊。

{{< sentence >}}
交朋友增体面，不如交朋友益身心。
{{< /sentence >}}

两种社交，一种让你出门更有面子，一种让你回家更安心。面子型社交的回报会通货膨胀，安心型社交的回报会复利增长。调用时机：犹豫要不要参加一场只为"认识人"的饭局。

{{< sentence >}}
但责己，不责人，此远怨之道也。
{{< /sentence >}}

不是说别人没错，是说"责人"这个动作本身不产生改善，只产生怨恨。你指出别人的问题，对方记住的不是问题，是你的态度。先管自己该管的部分，怨气自然减少。调用时机：想说"你怎么又……"的时候。

{{< sentence >}}
志不可不高，志不高则同流合污。
{{< /sentence >}}

志向不是梦想，是底线。志向高的人不一定成功，但他知道什么事不能做。没有底线的人不是没追求，是随波逐流——环境往哪推，他就往哪走。调用时机：发现自己在做一件"大家都在做"但心里不踏实的事。

{{< sentence >}}
常存仁孝心，则天下凡不可为者，皆不忍为。
{{< /sentence >}}

这是一条内建防线。不靠规则约束自己，靠"不忍心"。规则可以钻空子，不忍心没有空子可钻。你不是因为违法才不做某件事，是因为做了之后面对家人的时候会不安。调用时机：站在灰色地带、没人会发现但你知道不对的时候。

{{< sentence >}}
处世以忠厚人为法，传家得勤俭意便佳。
{{< /sentence >}}

选榜样不看能力最强的，看人品最稳的。忠厚人不一定飞得最高，但摔跤的概率最低。传给下一代的也不是财富，是习惯——勤和俭是两个最不起眼但最耐用的习惯。调用时机：选合作伙伴或决定教孩子什么。

{{< sentence >}}
求备于一人，则贤者不可得。
{{< /sentence >}}

要求一个人什么都好，最后连好人都留不住。每个人都有短板，你盯着短板看，优点就消失了。完美主义用在做事上是高标准，用在看人上是苛刻。调用时机：对同事、伴侣或朋友感到失望，觉得"他怎么连这都做不好"。

{{< sentence >}}
和气迎人，平情应物。
{{< /sentence >}}

八个字，两件事。对人温和，对事平静。做到第一条不难，遇到麻烦还能做到第二条才见功力。"平情"不是压抑情绪，是不让情绪替你做判断。调用时机：遇到突发状况、情绪先于思考涌上来的时候。
