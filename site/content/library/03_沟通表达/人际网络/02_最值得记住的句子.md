{{< sentence >}}
给你带来最关键机会的人，通常不是你最亲近的朋友。
{{< /sentence >}}

你的密友和你看同样的信息、知道同样的机会。真正能带来新信息的是那些"偶尔联系"的人——他们连接着你接触不到的圈子。找工作时帮你传递消息的，往往是半年没说过话的前同事，不是天天一起吃饭的搭档。

{{< sentence >}}
你站在两个圈子的交叉点上，比你在一个圈子里认识所有人更有价值。
{{< /sentence >}}

网络的核心资产不是节点数量，是桥梁位置。同时接触技术圈和业务圈的人，比只在技术圈认识一百个人的人掌握更多信息差。位置决定信息流向，信息流向决定机会。

{{< sentence >}}
维护三段深度关系的回报，大于开拓三十个新联系人。
{{< /sentence >}}

关系的价值来自信任，信任来自重复互动和兑现承诺。一次聚会认识十个人，三个月后一个都叫不出名字。同样的时间拿来和三个老朋友各深聊一次，每段关系的密度都在增长。

{{< sentence >}}
社交圈越同质化，你得到的信息越重复。
{{< /sentence >}}

和相似背景的人待在一起很舒服，但舒服的代价是信息冗余。每个人推荐同样的书、讨论同样的话题、知道同样的机会。打破这个循环不需要"广撒网"，只需要有意识地跨出一个你不熟悉的领域。

{{< sentence >}}
最可靠的社交货币不是人情，是你持续提供的价值。
{{< /sentence >}}

靠请客吃饭和帮忙办事维持的关系有天花板。持续被人想起的人，不是最会做人的人，而是能稳定输出有用信息、专业判断或实际帮助的人。让别人需要你，比让别人欠你更持久。

{{< sentence >}}
关系不维护就会衰减——不是因为对方变了，是因为信任有保质期。
{{< /sentence >}}

三年没联系的老朋友，你突然找他帮忙，他会犹豫。不是他不想帮，是你们之间的信任已经过期了。关系维护不需要频繁，但需要规律——每隔一段时间有一次有质量的互动，信任才能保鲜。
