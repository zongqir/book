# 在你下次掏钱之前会冒出来的七句话

每句话背后都有一个控制实验，每句话对准一种你在日常决策中反复踩的坑。

看到促销犹豫了——翻一遍。和人谈价格——翻一遍。又舍不得关掉某个选项——翻一遍。

## 环境替你选好了比较对象

{{< sentence no="1" quote="人们很少做不比较的选择。我们的决定总是相对的——而环境决定了你拿什么来比。" >}}
你以为自己在独立判断，实际上在比较菜单上刻意安排的选项。下次看到三个选项、其中一个明显不如另一个时，先问自己：谁放了这个诱饵？
{{< /sentence >}}

## 零价格是一个情绪开关

{{< sentence no="2" quote="免费不只是一个价格。它是一个情绪热键——一按下去，正常的成本收益分析就停摆了。" >}}
"免费"触发的是"不拿就亏"的冲动。你为了免费赠品排四十分钟队，为了凑免费邮费多花一百块——付出的远超省下的。
{{< /sentence >}}

## 第一个数字锁定了判断范围

{{< sentence no="3" quote="一旦我们按某个价格买了某样东西，这个价格就成了后续所有相关判断的锚。" >}}
锚定不需要你相信那个数字，它只要出现过就行。谈薪水时谁先开价、看房时第一套什么价位——第一个数字的权力远超你的感觉。
{{< /sentence >}}

## 社会规范碰到钱就碎

{{< sentence no="4" quote="社会规范和市场规范各自运行良好，但一旦市场规范挤入社会规范的领地，社会规范就很难恢复了。" >}}
帮邻居搬家，他递来三百块。你突然觉得不对劲。一段"朋友互助"的关系刚被切换成"雇佣交易"——切换之后，回不去。
{{< /sentence >}}

## 拥有就开始高估

{{< sentence no="5" quote="我们对已经拥有的东西赋予了更多的价值，仅仅因为它是我们的。" >}}
你卖二手觉得对方出价太低，买家觉得你要价太高。双方都没说谎。所有权给卖家装了一副滤镜，把相处记忆和情感当成了市场价值的一部分。
{{< /sentence >}}

## 预期直接改变感受

{{< sentence no="6" quote="当你预期某种体验会好的时候，它通常就会好。预期不只是心理期待——它改变了你的生理感受。" >}}
贴了品牌标签的止痛药比相同成分的便宜药更有效。预期重塑了神经系统的实际反应。你觉得贵酒好喝，一部分原因是它在你嘴里真的变好喝了。
{{< /sentence >}}

## 关不掉的门正在消耗你

{{< sentence no="7" quote="我们不愿关闭任何一扇门——即使它通向的房间对我们毫无价值。" >}}
同时备着三份工作机会、两段暧昧关系、四个副业方向。看起来灵活，实际上哪个都没认真推进。保留选项的代价是隐形的，但它一直在消耗注意力和执行力。
{{< /sentence >}}
