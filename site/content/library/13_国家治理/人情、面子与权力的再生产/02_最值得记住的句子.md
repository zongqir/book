## 面子作为社会货币

{{< sentence >}}
面子不是虚荣，是一种可以积累、兑换和透支的社会货币。
{{< /sentence >}}

给面子是投资，收面子是收益，丢面子是资产缩水。一个人在社会网络中的行动能力，很大程度上取决于他的"面子账户"余额。这套系统和金融系统的逻辑惊人地相似，只是交易介质不是货币，是社会认可。

{{< sentence >}}
面子的价值取决于给你面子的人是谁。有些面子比另一些面子更值钱。
{{< /sentence >}}

一个普通朋友给你的面子和一个有权力的人给你的面子，在社会交换系统中的"兑换率"完全不同。面子背后是权力等级——越高位的人给出的面子越有分量，因为它隐含了更大的潜在资源调配能力。

## 人情作为社会债务

{{< sentence >}}
收了人情就欠了债。这笔债没有合同、没有利率、没有到期日，但必须还。
{{< /sentence >}}

人情债的特点是模糊性：什么时候还、怎么还、还多少，全看关系和情境。但"欠着"这个状态是确定的。这种模糊性不是缺陷，恰恰是人情系统的设计特征——它让债务关系持续存在，把一次性交换变成长期绑定。

{{< sentence >}}
做人情的人不一定期待等额回报。他期待的是关系本身——一种可以在未来某个不确定时刻兑现的支持承诺。
{{< /sentence >}}

人情不是一手交钱一手交货的交易。它更像是在社会网络中储蓄信用。今天帮了别人一个忙，不是为了明天让他帮一个同等的忙，而是为了在社会网络中维持一个"可信赖"的位置——这个位置本身就是资源。

## 权力的非正式运作

{{< sentence >}}
正式制度决定了谁有权力。人情面子系统决定了权力怎么被实际使用。
{{< /sentence >}}

一个领导签字批准一个项目，是正式权力在运行。但决定这个项目是否能排到领导面前、领导是否愿意优先处理、处理时是从严还是从宽——这些都在人情面子系统的控制之下。制度是骨架，人情是肌肉。

{{< sentence >}}
制度越模糊的地带，人情面子的空间越大。
{{< /sentence >}}

规则明确、执行刚性的领域，人情运作的空间有限。但在自由裁量权大、规则弹性高、信息不对称的领域——人情面子系统几乎全面接管了实际的资源分配。
