# 看清选择代价的几句提醒

## 看起来安全的选择，往往隐藏最深的代价

{{< sentence >}}
听话和自主，从来都是反向的权衡。
{{< /sentence >}}

能帮你规避风险的建议，往往也规避了你自己发现的机会。

{{< sentence >}}
成功的定义，决定你会牺牲什么。
{{< /sentence >}}

当你追求的成功和社会期待一致时，别以为那就是你自己的选择。

{{< sentence >}}
最容易被忽视的代价，往往是时间。
{{< /sentence >}}

每一个选择都在消耗你无法重新获得的时间，这比金钱更值得警惕。

## 忠告的真实价值在于它让你看清什么会被放弃

{{< sentence >}}
真正的忠告，会让你感到难受，而不是放心。
{{< /sentence >}}

如果一个建议让你觉得如释重负，很可能是它只说了你想听的。

{{< sentence >}}
你在模仿别人的成功，就在放弃自己的独特性。
{{< /sentence >}}

照搬一条成功之路，往往意味着你也在复制那条路上无法得到的东西。

{{< sentence >}}
舒适的选择，往往是最昂贵的选择。
{{< /sentence >}}

避免冲突、保持稳定的路，到头来会付出最高的代价。

## 人生方向的决定，本质上是关于你愿意放弃什么

{{< sentence >}}
你无法拥有所有的可能，所以问题不是选什么，而是放弃什么。
{{< /sentence >}}

清楚地看清自己在放弃什么，比盲目地追求什么更重要。

{{< sentence >}}
不做决定本身，也是一个决定。
{{< /sentence >}}

躺平不行动的代价，往往比主动选择的代价还大。
