# 创业验证中最值得记住的几句话

## 计划和现实永远有偏差

{{< sentence >}}
再完美的商业计划，在客户面前也会被打脸。
{{< /sentence >}}

问题不在你没想周全，而在于你根本无法在办公室里想清楚。

{{< sentence >}}
你的假设，都是假设，直到被客户验证才是事实。
{{< /sentence >}}

不要在写计划书上花太多时间，把时间花在客户面前。

{{< sentence >}}
计划是必要的，但执行中的灵活性更必要。
{{< /sentence >}}

好的计划应该像一个指针，而不是一条死路。

## 关于客户验证的关键原则

{{< sentence >}}
客户验证永远比完美计划更重要。
{{< /sentence >}}

用最小的成本、最快的速度去验证你的假设。

{{< sentence >}}
你的客户就在那里，不是在脑子里，而是在现实中。
{{< /sentence >}}

别埋头写计划，出门去找客户。

{{< sentence >}}
最有价值的数据，往往来自于失败的尝试。
{{< /sentence >}}

快速失败并学习，比慢慢规划更经济。

## 关于转向和商业模式

{{< sentence >}}
转向不是失败，而是学习的结果。
{{< /sentence >}}

每一次转向都说明你了解了客户一点点。

{{< sentence >}}
商业模式不是先制定，而是在验证中浮现。
{{< /sentence >}}

四个步骤不是顺序，而是在你建立模式过程中的反复循环。

{{< sentence >}}
最大的风险，往往不是市场反应，而是你的坚持。
{{< /sentence >}}

一旦开始，最难的就是保持客户中心而不陷入自己的逻辑。
