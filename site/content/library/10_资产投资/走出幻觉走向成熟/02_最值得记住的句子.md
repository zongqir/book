# 八句话，把交易从赌运气拉回做概率

## 系统比信号重要

{{< sentence >}}
没有交易系统的人，赚钱是意外，亏钱是必然。
{{< /sentence >}}

散户最常见的状态：有买入的理由，没有卖出的规则。进场靠消息或指标，出场靠情绪。赚了一笔觉得自己开窍了；连亏三笔开始怀疑人生。

根源不是运气差，是根本没有一套完整的决策框架在运行。

{{< sentence >}}
交易系统的价值不是让你每笔都对，是让你在大量交易后胜出。
{{< /sentence >}}

新手追求每笔都赚。成熟交易者追求系统的期望值为正。

期望值 = 胜率 × 平均盈利 - 败率 × 平均亏损。胜率 40% 也可以赚钱，只要盈亏比够大。

## 放弃预测，转向应对

{{< sentence >}}
你不需要知道市场明天往哪走，只需要知道市场到了某个位置你该怎么做。
{{< /sentence >}}

预测是在猜答案。应对是把每种可能性都列出来，提前定好反应规则。价格突破了，按规则进；跌破了止损线，按规则出。

对错不由你判断，由系统判断。

{{< sentence >}}
正确的亏损和错误的盈利——前者是系统运行的成本，后者是系统崩溃的种子。
{{< /sentence >}}

按规则止损，亏了，这是正确的亏损。系统在运行，成本可控。

不按规则乱买，赚了，这是错误的盈利。它强化一个危险的信号：违反纪律也能赚钱。下次你会押得更大，直到一笔亏光。

## 资金管理决定生死

{{< sentence >}}
仓位大小不是胆量问题，是数学问题。
{{< /sentence >}}

全仓一只，涨 20% 觉得自己很牛。跌 20% 已经伤筋动骨。连跌两次 20%，本金只剩 64%，需要涨 56% 才能回本。

仓位计算的起点是你能承受的单笔最大亏损。用止损幅度和可承受亏损金额反推每笔该买多少——不是"我觉得这次挺有把握"。

{{< sentence >}}
截短亏损，让利润奔跑。
{{< /sentence >}}

六个字，整个趋势跟踪策略的压缩。做到前半句靠止损纪律。做到后半句靠抑制"赚了快跑"的冲动。

大部分人恰好做反了：亏了死扛，赚了急跑。

## 系统执行靠纪律，不靠信念

{{< sentence >}}
最大的敌人不是市场，是你在连续亏损时想改系统的冲动。
{{< /sentence >}}

连亏五笔，你开始怀疑系统有问题。其实连亏五笔可能完全在系统的统计预期之内。

一套胜率 40% 的系统，连续亏损五次的概率接近 8%。交易一年，这件事几乎一定会发生。分清"系统正常波动"和"系统失效"，是成熟交易者最硬的心理关。

{{< sentence >}}
成熟交易者做的事很无聊：重复执行已经验证过的规则。
{{< /sentence >}}

新手觉得交易应该刺激。涨了兴奋，跌了紧张，每天都有故事。

成熟的状态恰好相反——你已经知道该怎么做，剩下的只是执行。没什么可想的，没什么可激动的。

无聊，才是系统在场的证据。

## 什么时候回来翻这一页

- 准备下单前，先回到第 3 句：你是在预测还是在应对？
- 止损犹豫的时候，回到第 4 句：正确的亏损好过错误的盈利。
- 想重仓押一把的时候，回到第 5 句：算一下仓位。
- 连续亏损想换策略的时候，回到第 7 句：先分清是波动还是失效。
- 觉得交易太无聊的时候，回到第 8 句：无聊就对了。
