## 关于语感的本质

{{< sentence >}}
语感不是天赋，是辨认的次数。
{{< /sentence >}}

打破"语感是天生的"这个误解。品酒师不是天生能分辨年份，是辨认了足够多次。读文字也一样——有意识地停下来辨认"这里为什么舒服""那里为什么别扭"，语感就在生长。

{{< sentence >}}
好文字先过身体，再过大脑。
{{< /sentence >}}

读到一个句子起鸡皮疙瘩，这个反应比任何分析都快。语言的声音、节奏和呼吸感直接作用于身体感受，不需要经过意义理解的中转。

## 关于节奏的力量

{{< sentence >}}
连续三个同样长度的句子放在一起，读者就开始走神了。
{{< /sentence >}}

节奏单一是阅读疲劳最隐蔽的来源。内容没问题，论证没问题，但句子长度不变化，读者的注意力就找不到落脚点。这条规则适用于任何类型的写作。

{{< sentence >}}
标点是呼吸，不是语法。
{{< /sentence >}}

从修辞感觉的角度，逗号、句号、分号的核心功能是控制阅读时的气息节奏。把标点当语法规则来用会写出语法正确但读起来喘不过气的句子。

## 关于审美判断力

{{< sentence >}}
能说出一段话好在哪里的人，修改自己的文字时才有内部标尺。
{{< /sentence >}}

如果对好文字的感觉停留在"我觉得好但不知道为什么"，修改时就缺少方向。把"觉得好"转化成"知道好在哪里"，是从感性阅读走向有意识写作的关键一步。

{{< sentence >}}
删掉一个漂亮但不必要的形容词，比加一个漂亮的形容词更难。
{{< /sentence >}}

语感成熟的标志之一是学会割舍。知道什么时候该加，更知道什么时候该减——减的能力需要更强的审美判断力。
