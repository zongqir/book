# 看到热闹公司时，先拿这几句压住自己

## 先把研究顺序记住

{{< sentence >}}
个人投资者最稀缺的优势，不是消息更快，而是更早在生活里看到变化。
{{< /sentence >}}

看到这句，就别再下意识把自己放在机构的反面。林奇不是让你认命，而是让你先承认自己的起点在哪里。

{{< sentence >}}
你看见的热闹，只配当研究起点，不配直接当结论。
{{< /sentence >}}

这句专门防“我用过、我见过、我很熟，所以我懂了”。熟悉感只能帮你早一步，不帮你少做后面的功课。

{{< sentence >}}
买股票之前，先说清它是哪一类公司，再决定该问什么问题。
{{< /sentence >}}

这是林奇最耐用的一层。公司分类一乱，后面的增长、估值和持有逻辑都会一起乱。

{{< sentence >}}
快速成长很迷人，但负债、增发和现金流会告诉你它到底是真长还是硬撑。
{{< /sentence >}}

每次看到高增长公司，先把这句翻出来。很多“好故事”就是死在这里。

{{< sentence >}}
好生意也会买贵，热产品也会看错。
{{< /sentence >}}

这句能给热情降温。你喜欢的公司，不等于任何价格都能买。

## 持有以后更该反复看

{{< sentence >}}
分批买入不是胆小，是给判断留修正空间。
{{< /sentence >}}

尤其适合那种“我已经很想下单”的时刻。林奇的方法不是憋住不动，而是别一次把自己送到没法回头的位置。

{{< sentence >}}
如果原来的逻辑已经变了，就别拿熟悉感替它续命。
{{< /sentence >}}

很多人买前会研究，买后只会解释。看到这句，就回头查最初那套判断还在不在。

{{< sentence >}}
最大的错误，常常不是没抓到十倍股，而是把根本没看懂的公司脑补成十倍股。
{{< /sentence >}}

它提醒你，林奇这本书最该治的不是错过焦虑，而是想证明自己眼光好的冲动。

## 最该拿它出来的几个时刻

- 你刚在生活里看到一家突然变热的公司，心里已经开始想买
- 你已经很喜欢一家公司的产品，准备直接跳过报表
- 你发现自己把很多公司都往“高成长”里塞
- 你因为股价波动，开始忘记当初到底为什么买
- 你拿不准该继续跟踪，还是该承认自己没看懂

这页最好的用法，不是背熟全部，而是在每次心里已经有答案的时候，先拿一两句出来拆自己的冲动。
