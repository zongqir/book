## 反馈：系统的自我调节能力

{{< sentence >}}
正反馈放大变化，负反馈抵消变化。系统的稳定性往往取决于负反馈的强度。
{{< /sentence >}}

一个简单的例子：房间里温度上升 → 人们打开窗户 → 温度下降 → 人们关上窗户。这是负反馈，维持了温度的稳定。

{{< sentence >}}
很多看起来是外部原因造成的结果，其实是系统内部的反馈导致的。
{{< /sentence >}}

失业人数增加，导致消费下降，导致企业生产减少，导致失业人数进一步增加。这是一个恶性循环，由系统内部的反馈驱动。

## 延迟：时间差导致的误判

{{< sentence >}}
原因和结果之间的时间延迟，让人很容易误判因果关系。
{{< /sentence >}}

今年的教育投入，可能要 10 年后才能看到人口素质的提升。在这 10 年间，很多人会觉得"投入了也没有用"。

{{< sentence >}}
反应太快会导致过度矫正。反应太慢会导致问题恶化。找到合适的反应速度很困难。
{{< /sentence >}}

这是调控系统时最常见的问题。要么反应过度，要么反应不足。

## 非线性：投入和产出不成比例

{{< sentence >}}
线性思维假设：投入翻倍，产出也翻倍。现实往往不是这样。
{{< /sentence >}}

一个销售人员，从 5 个增加到 10 个，销售额可能从 100 万增加到 300 万（因为团队协作的效应）。但从 100 增加到 200，销售额可能没有翻倍（因为遇到了市场天花板）。

{{< sentence >}}
系统的饱和点、临界点、折返点会让投入产出关系产生跳变。
{{< /sentence >}}

达到某个关键点后，情况可能会突然改变。一点点的额外投入，可能会导致大的变化。或者反之。

## 系统结构决定了行为

{{< sentence >}}
改变人、改变意识，往往不如改变系统结构有效。
{{< /sentence >}}

不要期待"激励人更努力"就能解决问题。如果系统结构导致了某种行为，激励往往无效。

{{< sentence >}}
看似是道德或性格问题的现象，很可能是系统结构问题。
{{< /sentence >}}

官员贪腐，看起来是道德问题。但如果系统设计导致了"贪腐收益大、被抓风险小"，再有道德的人在这个系统里也容易出问题。改变系统比改变人更有效。
