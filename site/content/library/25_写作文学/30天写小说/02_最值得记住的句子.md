## 关于完成的优先级

{{< sentence >}}
你不可能修改一张白纸。
{{< /sentence >}}

所有关于"写完再改"的论证，压缩到最短就是这句。白纸上没有任何东西可以被改善、被打磨、被推翻。先有初稿，才有修改的起点。

{{< sentence >}}
截止日不是敌人，是唯一能帮你打败完美主义的盟友。
{{< /sentence >}}

没有外部时间压力时，完美主义有无限的空间扩张。截止日把这个空间压缩到必须做出选择、必须放弃某些纠结的程度。

## 关于日常写作纪律

{{< sentence >}}
灵感是访客，纪律是住客。你不能把房子建在访客身上。
{{< /sentence >}}

等灵感来再写的人，一年里能写作的日子不超过十天。把写作变成不依赖灵感的日常行为，是完成任何长篇作品的基础条件。

{{< sentence >}}
今天写了一千个烂字，比今天想了一千个好点子更接近终点。
{{< /sentence >}}

纠正"好的构思就是好的进展"这个错觉。构思停在脑子里不落纸面，对完稿进度的贡献是零。

## 关于过程中的心态

{{< sentence >}}
中段是所有小说的坟场，活着走过去才能看到结尾。
{{< /sentence >}}

开头有新鲜感，结尾有冲刺感，中段什么都没有——只有持续推进的枯燥。大部分放弃都发生在这里。知道这一点本身就是一种武装。

{{< sentence >}}
写到第十五天觉得自己写的全是垃圾，是正常现象，不是失败信号。
{{< /sentence >}}

中途自我怀疑不是能力问题的信号，是长期写作的正常阶段。把它当成路标而不是红灯——穿过去就好。
