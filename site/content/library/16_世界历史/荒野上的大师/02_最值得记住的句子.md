{{< sentence >}}
福塞特每次从亚马逊回来都更瘦、更病、也更确信Z城存在。证据不足反而强化了他的信念——因为"还没找到"在他看来说明还需要更深入。
{{< /sentence >}}

缺乏证据被解读为"还没找到"而不是"可能不存在"——这是执念最常见的自我维持机制。

{{< sentence >}}
福塞特不信任任何同行者。他的几次探险都因为同伴崩溃而中止——但他从不认为是行程太危险，而是认为同伴太弱。标准永远在别人身上。
{{< /sentence >}}

当一个人把所有失败都归因于他人的不够强——问题可能在他对风险的评估上。

{{< sentence >}}
格拉恩站在福塞特最后一次被目击的地点——周围是密林和沉默。不是所有谜团都有答案。有些人消失了就是消失了。
{{< /sentence >}}

接受"不知道"比编造一个答案更诚实也更困难。

{{< sentence >}}
福塞特最后一次出发时带上了自己的长子。他的执念不只赌上了自己的命也赌上了下一代。执着者最容易伤害的是最亲近的人。
{{< /sentence >}}

个人的执念会拖入身边的人——他们可能并不真正理解风险也无法拒绝。

{{< sentence >}}
亚马逊丛林不像北极那样用冰雪杀人——它用密度杀人。蚊虫、疾病、湿热、方向感丧失。敌人不在某一个点而是到处都是。
{{< /sentence >}}

有些环境的危险不是集中爆发的而是持续渗透的——这种危险更难被感知也更难被应对。
